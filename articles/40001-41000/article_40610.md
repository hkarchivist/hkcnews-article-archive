### 「海事考古」的聯想

聽到「海事考古」四字，不少人該會聯想到沉船、海下尋寶、深海異類之事；尤其是沉船事件，因「鐵達尼號」電影情節而變得家傳戶曉，就連香港海事博物館，今年也以此作公眾活動題目之一。而考古界外，時至今天，對海事考古有興趣，甚或是聞聽過「海下」考古之人，寥若晨星。 難道，海事考古就只有沉船嗎？而香港，有海事考古的例子嗎？

### 考古：海洋應用

2021年3月初，水下考古學先驅者之一，左治巴思 ( George Bass ) 逝世，終年88歲。他的父親是文學教授，子如其父，文章生動，一針見血。他年輕時，有幸被邀請參與一宗歷史沉船事故的調查，後來在《國家地理雜誌》亦頻頻出版文章，致力在學術界內、外，推廣當年較新穎的潛水技術，應用到深入海下的考古發掘，尋覓人類歷史的另一面。巴思最大的成就，正正就是他的前瞻性，意識到考古在海洋應用的可能性，將「陸地為主」的考古學科 ( Land-based archaeology ) 推展應用到海域 ( Water-based archaeology ) 。

### 海事考古遲到因由

考古界內，向來做考古發掘的，大都以陸地為主，眾所週知；位處海域、川河流水內的考古文物，卻無人問津。若要究其原因，可從兩面剖析： 一、意識也；二、技術也。

### 海洋與人類文明的關係

意識上，先人對「未知」的海洋，固然感到興趣，但興趣是傾向於開拓海洋貿易、獲得生態資源，諸如捕魚、濕地耕耘、拓闊水源、甚至海上以物換物等。過程中，衍生大量關於航海技術、造船、針路、農業、生態物種、海島、海洋法例等資訊，用作軍事、貿易之用。換言之，以「考古角度」去看「海洋對人類文明發展的關係」的人，本已稀少無幾，而有類似「求知」心態之人，要不是純粹為了資源商貿、邊界防守，就是鑽研航海技術、航海網絡、甚或用作彰顯國力而已。

### 欲知曉瀚海，卻疾於技術

世界寬廣，當中定有例外。若小草是位古人，幸而生活富庶，可有閒情逸致，對海底下的考古文物提起興趣，同時又具資金、人脈，當然希望「下海」尋寶。但第二難處，就是技術。 先人對海的種種迷思、遐想，很大程度是因為「未知」，受制於技術，一來要跑出自己的村落已有難道，更何況是找對的船夫出海；二來，要在無現代科技配合下潛水，抵受氣壓、自然因素，令古人不能太深入做考察。

### 水肺：海事考古的推進劑

基本上，在四十年代之前，若要下海「尋寶」，甚為艱鉅，下海工作也要用上鋼造、重約幾十公斤的傳統潛水衣，接駁母船喉管呼吸；傳統潛水衣的「衣」實際上是「容器」，因此不受水壓影響。傳統潛水衣後來發展成潛水深度可達幾百米，但可惜因為衣物笨重，對用者的活動能力加上諸多限制，令它應用到海事考古學的熱情不大。海事考古學所涉獵的技能繁多，譬如潛水、深海求生、探索海底、掌控視野、海下記錄、生態意識要強，更要預測天氣、奠定安全機制、預算資金、減低風險等等，加上最終尋獲「寶藏」的可能性（因為技術所限，比起陸地考古，通常是比較低），種種原因，令到海事考古學遲遲也未萌芽。那麼，海事考古是到了何時才萌芽呢？簡單來說，就是水肺發明之時。大約是自上世紀四十年代起，水肺潛水技術 ( SCUBA, 全稱Self-Contained Underwater Breathing Apparatus ) 誕生，令人類可在足夠氧氣下、不笨重地在水中逗留更長時間，同時也帶領了潛水熱潮，史無前例。海事考古學 ( Maritime Archaeology)、海洋科學 ( Marine Science )、海洋生物學 ( Marine Biology ) 等亦因此逐漸興起。

### 海事考古 ＝ 海下考古？

本文一直使用「海事考古」這個名詞組，在香港的研究學府，極其少用（因為考古也很少人吧！），而內地的研究單位也通常寫成「海下考古」。但值得一提，海事考古( maritime archaeology ) 屬於一個大專業，它所涵蓋的專業領域包括「海下考古」 ( underwater archaeology ) 、航海考古 ( nautical archaeology ) 、濕地考古 (wetland archaeology) 等等。

### 香港有潛在的海事考古文物嗎？

小草可以不用遲疑的回答：絕對有，而且許多。只是因為香港跟大多數地區一樣，擁抱「陸地為主」的考古思維，才未有海事考古人員研究、出版香港海域考古相關的詳細記錄。小草現列舉幾個例子，供各位參考，希望引起大眾關注：

**陶片、青花瓷片**

大嶼山竹篙灣一直也有許多陶片、青花瓷片等等的文物，陸地上有，海下亦然。2008年5月份，船灣郊野公園進行了郊野公園實地考察，而赤洲水域附近發現的陶片、青花瓷片和陶磚塊等就被正式記錄在案。如其他發現碎片的海域一樣，這些陶瓷碎片早已被本地潛水員、鄰近居民被發現，尤其是在進行公海潛員訓練期間，很多訓練學員也發現不同形狀、顏色、大少的碎片，零零散散，分佈各區。2010年，香港民間有水下文化遺產小組聯同海事博物館，在西貢郊野公園對出海域進行了為期五日的海底考察，記錄了313件海下發現的碎片，當中有22件文物被收集。 但因「陸地為主」的考古思維，令香港的海域未有海事考古人員研究出版相關的詳細記錄。順帶一提，他們用了約三年的時間進行海水淡化 ( desalination )，才將文物的含鹽成份降至零。

**塔門水下石獅雕像**

早已被眾多潛水人員發現的，是塔門水下的一頭石獅雕像，媒體也報導過。據聞，因為天后廟要更換石獅子的原因，將舊的石獅子棄在海下，乃便利和經濟考慮而已；亦有村民估計是與風水有關，當年曾經有村民外出進行海事活動時亡於大海，因此故意放置石獅。真實原因，無從判斷。

**萬宜水庫明代古船**

1974年，在興建萬宜水庫東壩時被建建築人員意外發現，1974年5月24日，香港考古學會聯同和香港大會堂博物館官員組成小組，視察遺址，並確認了這艘長80 x 22呎的古代舊貨船的考古價值。1974當年的考古學家只有五天的搶救式發掘，後來至3年後政府在古船附近才築起保護性的堤岸，包圍成小水塘，才繼續進行堪察考古工作。據1977年的媒體報到，考古學家稱附近傾倒的土樣也坍落在船沉的範圍內，要清除塘內的泥漿費時、昂貴，而且用機器去掉泥漿更有機會破壞船隻的整體結構。後來用水槍清理數週後，發掘了約1%的文物，當中包括一些燒焦的明代古船木板（經年代測試後，推算木板年代為1290年，最多85年的誤差）、泰國十四世紀的Sawankholok陶片、中國古錢、其他馬來西亞和東南亞的瓷片、波斯玻璃製造的小珠等等。如有興趣看實物，部分瓷片、陶碎片可在海事博物館找到。

**滘西洲沉船**

有一西貢滘西洲的沉船，沉船年代不詳，但經遙距水下攝影和側掃聲納查看後，推測是近代沉船，因此未有進行相關考察活動。

**伙頭墳洲打炮和糧船灣洲宋代碇石**

2016年，香港水下文化遺產小組在西貢糧船灣洲發現了碇石（錨上方的橫石），現在放在香港海事博物館的展覽館中。另外，小組在伙頭墳洲亦發現了兩枚18世紀的大炮，也有媒體高調報到過，但時至今天，大砲還沒有向公眾展示過。

**大浪灣燕子岩水泥沉船**

記載在英國水航道測量局（The United Kingdom Hydrographic Office）的資料庫內，一艘1965年因大霧天氣而意外沉沒於燕子岩近岸位置，是海安船務有限公司（2006年已停業）水泥貨船，船面甲板水深6米至船底10多米，船身分成2截，一袋袋排列好的水泥清晰可見，是潛水界內已頗有名氣的拍攝地點之一。

**印洲塘海岸公園**

筆者最喜愛的海岸公園就是印洲塘，遇上過八十七歲的水上人，今年教筆者數多海岸區內的大少事。筆者在岸邊的考察也發現有大大小小的沉船，跟本不用潛水也清晰而見。話說回來，筆者第一隻在印洲塘發現的沉船，也是因為當天的同伴眼力好，在筆者悠閒攝影之時，講了一句：「這裡好像是有一隻船沉了」。千禧年之前，印洲塘是棄船聖地，不知還有多小隻船故意的被沉在此處。

**果洲群島（果洲南）沉車**

最後，除了沉船，也有海底沉車。有一輛私家車沉在十米內水深處，據Raymond先生所講「估計早年香港走私猖獗，懷疑走私客逃避不了追捕，所以把車輛棄置為求脫罪」。一來是近代產物，二來年份、原因也不詳，因此也就未有考古人員考察。但考古是一定要幾百年的東西才算是「古」嗎？

### 草草結語：

常言道：「香港地少人多」，小草覺得，我們重視陸地的發展，卻省略了水下的文化；看似老舊殘缺的東西，只要有人力去考證、探索，其潛在的價值可有震憾性的影響。香港沒有常駐的海事考古學家，但我們作為小市民，就算不潛水、不懂考古，也是可以從個人思想開始，思索一下：香港作為一個沿海的、多元文化的城市，她所擁有考古價值會只在陸地上嗎？
