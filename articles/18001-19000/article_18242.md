# 馮檢基選舉開支103萬元　夫妻自資81萬　逾5萬捐款來自親建制

選舉事務處周一（28日）公開馮檢基申報的選舉開支。根據申報紀錄，馮檢基在九龍西補選中，共花費102.6萬元。他共得31.3萬元選舉捐款，當中最大一筆、價值10萬元的現金捐款來自其妻陳敏慈，換言之，馮檢基夫婦為補選付出逾81萬元。

馮另獲「小巴大王」馬亞木兒子、潮州商會常務會董馬僑生、前全國政協委員張家敏、前民建聯成員蔡東洲、西貢區議員張美雄等親建制人士捐款，金額由1,000至3萬元不等。

**馮檢基選舉開支分布**


| 選舉廣告 | $51.8萬 |
| --- | --- |
| 選舉聚會 | $23.1萬 |
| 代理人及選舉助理 | $13.6萬 |
| 租金、辦公室及運輸 | $12.2萬 |
| 候選人個人開支 | $45 |
| 雜項 | $1.8萬 |
| **總計** | **$102.6萬** |


馮檢基102.6萬元的選舉開支，當中逾半、51.8萬元用於選舉廣告，包括傳單、貼紙、標語、橫額、小巴車身廣告等，以小巴車身廣告為主，涉及逾15萬元；其次為選舉聚會，包括街頭宣傳、集氣大會及車隊巡遊，花費23.1萬元。

**馮檢基獲得選舉捐款分布**


| 1,000元以上捐款 | $27.2萬 |
| --- | --- |
| 1,000元或以下捐款 | $4.1萬 |
| **總計** | **$31.3萬** |


馮檢基收到1,000元以上捐款共271,611元，當中10萬元來自其妻陳敏慈，其次為一筆5萬元來自軒通有限公司（High Express Limited）的捐款。

馮參選時被指「收錢」、「鎅票」，其選舉捐款名單上確有數名親建制人士，包括：


| 「小巴大王」馬亞木兒子、   
 潮州商會常務會董馬僑生 | $3萬 |
| --- | --- |
| 前全國政協委員張家敏 | $2萬 |
| 前民建聯成員蔡東洲 | $5,000 |
| 方國珊前助理、   
 西貢區議員張美雄 | $1,111 |


至於1,000元或以下的小額捐款共41,032元，當中有40,135元為眾籌所得，其餘是紙張、信封、USB、電話上網費等物資捐贈。

馮檢基獲得選舉捐款總數為31.3萬元，與選舉開支相差71.3萬元，連同其妻捐出的10萬元，馮氏夫婦為補選付出81.3萬元。其對手陳凱欣，選舉開支共146.3萬元，卻有多達111.6萬元捐款可用於補選，她僅自費34.7萬元。
