# 4張圖睇大灣區香港被規劃

大灣區香港被規劃，仲話「蘇州過後冇船搭」？林鄭月娥同中央幾時問過香港人有幾多人想上呢隻船？簡單疏理咗《粵港澳大灣區發展規劃綱要》呢份文件（下稱《規劃》），用下面4張圖睇吓香港被規劃咗咩嘢角色，同埋呢啲角色有咩影響，再睇下盤數，因為香港用喺大灣區嘅公帑，唔止預算案果啲數咁少，咁就明白上呢隻船，香港點解失多得少，危多過機。

睇完香港被規劃嘅角色點樣危機重重，再睇下另一個天大危機——走去大灣區嘅香港公帑，唔止財政預算果啲數咁少。

林鄭月娥喺政府打滾多年，熟悉政府架構運作同條例，「最叻」諗啲「絕世好橋 」，所以佢將來肯定會成立呢類大灣區機構同基金，避過公眾同立法會監管，向中央交單交心；西九故宮博物館一役， 佢已完美示範「深厚嘅功力」。香港百年一遇嘅「搬金奇才」，林鄭月娥當之無愧，我哋會銘記於心，出賣港人利益嘅名單，一定會留番個位俾佢。   
   
 睇完3張圖，最後就總結一下大灣區香港被規劃嘅得同失。

得，咪就係所謂嘅四個流通，但香港將失去嘅會好大鑊，除經貿外，各方面嘅政策自主權被嚴重削弱；香港各方面嘅標準要求，一向比中國高，而家同大灣區大融合，咁我哋咪要去遷就中國，降低香港嘅標準要求囉。要人才同資金流通，咁將來大灣區嘅社工、醫護人員、教師，將來甚至任何專業人士，都可以嚟香港打工，咁我哋香港嘅專業水平咪降低囉；要貨物流通，食物同環境安全會降低，奶粉變毒奶，疫苗變毒苗。專業水平同規管 ，會全方位大幅下降；大灣區人同車走嚟香港，人口、交通、社福民生醫療服務全面逼爆。要資金流通，大搞跨境人民幣交易投資， 咁香港嘅金融規管就會越嚟越鬆，風險激增，金融系統穩定性受衝擊 。

香港各方面嘅標準安全同監管受嚴重衝擊，一萬字都寫唔完，呢啲正正就係劣幣驅逐良幣嘅現象，令香港加速失去獨有優勢 ，危及香港嘅地位同前途。

 附2： 探索成立聯合投資開發機構和發展基金——《規劃》中和香港有關的重點撮要如下：


| **1. 引進國際先進技術**   
 •引進國際先進技術：吸引跨國公司總部和國際組織總部落戶大灣區和設立全球研發中心、實驗室   
 •三地企業合作跨國兼併收購和綠地投資 |
| --- |
| **2. 創新科技**   
 **粵港澳合作的投資開發項目：**   
 •國家科技計劃專項、基金   
 •粵港澳企業、高校、科研院所共建的協同創新平台   
 •粵港聯合創新資助計劃   
 •粵港澳產學研創新聯盟   
 •軍民融合創新示範區   
 •港深創新及科技園   
 •港澳國家重點實驗室夥伴   
 •粵港澳聯合創新專項資金, 允許相關資金在大灣區跨境使用   
 •廣東科技計劃 |
| **3. 金融與保險**   
 **香港主要任務：**   
 •金融產品跨境交易；區內基金、保險等   
 •豐富投資產品類別和投資渠道，建立資金和產品互通機制   
 •在亞投行和絲路基金中發揮積極作用，相關金融機構在港澳設立分支機構   
 **金融：**   
 •服務「一帶一路」建設的投融資平台   
 •擴大大灣區內人民幣跨境使用規模和範圍：(1)區內銀行可開展跨境人民幣拆借、即遠期外匯交易、衍生品、理財產品等業務。(2)區內企業可跨境發行人民幣債券(3)香港機構投資者在區內募集人民幣資金投資，投資境內私募股權投資基金和創業投資基金   
 •大灣區高新技術產業融資中心： (1)香港私募基金參與大灣區創新型科技企業融資(2)創新型科技企業進入香港上市集資平台   
 •內地企業在香港設立資本運作中心及企業財資中心   
 •香港開發大宗商品及其他風險管理工具   
 •知識產權證券化試點   
 •香港打造大灣區綠色金融中心， 建設國際認可的綠色債券認證機構   
 •深港金融市場開展科技金融試點   
 •粵港澳大灣區發展相適應的賬戶管理體系   
 保險：   
 粵港澳保險機構合作開發   
 •跨境保險：機動車和醫療保險產品   
 •跨境人民幣再保險業務 |
| **4. 司法**   
 **香港主要任務：**   
 •國際法律服務、國際商事爭議解決和「一帶一路」國際仲裁中心   
 •借鑒港澳信用建設經驗，探索對區內企業聯動實施信用激勵和失信懲戒措施   
 •深港司法合作交流   
 •粵港澳合夥聯營律師事務所試點   
 •港澳律師在珠三角九市執業和業務範圍問題 |
| **5. 私隱**   
 •粵港澳互聯網、大數據、人工智能和實體經濟深度融合:(1)珠三角國家大數據綜合試驗區(2)智慧城市合作和時空信息雲平台(3)建立統一標準，開放數據端口，建設互通的公共應用平台等   
 •大灣區電子支付系統互聯互通   
 •醫療數據和血液等生物樣品跨境在區內限定高校、科研機構和實驗室使用和優化管理 |
| **6. 產品安全**   
 •大灣區企業使用香港的檢驗檢測認證等服務   
 •中藥質量研究國家重點實驗室 粵澳合作中醫藥科技產業園開展中醫藥產品海外註冊公共服務平台建設   
 •障內地供港澳食品安全，港澳參與廣東出口食品農產品質量安全示範區和「信譽農場」建設，惠州粵港澳綠色農產品生產供應基地 |
| **7. 人才**   
 **教育社工醫療**   
 人才培養、 資格互認、標準制定等方面加強合作：   
 •資格互認：高校相互承認學分、社會工作領域職業資格   
 •港澳醫務人員到九市開展學術交流和私人執業醫務人員短期執業   
 •港澳中小學教師、幼兒教師到廣東考取教師資格並任教   
 •港澳居民中的中國公民可報考內地公務員工作   
 •非急重病人跨境陸路轉運服務，在指定公立醫院開展跨境轉診合作試點   
 •粵港澳合作辦學：高校、職業教育   
 •培訓交流合作：高校、職業教育、醫療衞生人員、基礎教育，三地幼兒園締結「姊妹園」   
   
 **青年事務**   
 幫助香港青年在大灣區創業就業的項目：   
 (1)「青年發展基金」   
 (2)深港青年創新創業基地   
 (3)前海深港青年夢工場   
 (4)南沙粵港澳（國際）青年創新工場   
 (5)中山粵港澳青年創新創業合作平台   
 (6)中國（江門、增城）「僑夢苑」華僑華人創新產業聚集區   
 (7)東莞松山湖（生態園）港澳青年創新創業基地   
 (8)惠州仲愷港澳青年創業基地等 |
| **8. 交通旅遊**   
 •兩地牌機動車通過多個口岸出入境   
 •粵港澳共享區域旅遊資源   
 • 粵港澳遊艇自由行 ，並建設國際遊艇旅遊自由港 |
| **9. 與地方合作項目**   
 (1)佛山：離岸貿易合作   
 (2)深圳前海：深港現代服務業合作區，香港交易所前海聯合交易中心   
 (3)廣州南沙：合作示範區，南沙打造為華南科技創新成果轉化高地，重點領域新一代信息技術、人工智能、生命健康、海洋科技、新材料等   
 (4)落馬洲河套：港深創新及科技園和深方科創園區   
 (5)江門：大廣海灣經濟區   
 (6)東莞：東莞濱海灣地區   
 (7)佛山：粵港澳高端服務合作 |
