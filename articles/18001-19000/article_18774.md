 後顧而無憂

每一個人都會有死亡的一天，如何在離世前後顧無憂，安然離世呢？對一些無親無故兼生活困難的長者而言，可能在自己「最後的一程」時，不願任由人「擺佈」，希望自己也有「最終的話事權」。此是對一些長者而言，於離世前總有「千千結」心事放不下，致不釋懷的原因，因而令他們在面對身體不如前下，更加影響了他們的「自控生活」不如前的日暮感覺 ; 此更何妨是逝世後，可有人代勞通知他們的遠方親友，他的去世消息及遺言……呢！

  
 安老服務的全人照顧

筆者工作早年接觸的多是無親無依的匱乏老者，他們盡多是體弱及自認是「沒有明天」的一群，部分因頑疾入院，深知不能回到社區渡餘生時，多會委托别人聯絡筆者，希望前往和他們見最後一面。於往探問他們時，總聽到他們千般憂慮，如擔憂他們遺體無人理會、逝世後無法通知親友、家當不知如何處理……等等。

安老服務的要旨是要令長者「安心養老 後顧無憂地生活」，很多長者由於忌諱及迷信問題，難渲「人生限期」到臨之希望，致生活悶悶不樂 ; 也有老者深恐逝去後缺財力殯葬而「不吃不喝」積蓄過多金錢，因而大大影響他們的日常生活，進而影響健康來 ; 部分更不願被當局「擺佈」將其骨灰撒在「群灰亂放」的沙嶺公墓 ; 當然部分人士希望有「最終話事權」，目的是能可在離世前自己事先可安排「後事」下，掌控其殮葬的處理，以令自已生前及死後安心及釋懷 ; 此正如有遺產的人士般早訂遺囑一樣。況且對「身後蕭條」的孤老而言，若身故後若有何服務可知會其遠方親友他們的仙逝事，確也是令獨老有免去無限的掛慮。 如何令一些長者可「死而無憾」，後顧無憂地生活呢？於努力為長者提供衣食住行文康醫療等服務、照顧及關懷時，為孤寡無助弱勢長者提供最後一程、逝去後的殮葬及跟進服務，該才是社工對他們的「生前照顧 死後服務」的「全人(人生)服務」工作。

  
 鍥而不捨 「死」也要做

事實上，欲為長者提供「後顧無憂」的規劃服務，必先也要打破機構内在「禁忌」的阻力，加上機構的直屬上司也認為此需要性不大、坊間也有機構提供類似的服務……。總之，不准開辦是最後決定。筆者爭取開創此服務與上司爭取多時，在一次上司盛怒時問她在甚麼條件下，才允開辦，欣聞她開出苛刻條件-必要有員工兩年的薪酬，為免她反口不認，筆者也要她重覆再說出要求，再加上我說 : 「你要記住所說的！」以令她留下「深刻難忘」盛怒時的承諾。結果不消一星期有幸得一名導演及一牙醫全數捐出所需金錢，致可「逼令」上司開出綠燈於03年開辦「後顧無憂 規劃服務」。

推創新事工並不是憑賴一時衝動而來，於90年代，筆者已多方探討，如徵詢長者意見(超乎想像，長者極盼望可委託福利機構提供服務)、社會接納程度(市民深信福利機構的收費不會「襯火打劫」)、徵詢正營運中的機構意見時(拒絕給予資料及意見，因恐我們分薄他們的「生意」)、長生店營運者(有幸我們認識了殷實在業者的支持)，在面對人口日漸老化，獨居無親長者增加勢頭下，完善長者「生」的服務之餘，開設為他們「最後一程」的「死」服務，使安老工作真正起著「全人(人生)關顧」的理想，是筆者退休前的職志。

  
 為「死」工作作探討籌備

工作單憑「匹夫之勇」，而開辦新服務，不掌握社區人士對殯葬的「脈搏」意向及需要，將會是失敗的要因，此不但是人力及資源的「白費」，也引來「安於現狀」的上司「企硬」反對，責之「冒險行動」開辦新服務 ; 故筆者於90年代開始以探索手法，邀請得資深殯儀業人士及律師主持「如何妥善處理先人後事」講座，以上述名稱舉辦教育活動，目的是為避免觸犯社區人士的忌諱，以「死亡不是我」的「離身感覺」，而握動市民的報名參加講座，以引起巿民對「不吉祥」的各種殯儀及殮葬常識的好奇心 ; 同時由律師講解署立遺囑的問題等 ; 結果初期大受年青人的歡迎，而隨著以後同樣講座的舉辦，參加者的年齡開始「老化」，由為父母後事籌備的年青人，年齡群逐漸「老化」至中年及老年人 ; 後者初期由「閃躲」報名，至攜同老伴明確指定報名參加 ; 此正也咸稱向我們說 : 「為甚麼我們可以立遺囑，而不為我們的後事作安排呢？人終會有一天離世，及早作出安排，可避免後人徬徨及互相猜想，而影響他們的關係 ; 胡亂殮葬我們，後悔莫及，因為此是不能再辦第二次喪事！」基於掌握大部分社區及長者對後事安排的無畏無懼，故筆者有上文的堅持「企硬」態度，和機構爭取開創「後顧無憂 規劃服務」。

  
 破忌諱 推實務

2003年隨著機構開綠燈開辦此社福界首個推行的「接受委托的殮葬事工」 -「後顧無憂 規劃服務」，我們很幸運可聘得一位很有創新意念的專責社工全盤策劃及執行所有工作，以及推出不同創意的社區宣傳及教育活動，以跳出傳統殮葬業的神秘形象。

孤寡無依兼經濟困難的綜援受惠者，是我們的服務對象，他們初期乃經社工轉介而來 ; 為令服務對象清楚知悉所掌控的後事安排，我們先由專業的承包商(殯葬業人士)，講解各種為他們有關的服務，由遺體領取、入殮的壽衣、棺木、祭品、儀式、火化、骨灰位、靈位上所刻的字句等，鉅細靡遺地解答他們的問題 ; 在他們清醍下(我們鼓勵他們有人陪同出席作出見証)，我們更由律師辦理他們的簽署委托，以合乎法律上我們承諾的服務責任(此實也令長者深覺真正後顧無憂，因為恐一旦違約，我們是會負上法律責任)，令他們確實有「掌控」的安心 ; 服務費方面，感謝社署的批肯，在綜援受助人生前委託下，當他們離世後，社署便會撥出殮葬費用予我們辦事，此也令長者安心，無需節省生活開支而影響生活質素。我們也會每兩年與長者會面一次，看他們有否改變決定，會重新和他們也在律師見証下，簽署服務承諾書，以示我們為他們服務的心志不改。

為破解社會人士對殯儀殮葬的神秘認識及誤解，消除忌諱及迷信，讓社區人士親身「走到現場」在業者沿途講解各種風俗及禮儀下，由參觀實際的由醫院或殯儀館「出殯」開始、至送遺體至火葬場，以及包括海上與公園撒骨灰等，是我們大膽的「社區教育」項目嘗試 ; 結果反應異常熱烈，正如前述的講座一樣，初期是以追求知識不怕「邪氣」的中青年者參加為多，随後年長老者竟佔了很大的比例。很感謝各大傳媒多次隨隊作出報導，令香港人大開「眼界」，消除很多人對死亡及殯葬的恐懼及不安 ; 可惜我們得不到當年食環署的支持，不准新聞界拍攝於墳場下葬、火葬場及在撒灰公園的儀式，深感遺憾非常。

  
 安老服務最後一環的需要

隨著本港的人口正在老化，預計未來人口老化的情況愈趨明顯。在2015年，本港有大約112萬長者，佔總人口約730萬的15%，即大概7人便有1人是65歲或以上長者。到2040年，香港總人口將達到820萬，當中有接近三分之一（大約250萬）為65歲或以上長者。雖然年長並不是等同死亡，但在人必會死的事實下，而由於人口老化，整體死亡率及死亡人數呈現上升是自然的趨勢 ; 但為甚麼人們生前只關注生前「預立遺囑」、結婚前作婚前、嬰兒誔生前………等等作準備，而對忌諱的「殮葬意願」而不顧呢？另一方面，善終服務、紓緩治療、預設醫療指示、器官捐贈、遺體捐贈……等，不也是人們選擇面臨死亡或離世後的意願安排嗎？為能令「老者安之」，香港的安老服務近年漸趨多元及綜合化，但對迎合很多長者生前可能藏於心底的「最後遺願」服務，以可令其預早安排後事，卻鮮見普遍推行，令他們可有「最終話事權的掌控」，「後顧無憂。規劃服務」的社區教育及工作項目，也確是令長者生前可有「老者安之」之效。事實上，有部分長者也不願其身故後，由當局處理其遺體，及將其骨灰撒於沙嶺的公墓。
