# 整治醫管局　請審計署出手衡工量值

財政司司長陳茂波在網誌稱，「預算案中有關改善公營醫療服務的四項措施，不足以根本性地解決所有積存已久的問題……作為本港公營醫療系統的前線機構，醫管局有責任迫切地改善其行政管理效率、聆聽員工心聲並切實作出調整，才能把問題緩解。」

食衛局局長陳肇始亦在《晴朗》表明，醫管局要檢視其行政效能。

畢竟，負責管理公營醫院的是醫管局，食衛局只是負責整體醫療政策的決策局，關於人手行政管理，如陳肇始所言，他們只能要求醫管局提交報告，最後亦只能盼望醫管局自我完善。不過，我們是否真的對醫管局束手無策？

政府內有一個部門，正正不受其他部門歡迎，卻發揮著監察的作用，那就是審計署！我們提出，審計署可否就衡工量值方面審視一下醫管局的人手行政管理？

翻查審計署過往的工作，除了每年向立法會提交的審計報告外，原來審計署本身已不時對醫管局的不同範疇作出審核。

年份與題目如下：


| 年份 | 題目 |
| --- | --- |
| 2017 | 醫院管理局對公營醫院工程項目的管理 |
| 2016 | 醫院管理局的藥物管理 |
| 2014 | 為長者提供的健康服務 |
| 2012 | 醫院管理局：公私營協作計劃 |
| 2006 | 醫院管理局 : 未清繳醫療費用的管理 |
| 2006 | 醫院管理局及社會福利署 : 減免醫療收費的管理 |
| 2002 | 長者住宿服務 |
| 2001 | 醫療設備的管理 |


在司長與局長均開聲要求醫管局要檢視行政效能之下，加上各界亦對醫管局不同問題作出質疑。例如，過去十個財政年度，公立醫院的醫生增加了23%（由約5100人增加至約6300人）；護士的增幅為34%（由約19000人增加至約26000人）。但同時，醫管局總部的人手卻增加了85%（由約1200人增加至約2200人）！

至於員工成本支出則大增71%至超過451億元，但如醫學界議員陳沛然提出，醫生與護士薪酬累計加薪只有約38%及46%，質疑醫管局薪酬開支增幅與醫護人工加幅不成正比。

如此種種問題，請審計署拿出「算盤」為醫管局衡工量值！
