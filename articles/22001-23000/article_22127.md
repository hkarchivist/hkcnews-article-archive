# 政府沒統計移民　領事館去年數字顯示　港人定居加拿大台灣創13年新高

究竟近年有幾多香港人移民？特區政府未有相關統計。眾新聞向多國駐港總領事館、駐港辦事處，查詢港人成功申請移民人數。數字顯示，2005年移民加拿大的港人人數僅729人，2015年起逐年攀升，由895人升至2018年的1,525人，三年升幅達七成，並為13年來新高。申請人數方面，申請加拿大永久居留的人數，由2013年的795人升至2014年的1,251人，2016至2018年繼續按年遞增。

另外，港人定居台灣的人數於2005年僅592人，數字由2014年起逐年攀升，由697人攀升至2018年的1,090人，四年內增加近六成，亦為13年來新高。居留台灣的人數亦由2005年僅1,643人升至2018年4,148人，2014年及2015年各達6000、7000人（註：2015年以前的計算方法將港澳人數合併，2016年起只計算港人）。

特區政府又有做什麼統計？多份中文報章（2018年5月）引述保安局數字估算， 2017年港人移民外國人數僅為6,500人 （不但創新低，數字更僅為1992年66,000人的十分之一；2008年估算有9,100人移民，之後2011至2013年每年約8,000人）。另一方面，英文報章（2018年5月《南華早報》）引述政府統計處的數字指出，2017年有24,300港人移民外國。

**眾新聞向保安局查詢，局方估算移民外國港人數目如下：**


| **年份** | **保安局估計移民外國人數** |
| --- | --- |
| 2008 | 9,100 |
| 2009 | 7,200 |
| 2010 | 7,200 |
| 2011 | 8,300 |
| 2012 | 7,600 |
| 2013 | 7,600 |
| 2014 | 6,900 |
| 2015 | 7,000 |
| 2016 | 7,600 |
| 2017 | 6,500 |


保安局如何得出上述估算數字？發言人表示：「香港居民離境外遊時不需向政府申報外遊目的，因此我們並沒有港人移民外國的直接統計數字。本局是根據向警務處申請無犯罪紀錄證明書（註：俗稱良民證）的數字和其他相關資料，估算港人移民外國的人數。然而，這估算有其限制，例如，申請無犯罪紀錄證明書的目的除移民外，亦包括海外升學，交換訪客等。至於目的是移民的，亦並非所有獲發無犯罪紀錄證明書的人士都能成功申請有關國家的移民簽證。此外，並非所有取得移民簽證的人士都會移居有關國家。在詮釋時，須考慮有關估算的限制。 」

眾新聞亦向統計處查詢，處方回覆表示沒有編製香港居民移民海外的統計數字。對於該處歷年的 [網上資料](https://web.archive.org/web/20210712031120/http://gia.info.gov.hk/general/201902/19/P2019021900368_304222_1_1550561459706.pdf) ，統計處表示數字是指在某一個統計期間香港居民（不包括單程證持有人）的淨遷移（即「移入」減「移出」）數字。由於資料來源的限制，除了單程證持有人的移入數字外，該處並沒有其他「移入」和「移出」的獨立分項數字。

**統計處香港居民淨遷移數字：**


| **年份** | **淨移入** | **淨移出** |
| --- | --- | --- |
| 2011 | -- | 45,700 |
| 2012 | -- | 41,600 |
| 2013 | -- | 17,900 |
| 2014 | -- | 15,700 |
| 2015 | 5,500 | -- |
| 2016 | -- | 6,100 |
| 2017 | -- | 24,300 |
| 2018 | 20,400 | -- |


*「註：香港的 [人口數字](https://web.archive.org/web/20210712031120/https://www.censtatd.gov.hk/hkstat/sub/sc20_tc.jsp) 是以「居住人口」的方法編製。「居港人口」包括「常住居民」（在港逗留最少3個月的香港永久性居民（在統計時點之前或後6個月內））和「流動居民」（在港逗留最少1個月但少於3個月香港永久性居民（在統計時點之前或後6個月內））*

*「移入」是指在上一個統計時點並不屬於「居港人口」，但在此統計時點成為屬於「居港人口」的人士。常見的例子包括在該段期間到港的香港非永久性居民，如單程證持有人、外籍家庭傭工或其他來港工作或求學的人士；亦包括之前不論任何原因逗留在外地的香港永久性居民，在統計時點的前或後6個月回到香港逗留1個月或以上。*

*相反，「移出」則指在上一個統計時點屬於「居港人口」，但在此統計時點並不屬於「居港人口」的人士。常見的例子包括在該段期間完成工作合約或學業而離港的香港非永久性居民；亦包括那些原本通常在港居住的香港永久性居民，因工作或求學原因須到內地或海外長時間生活，在統計時點的前及後6個月在港逗留少於1個月的人士。」*

## **學者：每年3萬多人移出香港**

香港大學社會工作及行政學系講座（人口健康）教授葉兆輝表示：「近年港人移居外地的確有所增加，我透過方程式推算，一年移出香港的人數應該有最少3萬多人（註：根據人口增長率、生育數字、死亡數字、移入人數，從而推算移出人數）。另外，申請良民證是估算移民人數的方法之一，然而部分港人於九七前移民、之後回歸香港，他們本身已經有外國護照，如果他們要再回到移居地居住未必需要再申請；也有年輕人先留學後移民定居，這些數字未必能透過良民證申請人數反映出來。此外，入境處亦不一定知道港人有沒有其他護照。」

葉兆輝續指：「之前政府曾經向市民派6000元（2011年），當時不少移居外國的非本地居民申請。我們於是根據該些數字估計，推算有60萬名18歲以上港人於外國居住，當中包括留學生。」他說，如果政府需要知道二次移民、或所謂隱形移民的數字，其實可以用出入境紀錄來計算。

專欄作家畢明指出，「隱形移民」數字難以估算：「現在我每個月都有朋友移居外地，尤其是有孩子的同輩朋友，移居地包括英語國家、台灣日本東南亞都有。現在有一種『隱形移民』，他們不一定在法律上做足移民手續或連根拔起，但卻選擇一年內大部分時間不在香港居住，在外地買房子也好、租 Air BnB 也好，由於現代工作在空間選擇上更具彈性，他們只需要間中回港打理業務或探望家人。他們已經opt out from Hong Kong（選擇離開香港的人和事），而且情況非常普遍。正式移民數據無法反映這個情況，相信人數亦難以準確計算。」

除「隱形移民」外，畢明相信難以有確切統計的，還包括「二次移民」人士，即大批曾經移民海外，之後回流香港，現在打算再到移居地居住者。另外，有九七前移民人士於當地結婚生子後，連同孩子一同回流香港，或者特地坐飛機往移居地生孩子；這些孩子現在可能已經超過20歲，也有外國身分。「從前大家未有首次移民的身分，移民數據才成立；現在潛在可以離開的人多了，移居外地的方式亦增加了，難以用簡單數字概括。」

中文大學亞太研究所今年1月進行的 [電話調查](https://web.archive.org/web/20210712031120/http://www.hkiaps.cuhk.edu.hk/wd/ni/20190103-123827_1.pdf) ，訪問 708 名18 歲或以上市民，約有三分之一受訪者表示，如有機會，將打算移民或移居外地。有移民或移居打算的市民中，一成六（16.2%）表示近期曾為移民作準備（按整體受訪者計佔 5.5%）。年輕的受訪者表示如有機會將打算移民或移居外地的比率較高，18 至 30 歲受訪人士為 51%，而51 歲或以上的僅有 21.1%。按教育程度分類，具大學或以上學歷者移民的意願較高，為 47.9%。最多人提及的推動因素為不滿本地政治爭拗太多、社會撕裂嚴重，以及對本地居住和生活環境不滿。最多人提及的拉動因素則是外地居住生活環境更寬敞。
