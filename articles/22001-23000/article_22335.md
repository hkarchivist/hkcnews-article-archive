# 連登絲巴今起英國7媒體、5地區登廣告　控訴中國違反《中英聯合聲明》

英國前外相約翰遜擊敗現任外相侯俊偉，當選保守黨黨魁，今日（24日）起接替文翠珊成為英國新任首相。連登絲巴藉此機會，再度發動國際輿論攻勢，即日起於英國7間媒體、5個地區的街頭廣告牌刊登廣告，控訴中國違反《中英聯合聲明》，同時促請新首相領導下的英國信守承諾，採取措施捍衛香港的人權自由。今次行動由連登巴打「我要攬炒」於7月12日發起，網上眾籌展僅12小時已達300萬港元目標。

「我要攬炒」今日下午在連登 [發布](https://web.archive.org/web/20190811072422/https://lihkg.com/thread/1358876/page/1 ) 英國廣告攻勢詳情，即日起於英國7份報章、雜誌、網上媒體刊登紙本及網上廣告，並在倫敦 Chelsea Yacht Club 及 Lots Road、利物浦Prescot Road、伯明翰Upper Dean Street、愛丁堡、加的夫的街頭廣告牌連續3日刊登廣告。報章、雜誌、網媒廣告發布詳情如下：

據「我要攬炒」公布的《倫敦標準晚報》及《衛報》全版廣告，展示《中英聯合聲明》被剪碎、壓皺的圖像，文內指中、英雙方在1984年簽訂《中英聯合聲明》承諾，香港在主權移交後會享有言論、新聞出版、集會等基本自由，惟中國屢次公然違反此國際條約，侵蝕香港的行政、立法、司法機關，而最近一次違反條約見於引渡修例，事件觸發大型及持續的示威，近200萬人走上街頭。儘管特首林鄭月娥指修例草案「壽終正寢」，但未有正式撤回。香港的基本自由正受攻擊，警察殘暴對待示威者及記者的情況加劇，而警隊受到不向人民問責的政府袒護。廣告文內指出，英國對香港有法律及道德上的責任，確保《中英聯合聲明》有效實施。港人促請英國政府制裁壓抑香港人權自己的人士，維護香港的人權、公民權利及民主化，與香港並肩同行。

「我要攬炒」今日發帖呼籲旅居英國的港人拍下實體廣告的照片，上載至社交媒體，加上「#FightforFreedom」及「#StandwithHK」hastag，加強聲勢。團隊即日在 [Facebook](https://web.archive.org/web/20190811072422/https://www.facebook.com/standwithhk/) 、 [Twitter](https://web.archive.org/web/20190811072422/https://twitter.com/Stand_with_HK) 、 [Instagram](https://web.archive.org/web/20190811072422/https://www.instagram.com/standwithhk_/) 及 [YouTube](https://web.archive.org/web/20190811072422/https://www.youtube.com/channel/UCyi42Y5o1ksO_yFItyHW7Jw) 開設專頁，發布登報攻勢的最新消息及後續行動等資料。

聯合聲明登報隊義工Chris透過Telegram接受眾新聞訪問。被問到首輪攻勢涉及金額、為下一輪行動預留金額，Chris表示因為法律問題，刊登前未能透露與媒體商討的內容、廣告費等。至於下一波攻勢，Chris指為免「漏風」、失去效果，亦不便透露，「只可以話，嚟緊睇實英國，仲會有嘢發生。」

繼上月籌集逾500萬元，在二十國集團（G20）峰會期間，在美國、英國、德國、法國、加拿大、意大利、瑞典、韓國、日本、台灣等逾10個國家及地區的主要媒體刊登「反送中」廣告，連登絲巴本月12日再度發起 [眾籌](https://web.archive.org/web/20190811072422/https://www.gofundme.com/f/uk-ad-campaign-to-defend-sinobritish-jd ) 登報行動，以道義、人權、英國利益作切入點，爭取英國關注中共違反《中英聯合聲明》，將「民間外交戰」升級。

「我要攬炒」當日在連登 [發帖](https://web.archive.org/web/20190811072422/https://lihkg.com/thread/1308609/page/1 ) 稱，團隊已設計了十幾款登報文案，並提供其中一款供大眾參考，惟汲取G20時 [誤傳青島啤酒以天價搶走韓國《中央日報》頭版](https://web.archive.org/web/20190811072422/https://thestandnews.com/politics/%E5%82%B3%E9%9F%93%E5%9C%8B%E5%A0%B1%E7%AB%A0%E9%A0%AD%E7%89%88%E8%A2%AB%E9%9D%92%E5%B3%B6%E5%A4%A9%E5%83%B9%E6%90%B6%E8%B5%B0-%E9%80%A3%E7%99%BB%E4%BB%94-%E6%BE%84%E6%B8%85%E5%9B%A0%E9%8C%AF%E9%81%8E%E6%94%AF%E5%B8%B3%E6%9C%9F-%E5%B0%B1%E9%8C%AF%E8%AA%A4%E6%95%98%E8%BF%B0%E8%87%B4%E6%AD%89/ ) 的經驗，團隊會將登報日子嚴格保密，以策安全，僅透露會分開幾波攻勢、「突襲式」登報。翌日（13日）「我要攬炒」 [發帖](https://web.archive.org/web/20190811072422/https://lihkg.com/thread/1310537/page/1 ) 指，僅用12小時，便達到300萬元的眾籌目標，即時起停止眾籌。如有餘款，將捐予「612人道支援基金」。

今次聯合聲明登報隊義工Chris指，兩次登報由不同的團隊籌備，兩者並無「延續」關係，有G20登報的團隊成員以個人身份加入今次聯合聲明登報隊，「有啲大大小小嘅位就分享下經驗咁」。他表示，兩次都是有人在論壇或其他渠道有構思，得到很多人贊成後，有人付諸實行才成事，「一如今次成個運動 leaderless同『各自爬山』嘅模式咁，有心嘅人嚟幫手，一齊有同一個信念，一齊做好件事。香港人，加油！」。

Chris坦言，在籌備登報的過程中，團隊成員基本上不知道彼此的身份、背景，甚至不知道整個團隊有多少人，「喺個 organic 嘅環境下，唔同有正式架構嘅機構等，基本上無一個成員會知道哂成條 team嘅人數甚至係 group數，以至連自己到底有幾多唔知嘅嘢都唔知，例如A搵B幫手、 B又搵C、D幫手，喺香港人憑信念一齊做嘢嘅大前提下，無需要亦都唔會知道哂 full picture……大家都唔著眼於大家嘅背景定專業背景，邊度幫到手就幫，output緊要。」

「我要攬炒」在過去個多月發起多輪行動，爭取國際社會關注香港的人權自由狀況，包括 [要求外國政府取消港府官員及建制派人士及其家人的外國護照](https://web.archive.org/web/20190811072422/https://lihkg.com/thread/1196704/page/1 ) ， [要求聯合國介入](https://web.archive.org/web/20190811072422/https://lihkg.com/thread/1217474/page/1 ) 警察追打市民、襲擊各地記者、 阻礙人道救援等問題。
