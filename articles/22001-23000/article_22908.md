# 央視率先披露　國泰行政總裁何杲、顧客及商務總裁盧家培離職　主席史樂山：全力支持《基本法》賦予的「一國兩制」

[中央電視台](https://web.archive.org/web/20220812171858/http://news.cctv.com/2019/08/16/ARTIcztdXBKk1UXxmKikwR1j190816.shtml) 今日下午率先披露，國泰航空管理高層人士變動，行政總裁何杲（Rupert Hogg）和顧客及商務總裁盧家培離職。二人職位分別由鄧健榮及林紹波接任，本月19日生效。央視網今午5時02分發稿，比國泰航空（293）5時14分於 [港交所發通告](https://web.archive.org/web/20220812171858/https://www1.hkexnews.hk/listedco/listconews/sehk/2019/0816/ltn20190816435_c.pdf) 早12分鐘。

[央視網](https://web.archive.org/web/20220812171858/http://news.cctv.com/2019/08/16/ARTIcztdXBKk1UXxmKikwR1j190816.shtml ) 引述國泰航空主席史樂山指，何杲及其團隊執行為期3年的企業轉型策略，對國泰航空的復甦及持續發展，建立了穩健的平台。然而，最近發生的事件，令國泰航空對飛行安全和保安的承諾受到質疑，亦令國泰聲譽和品牌受壓，「我們認為這是合適時機任命新的管理團隊，重建信心以及帶領國泰航空再創高峰。國泰航空全力支持香港實行《基本法》賦予的『一國兩制』原則。我們對香港的美好未來充滿信心」。 [港交所通告](https://web.archive.org/web/20220812171858/https://www1.hkexnews.hk/listedco/listconews/sehk/2019/0816/ltn20190816435_c.pdf ) 指，何杲及盧家培呈辭是作為公司的領導者「對公司近月所面對的事件負責」。

根據 [港交所通告](https://web.archive.org/web/20220812171858/https://www1.hkexnews.hk/listedco/listconews/sehk/2019/0816/ltn20190816435_c.pdf ) ，新任行政總裁鄧健榮現年60歲，1982年加入太古集團，現為太古集團董事。接任顧客及商務總裁的林紹波現年46歲， 1996年加入太古集團，曾駐公司在香港、日本及斯里蘭卡的辦事處，前年6月至今年7月擔任商務及貨運董事，及後出任香港快運航空有限公司行政總裁。鄧健榮及林紹波將分別獲665萬港元及361萬港元年薪連津貼，並可收取按表現發放的花紅，有權享有其他實質福利和參與公積金計劃。

[國泰網頁](https://web.archive.org/web/20220812171858/https://www.cathaypacific.com/cx/zh_HK/about-us/about-our-airline/management-team.html) 顯示，何杲1986年加入太古集團，至今33年。他於2017年5月1日起任國泰行政總裁，亦是港龍航空和華民航空主席、香港太古集團和中國國際貨運航空董事。盧家培則於1991年加入國泰，出任顧客及商務總裁前為企業發展及資訊科技董事。

近月反送中運動中，國泰航空屢受壓力。中國民航局本月初向國泰發出重大航空安全風險警示，要求國泰停止所有參與和支持非法遊行示威、暴力衝擊活動的機組人員飛內地航班，並指示國泰向內地審查機構提交所有飛往內地及飛越內地領空的機組人員身份資料。國泰表明會配合民航局要求。

7月28日上環衝突中，國泰機師廖頌賢被捕，及後被控以暴動罪。本周一，大批市民癱瘓機場，當日網上流傳一張疑似國泰客機駕駛艙屏幕的相片，顯示當晚有航機返港。國泰前日（14日） [證實](https://web.archive.org/web/20220812171858/https://news.cathaypacific.com/%E5%9C%8B%E6%B3%B0%E8%88%AA%E7%A9%BA-%E5%82%B3%E5%AA%92%E5%9B%9E%E6%87%89-2019%E5%B9%B48%E6%9C%8814%E6%97%A5) ，已中止兩名機師的僱傭合約，一人正涉及司法案件，另一人涉及運作8月12日CX216航班時不當取用公司資訊。
