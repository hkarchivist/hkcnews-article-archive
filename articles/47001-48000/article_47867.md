# 【立會選舉】6參選人擁外國國籍或居留權　參選人：愛國與擁有幾多國籍沒關係

立法會選舉提名期上星期五結束，在「愛國者治港」的新制度下，眾新聞統計共6名功能界別參選人申報持有外國國籍或居留權。其中建築、測量、都市規劃及園境界兩名參選人均申報擁有居英權，現屆議員謝偉銓強調居英權不影響他效忠國家和特區，又反問記者：「點解你哋又咁關注有無居英權嘅問題呢？」其對手、建築署前助理署長陳澤斌亦指：「愛國就不嬲都愛國，呢個我諗同你攞幾多個國籍無咩關係。」

根據《立法會條例》，12個指定功能界別容許持外國國籍或居留權的人士擔任議員。眾新聞翻查參選人提名表格，發現12個界別當中，共有6名參選人申報持有外國國籍或居留權，另外會計界黃宏泰申報自己擁有過期的美國居留權。


| **功能界別** | **參選人** | **持有外國國籍/居留權** |
| --- | --- | --- |
| 建築、測量、都市規劃及園境界 | 謝偉銓 | 英國居留權 |
| 建築、測量、都市規劃及園境界 | 陳澤斌 | 英國居留權 |
| 法律界 | 林新強 | 英國居留權 |
| 地產及建造界 | 龍漢標 | 英國、加拿大居留權 |
| 工業界（第一） | 梁日昌 | 英國國籍 |
| 會計界 | 黃俊碩 | 英國、加拿大居留權 |
| 會計界 | 黃宏泰 | 美國居留權（過期） |


### 謝偉銓稱居英權「無乜價值」  惟仍未放棄

其中建築、測量、都市規劃及園境界兩名參選人均申報擁有居英權。現任議員謝偉銓表示，他早於1992年獲得居英權，但現時主要使用特區護照，居英權對他的價值已經「無乜」，直言「我可以放棄就放棄」。記者追問為何仍未放棄時，他回應：「我無話唔放棄，點解你哋又咁關注有無居英權嘅問題呢？係咪放棄咗就有另一個諗法呢？同埋如果大家覺得呢樣嘢，對工作方面造成障礙，咁我當然會放棄，不過我睇唔到（有障礙）。」他又強調，擁有居英權不會影響他效忠國家和特區。「完全無影響，你睇返過去就知啦，同埋我亦都係全國政協。」

### 建築署前助理署長陳澤斌：愛國同你攞幾多個國籍無咩關係

挑戰謝偉銓的建築署前助理署長、建築師註冊管理局主席陳澤斌承認，早於8、90年代時已申請居英權，一直持有至現在。他表示，由於法例沒有規定，會到當選後才決定是否放棄居英權。對於持有外國居留權，會否影響對國家和特區效忠，陳澤斌認為沒有影響。「愛國就不嬲都愛國，呢個我諗同你攞幾多個國籍無咩關係，呢樣嘢喺香港嚟講好普通㗎啫，都唔係淨係我係咁。」他又指，由於法例容許他持有外國居留權，相信跟足選舉法例就沒有問題。

會計界4名參選人當中，亦有2人申報持有外國居留權。其中民建聯黃俊碩申報擁有加拿大和英國居留權。他向記者表示會著同事回覆查詢，惟直至截稿前仍未獲回覆。另一參選人、灣仔區議會主席黃宏泰，在表格報稱擁有美國（過期）居留權。他接受查詢時指，曾經擁有美國居留權，但現時已過期失效，如入境美國需重新申請，申報只為公正起見，提供清楚資料予當局審批參選資格。

參選法律界的前律師會會長林新強亦申報持有居英權，他書面回覆稱，90年代初因為孩子的學業問題而取得居英權，但未有回應有否計劃放棄居英權。至於會否影響效忠國家和特區，他僅稱《基本法》容許立法會有不多於20%外籍議員。參選地產及建造界的地產建設商會秘書長龍漢標則申報擁有加拿大和英國居留權，他至截稿前未有回應。參選工業界（第一）的自由黨梁日昌則申報擁有英國國籍，記者至截稿前未能聯絡他回應。

根據《立法會條例》，除12個指明功能界別以外，立法會候選人必須是香港永久性居民中的中國公民，不能擁有外國居留權。12個容許候選人擁有外國居留權的功能界別包括：法律界；會計界；工程界；建築、測量、都市規劃及園境界； 地產及建造界；旅遊界；商界(第一)；工業界(第一)；金融界；金融服務界；進出口界；保險界。

### 政制局：容許有外籍議員反映包容、多元  助維持國際都會形象

《基本法》第67條訂明，非中國籍或在外國擁有居留權的香港永久性居民可擔任立法會議員，但其比例不得超過全體議員20%。政制及內地事務局指出，現有安排可吸納不同人才進入立法會，反映香港的包容性和多元化，讓非中國籍人士對香港作出貢獻，並有助維持香港作為一個國際都會的形象。

今年4月在修改選舉制度的草案委員會上，立法會法律顧問曾去信政府，查問當局有否諮詢相關的持份者，應否保留現行12個界別容許議員持外國國籍。當局回覆指，不建議將容許持外國國籍的功能界別，隨議席增加而上調至18席（90席中的20%），解釋指2010年政改時，亦沒有因立法會增至70席而增加相關議席。

### 至少35功能界別參選人  非參選界別選民

眾新聞翻查參選表格，亦發現至少35名參選功能界別的參選人，本身並非參選界別的選民，亦沒有申報為其他功能界別選民，但因為申報與界別有密切聯繫而參選。當中有8名現任議員，包括漁農界何俊賢、勞工界郭偉强、商界（第二）廖長江、工業界（第二）吳永嘉、金融界陳振英、金融服務界張華峰。立法會主席、參選工業界（第一）的梁君彥，以及擔任議員20年的飲食界張宇人，同樣不是所屬界別的選民。

另外，亦有多人在報名參選時，一度誤以為自己是界別選民，需要在表格上作出更正。包括法律界林新強，他稱已登記為「香港特別行政區全國人大代表香港特別行政區全國政協委員及有關全國性團體代表界」的選民。另外，勞工界李廣宇、工業界（第二）羅程剛、金融界陳振英、金融服務界張華峰、體育、演藝、文化及出版界蘇惠良、紡織及製衣界陳祖恒均在報名時一度誤報。
