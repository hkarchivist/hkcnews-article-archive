# 【不信任】北京代港訂國安法實施細則　立法期間基本法委員會要簽保密協議

繼人大常委繞過立法會直接頒布實施國安法後，特首林鄭月娥會同國家安全委員會周一根據國安法第43條，訂立國安法執法實施細則。據悉，國安法實施細則仍然主要由北京制訂，港府主力負責法律適應化功能，主動權恐已不復存在。

特首林鄭月娥周二出席行政會議前則推翻自己兩星期「無看過（國安法）法律條文」，港府在立法過程中曾獲諮詢，反駁外界說港府遭蒙在鼓裏、乜都唔知，「呢個幻想力及誇大能力令我大開眼界」。

人大常委會在6月30日通過國安法，晚上11時刊憲生效。一個星期後，政府在特首會同國安委決定後，在周一晚再刊憲公布全長116頁的《國安法第43條實施細則》。細則第4條列明：「本實施細則的中文文本為真確本，解釋本實施細則須以此為根據。英文譯本僅供參考。」（The Chinese text of these Implementation Rules is an authentic text, and these Implementation Rules are to be construed accordingly. The English translation text is for reference only.）港區國安法的主體條文，並沒有類似的說明。不過，它的英文版本並非同步刊憲，而是先有新華社英文譯本（注明只是參考），其後刊憲的英文版與新華社英文版本一樣，也標注for information，即並非具法律效力版本。

[國安法第43條實施細則（英文版）](https://web.archive.org/web/20200716213710/https://bit.ly/3gzMWgm)

政界消息向眾新聞指出，細則雖然提及實施情況，但主要框架及執法流程仍由北京主導，保安局等主要負責法律適應化工作。

有資深親中陣營人士也相信，中央自去年8、9月已開始研究國安法立法，由中央直接制訂細則「完全唔出奇」。該人士補充，國安委周一才開會，但中央希望盡快落實，所以中央完全可以直接「櫃桶底攞一份（細則）出來」給特區政府，做法切合國安法立法過程中，由中央主導的作風。

保安局局長李家超傍晚出席立法會會議後，未有正面回應是否中央主要制訂細則：「細則係由國家安全委員會制訂的，國家安全委（員會）的事情我地唔會公開。」

行政會議成員湯家驊則認為，國安法第43條指定大框架，涉及香港本身執法程序，都要根據香港程序辦事，所以特首會同國安委制訂細則，「情況等同附屬法例，香港係有參與啊」。

林鄭月娥在兩個星期前回應記者提問承認，並無看過國安法完整條文，只是基於當時剛通過的全國人大決定和新華社公布人大決定說明所見作出評論。不過兩周後，林鄭月娥今天早上稱，只是無見過「全套法律」，反駁外界指特區政府被蒙在鼓裡、什麼都不知道等說法，她形容這些批評「幻想力及誇大能力令我大開眼界」。

不過，中央在整個國安法立法過程中，是否信任香港確成疑問。據眾新聞了解，基本法委員會委員需要在立法過程中簽署保密協議，做法前所未見。

目前基本法委員會港方委員包括副主任譚惠珠、委員陳弘毅、莫樹聯、黃玉山及身兼立法會議員的梁美芬。

梁美芬不評論是否要簽署保密協議。她說：「唔講得喎，呢啲係內部運作。真係唔好意思，講其他、講法律解讀。」
