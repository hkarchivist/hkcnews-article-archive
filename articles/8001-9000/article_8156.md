# 朱經緯供稱　928後警方不再考慮放催淚彈

退休警司朱經緯，在2014年佔領運動期間涉以警棍在旺角毆打青年鄭仲恆，被控一項襲擊致造成身體傷害罪。朱經緯作供期間，曾透露警方在當年9月28日施放87枚催淚彈後，便不再考慮使用此戰術。除此之外，朱經緯自辯期間，亦提及他在佔領行動時，曾使用的部分戰術。

**朱經緯供詞整理：**


| **2014年** | **佔領事件** | **朱經緯提及警方布防** |
| --- | --- | --- |
| 9月28日 | 佔領運動開始 | 警方防護封鎖線、包圍及驅散示威者的對策中，其中一個是使用無毒、非致命的催淚氣體。自9月28日警方施放87枚催淚彈後，施放催淚氣體的戰術無「不使用」的明確指示，但所有行動計劃中，此戰術不被考慮。 |
| 10月17日 | 市民在晚上聚集旺角。除了重新佔領彌敦道北行線外，佔領的範圍比之前更大。 | 警方在凌晨成功開通彌敦道北行線，恢復通車。當晚聚集旺角的人約有9000人，是佔旺期間最多人的一日。   
   
 朱經緯稱：「由嗰日開始，旺角嘅情況越來越壞，警方嘅容忍度、對法律嘅底線越退越後」，又指很多衝突、挑釁、身體暴力差不多每日發生。 |
| 10月19日之後的一段時間 | ／ | 隸屬第一梯隊和部分第二梯隊的警員獲發多一支警棍。該警棍「有新戰術跟埋」，名稱為「check drill」，要使用該警棍須接受其他訓練。   
   
 朱說，該款警棍的其中一個用途，是面對人群「全部舉手，好被動地行過來，迫到嚟，同你面對面」的情況，形容該人群「表面passive，其實好active。」 |
| 11月25日 | 潮聯小巴成功申請禁制令後，其代理律師及執達主任到亞皆老街宣讀禁制令，並派人清除障礙物。警察晚上在旺角驅散群眾，期間曾使用胡椒噴霧、催淚水劑驅趕示威者。 | 所有在場帶隊的指揮官都需要聆聽訓示，內容包括分配工作、過去24小時發生的事（尤其清拆、禁制令的情況）、解釋旺角的最新情況。   
   
 朱經緯晚上7時抵達旺角，職責負責守衛亞皆老街。他形容當晚是高度緊張、不穩定，很多人蒙面、戴頭盔、身穿保護工具、拿自製盾牌及其他工具，並嘗試在不同地方重行佔據道路。   
   
 警方整晚透過公共媒體或現場作不同警告，但沒有效果，警方亦需用武力恢復公共秩序。當晚約90多人在現場被捕，20多名警員受傷。 |
| 11月26日早上至晚上7時45分前 | 的士團體的代表律師及執達主任到達旺角，宣讀臨時禁制令內容。社民連黃浩銘及學民思潮召集人黃之鋒等人要求執達主任及律師，說清楚會否清走在道路上的人。 | 朱經緯在早上7時半離開旺角。   
   
 朱經緯再到旺角執勤前，曾聽取半小時的訓令，又向其下屬作15分鐘的訓示，內容包括前一晚發生的事、當日早上警方執行禁制令時遇到的困難、當時的最新情況、當晚的工作，以及他分配予其小隊的工作。 |
| 11月26日晚上7時45分後 | 晚上大批市民開始聚集旺角 | 朱經緯及其大隊到達旺角現場。當時旺角有千計的人聚集，朱曾在警方電台通訊器聽到，西洋菜南街有好大群人。主要人群一路向亞皆老街移動，警方亦部署很重的警力。   
   
 當時人群企圖衝出彌敦道，希望再佔領彌敦道，警方曾作多次警告，很多衝突需使用武力恢復秩序，朱經緯形容是「你進我退」、「拉鋸」的狀態。除了主要的人群，旺角其他地方均有人「搞事」。 |
| 11月26日晚上9時 | 大批市民聚集旺角，警方曾作多次驅散。 | 當時警方仍容許群眾過馬路，但馬路上突然多出幾百人，基本上「我行返過去，你又行返過嚟」，又以綁鞋帶、跌散紙等方式在馬路中停下，要花費5、6次紅綠燈次數才能讓他們完全橫過馬路。警方其後需用人手將他們攔住。   
   
 朱又指，當晚故意拖慢過路的人，主要聚集在惠豐中心前往匯豐銀行的路段。 |
| 11月26日晚上10時 | 大批市民聚集旺角，警方曾作多次驅散。 | 警方決定封鎖亞皆老街和彌敦道路口，朱經緯形容當時人群情緒越來越高漲。當時警方為市民提供其他路段過路，並用大聲公作呼籲。   
   
 當時警方共面對三大人群，分別是西洋菜南街聚集，企圖想出到亞皆老街的人群；惠豐中心外的人群；彌敦道對面、匯豐銀行外的人群。朱表示，警方要推開他們，防止他們接近行人過路線，期間有警方拔出警棍，使用「check drill」戰術。   
   
 裁判官曾問警棍有否接觸人的身體，朱經緯先答沒有，其後答有，指「check drill」會構成身體接觸。 |
| 11月26日晚上約10時20分 | 大批市民聚集旺角，警方曾作多次驅散。 | 朱經緯指，他當時在亞皆老街與彌敦道交界，面對惠豐中心外的人群，又聽到惠豐中心的人群突然起哄，令警方的注意力移向該處，忽略背後的上海商業銀行。 |
| 11月26日晚上約10時半 | 大批市民聚集旺角，警方曾作多次驅散。期間，鄭仲恆曾被朱經緯以警棍打。 | 朱經緯作供指，有下屬急忙向他表示， 「上海商業銀行個邊出事」，他回頭發現，該段行人路都是「密麻麻」的人，並開始衝擊警方防線。他見狀後，與身穿反光衣、約20人的下屬一同增援，期間曾以警棍擊打鄭仲恆。 |
