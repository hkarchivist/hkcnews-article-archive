# 高血壓、高膽固醇35歲浮面　醫生批60歲做篩查太遲

衞生署上周公布 [2014-15年度人口健康調查報告](https://web.archive.org/web/20210629150506/http://www.chp.gov.hk/tc/static/51256.html) ，關注公共衛生政策的醫護行者主席、中大醫學院榮譽臨床助理教授范寧深入分析報告，指出雖然報告提及中風、冠心病在55至64歲開始「爆發」，但這些疾病的風險因素，包括：高血壓、高膽固醇、糖尿病等，往往早10至20年、即早至35歲便可能會浮現，但政府只資助六旬長者做篩查「為時已晚」，市民應在30餘歲開始做身體檢查，及早防範。

衞生署於2014年12月至2016年8月期間，透過實地問卷調查及身體檢查，訪問5,435個家庭住戶、共12,022名15歲或以上人士，並為當中2,347人進行身體檢查。結果顯示，整體有39.6%人士患有至少一種慢性疾病。


| **常見慢性疾病** | **平均患病率** |
| --- | --- |
| 高血壓 | 17.8% |
| 血膽固醇過高 | 14.4% |
| 糖尿病 | 5.5% |
| 冠心病 | 2.1% |
| 哮喘 | 1.8% |
| 癌症 | 1.5% |
| 中風 | 1.4% |
| 慢性阻塞性肺病 | 0.5% |


上述部分慢性疾病亦會互為影響，例如 [高血壓](https://web.archive.org/web/20210629150506/http://www.chp.gov.hk/tc/content/9/25/35390.html) 、 [高膽固醇](https://web.archive.org/web/20210629150506/http://www.cheu.gov.hk/files/her/cholesterol_lft.pdf) 可導致冠心病和中風，而 [糖尿病](https://web.archive.org/web/20210629150506/http://www.elderly.gov.hk/tc_chi/common_health_problems/diabetes/diabetes.html) 的慢性併發症亦可引致高血壓、冠心病和中風。

根據報告結果，冠心病和中風在55至64歲年齡組別開始「爆發」。

冠心病平均患病率為2.1%。35至44歲受訪者中僅得0.2%人患有冠心病；到45至54歲年齡組別的患病比率則為0.9%；但到55至64歲，患冠心病比率大幅躍升2個百分點至2.9%，此後隨年齡增加，患病率幾乎以倍數上升（下圖）。

中風的患病率亦有類似趨勢。35至44歲年齡組別有0.2%曾被醫生診斷患有中風；45至54歲則有0.5%；到55至64歲，患病率是前一個年齡組別的3倍，即1.5%；此後65至74歲、75至84歲兩個年齡組別的患病率，均較前一個年齡組別多約3個百分點（下圖）。

<figure>
<img src="https://web.archive.org/web/20210629150506im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/12/人口健康調查-中風-20171204195950_2bb1_large.jpg" alt="">
</figure>

高血壓、高膽固醇及糖尿病的患病率，則較冠心病和中風早10至20年、即35至54歲開始浮現。

調查顯示，高血壓在34歲或以下的患病率維持在1%或之下，自35歲開始升至4.7%；到55至64歲年齡組別，患病率達27.9%，已比平均值17.8%高逾10個百分點（下圖）。高膽固醇各年齡組別的患病率走勢相若。

<figure>
<img src="https://web.archive.org/web/20210629150506im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/12/血壓高-人口健康調查-20171204194908_1854_large.jpg" alt="">

糖尿病則在45至54歲年齡組別見明顯升幅，44歲或以下的年齡組別患病率少於1%；到45至54歲飊升至3.6%，下一個年齡組55至64歲起，已超越5.5%的平均值，比率大致隨年齡上升而增加（下圖）。

<figure>
<img src="https://web.archive.org/web/20210629150506im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/12/糖尿病-人口健康調查-20171204200008_6ac9_large.jpg" alt="">
</figure>

醫護行者主席、中大醫學院榮譽臨床助理教授范寧分析，調查結果顯示出高血壓、高膽固醇、糖尿病與中風、冠心病的關係，相關疾病的篩查年齡亦應該推前，他建議在30多歲開始做身體檢查。「35歲開始爆上，比中風、冠心病早咗10至20年，呢個果（中風、冠心病）係10幾20年功夫。所有嘢應該喺55歲前要做，所以而家好多篩查，其實都唔應該等到60、70歲先做。」

調查顯示，35至44歲年齡群組之中，曾接受膽固醇檢驗、血壓量度、血糖檢驗者，比率分別為52%、72.1%、56.3%。衞生署指出，在調查中被驗出有高血壓、糖尿病的受訪者，約半數事前並不知道自己患病，而被驗出有高膽固醇者，更有7成事前不自知。

[癌症第二號殺手](https://web.archive.org/web/20210629150506/http://www.healthyhk.gov.hk/phisweb/zh/healthy_facts/disease_burden/major_causes_death/cancers/cancers/) 大腸癌，與冠心病的風險因素類同，包括：肥胖、缺乏運動、有糖尿病史及高血脂等。政府去年9月起推出為期3年的「 [大腸癌篩查先導計劃](https://web.archive.org/web/20210629150506/http://www.chp.gov.hk/files/pdf/3_colorectal_cancer_prevention_and_screening_chi.pdf) 」，分階段資助於1946至1955年出生、沒有大腸癌徵狀的市民接受大腸癌篩查，換言之，是61歲或以上人士才受資助。

<figcaption>在衞生署調查中被驗出有高血壓、糖尿病、高膽固醇的受訪者，為數不少事前並不知道自己患病。資料圖片</figcaption>

范寧提到，透過「增加體能活動量或運動量」、「控制或降低體重」等生活行為去控制高血壓、高膽固醇、糖尿病的比率偏低，只是介乎2至5成，情況值得深入了解，「生活行為做唔到，係佢唔知，定係逼於無奈做唔到呢？係咪成個生活環境令佢做唔到？有冇嘢幫佢做到？呢個係社區可能幫佢，而醫生、醫療制度幫唔到。」他引述一項 [美國研究](https://web.archive.org/web/20210629150506/http://www.hpoe.org/Reports-HPOE/2017/determinants-health-food-insecurity-role-of-hospitals.pdf) 指，市民的健康主要受社會經濟因素（40%）、臨床護理（20%）、外在環境（10%)影響，個人可以控制的比例僅30%。

<figcaption>透過「增加體能活動量或運動量」、「控制或降低體重」等生活行為去控制膽固醇的比率偏低，高血壓及糖尿病的情況相若。《2014-15年度人口健康調查報告》截圖</figcaption>
</figure>

透過控制飲食以控制膽固醇，亦非所有人都做得到。范寧指，減少進食高脂或高膽固醇的食物未必是個人意願，尤其是低入收家庭，鮮有機會在家煮食， 往往一日三餐都在街外吃，政府對食品供應商、餐廳或可作監控、抽查，「既然食衛局會抽查食物係咪安全，咁係咪可以查埋『三高』（高血脂、高血壓、高血糖），查出就罰款、俾警告佢？會唔會整個『三低餐廳』，俾個標誌佢，唔係淨係『有營食肆』咁簡單，如果佢有菜，但全部都係『三高』菜，咁點搞呢？」衞生署今個學年推出學校午膳減鹽計劃，現時有13間學校午膳供應商參與，以每年減少供校午膳的鈉含量最多10%為目標，范寧建議政府考慮將計劃以強制性質推行，從小控制飲食習慣。

范寧又留意到，被醫生診斷為患有高血壓、高膽固醇、糖尿病的受訪者，均有約1成人選擇「服用不需要醫生處方的成藥」，他警告：「呢10%係炸彈。」范寧指出，相比急性疾病，慢性疾病患者為「已知人口」，本應受到控制，尤其是血壓、血糖水平該受持續、準確監控，並由專人處方相應的藥物，惟調查反映高達1成高血壓、高膽固醇、糖尿病患者自行用藥，他認為情況危險，「呢班人有冇醫生跟緊？有冇人幫佢哋？佢哋有冇自行monitor（血壓、血糖水平）？呢個係好嚴重嘅健康管理問題，甚至比亂咁食抗生素更危險……血糖太低係會死人。」
