# 性侵指控浮現　美國再有著名電視主持下馬

<figure>
<img src="https://web.archive.org/web/20210630215115im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/11/革職-性醜聞-勞爾-20171130153220_10b9_large.jpg" alt="">
<figcaption>勞爾是美國家傳戶曉的電視節目主持。美聯社</figcaption>
</figure>

美國全國廣播公司（NBC）周三突然解僱王牌早晨節目《今日／Today》主持人勞爾（Matt Lauer），原因是「在工作場所不當性行為」。勞爾是美國家傳戶曉的電視節目主持人，年薪達2500萬美元。NBC新聞部表示，在一名女子周一會晤公司行政高層後，公司決定解僱勞爾。

勞爾之前，近期已有兩名美國傳媒人因性醜聞被革職，包括哥倫比亞廣播公司（CBS）主持人羅斯（Charlie Rose）及MSNBC政治記者哈爾柏林（Mark Halperin）。NBC新聞部主席賴克在發給員工的通告表示，這是勞爾在NBC工作20年的第一個投訴，但電視台有理由相信這不是單一事件。在周三的編輯會議上，賴克表示勞爾涉及的一個投訴，是始於2014年俄羅斯索契冬季奧運會，而且回到紐約後仍然繼續。勞爾被解僱後，NBC再收到兩宗關於他的投訴。

在勞爾被革職數小時之後，娛樂雜誌《Variety》上載據稱是進行了兩個月的調查，訪問了幾十名不願透露姓名的現任及前任員工。報道說，有人指控勞爾曾給一名同事一個性玩具，還貼了一張紙說想用在她身上；有指勞爾在女同事面前露體；有稱勞爾問女性製作人她們的性生活；亦有指勞爾說到他想與哪些主持人睡覺。報道指勞爾的辦公室桌下有一個掣，他不必站起來就能夠鎖上房門。記者向勞爾及其經理人查詢，但是沒有回覆；而NBC沒有說明勞爾是否認抑或承認做過這些事。勞爾已婚，有三個孩子。

《紐約時報》報道說，根據對超過六名前《今日》員工的採訪，這個節目雖然播出時營造一種家庭的融和氣氛，但在熒幕背後，有的時候卻是男童會的氛圍。《紐約時報》引述《今日》前員工稱，用女性儀表開玩笑是經常的事。《今日》是美國電視行業的著名老牌早晨節目，首播是1952年，是全球首個早晨電視節目，內容包括新聞報道及專訪，長期是同類節目收視第一位，直至八十年代末被美國廣播公司（ABC）的《早安美國／Good Morning America》取代。到了1995年，《今日》奪回首席直至2012年4月。如今《今日》穩坐第二把交椅，收視人數約400萬。

NBC周三再收到兩宗有關勞爾性侵犯的投訴，其中一名投訴者對《紐約時報》表示，九十年代末她擔任《今日》製作人不久，勞爾就對她作出不當之言。一次，她與勞爾共同做一宗新聞，晚飯時勞爾提了一些不當的問題，例如有沒有欺騙過丈夫。在往機場途中的車上，勞爾坐得很接近她，令她感覺不安。她憶述說後來坐開，勞爾說「你真沒趣」。

她對《紐約時報》說，2001年，勞爾要她到辦公室討論一則新聞，進了辦公室後，勞爾鎖上房門，其後勞爾要她解開襯衣鈕扣並與她性交。其間她昏了過去，醒來時發現自己在勞爾辦公室地板上，勞爾要他的助理帶她去見護士。她說，沒有把事件告訴NBC，因為她覺得自己應該做更多些事來阻止勞爾。大約一年後，她離開NBC。這名女子把事件告訴仍在NBC工作的當時上司，事件周三報告給NBC新聞部。

| 機構 | 職位 |
| --- | --- |

| MSNBC | 政治評論員、作家兼記者哈爾柏林（Mark Halperin） |
| 《紐約時報》 | 駐白宮記者思拉什（Glenn Thrush） |
| 美國全國公共廣播電台 | 兩名高級新聞編輯奧拉斯基斯（Michael Oreskes）、斯威尼（David Sweeney） |
| 哥倫比亞廣播公司（CBS） | CBS This Morning 主持人盧斯（Charlie Rose） |
| Minnesota Public Radio (MPR) | 紅牌電台節目主持人基勒（Garrison Keillor） |
| Fox News | 高收視The O'Reilly Factor主持人奧萊利（Bill O'Reilly） |
| 美國全國廣播公司（NBC） | 《今日／Today》主持人勞爾（Matt Lauer） |


《紐約時報》周三發表長篇文章 [「A Failure of the Network News Star System／為甚麼新聞業的明星系統造就了這群惡霸」](https://web.archive.org/web/20210630215115/https://cn.nytimes.com/usa/20171130/matt-lauer-sexual-misconduct/) ，討論近期電子媒體的著名主播及主持人被指涉及性犯罪的情況。

文章指出，這些星級主持都是美國家傳戶曉的人物，角色深入民間。有像隨和父親的NBC勞爾；講故事搞笑話一流的古怪叔叔、公共廣播電台的基勒；舉止儒雅、愛問這問那的CBS主持人盧斯；讓觀眾覺得有他在身邊「一直在照顧你、支持你」的Fox News奧萊利。

這些媒體名人多年來扮演的角色讓他們在廣播界獲得了聲望，甚至具有全國政治影響力。現在，他們一個接一個陷入性醜聞的指控。在電視上扮演的角色外衣被剝去後，聲望盡失，代表著舊電視新聞的父權象徵正在崩壞。

從他們的年薪——例如勞爾和奧萊利各自年薪就超過2000萬美元，可反映出他們是電視網的金山銀山，電視台靠他們吸引觀眾收看。由於在他們身上的巨大投資，電視網姑息養奸，無視對他們不當行為的指控，導致罪行多年來多次重演。

這些人現在才被革退，報應可能來得太晚。這本來就是電視網早該要做的：畢竟，它販賣的主要產品是誠信，而代言人就是坐在主播台後面的人。電視網一旦失去觀眾的信任，整個企業就會崩潰。
