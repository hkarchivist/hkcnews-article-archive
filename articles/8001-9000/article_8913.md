# 飲唔飲？齊齊飲？

一位挪威裔朋友說，他兒時最怕過聖誕，怕？聖誕不是小孩子最高興的日子嗎？挪威人聖誕大過天，分散各地的家人會一連三天，在某家一起歡聚，除美食和糖果雪糕甜品，還一家大細齊齊玩的各樣遊戲，小孩子更拆禮物拆到手軟，怎會怕過聖誕？朋友的兒時經驗，原來不是孤例，聖誕連續幾天狂歡飲飲食食，問題出在飲酒。據挪威公共衛生研究所(Institute for Public Health)，2011年父母酗酒的兒童和青少年一共9萬名，佔該組別人口8.3％；他們的聖誕經驗，不是歡樂，是恐懼。

### 飲唔飲？

據WHO(世界衛生組織) ，2008-2010年歐洲的人均酒精類飲品量是10.9公升，挪威人不是最豪飲一族，只7.7公升，但問題在飲酒習慣。

挪威人一般冷靜理性克制，但不知是否一至五自我克制得太厲害，好杯中物的，一到週末便放縱狂隊酒，見過幾位平時温文爾雅的朋友，暴飲後提高8度聲浪，胡言亂語手舞足蹈，判若兩人。如果你去過挪威，週末晚上在市中心，暴飲一族你應碰過。週末尚如此，何況聖誕這長假？

挪威12月份酒的銷量驚人，酒類人均銷量是：啤酒43.7瓶，葡萄酒7.2瓶(2015)，大幅拋離其他月份，說明不少挪威人聖誕愛狂隊酒的現象，據一項調查，50萬挪威人(佔人口約9.4％)認為聖誕家庭聚會如有人暴飲的話，會令人不快。


{{< video src="https://web.archive.org/web/20210712012848if_/https://www.youtube.com/embed/ledVRZ7R4UQ" >}}


暴飲和酗酒，不單失儀兼傷身，更能亂性，特別對家人和兒女造成很大傷害，挪威如是，香港也如是：

挪威一小孩（挪威報章VG, 2014）：

> **爸爸對我和小弟弟從來不兇惡， 但如果媽媽批評他喝酒，他們往往會大聲吵架，有時甚至會動手動腳。 我通常會帶小弟弟躲進浴室，將門鎖上，然後用手摀住他的耳朵，令他不用聽到最可怕的吵架聲。有次聖誕節，爸爸喝了很多紅酒，他在沙發上躺了一段很長時間，之後吐了很多紅色的液體，我以為是血，以為爸爸會死掉。**


香港一位住院戒酒的病人：

> **從小常看見爸爸喝得爛醉，常常為了拿吃飯的錢去買酒而和媽媽不斷爭吵，甚至對我和媽媽拳打腳踢。每天我都生活在恐懼當中，想逃離這樣的環境卻又離不開。我從小告訴自己千萬不要像他，諷刺的是，如今我卻天天喝酒，竟然還需要住院。**


酗酒父母對小孩造成多年身心的後遺深遠，難怪有8.3％挪威兒童和青少年怕過聖誕。8.3％這數字較人太沉重，難怪挪威有一調查，認為在聖誕家庭聚會，成人不應在小孩和青少年面前喝酒的，高達80%。不少輿論認為，在小孩和青少年面前，最多一杯起兩杯止， 應等他們先睡，才盡情把酒談歡。

「在小孩和青少年面前，飲唔飲酒？」這問題，你怎看呢？再看香港兒童和青少年的飲酒情況，香港和挪威倒有些文化差異。

### 齊齊飲？

香港的人均酒精類飲品量是2.6公升(2016)，不單遠不及挪威(7.7公升)，也低於亞洲其他國家，不過，近年香港兒童和青少年飲酒比例增加，香港理工大學研究(2014)更指出，平均最早接觸酒精飲品的年歲為10.9歲，另外小學生飲酒比例偏高，令人擔心：


| 香港學生的飲酒情況(2014-15) |  |
| --- | --- |
| 小四至小六學生 | 26.00% |
| 中學生 | 56.80% |
| 專上學生 | 78.70% |
| 註：上述為以往曾經飲酒、過去1年內及過去30天內曾經飲酒的比例。   
 資料來源：衞生署 |  |


有點令人費解的是部份父母鼓勵兒童喝酒的態度：

*(1)有飲酒的小學生當中，超過50%表示酒精主要來自父母；20%來自祖父祖母等家庭成員；超過10%人是自己購買。　資料來源：香港大學公共衞生學院(2013)*

*(2)「父母鼓勵喝酒的行為和中學生喝酒的習慣與傾向有顯著的關係，當中以鼓勵子女喝酒和教子女喝酒練酒量的行為，最影響中學生每月喝酒和未來12個月可能喝酒的傾向。*

*本港不少家長有着錯誤的觀念，以為既然子女長大後交際應酬都難免要喝酒，倒不如早些「訓練」他們的酒量，減低將來被人灌醉或醉酒失儀的機會。然而，父母經常在子女面前不抗拒甚至鼓勵的態度，例如在子女面前喝酒甚至喝醉、要求子女買酒、開瓶、直接地鼓勵喝酒、教子女喝酒練酒量等，都會影響子女對酒的態度和行為，不知不覺地損害他們的健康。」* *資料來源：衞生署衞生防護中心*

聖誕家庭聚會，挪威人為「在小孩和青少年面前，飲唔飲酒？」而糾結，在香港，有部份父母不單在未成年子女面前喝酒，仲齊齊飲齊齊happy，認為要早些「訓練」他們的酒量，飲酒也要贏在起跑線？不過，衞生署並沒有提出父母鼓勵兒童喝酒的數字，是否只佔很少數，不得而知。

聖誕，是一年中最大的節日，難得一家歡聚，挪威人極度期待的同時，卻又為預備聚會而極度焦慮，挪威心理治療師A. Stefanusson 曾這樣描述挪威人過聖誕：「做挪威人並非易事，我們有世界上其他人沒有的挑戰。」這番本來是自嘲的話，但如果你想起每年有8.3％挪威兒童和青少年怕過聖誕，對挪威成人，究竟「飲唔飲酒」的確是一巨大挑戰，Stefanusson的戲言，不是笑話。怎辦？
