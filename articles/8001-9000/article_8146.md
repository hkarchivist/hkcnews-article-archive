# 當差36年　朱經緯曾鎮壓越南難民暴亂

退休警司朱經緯，在2014年佔領運動期間涉在旺角以警棍毆打青年鄭仲恆，被控一項襲擊致造成身體傷害罪。案件在東區裁判法院審訊。朱經緯自辯期間，講述他在警隊內的晉升過程，以及執勤經驗。朱經緯進行證人宣誓時，選擇天主教形式。

朱經緯加入警隊36年，主要在軍裝部工作，曾參與多個大型行動，包括90年代警方鎮壓越南難民暴亂的行動。

**朱經緯簡歷**


| 1979年9月24日 | 加入警隊 |
| --- | --- |
| 1980年代 | 駐守衝鋒隊，曾處理持械行劫案 |
| 1984年3月 | 晉升為督察，開始不時擔任機動部隊的訓練人員，直至退休 |
| 1990年代 | 參與鎮壓越南難民的暴亂 |
| 1993年 | 晉升為總督察 |
| 2003年 | 參與七一遊行的警務工作 |
| 2004年 | 第二次參與七一遊行的警務工作，負責護送主隊 |
| 2005年 | 香港主辦世貿會議部長級會議時，在警察總部行動組工作 |
| 2006年 | 晉升為警司，曾負責更新及改進機動部隊的訓練項目 |
| 2014年 | 在佔領行動中執勤 |
| 2014年11月26日 | 涉嫌在旺角用警棍襲擊鄭仲恆 |
| 2014年12月25日 | 退休前休假開始 |
| 2015年7月23日 | 退休生效 |
| 2017年3月27日 | 因本案被捕，被控一項襲擊致造成身體傷害罪 |
| 2017年11月6日 | 案件開始進行審訊 |
