# 潘小濤先生，你是否也想多了？

尊敬的潘小濤先生：

「識得睇一定睇留言」正如網絡名言所講，先生 Facebook Page 上的留言，應驗了魯立在「 [著了魔的曾俊華團隊](https://web.archive.org/web/20170323054533/https://www.hkcnews.com/article/2722/%E8%91%97%E4%BA%86%E9%AD%94%E7%9A%84%E6%9B%BE%E4%BF%8A%E8%8F%AF%E5%9C%98%E9%9A%8A) 」的預言：「香港連一點獨立意見也容不下，動不動就以莫須有的罪名，隨意向別人扣帽子，我們究竟是更團結？還是更撕裂？」

見到小濤先生的讀者，指魯立的文字是「高級五毛」；但最諷刺是這些扣帽子式的留言，偏偏就是「低級五毛」的水平。一句起兩句止的漫罵，究竟又有何意義？我相信「薯仔」是天真的，只不過他們見不到自己是如何自我矮化到連五毛都不如。

當然，讀者言論不代表樓主立場，所以我也只是想先生明察，在當今網絡生態，正是車公靈籤所云：「何為邪鬼何為神，神鬼如何兩不分，但管信邪修正外，何愁天地不知聞。」

其實寫「著了魔的曾俊華團隊」，早知會觸動「薯仔」的神經，但連小濤先生也跌入了誅心之論的思想陷阱，實在有點教人感到失望。

「我係睇完成篇文嘅！呢種手法，喺政改、佔中等見好多，話自己都支持民主撐普選，不過就唔認同咁爭取方法，甚至講成係受害者（「容姐」打上電台作故仔，講到聲淚俱下）。但實際上，佢由開始已經唔係薯粉，佢一直都只係梁粉，而假裝成前薯粉，就可以令到［薯片變了］嘅指控，更加有力！」 （摘自 [潘小濤 Facebook Page](https://web.archive.org/web/20170323054533/https://www.facebook.com/permalink.php?story_fbid=10155540832405016&id=27743445015) ）

首先，魯立要感謝小濤先生你讀完這篇蕪文。畢竟在今時今日，很少人有耐性聆聽別人的不同意見，至少小濤先生願意讀罷蕪文再回應，不過，小濤先生又憑甚麼去肯定魯立「一直都只係梁粉，而假裝成前薯粉？」小濤先生是猜想吧？可是先生的語氣，卻又彷佛像魯立肚裡的那條蟲。

至少在客觀效果上，我見到在「薯仔」圈子中，魯立在一時間變得聲名狼藉，人見人憎。說不定，魯立也只是正言若反，讓「薯仔」更團結一致呢？陰謀論厲害之處，就是正反皆可以講得通，誰都不可以推翻誰。

敢問先生一句，你可以推翻「魯立是曾俊華團隊的黃蓋」這個理論嗎？這就當是魯立向小濤先生的一份小小的挑戰。我無意改變先生的立場，但是我想觀摩見識一下，先生的論說功夫。

有人說魯立是衝著網上瘋傳黎女士的公開信，也有人說「煽動仇恨」是一頂帽子。要是先生有留意魯立的全文，應該可以見到魯立很刻意地分別「曾俊華」和「曾俊華團隊」。魯立說著了魔的，是「曾俊華團隊」，而不是「曾俊華」。小濤先生又可有想過，在先生的 Facebook Page 上留言的，有沒有「曾俊華團隊」的人？

魯立的疑問是：「假如曾俊華知道社會的撕裂，也見到有人因為不同的意見被人身攻擊，曾俊華有沒有挺身而出叫支持者冷靜理性一點？」

可惜，曾俊華沒有出來說一句公道話，這一點事實又說明了甚麼？曾俊華是團結了一群人，可是團結了這群人，卻在攻擊另一群人，這又算真的團結嗎？以團結作口號，不是偽善是甚麼？

無錯，魯立的確有讀過黎女士的公開信，更加對那種對偶像崇拜的情緒感到很不安。建制派或許在造王，但反對派何嘗不是在造神？這種狂熱的投入，對理性討論又有甚麼意義？

小濤先生是香港具影響力的公共知識份子，若然先生能夠認真回應魯立以上的疑問，理性討論，肯定是萬民之福。

最後，魯立也要感謝《眾新聞》刊登蕪文，沒有對言論作出審查。任何人若然相信真理越辯越明，正反的立場也應該要有得到表達的空間，否則這個社會就只剩下立場，不用討論的一言堂。
