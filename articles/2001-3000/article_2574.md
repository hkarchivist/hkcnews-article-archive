# 網絡白色恐怖？林鄭負評隨「公關災難」增減　情緒化稱呼佔少數

特首參選人林鄭月娥昨日出席教協論壇時，主動提到網民攻擊為她拍宣傳片的影星蕭芳芳，形容為白色恐怖。不過數據顯示，雖然針對林鄭的負評佔比一直維持在七成以上，但仔細分析下，大部分網民以中性稱呼對林鄭作出負評，而發表次數的波動隨著她的「公關災難」變化，由爆出「中央不任命論」和「廁紙論」之後開始飊升。而長期以「淋病」或「淋鄭」等情緒化名稱對她作出負面批評的網民僅佔少數。

《眾新聞》委託社交媒體數據收集及分析公司TAF DATA，自今年1月1日起，追蹤主要特首參選人的網絡輿情，隨著葉劉淑儀和梁國雄停止選舉工程後，目前僅追蹤3名特首候選人的網絡輿情，調查範圍包括Facebook等社交平台，及各個可在Google搜索到的網上論壇。截至3月12日晚上11時59分，這個社交媒體輿情系統共分析了730,415則留言，其中397,102則涉及林鄭， 189,850則曾俊華以及97,215則胡國興。

林鄭月娥早前曾指網上世界的特點是言論不負責任和情緒化，昨晚她更在特首論壇上自稱是白色恐怖的受害者，形容自己經常在網上被罵，連累到支持者蕭芳芳都被抨擊。林鄭直言：「這種白色恐怖是社會不能接受，需要大家團結、解決紛爭，唔好非黑即白，唔好唔鍾意一個人、一公里以內所有人都唔鍾意、要接受抨擊。」

到底是否大部分網民都如她所言的情緒化，事無大小都會砲轟她？《眾新聞》仔細分析數據，以對林鄭的稱呼將針對她的負面意見分為三種：

１）「林鄭／林太／奶媽」等中性名字和花名(下圖 **綠線** )   
 ２）「林病／淋病／淋鄭」等較情緒化稱呼(下圖 **紅線** )   
 ３）「梁振英2.0／CY2.0／689 2.0」等明顯政治取向稱呼(下圖 **黑線** )

分析發現，以「林鄭／林太／奶媽」稱呼她的網民，發表負評的波幅和她製造公關災難的時間大致吻合。林鄭月娥的負評佔比一直保持在七成以上，由1月初至今，負評次數出現了幾次高峰，分別是她宣布辭職、廁紙論、派錢給行乞婦、傳出張德江南下為她拉票、情人節翌日、民主思路論壇以及昨天晚上的教協論壇。

## **林鄭派錢內地行乞婦　引爆負評高潮**

從林鄭月娥的負評表看到，大部分以「林鄭／林太／奶媽」中性名字稱呼她的負評出現時間和事件發生日子大致接近。其中最明顯的上升是在林鄭在1月22日自爆「廁紙論」後，這類負評數量就一直持續上升，最高峰是在1月28日，即林鄭派$500元予內地行乞婦翌日。由當日起的一個多星期，直至2月7日傳媒報導人大委員長張德江南下深圳為林鄭拉票，林鄭這段時間的負評一直都處於高水平。隨後數日因林鄭沒有太多動作，所以負評開始回落，直至情人節，她在競選網頁貼出丈夫情書才再次回升。

其後以中性名字稱呼她的負評再現高峰的日子，包括情人節翌日、入閘翌日、民主思路論壇及昨晚的教協論壇。換言之，這批網民大多是回應林鄭月娥的事件而作出批評，並非無時無刻都針對林鄭。這類網民佔大多數，最高時期一日可以有4800多個負評，最低的時候一日幾百個留言。

值得留意的是，的確有一群網民會以「林病／淋病／淋鄭」等情緒化字眼稱呼林鄭，並且長期發文批評她，發文數量並未受新聞事件影響，但這類網民發表負評次數只屬少數。根據數據顯示，最高峰時期一日也只有2000多個此類留言，平常只在1000個留言左右徘徊。這類發言比較情緒化，有部分未必針對事件，反而涉及人身攻擊，只是純粹情緒宣洩，例如「淋病上腦毒蛾」、「淋病係女妖」等。不過，2月7日張德江南下消息傳出後，使用這些字眼稱呼林鄭的網民增加，反映這件事明顯觸動到網民神經，增加了不少情緒化的表達。至於第三類「梁振英2.0／CY2.0／689 2.0」稱呼的負比，佔的比率就更低，但數據很穩定，未出現太大波動，平均維持在一日500個以內。

當然針對林鄭的稱呼只能反映網民對她有較強烈情緒，未必能全面反映網民留言內容的理性程度。

--調查方法註釋--

本次調查的網絡輿情觀察系統，會收集網民就各參選人發表的留言，按其内容分為「正面／中立／負面」評價。TAF DATA已建立一個龐大詞庫，正面和負面的字眼，由調查1月1日開始時的9000多組，到近日已增加至各有近1萬組。系統透過詞義分析（semantic analysis），將留言內各個帶感情色彩（sentiment） 的字眼評分為+1或-1，正評的定義是留言内容中普遍使用+1的詞語，例如：「林鄭月娥好打得，政通人和，知人善任，盡心盡力，造福社會」；負評則是留言普遍使用負面情緒-1的詞語，例如：「上帝點解叫淋病選，唔係叫佢做特首？」

若程式遇上同時帶有+1及-1的內容，出現衝突情況，例如出現「支持林鄭月娥退休」既出現「支持」+1的詞語，也出現「退休」-1的詞語，系統就會啓動進一步分析發表者背後的社交網絡。基於網絡同溫層原理，若發表者背後的社交網絡有逾7成都是反對林鄭參選的，系統就會判斷為負面評價，並將這類留言標註出來，交由人手再覆核確認留言屬於正評抑或負評，未經人手判定前不會計算入正、負或中立評價的統計中。由於香港網絡環境充滿反諷意見，目前系統判定出現的誤差，至今約為+/-5%。
