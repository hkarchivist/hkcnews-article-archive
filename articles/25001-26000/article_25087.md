# 【數說區選】新民黨全軍覆沒　28參選人得票率低於建制平均值

新一屆區議會選舉產生452個議席，在破紀錄新高的71.2%投票率之下，民主派陣營共贏得 388個議席（85.8%），建制陣營只取得59席（13%），其餘5席屬立場不明確者。換言之，民主、建制議席比例大幅拉開至6.5比1。今次區選結果最重要的影響，是民主派可在下屆特首選委會中取得屬區議會的全數117席，足以在1200人的選委會中，影響下任特首的得票數字。此外，明年立法會選舉中的一個區議員互選議席，民主派料也能取得。

建制派主要政黨兵敗如山倒，其中新民黨更是「全軍覆沒」，派出28人參選，無一人獲勝。眾新聞整理新民黨得票情況，發現新民黨參選人的平均得票率為39.43%，略低於建制派整體得票率40.98%。

建制陣營中，以民建聯派出最多人參選（181人），其次是工聯會（62人）、新民黨（28人）及經民聯（25人）。當中新民黨選情最慘淡，全部無功而還。


|  | **參選人數** | **贏得議席** |
| --- | --- | --- |
| **民建聯** | 181* | 21 |
| **工聯會** | 62* | 5 |
| **新民黨** | 28 | 0 |
| **經民聯** | 25 | 3 |
| **自由黨** | 11 | 5 |


*部份參選人同時填報隸屬民建聯及工聯會，故有重複計算的情況

民主陣營方面，派出99人參加區選的民主黨，成功贏得91席，成為新一屆區議會的最大黨。而公民黨、民協、新民主同盟分別在超過八成參選選區中勝出。


|  | **參選人數** | **贏得議席** |
| --- | --- | --- |
| **民主黨** | 99 | 91 |
| **公民黨** | 36 | 32 |
| **民協** | 21 | 19 |
| **新民主同盟** | 20 | 19 |
| **工黨** | 7 | 7 |


民主、建制陣營「大洗牌」，粗略估計建制派今屆比上屆少逾200席，最新民主、建制議席比例約為6.5比1。然而，若以得票率計，民主、建制分別得57%及41%，仍然略低於「六四比」，建制票的穩固程度不容小覷。


| **陣營** | **議席** | **有效票數（佔比）** |
| --- | --- | --- |
| **民主** | 388 | 1,667,958（56.89%） |
| **建制** | 59 | 1,201,553（40.98%） |
| **立場不明確*** | 5 | 62,234（2.12%） |
| **總計** | 452 | 293,1745（100.00%） |


在今次區選中，多個中產選區錄得相當高投票率。建制派走專業、中產路線的新民黨，有派員參選的28個選區之中，包括東區太古城東、東區康怡、南區海怡西、深水埗美孚中、沙田第一城等選區，投票率均超過75%，高於整體71.2%，其中第一城及海怡西投票率更接近八成。在高投票率下，新民黨卻嚴重受挫，沒有取得任何議席。

*立場不明確：西貢環保北方國珊、西貢環保南張美雄、西貢都善張展鵬、深水埗下白田甄啟榮、荃灣郊區伍顯龍。

眾新聞整理新民黨派員參選的28區投票結果，發現新民黨參選人的得票率，介乎於30.03%至47.52%，平均為39.43%，低於建制派整體得票率40.98%。

新民黨成績最差的一區，為大埔舊墟及太湖選區，該區總投票率高達76.96%。新民黨的劉文杰，挑戰尋求連任的「埔向晴天」劉勇威，最終僅得2,310票（得票率30.03%），不及劉所得5,383票的一半（69.97%）。

[【新民黨28選區結果一覽】](https://web.archive.org/web/20200110232330/https://drive.google.com/open?id=1ZTHBuPpDoK5-adP2rAh7QPiZCPeV3xY30OQHCYJaBVA)

南區海怡西今屆投票率為79.36%。現屆區議員、被視為新民黨主席葉劉淑儀「接班人」的陳家珮，今次取得3,236票（43.73%），並以928票之差，敗予新人、黃之鋒的「Plan B」林浩波，後者取得4,164票（56.27%）。海怡西上屆投票率高達62.83%，顯著高於整體平均39.72%。在上屆區選中，陳家珮與人民力量袁彌明競逐議席，前者獲2,945 票（56.74%），後者得2,245票（43.26%）。

另一「大熱倒灶」的新民黨參選人，是有「第一城城主」之稱的黃嘉榮。他自2011年起出任沙田第一城區議員，上屆在沒有競爭下自動當選，今屆面對公民黨黃文萱挑戰，最終僅得2,753票（34.96%），而黃文萱得5,122票（65.04%）。換言之，黃嘉榮的得票僅佔黃文萱的一半。第一城今屆投票率為80.82%，比全港平均71.23% 高出9.59個百分點。

在屯門三聖，今屆投票率為68.57%，低於平均值。不過，新民黨蘇炤成同樣「老巢」不保。蘇炤成年屆79歲，擔任三聖區議員超過30年，他於2014年加入新民黨。報稱獨立的「立言香港」委員巫堃泰，今次挑戰蘇炤成，並以2,834票（56.32%）勝出，而蘇只得2,198票（43.68%），輸636票。

新民黨主席葉劉淑儀回應選區結果時，否認新民黨「亡黨」。她表示，新民黨參選人得票率接近4成，與建制派整體相若，又指新民黨在多區取得的票數均創歷史新高，反映選民支持。惟葉劉形容，今屆區選被「政治化」、「立法會化」，結果反映市民對政府有好大不滿。
