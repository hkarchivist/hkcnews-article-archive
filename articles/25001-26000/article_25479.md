# 【林鄭述職】習近平未再公開說「止暴制亂」、「依法懲治暴力」

<figure>
<img src="https://web.archive.org/web/20220120000019im_/https://www.hkcnews.com/news_新聞/hk-discuss/2019/12/北京-習近平-林鄭月娥-20191216202554_2dd9_large.jpg" alt="">
<figcaption>行政長官林鄭月娥（左）向國家主席習近平（中）述職。習近平表示，對於林鄭在香港非常時期顯示出的勇氣和擔當，中央是充分肯定的。政府新聞處圖片</figcaption>
</figure>

行政長官林鄭月娥周一下午到中南海瀛台，向國家主席習近平述職。習近平在公開發言時，開宗明義指2019年是香港回歸祖國以來，局面最為嚴峻複雜的一年。他重申，上月14日在巴西出席金磚國家領導人第11次會晤時，中央對香港局勢基本立場和態度，為三個「堅定不移」：維護國家主權安全發展利益的決心堅定不移、貫徹一國兩制方針的決心堅定不移、反對任何外部勢力干涉香港事務的決心堅定不移。


{{< video src="https://web.archive.org/web/20220120000019if_/https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRTHKVNEWS%2Fvideos%2F485826085381485%2F&show_text=0&width=560" >}}


習近平上月兩度開腔談論香港局勢，都提到林鄭月娥帶領的特區政府要「止暴制亂」。據新華社報道， [習近平11月4日在上海進博會期間見林鄭](https://web.archive.org/web/20220120000019/http://www.xinhuanet.com/politics/2019-11/05/c_1125192207.htm ) 時，明言止暴制亂、恢復秩序是「香港當前最重要的任務」。11月14日， [習近平在巴西出席金磚國家領導人第11次會晤](https://web.archive.org/web/20220120000019/http://www.xinhuanet.com/politics/leaders/2019-11/14/c_1125233663.htm ) 時，形容止暴制亂、恢復秩序是「香港當前最緊迫的任務」。

習近平會見林鄭述職的公開發言中，並沒有「止暴制亂」，傍晚 [新華社發稿](https://web.archive.org/web/20220120000019/http://www.xinhuanet.com//2019-12/16/c_1125353595.htm ) 亦沒有「止暴制亂」的字眼。不過，林鄭述職後見記者，引述兩名領導人習近平和總理李克強重申，目前最重要的任務是依法「止暴制亂」，恢復社會秩序，同時要堅守一國兩制方針，維護香港的長期繁榮穩定。

習近平又提到，中央將繼續：堅定支持林鄭帶領香港特區政府依法施政、堅定支持香港警方嚴正執法、希望社會各界人士團結一致，共同推動香港發展重回正軌。 [新華社傍晚發稿](https://web.archive.org/web/20220120000019/http://www.xinhuanet.com//2019-12/16/c_1125353595.htm) ，在兩個「堅定支持」後補上第三個「堅定支持」：堅定支持愛國愛港力量。這三個「堅定支持」與他上月在巴西講話中的三個「堅定支持」有所不同。

根據 [新華社報道](https://web.archive.org/web/20220120000019/http://www.xinhuanet.com/politics/leaders/2019-11/14/c_1125233663.htm ) ，習近平上月在巴西表示，中央將「繼續堅定支持行政長官帶領香港特別行政區政府依法施政，堅定支持香港警方嚴正執法，堅定支持香港司法機構依法懲治暴力犯罪分子。」此外，習近平11月初在上海見林鄭時，亦提過「依法制止和懲治暴力活動就是維護香港廣大民眾的福祉，要堅定不移」。但在今次林鄭述職，不論是習近平的公開發言或新華社發稿都未有再提及「依法懲治暴力」。新華社昨稿件內文，全篇都沒有「暴力」一詞。


{{< video src="https://web.archive.org/web/20220120000019if_/https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRTHKVNEWS%2Fvideos%2F583574402458633%2F&show_text=0&width=560" >}}


習近平公開發言時未有再提到「止暴制亂」，資深傳媒人、時事評論員程翔分析指，從事態發展而言，可能是習近平見到「止暴制亂」政策不但沒有解決暴力問題，更帶來對內、對外的負面影響：對於香港內部，北京愈強調「止暴制亂」，香港民眾愈反感，這呈現於區議會選舉建制派大敗；在國際層面，「止暴制亂」政策直接促成美國通過《香港人權與民主法案》，而且很多國家都考慮通過類似的法令，以制裁香港的官員。

程翔指出，雖然習近平不公開講「止暴制亂」，林鄭卻引述有此說法，或反映中央仍希望繼續推行「止暴制亂」，只是不會公開說，以免引起反感。他補充指，類似情況經常出現，以「中國製造2025」製造業政策為例，中國一公開提出該計劃，美國就有很大反應，此後中方未有再公開提及「中國製造2025」，以免挑起美國關注，但中方私下仍然照做，沒有因為美國反對而叫停計劃。

中文大學歷史系兼任教授、中國政治評論專家林和立認為，不宜對習近平今次沒有公開說「止暴制亂」解讀太多，因為可以純粹是字面的交替，但中共對香港的強硬路線沒有改變。他分析指，不排除習近平是想與香港的反對派營造一個互動、「有得傾」的態勢，但習仍提到支持警察，其實等同支持「止暴制亂」，他相信其態度依舊強硬。
