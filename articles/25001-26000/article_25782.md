# 2020，Keep Calm and Write On──眾新聞踏入4周年感言

【撰文：眾新聞執行總編輯盧曼思】

眾新聞是一間山寨廠，辦公桌是長長的豬肉枱，我的右邊是80年代在美國跑過新聞的主筆安裕，我的左邊是90後記者HT。右邊那位經歷過香港傳媒的黃金時代，左邊那位正在經歷不知是最好還是最壞的時代。

坐在他們兩人中間，我這個生於70年代末的，有時感到孤單。我這一代人仍留在傳媒行業的只有少數，在香港社會也是很尷尬的夾心人。

這3年來，每當我在辦公桌上抬頭，看到的，是我親手貼在牆上的一句話：Keep Calm and Write On。

自己攞嚟。

2019年最記得是7月21日晚，我一個人在公司，望著這句 Keep Calm and Write On，眼淚長流不止。

和大家一樣，有很多很多個星夜，都與淚水作伴。

點樣Keep Calm and Write On？我深呼吸一口氣，但發現自己很喘氣。

記得2017年的元旦，我和總編輯李月華在眾新聞辦公室倒數後，便在跑道上飛奔了3年。

2017年1月1日，是眾新聞的生日。當年正是林鄭月娥做特首前夕、香港社運最低迷的日子，也是一些人所謂的傘運後抑鬱期，人人最不願看新聞的時空。

沒有人想到，2年多後，我們會忙到差點兒暈低。2019年的反送中運動採訪，對眾新聞這個初生媒體，是一個很大的挑戰。我們全職的前線記者只有5人（另有幾位特約及實習記者幫忙），李月華和我負責編輯工作，另有主筆安裕和楊健興做分析評論。

一個只有約10人的團隊，長幼各半，我是唯一一個中年人，夾有中間，過去3年負責眾新聞的編採，坦白說，一點也不容易。在反送中運動期間，我們因資源的限制，無法負擔長時間開live，加上各媒體已經在做，我們希望在直播以外，做出有自己特色的新聞。

甚麼是有自己特色？

3年前，我問過李月華一個問題：眾新聞的定位是什麼？她跟我說：要做到人哋信任你。

這句話，看似無難度，但當走進網絡世界，人人講求要快、要有圖、要搶眼、要有標題、要有hit rate先係王道的時光隧道，很多時，你會發現，「要做到人哋信任你」似乎有點難度。

還有，當社會處於極度撕裂、人與人之間充斥懷疑與不信任、人們只願意相信他們認為要相信的，「要做到人哋信任你」，過去一年，我多次想召喚天上的智者，在星夜下給我指點迷津。

很多中老年人說，2019年思想上受到很大衝擊，是啊，我也有很多想法未能沉澱，像氣泡在空氣迴盪，包括如何在記者路上繼續奔馳。

「要做到人哋信任你」，好吧，我召喚昔日的我，入行以來被人罵過狗血淋頭的包括：人名地名基本資料數字不可寫錯對過晒未；5W1H問晒未齊晒未；有沒有找不同人求證......

要的，是事實。

但今時今日講呢啲，可能會被人話你慢十拍啦、保守不合時宜啦、唔識做網啦、新聞唔juicy唔搶眼啦、又無圖又無片啦、老餅啦...…更令人苦惱的，是經求證過的事實，當與群眾的立場和想法有落差時，卻可能被人質疑你在漂白、抹黑、做打手...…

我害怕的，是年輕記者抵受不住網絡上的輿論批評而感到恐懼。在我們眼前的，不單是強權，還有愚昧和人云亦云。

2017年，我和一班前輩組成眾新聞，是希望 [尋回新聞的初心](https://web.archive.org/web/20210701192746/https://www.hkcnews.com/article/318/%E3%80%8A%E7%9C%BE%E6%96%B0%E8%81%9E%E3%80%8B%E6%88%90%E7%AB%8B-%E7%9B%BC%E5%82%B3%E6%89%BF%E6%96%B0%E8%81%9E%E5%88%9D%E5%BF%83) 。

2019年，在強權控制輿論、社會前所未見的撕裂、假新聞泛濫的年代，我們更加明白，有一些價值不能放棄。

每天你掃手機，閱讀的都是搶眼的標題、張圖好fun的帖文，有時候連click入去都費事吧？但你的腦袋、你的思想，有因為這段新聞而變得豐富嗎？

人，總不能只食即食麵和快餐吧？

因為資源所限，眾新聞每天的新聞不多。但我們希望，你可以讀到一些有內容、有意思的文章，盼望你可以click入來看看。

或許，你覺得文字又長又悶，但一天5分鐘的閱讀旅程，我們希望你有多一點營養、多一點感動、多一點新想法。

我們，還有很多題材想做、很想搞搞多些新意思，但礙於資源所限，未能做到。

除夕寫下這篇文章，因為我感覺到自己的狀態，像一個馬拉松選手，很喘氣卻要繼續跑下去；而我的年輕同事，前面還有漫漫長路，我希望他們可以走上一條新聞的正路。

我們走著的這條路，堅持初心，是否可以走下去？

4歲的孩子，剛剛入幼稚園，開始人生新階段。2020年，4歲的眾新聞也要步入新里程，希望大家多多支持，特別是要救救我身邊的李月華，她平均每天工時長達16小時，我們都很擔心她。2020年，希望可以有更多資源，成就我們的理想。（PS：抱歉因資源限制，有時我們發稿、回覆比較慢）

2019年，我意識到自己開始老了，交棒予年輕記者，將是我未來的主要工作。經過反送中一役後，我很希望香港能夠有一個具養份的新聞平台，給有心入行的青年。眾新聞每一位年輕記者都很用心，也不怕辛苦（當然啊，是精挑細選的）。未來，這個平台屬於他們。

2020年，是時候，為你每天閱讀的新聞，加多些營養。

我們會，Keep Calm and Write On。
