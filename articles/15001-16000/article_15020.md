# 利潤．方便．揾命搏

[請支持《眾新聞》，成為訂戶](https://web.archive.org/web/20210124093520/https://www.hkcnews.com/aboutus/#subscribe)

  
 【撰文︰工黨副主席李卓人】

颱風山竹來勢兇兇，環保人士關注的是颱風與氣候變化的關係，根據氣候科學，兩者絕對有關，而特朗普放寛甲烷排放標準，其實在為地球掘墳墓。

香港亦然，我們絕對有責任為地球未來，關注氣候變化，現在就要防範於未然，否則將來可能會有災難性的影響。我在未講勞工權益之前，都希望提出氣候變化的關注，因為在人禍製造的天災中，大家都無可逃避，必須全球盡快共同應對氣候變化。   
   
 對打工仔女來說，最緊張的是颱風幾時打到香港，有否假期，還是李氏力場會否發功，大家照樣返工。但我們不要忘記很多打工仔女，也要冒大風大雨照常上班，他們打風沒假放，而是要搵命摶。

這群要冒險上班的打工仔女，首先要面對的是如何返工的問題。沒有公共交通就只有步行或乘搭的士，但步行步步驚心，乘搭的士則要額外開支。大家會問，勞工法例有什麼保障？答案是沒保障，勞工處只發出沒有法律效力的指引，而一切都視乎勞資雙方的安排協議，但大家都知道，除非有集體談判權，否則什麼協議都是大石壓死蟹，僱員不得不遵從。

### 打風要員工搵命摶？

勞工處的指引指出，僱主須首要考慮僱員在工作地點及往返工作途中的安全，並評估對必要人員的需求，並只要求絕對必要人員在八號或以上颱風警告期間上班。簡單來說，僱主應只要求「必要人員」上班，那麼什麼是「必要」﹖為何現時這麼多零售舖、酒樓食肆等，都在颱風期間營業？這有什麼必要？

這是香港的文化問題。老闆要搏到盡，市民什麼都要方便，但方便背後，其他員工的犧性和辛酸，又有多少人知道？我明白颱風期間，醫院要開，消防員要當值，電廠員工要緊守崗位，必要服務不能停。但為何7-11、CIRCIE K、惠康、百佳、酒樓、麥當奴和連鎖快餐店都要開門營業？回顧香港歷史，大家都在打風前入定貨，準備颱風時留在家不外出。但到今時今日，一切如常營業，市民方便了，但員工呢？我們的方便是否要別人冒生命危險來提供？在此呼籲香港人廣儲糧，颱風期間不外出，使開門營業的零售、食肆沒有預期市場收益，用腳投票，使非必要服務不要用員工的安危做賠注。

### 打風指引冇法律效力　僱員冇保障

<figcaption>八號風球連公共交通都停駛，僱主有責任提供交通接送服務給要上班的僱員。美聯社</figcaption>
</figure>

勞工處指引亦有呼籲僱主對必要上班的僱員提供交通接送服務，而且也不扣除缺勤員工的工資，及考慮以颱風津貼鼓勵上班員工。但指引歸指引，對僱主毫無制約力，僱員根本毫無保障。每次颱風過後，職工盟都會接到僱員查詢電話，問僱主能否因僱員颱風期間缺勤扣人工，我們只有答視乎雙方協議。以下是僱主七大陰招：

# 非必要服務要僱員揾命摶照常上班，勞工處的指引要僱主先評估是否必要人員，才安排返工，但指引無任何法律效力，等於零保障；   
 # 打風不上班當缺勤扣人工丶勤工獎等，但法例無保障；   
 # 打風貼錢返工，自掏腰包搭的士，但法例無規定有交通津貼；   
 # 打風不上班扣年假丶法定假，是違反法例因沒有足夠通知時間；   
 # 直踩24小時，頂替下更工作。無加班額外補水；   
 # 為晨早照返工，留在工作地點過夜。   
 # 威嚇員工打風要返工，否則視作不服從合理命令，威脅要即時解僱且無任何補償。做法違反勞工法例，但威嚇員工目的已達。   
   
 但憤憤不平的僱員問職工盟：為何不修訂法例去保障僱員？

沒有抗爭，哪有保障？大家一起爭取。   
   
 職工盟希望香港社會能重新反思，應優先考慮從業員的健康安危，在颱風期間暫停一切不必要服務。對從事必要服務而要上班的工人，應立法規定僱主提供交通津貼和規定颱風值勤，僱主須支付雙工給颱風值勤的僱員，並在僱員因風不能值勤不得扣減工資。   
   
 有張宇人之流的人士或會指控我們什麼都要僱主埋單，但不要忘記僱主有職安健的責任，確保僱員和僱主自己的安全。 颱風期間停止營業，不用上班而不扣人工或颱風返工雙工計，都是希望僱員作為弱勢一方，不用承擔一年也最多只是數天的天災風險。而僱主提供良好的工作環境和條件，僱員快樂上班，在颱風前後更勤力工作，提高生產力回報僱主，這不是雙贏嗎？
