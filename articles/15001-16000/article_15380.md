# 睇《延禧》，要用腦！


{{< video src="https://web.archive.org/web/20210712021805if_/https://www.youtube.com/embed/O2nIhiaLGQU" >}}


【撰文：曾經的電視迷】

難得在家吃飯，電視傳來大台正播放的國產大熱劇集《延禧攻略》主題曲，老媽即大叫：「做啦做啦，唔好嘈我。」

我問：「點解你咁鍾意睇呢套劇？」

老媽：「劇情緊湊，演員做得好囉。」

我：「套劇講來講去，咪又係點樣為咗自保、人不為己天誅地滅、宮廷無個好人呢啲，同以前的《金枝慾孽》、《宮心計》咪一樣，你唔怕悶咩？」

老媽：「我唔記得以前睇過乜。同埋唔睇的話，我又有乜嘢做呢！」

我問旁邊的姊姊：「乜香港人唔係話頂唔順國產嘢咩，點解咁多香港人睇呢套《延禧》？」

姊：「你咪咁認真啦，睇電視，就係唔想用腦。」

係真，唔係呀？


{{< video src="https://web.archive.org/web/20210712021805if_/https://www.youtube.com/embed/zbVP9sd316k" >}}


老媽的大半生，幾乎每晚都在看大台的電視劇，賣廣告就入廚房洗碗 （她的洗碗時間破晒紀錄）。我家每壞一部電視機，老爸便笑道：「你阿媽又睇爛一部啦。」對老媽來說，電視劇是她消磨時間的寄託，她沒有足夠的知識水平明白電視劇所蘊含的社會文化意義。其實除了老媽，我身邊有一些高學歷、在職場任高位的OL朋友，近日也在討論《延禧》，內容都是劇情和演員花邊新聞。

我一集《延禧》也沒有看過，不知道老媽口中的魏瓔珞有幾吸引。我看著老媽的狂熱，令我想起小時候和她一起看 [《新紮師兄》](https://web.archive.org/web/20210712021805/https://zh.wikipedia.org/zh-hk/%E6%96%B0%E7%B4%AE%E5%B8%AB%E5%85%84) 和 [《壹號皇庭》](https://web.archive.org/web/20210712021805/https://zh.wikipedia.org/zh-hk/%E5%A3%B9%E8%99%9F%E7%9A%87%E5%BA%AD) 的日子，可惜她已忘記得一乾二淨。

《新紮師兄》是80年代我讀小學時看的，記得演員是梁朝偉劉青雲呂方吳孟達，還有小虎隊唱的主題曲「伴我啟航」；90年代看《壹號皇庭》在讀中學，聽著那色士風主題音樂會幻想做大律師可以晚晚落酒吧好有型 （後來當然知道是假的）。這兩套劇集曾經陪伴我成長，直至近年，我才明白它們的含意。

《新紮師兄》和《壹號皇庭》都由香港人製作、取自本地題材、香港演員在香港拍攝用廣東話演出。兩劇內容雖然沒有直接批判社會、個案內容虛構，但故事卻是來自某個時空下，香港獨特體制和社會氛圍所引伸的集體共鳴。

《新紮師兄》梁朝偉飾演的杰佬，由基層小子捱過學堂的刻苦訓練變成熱血警察，是廉政公署成立後，警察形象由反變正的時代寫照；《壹號皇庭》由歐陽震華、陶大宇、林保怡一班大狀經手的案件，最後大多邪不能勝正，他們出入的香港高等法院是個有血有肉、能夠伸張正義的地方，因為這兒有司法獨立。《壹號皇庭》的英文名，叫The File of Justice。

《新紮師兄》和《壹號皇庭》令人懷念的，不單是幕後製作團隊的創意，更重要是劇集所蘊含的香港元素。早前內地奇女子唐琳玲闖入高院引起風波，原來早有內地網民談及，去香港很想去高等法院，因為看完《壹號皇庭》很想了解香港的陪審團審訊制度，這是大陸所沒有的。

曾幾何時，香港電視劇集所盛載的香港風味，不知不覺滲入了香港人成長的血液，更成為內地人對香港的憧憬。如今，香港人扭開由紅色資本話事的大台，看到的是《延禧》一類的多套大陸製作配音劇集，它們講卡士派場、講政治正確的歷史、講人工美學。香港人覺得好看的同時，難道看不出，香港電視劇集曾經有過的獨特靈魂，已經消失得無影無蹤？看著《延禧》成長的年輕一代，將會有著我年少時看《新紮師兄》和《壹號皇庭》的記憶與回甘嗎？

<img src="https://web.archive.org/web/20210712021805im_/https://www.hkcnews.com/news_新聞/hk-discuss/2018/10/中國-電視劇-延禧攻略-20181003020743_2007_large.jpg" alt="">
</figure>

或許，《新紮師兄》與《壹號皇庭》只屬於過去的香港，兩劇假如要開拍2018年版的話，杰佬在眼前的社會或者已經變得太遙遠，法庭佈景中響起的色士風配樂也可能夾雜陣陣憂傷。不過，沒有了《新紮師兄》和《壹號皇庭》，是否代表香港人甘心從《延禧》中得到慰藉而滿足？紅色資本的餵食，難道真的可以不經思索，有電視睇唔通就真係唔使用腦？

有腦的話，是時候思考，屬於香港人的新一場流行文化革命。
