# 「山竹」善後　港大民調：林鄭得50.8分　民望新低

超強颱風「山竹」早前襲港，林鄭政府的善後工作備受批評，港大民研計劃公布 [最新民調](https://web.archive.org/web/20210724192118/https://www.hkupop.hku.hk/chinese/release/release1548.html) ，林鄭月娥的評分較兩星期前大跌6.5分至50.8分，為上任以來新低，而其最新支持率為40%，反對率為45%，民望淨值為-4個百分點，較兩星期前明顯下跌15個百分點，是其上任以來第二低。而林鄭 [民望淨值最低](https://web.archive.org/web/20210724192118/https://www.hkupop.hku.hk/chinese/release/release1530.html) 時為今年5月21至25日，六四事件29周年前夕，當時她的支持率為39%，反對率為44%，民望淨值為-5個百分點。

最新這次民調在9月18至20日進行，山竹過後17日（周一）上班日出現交通大癱瘓。港大以隨機抽樣方式電話訪問1,002名香港巿民，調查的回應率為56%。特區政府方面，民望則與一個月前變化不大，最新滿意率為37%，不滿率為45%，滿意淨值為-8個百分點。社會狀況評價方面，在經濟、民生及政治狀況當中，三項滿意淨值分別為+2、-20和-37個百分點，反映市民最不滿現時的政治狀況。而過去兩次調查中，市民同樣最不滿政治狀況，分別為7月的-42%和8月的-47個百分點。

另外，調查深入分析了受訪市民的年齡和學歷，發現愈年輕的人，愈反對林鄭月娥出任特首：


| 年齡組別 | 反對率 |
| --- | --- |
| **18-29歲** | **72%** |
| 30-49歲 | 49% |
| 50歲或以上 | 31% |


而學歷愈高的人，愈反對林鄭月娥出任特首：


| 教育程度 | 反對率 |
| --- | --- |
| 小學或以下 | 25% |
| 中學 | 45% |
| **大專或以上** | **56%** |


由於最新調查顯示林鄭的支持率淨值，與上次調查出現非常明顯變化，今次港大以大數據追蹤香港Facebook專頁及本港各大論壇提及某特定政治人物及團體的熱帖，歸納出十大網上條目表列如下（已剔除重覆項目），由讀者自行判斷有關事件是否影響民調數字，當中大部分條目與颱風「山竹」襲港後特首的處理有關：


| 颱風「山竹」襲港後林鄭月娥見記者 |
| --- |
| 網民討論颱風「山竹」襲港後的上班情況 |
| 途人被大樹塌下壓住失去知覺 |
| 森美為偽電影預告《返工》配音 |
| 網民到林鄭月娥Facebook留言希望政府宣布停工 |
| 林鄭月娥指若宣布停工是不負責任 |
| 有僱主扣減颱風「山竹」襲港後遲到或缺勤員工薪金 |


另外，港大又進行市民對施政報告期望第二階段調查，第一階段調查在2018年9月3至6日，最新調查顯示，最多巿民認為特首林鄭月娥的施政報告重點應放在房屋問題上，佔95%，包括82%認為「好需要」和13%認為「幾需要」，結果是歷來新高，調查又按照需要程度以1分最低、5分最高要求受訪者評分，房屋問題的平均量值為4.8，接近「好需要處理」，亦屬五個項目之中最高。其次為醫療政策、教育、經濟發展和政制發展。
