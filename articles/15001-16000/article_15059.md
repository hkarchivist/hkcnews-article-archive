# 港大畢業生議會調查：新校長首要工作「維護大學核心價值和捍衛大學利益」

香港大學畢業生議會常務委員會公布「大學校長工作重點」意見調查結果，發現受訪會員認為，「維護大學核心價值和捍衛大學利益」及「捍衛大學自主」屬新任校長張翔的首要和次要工作，而「與內地大學合作」和「募捐」則屬重要性最低的2個選項。

調查於3月28日至4月13日期間進行，以電郵向12萬名會員發出問卷，共有899名會員完成問卷。問卷提供10個工作範疇，讓受訪者評估張翔應先集中和首要處理那項工作範疇，1分為完全不重要，5分為最重要。調查發現，「維護大學核心價值和捍衛大學利益」和「捍衛大學自主」各得平均4.79分和4.61分。

張翔未上任前，交予港大遴選委員會的計劃書中，表示港大應加強與內地政府及內地大學合作。但「與內地大學合作」的選項只得平均2.44分，得分最低，反而「國際化」得4.01分，排名第4，比「與內地大學合作」的排名高。


| 新校長工作重點 | 平均分 |
| --- | --- |
| 維護大學核心價值和捍衛大學利益 | 4.79 |
| 捍衛大學自主 | 4.61 |
| 提升大學卓越研究成就 | 4.21 |
| 國際化 | 4.01 |
| 加強大學管治 | 3.96 |
| 創造影響力 | 3.95 |
| 優化創新和創業精神 | 3.87 |
| 提高國際排名 | 3.54 |
| 募捐 | 3.42 |
| 與內地大學合作 | 2.44 |

委員會今日與張翔會面，討論調查結果，歷時約1小時。畢業生議會主席潘燊昌表示，會面氣氛熱烈，形容張翔「好有活力、坦白講嘢、對高等教育有熱誠」。至於討論有否觸及大學校園能否討論港獨的議題，畢業生議會大學事務召集人傅景華表示，委員會對事件沒有既定立場，會面亦沒有談及具體的議題，但已向張翔反映會員對言論和學術自由的關注。
