# 黃金寶接任「健康大使」　陳凱欣宣傳攻勢未了

食物及衛生局前政治助理陳凱欣，周一（17日）正式卸任九龍社團聯會「健康大使」，由退役金牌單車運動員黃金寶接任，位於紅磡大廈外牆上的巨型海報，也換上黃金寶。不過，盛傳會代表建制派參加九龍西立法會補選的陳凱欣，她的「健康大使」宣傳廣告，仍出現在深水埗和長沙灣港鐵站月台和小巴車身；前兩天，有油尖旺居民收到陳凱欣伙拍九龍社團聯會成員的「工作報告」小冊子。

九龍西立法會補選11月25日舉行，提名期將於10月2日開始。陳凱欣8月12日起擔任親中組織「九龍社團聯會」的健康大使，宣傳一浪接一浪，被質疑偷步開始選舉宣傳。陳凱欣表示，本月12日已辭任九社聯的健康大使。不過，有住奧運居民收到「油尖旺青年社會長、樂活富柏主席陳德立」的工作報告，全冊共12頁，當中三分二是陳德立的工作報告、三分一是陳凱欣交代「健康大使」的工作，包括底頁的「健康大使」宣傳照，還有「陳凱欣跑步遊戲」的QR code。小冊子內頁有一張陳德立及陳凱欣的合照，旁邊寫上「德立與健康大使陳凱欣出席全城投入活出健康啟動禮，除了一同做運動外，都會繼續拍住上，在地區宣傳健康知識，推廣積極生活態度。」但除上述照片，未見有二人同場的合照。

另有住太子居民收到經民聯區議員黃建新的工作報告；土瓜灣居民收到西九新動力區議員林博的工作報告，陳凱欣也佔約三分之一篇幅。林博是九社聯理事；陳德立與黃建新均是九社聯地區委員會「油尖旺地區委員會」成員。

陳凱欣擔任九社聯「健康大使」，宣傳規格相當高，包括推出為她特別設計的手機遊戲。九社聯會長陳振彬被問到接任的黃金寶，宣傳品會否與陳凱欣同等規格，他稱：「我相信，嗱，整個活動都係推廣健康，咁可以做的我一樣照做，亦都要睇番廣告牌畀我哋個時間去到幾時啦。如果有的，我們一定去推廣。」陳振彬表示，當現有廣告檔期結束，及後又有廣告檔期的話，會全部換上黃金寶。

陳凱欣在紅磡嘉麗閣大廈外牆的巨型海報，於8月20日出現，記者早前向大廈發展商裕泰興子公司查詢，獲告知租至9月19日，而目前已經掛上黃金寶，陳振彬表示希望能夠續租，籲記者向九社聯秘書處查詢詳情。但陳凱欣海報上的QR code介紹「掃QR code 玩遊戲 跑里數 做善事」，到了黃金寶海報，則變成四個字「活動詳情」。掃瞄QR code後亦不再是進入陳凱欣跑步遊戲，而是顯示九社聯「全城投入 活出健康」的網頁。

至於陳凱欣之前推廣的「學童眼科檢查」服務，陳掁彬稱要視乎服務機構能否提供人手，有的話會繼續做。驗眼活動由希瑪眼科提供，該公司是上市公司（3309），公司主席是著名眼科醫生林順潮，亦是港區全國人大代表。

陳凱欣擔任「健康大使」期間推出的手機遊戲，說明「遊戲中每跑一公里，主辦機構『九龍社團聯會』將會捐出一元予慈善團體『藝恆愛心行動』，用於推動健康主題活動。」捐款設上限30萬元，活動有效期至今天晚上23:59:59，記者昨日下午4時9分打開遊戲，顯示「目前全民累積總里數：047599公里」，以此估算九社聯將捐出至少47,599元。陳振彬稱遊戲目前仍然繼續，「繼續架呢啲，我哋遊戲獎金有上限，到人攞晒我哋就會停架喇。」他指原本估計玩到今日就達標，但未達上限的話都會繼續，後又補充要開會詳細傾。

記者又向陳振彬查詢，幾時邀請黃金寶出任「健康大使」，他回應指「我識佢嘅，識佢好耐。我哋一路不只搵一個的，其實我哋希望搵多幾個，仲有其他人傾緊做健康大使。」他表示歡迎所有形象健康的人，如果雙方都覺得合適，便有機會出任「健康大使」。他又指，整個活動預計維持半年，至「明年農曆新年、流感高峰期過後」。

黃金寶曾三奪亞運金牌，去年轉到港協暨奧委會擔任項目發展幹事。黃亦曾於去年為當時的特首候選人林鄭月娥拉票，拍片表示支持林鄭。
