# 【傘運4周年】傘後紀錄片《分域大道》上映　導演杜浩綸：「講一個更大的香港故事」

<figure>
<img src="https://web.archive.org/web/20210628172615im_/https://www.hkcnews.com/news_新聞/hk-discuss/2018/09/導演-杜浩綸-分域大道-20180927030407_2beb_large.jpg" alt="">
<figcaption>導演杜浩綸從英國抵港，將會留港至十月初。何君健攝</figcaption>
</figure>

37歲的英國導演杜浩綸（Matthew Torne）曾製作電影《未夠秤》（2014）、《黃之鋒：熱血青年VS超級強國》（2017），今年推出傘後紀錄片《分域大道》（Last Exit to Kai Tak) ，昨晚在香港藝術中心首映。

電影取名 Last Exit to Kai Tak ，是受到小說《 Last Exit to Broolyn》啟發，該書講述了最社會底層的人們如何生活，另外亦因為大部分鏡頭在牛頭角、觀塘一帶拍攝，杜浩綸可以經常見到前往舊啟德機場的馬路，他指著眼前的金鐘夏愨道形容：「像這樣，向上的馬路，但到了一半在空中斷開。想講的是，有些人可以選擇離開香港、移民到其他地方，但有些人沒有這個選項。」

### [【 雨傘運動4周年專訪系列】](https://web.archive.org/web/20210628172615/https://bit.ly/2NBWn5a)


{{< video src="//web.archive.org/web/20210628172615if_/https://www.youtube.com/embed/YXq7yREKdVk" >}}


《分域大道》講的是傘運後的香港，電影取材主要是五個香港人的生活，分別是黃之鋒、林淳軒、劉偉德、黃洋達及何韻詩。電影裡的時空，黃之鋒與林淳軒同屬香港眾志、劉偉德準備參選區議會、熱血公民的黃洋達接受建制派人士挑戰打拳賽、何韻詩剛剛宣布轉為獨立歌手。傘運之後，政治、社會的力量反彈，對他們的生活造成很大影響。

選取這五個人的故事，是個令人好奇的決定。杜浩綸說：「我嘗試拍攝一群人，在交疊的時空出現的一群人，這樣可以講一個更大的香港故事。」他希望觀眾可以在這五個人之中找到與自身有共嗚的角色，但他反覆強調：「我不支持、不指責這些角色，我不是說這個是好人、這個是壞人，我只是展示他們在做甚麼，觀眾入場後應該有自己的思考、想法。」

杜浩綸被問到是否刻意找不同政治光譜的人，「是的，香港政治光譜很闊，還有黃毓民、梁天琦等等，重點是他們的意見都是有效的 （valid），即使我不同意你的看法。但政府不想這樣，建制派不想有辯論，他們想制止陳浩天在FCC演講，這樣是沒有言論自由。言論自由不會因為你不同意對方的意見，而突然終結。」

電影主題是傘運後的香港，但杜浩綸在傘運期間遠在英國倫敦。他記得，9月28日那天，他與社民連的吳文遠在倫敦見面，吳文遠翌日就搭飛機回港，杜浩綸則因為「愚蠢的原因」沒有跟上吳文遠。他很後悔，「因為第二日是我的生日，我父母從法國飛來，我不能告訴他們我要搭飛機離開了。」但他仍然每日追蹤傘運的最新消息，他能夠聽懂大部分的廣東話，所以經常透過網上片段關注事件。

他忘記自己在傘後是甚麼時候再來香港，但記得在2015年4月開始拍這部電影。早在傘運之前，他就認識黃之鋒與林淳軒，傘運後才認識其餘三人。他記得何韻詩約他到翠華見面，二人談起作為獨立歌手、獨立導演的話題，明白對方的難處；與黃洋達則是因為有common friend（共同朋友），他記得有次與黃洋達在九龍塘食晚飯，黃說當晚有場拳賽，叫他來拍攝，他覺得這很有趣就決定拍黃洋達。

還有劉偉德，有次杜浩綸與黃之鋒在美國落杉機宣傳《未夠秤》，其中一朝早食早餐時，杜問起有甚麼人值得拍攝，黃之鋒講了很多個人名，都一一被拒絕，直到這個「玩健身、有紋身、讀法律」的劉偉德，杜浩綸終於感興趣。杜第一次在香港與劉偉德見面就非常投契，因為大家都在英國長大、接受英式教育。對於每一個受訪者，杜浩綸都很喜歡，「如果你不喜歡他們，是很難拍攝的，試想像我要怎樣拍葉劉淑儀？」

那麼可有想過找梁天琦？杜浩綸表示拍攝開始前梁天琦還未具名氣，是在拍攝中段才冒起，所以錯過這個對象。「我在2015年一次與黃毓民的會面見到梁天琦，當時我心想：這是誰？這是我的錯誤。這就是紀錄片，你押下賭注希望會有事情發生，有時撞中了，但更多時候撞錯。幸好Nora Lam 拍了《地厚天高》，香港人不用擔心，他的故事有人講了。」

而黃之鋒的故事，杜浩綸這是第三次講了。2011年8月21日，是他們二人第一次見面。杜浩綸記得很清楚是這天，因為那是學民思潮第一次抗爭。他那天本來與《未夠秤》的另一主角馬雲祺（馬仔）一起，但後來馬仔在抗議行動中被捕，杜浩綸在北角警署外無所事事，想到要等幾個鐘，就不如去學民思潮的集會看看。他原先覺得非常無聊，直到黃之鋒拿起咪演講，引起他的注意，「之後我找他聊天，他說他只是14歲，我就：噢～OK。」

杜浩綸形容黃之鋒是個天生的政治家，「Joshua 很習慣鏡頭，但他不習慣表現自己，他在鏡頭前會掛上『政治面具』（political face），我要的不是這些。」這次拍《分域大道》，杜浩綸感覺彼此信任多了，對方明白自己不是要把他表現得很差或很壞，所以較能除下面具。但杜亦很欣賞黃之鋒的堅持，「他過去6年都生活在鎂光燈之下，但香港需要更多不會放棄的人，我不知道自己能否做到。」

「我仍然很愛香港，我覺得這裡是很好的地方。是的，政治上越來越差，我同意肥彭的講法，一些香港人在出賣香港，但正因為有人要出賣香港，所以要有人願意站出來。」杜浩綸說。
