# 香港「搞成咁」　望望林鄭行會有咩人就知點解！

羅范椒芬 [在英文電台節目中表示](https://web.archive.org/web/20211027071942/https://news.rthk.hk/rthk/en/component/k2/1479601-20190909.htm) ，她確認（I think we have confirmed that this is a true case）有年輕女子被誤導向前線示威者提供免費性服務，至於消息來自何處？出席同一節目的吳文遠表示，羅范指消息來自船河朋友聽聞的「一個朋友的朋友的女兒」。

事後，一眾傳媒追問羅范椒芬，她只以短訊回覆，指基於私隱，不能透露更多細節了。不過，大家都質疑羅范所謂的消息，其實只是網上流傳的一個「愛香港的14歲女仔」訊息，而根據《香港01》指，分別有兩位建制派人士接受查詢證實，羅范椒芬曾在WhatsApp群組內傳播過這篇文章。

首先，除非羅范能夠再提出證據，否則，這次「九成九」甚至「百份百」是「Fake News」！現時，網上假新聞多的是，有人刻意抹黑，亦有人誤信為真，對於羅范椒芬而言，我相信她是後者，但你可能即時會問，她會信以為真嗎？她是大紫荊勳賢、前教統局局長、前廉政專員、前科技園主席，也是現任行政會議成員呀！一個英殖時期培養出來的社會精英，豈會如此「是非不分」？

一個可能是誤會，但接二連三，這是巧合嗎？我指的是，現時在林鄭身邊的是甚麼人？只要細心數一數，就知道現時香港「搞成咁」，並不是偶然。

除了羅范「Fake News」當「執到寶」之外，同樣做過廉署的余黎青萍竟然單方面真心相信警員不會「無端端」使用過份武力！

再看現時的行會成員，葉劉認為示威者能夠善用telegram、連登、AirDrop，就肯定背後有「大佬」，葉國謙又說竟然有人「擺散錢」在港鐵讓示威者「買飛」，肯定有外國勢力！至於羅范的「好拍檔」李國章，他曾於2018年批評過年輕示威者就如「吸了毒」失去理性！

就連陳智思，一位相對而言比較親民的人，在修例之初，也曾經表示「未聽過商界有憂慮」！

再加上林大輝在《晴朗》形容，上次見林鄭班底，除了聶德權和邱騰華，幾乎問責班子所有人都缺乏士氣，根本沒有戰鬥力！

一個可能是誤會，但接二連三，這是巧合嗎？

林鄭身邊的人就是班「精英」，香港現時「搞成咁」，並不是偶然呀！
