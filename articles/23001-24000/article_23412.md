# 再談少女向「義士」提供Free Sex傳聞　羅范椒芬向《環時》稱：此事為真

行政會議成員羅范椒芬，昨早在香港電台一個英文節目，回應聽眾電郵指社交媒體上流傳，有少女被誤導向前線示威者提供免費性服務（free sex），羅范椒芬說：「I think we have confirmed that this is a true case. （我想我們已經確認這是真實個案）」在聽眾及節目主持人追問有何證據時，羅范稱：「The child is the daughter of a friend's friend. （小孩是朋友的朋友的女兒）」。羅范昨午接受中國官媒《環球時報》訪問，報道寫道：「（羅范）再次強調此事為真，她表示，她的一位值得信賴的朋友認識受害者，而且受害者只有14歲。」

《環球時報》報道又引述羅范稱：「初相識的暴徒男女都會約會、喝酒、吸食大麻，容易出事。」「當然可以理解的是，受害女孩和她的家長都不願意報警。」但報道未有提及她所指個案的具體證據。近日網上流傳的消息，是一篇下款「愛香港的14歲女仔」的文字（見下圖）。

為性暴力受害人提供支援的「風雨蘭」回應眾新聞查詢指出：「如未能肯定消息真實性與未得到事主同意，任何人都不應散播該消息或謠言，特別是作為專業人士或公眾人物。風雨蘭一般都不會評論此類言論，亦希望謠言止於智者。」

前教育統籌局常任秘書長、前廉政專員、現任行政會議成員羅范椒芬，昨早接受港台節目 [《Backchat》訪問](https://web.archive.org/web/20210712032437/https://www.rthk.hk/radio/radio3/programme/backchat) ，同場還有社民連主席吳文遠。主持人首先讀出一些聽眾寄來的電郵，其中兩篇提及「free sex」（免費性服務），包括有人指「年青人激進的四個主要原因：同儕認可、友誼、女示威者提供免費性服務、免費毒品」；另一則電郵則提及「一個14歲女生在社交網站分享」，指女生被誘使為「前線」提供免費性服務，每次都是不同的男性，更稱涉大麻，最後發現懷孕並感到後悔。

羅范椒芬在其後的討論中，引述兩則電郵的說法，她指：「我想我們已經確認這是真實個案。（I think we have confirmed that this is a true case. ）我為那些被誤導提供性服務的女生感到傷心。」羅范又形容，在太子站跪求取得閉路電視的舉動全都是「做戲（acting）」，旨在煽動憤怒。同場的吳文遠反駁指，年青人無必要冒著生命危險、承受槍彈傷害，以換取性服務。

節目後半段，有聽眾phone-in，指羅范椒芬散佈謠言、主持人亦都邀請羅范椒芬拿出證據。羅范椒芬表示，受害人是朋友的朋友的女兒，「是二手資料，但是直接的，亦都真確的（Second-hand knowledge, but it's direct. It's real.）」再被要求澄清的時候，她則指是14歲女孩寫的電郵，最後更稱：「想信就信，不信就不信。（If you want to believe, believe, if you don't want to believe, just don't believe.）」

羅范椒芬下午回覆香港電台查詢，指：「自己被一些人批評散佈謠言，因此不再接受訪問，市民有自由決定是否相信她所講的内容。」羅范昨午也接受了《環球時報》以及《南華早報》的訪問。

[《環球時報》](https://web.archive.org/web/20210712032437/https://3w.huanqiu.com/a/c36dc8/9CaKrnKmK4X?agt=8) 晚上7時30分的報道寫道：「9日下午，羅范椒芬向環球時報 - 環球網記者再度證實了此事的真實性。她稱，自己講述這段經歷的目的是想告訴女孩們要警惕不要受騙。因為初相識的暴徒男女都會約會、喝酒、吸食大麻，容易出事。另外，她還說，雖然『上當受騙你情我願』，但因為被害者未成年，所以事件已經構成了犯罪。她也說，『當然可以理解的是，受害女孩和她的家長都不願意報警』。但羅范椒芬稱，自己也希望藉此警告其他的年輕女孩不要以身犯險。」

[《南華早報》](https://web.archive.org/web/20210712032437/https://www.scmp.com/news/hong-kong/politics/article/3026426/senior-adviser-hong-kong-leader-fanny-law-stands-her-claims) 晚上9時39分的報道則引述了羅范受訪的說法，指：「人們有自由決定相信或不相信。當然，我可以透過值得信任的朋友的朋友，即認識受害女生那人，追蹤到訊息的源頭。但透露更多細節，會造成精神創傷（traumatic）。」

眾新聞以電郵分別向羅范椒芬及特首辦，查詢羅范椒芬作為行會成員、特首智囊，有否告知特首林鄭月娥相關事件？林鄭是否已知悉？有何證據確認事件屬實？如何跟進處理及有否向警方報案等問題。羅范椒芬沒有回覆，其電話沒人接聽。特首辦新聞主任回應指，羅范椒芬已就事件作出回應，行政長官辦公室沒有補充。

眾新聞向為性暴力受害人提供支援的「風雨蘭」查詢，問有否收到這類個案，發言人指：「除非有相關受助人以及該受助人欲公開資訊，否則我們都不便透露，一切有關受助人的資訊均絕對保密。所以我暫時答唔到你有冇類似嘅case。」

對於羅范的言論，風雨蘭總幹事王秀容表示：「正常聽到性暴力事件都應該鼓勵當事人求助，以保護當事人及其他受害者，同時，如未能肯定消息真實性與未得到事主同意，任何人都不應散播該消息或謠言，特別是作為專業人士或公眾人物。風雨蘭一般都不會評論此類言論，亦希望謠言止於智者。」

民間記者會發言人，批評羅范椒芬在完全沒有事實根據的情況下，作出完全無理指控，「一個前高官，竟然可以用假新聞來誤導大眾，完全唔明白一個受過高等教育的人，可以說出這些不負責任的言論，希望佢可以好好檢討，唔好行出來獻醜。我哋唔係妄求這些高官的道歉，因為已厭倦這些在上位者的荒謬言論，我哋唔需要佢道歉，只希望佢檢討自己的言論。」

被問到羅范椒芬言論是否代表政府攻擊示威者的「戰術」、如何對抗時，民間記者會發言人說：「唔敢擔保這個是否極權打壓的一個方法，但我們心存懷疑，點解咁多假新聞抹黑我們的前線示威者，甚至普羅大眾。極權政府好想分化和理非和前線示威者，先不論是否政府所為，但這些高官、當權者，用不盡不實消息去抹黑前線示威者，不能夠被接受。有關的性言論、侮辱，這些消息透過一個女人講出來，佢係極度唔尊重自己，亦都唔尊重女性，所以我們極之失望。」發言人指，前線手足和鍵盤戰士，會多做文宣工作闢謠。
