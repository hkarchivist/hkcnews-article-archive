# 土地小組諮詢結果　填海、發展郊野公園青年支持率偏低

土地供應專責小組昨日就「土地大辯論」公眾諮詢結果，向特首林鄭月娥提交長達123頁的 [報告](https://web.archive.org/web/20200804150448/https://www.landforhongkong.hk/pdf/Report%20(Chi).pdf) 。

[報告](https://web.archive.org/web/20200804150448/https://www.landforhongkong.hk/pdf/Report%20(Chi).pdf) 的量化意見，主要來自問卷及電話調查。問卷由港大研究中心設計，小組最終收到29,065份問卷，包括 21,608份網上問卷，以及7,457份實體問卷。電話調查則由中大香港亞太研究所負責，成功訪問3,011名18歲或以上的香港居民，當中2,005名來自家居固網電話訪問，另外1,006名來自手提電話訪問。

眾新聞翻查 [港大研究中心](https://web.archive.org/web/20200804150448/https://www.landforhongkong.hk/pdf/PE%20Report.pdf) 及 [中大香港亞太研究所](https://web.archive.org/web/20200804150448/https://www.landforhongkong.hk/pdf/Appendix%202%20(Chi).pdf) 的分析數據，發現不同年齡層對於填海、發展郊野公園相關選項的支持度有明顯差距，愈大年齡群組支持度愈高；與此同時，認為「對自然環境影響較小」 是土地發展最優先要考慮的因素者，則愈小年齡群組佔比愈高，結果均反映年輕人較重視保護自然環境，對於被指破壞環境的選項接受程度普遍較低。

發展私人農地選項則是愈大年齡群組支持度愈高，收回發展私人遊樂場地契約用地（包括是否收回粉嶺高爾夫球場）選項則見相反走勢。

### **填海、發展郊野公園、發展私人農地：支持率隨年齡遞增**

根據中大香港亞太研究所的的電話調查報告數據，被問到會否選擇「維港以外近岸填海」、「發展東大嶼都會」、「利用私人的新界農地儲備」、「發展郊野公園邊陲地帶兩個試點」、「發展郊野公園邊陲地帶其他點 」選項時，答「會」的受訪者，比率大致都隨年齡遞增；而答「不會」 的受訪者，比率大致隨年齡遞減。

| 30至39歲 | 47.6 | 53.8 |
| 40至49歲 | 56.9 | 52.0 |
| 50至59歲 | 66.4 | 62.9 |
| 60至69歲 | 78.0 | 68.2 |

問卷與電話調查的詢問方法、劃分的年齡組別均不同，但兩者反映受訪者對以上選項的支持率按年齡分布的趨勢相若。

### **回收發展私人遊樂場契約用地：支持率隨年齡遞減**

電話調查結果顯示，「利用私人遊樂場地契約作其他途 」選項，隨年齡增加，支持率遞減。問卷調查情況類似。


| **年齡** | **「利用私人遊樂場地契約作其他途 」支持率（%）** |
| --- | --- |
| 18至29歲 | 69.4 |
| 30至39歲 | 65.0 |
| 40至49歲 | 64.9 |
| 50至59歲 | 59.7 |
| 60至69歲 | 53.5 |
| 70歲或以上 | 49.5 |


### **逾三成人最關注發展對自然環境影響　年紀愈小愈重視**

電話調查其中一題問到，如果香港需要發展更多土地 ，受訪者認為最優先要考慮什麼因素，「對自然環境影響較小」 是最多人選的答案（34.4% ），其次為「可以快些提供到土地」 （20.5% ）及「提供到多些土地 」（18.2% ） 。
