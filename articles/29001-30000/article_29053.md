# 【數字創新高】港府2019上半年14次要求Google刪內容　佔中後曾以「國家安全」為由提出　涉民族黨YouTube頻道

香港眾志成員周庭周四（16日） [披露](https://web.archive.org/web/20201020003040/https://www.facebook.com/agneschowting/posts/1651812338292724 ) ，警方今年初以法庭搜查令，要求Facebook香港辦公室交出 [周庭專頁](https://web.archive.org/web/20201020003040/https://www.facebook.com/agneschowting) 在去年6月21日警總示威期間發布的一則 [帖文](https://web.archive.org/web/20201020003040/https://bit.ly/2XH33T5) 資料，包括訂閱者（subscriber）資料、IP地址和登錄紀錄（log record）。雖然Facebook香港辦公室最終拒絕警方要求，但事件觸發不少網民unlike、unfollow周庭專頁，周庭形容市民活於「警察都市」、面對「無形的恐懼」。

眾新聞翻查 [Facebook](https://web.archive.org/web/20201020003040/https://transparency.facebook.com/government-data-requests/country/HK ) 及 [Google](https://web.archive.org/web/20201020003040/https://transparencyreport.google.com/government-removals/by-country/HK?hl=zh_TW&country_breakdown=period:Y2017H2;country:HK&lu=country_request_amount&country_request_amount=group_by:totals;period:;authority:HK&country_request_explore=period:Y2016H2;authority:HK&country_item_amount=group_by:totals;period:;authority:HK ) 每半年發表一次的《資訊公開報告》（Transparency Report），發現香港政府在去年1月至6月，向Google提出14次移除要求（其中3次有法庭命令），共涉28個項目包括網頁搜尋、YouTube、Blogger及Gmail，要求次數創2009年起、有紀錄以來新高。報告未有披露港府移除要求的詳情，而Google評估各個要求後，將當中21%內容移除。

Google的《資訊公開報告》披露，早於2016年下半年，港府曾3次以「國家安全」為由，要求Google移除多達151項YouTube內容，以「國家安全」理由提出的佔當年港府要求總數99%；在2018年下半年，警方亦至少兩次要求Google移除香港民族黨的Google+及YouTube頻道。

至於Facebook，2019年上半年接獲143次香港政府索取資料的要求，略低於前兩年的數字。Facebook及Google尚未發布2019年下半年7至12月的數據。

Google自2009年起紀錄世界各地政府機關、法院及民事訴訟當事人，要求 Google移除資訊或提供資訊的數據，並每半年於 [《資訊公開報告》（Transparency Report）](https://web.archive.org/web/20201020003040/https://transparencyreport.google.com/government-removals/overview?hl=zh_TW&removal_requests=group_by:branches;period:&lu=removal_requests) 公開。報告現涵蓋145個國家或地區，其中， [香港](https://web.archive.org/web/20201020003040/https://transparencyreport.google.com/government-removals/by-country/HK?hl=zh_TW&country_breakdown=period:Y2017H2;country:HK&lu=country_request_amount&country_request_amount=group_by:totals;period:;authority:HK&country_request_explore=period:Y2016H2;authority:HK&country_item_amount=group_by:totals;period:;authority:HK ) 的司法及行政部門在2019年1月至6月，向Google提出14項移除內容要求（當中3項要求具備法庭命令），次數創紀錄新高，Google於2018年全年才接獲合共10項移除要求。

據 [《資訊公開報告》](https://web.archive.org/web/20201020003040/https://transparencyreport.google.com/government-removals/by-country/HK?hl=zh_TW&country_breakdown=period:Y2017H2;country:HK&lu=country_request_amount&country_request_amount=group_by:totals;period:;authority:HK&country_request_explore=period:Y2016H2;authority:HK&country_item_amount=group_by:totals;period:;authority:HK ) 披露，港府14次要求移除內容的理據，包括誹謗（5次）、詐欺（4次）、隱私權和安全性（2次）等，總共要求移除28個項目：網頁搜尋（17項）、YouTube（6項）、Blogger（3項）、Gmail（2項）。

報告未有披露各個港府要求的具體內容，以及Google最終有否及如何配合，僅公布2019年上半年整體有21%要求內容被移除，比率較2018年上半年及下半年的67%及71%為低。

報告解釋，Google如未有按要求移除內容的原因有很多，包括某些要求不夠明確，例如要求中沒有指明網址，Google不確定政府機關希望移除哪些內容；有些政府機關指控內容涉及誹謗，但有關機構透過非正式信函（ informal letters）提出移除要求，並沒有法庭命令（court orders）。

### 警方2018年要求刪民族黨Google+及YouTube頻道被拒絕

Google指出，報告會標示出與公共利益相關的政府要求。不過，Google並非每次都會披露該半年內接獲的移除要求內容，以近幾年為例，2019年上半年、2018年上半年、2017年上半年、2016年全年，報告都沒有披露期內接獲的要求詳情。而報告披露2018年下半年及2017年接獲的3項移除要求，均由警方提出，全數被Google拒絕。

**報告列出2018年下半年接獲兩個由警方提出的要求，均涉及香港民族黨：**

(1) 警務處處長聲稱，有 1 筆 Google+（由Google推出、類似Facebook的社群平台服務）個人資料屬香港民族黨（HKNP），而該黨是香港政府禁止運作的獨派政黨，故要求Google將該Google+ 個人資料移除。不過，Google指警務處處長未有提供移除要求的法律依據，故沒有封鎖有關資料。

(2) 警方表示，希望Google配合香港政府對香港民族黨的禁令，終止該黨的 YouTube 頻道，Google最終拒絕警方的要求。

至於2017年下半年，Google收到警務處網絡安全及科技罪案調查科要求，希望Google移除某位記者在 Blogger 發表的一篇政治評論文章。網絡安全及科技罪案調查科指，該網誌文章含有20 元香港紙幣的圖片，違反香港《刑事罪行條例》第 200 章第 103 條，該條款禁止「在任何物質上…複製任何香港流通紙幣…」。Google審核要求後，並沒有移除相關網誌文章。

### 港府2016年以「國家安全」為由，要求Google移除151項內容全數被拒

根據Google紀錄，在2016年下半年，香港行政機關或警方在2016年下半年，曾5次要求Google移除合共153項內容，其中3次以國家安全為由，要求Google移除多達151項YouTube內容，即佔總數99%。報告沒有進一步披露港府要求的詳情、發布或擁有該些內容的用戶，惟根據報告紀錄，期內並沒有港府要求的內容被移除。

2016年2月9日、農曆年初二凌晨時分，旺角發生「魚蛋革命」。2016年下半年有立法會選舉，之後梁頌恆、游蕙禎捲入宣誓風波，在人大釋法後被DQ。

至於Facebook，其 [《資訊公開報告》](https://web.archive.org/web/20201020003040/https://transparency.facebook.com/government-data-requests/country/HK ) 披露由2013年至2019年上半年的數據，同樣未公布2019年下半年的數字。報告顯示，2019年上半年接獲143項香港政府索取資料的要求，略低於2017年及2018年各半年數字。Facebook在2019年上半年最終就66%要求提供了部份資料。報告未有透露各要求的內容詳情。

### 莫乃光：未收過市民求助　或陷白色恐怖

資訊科技界立法會議員莫乃光回應Google《資訊公開報告》指，不知悉港府要求Google移除的內容詳情，他相信港府及Google亦未必願意向他披露。他續指，即使目前只有2019年上半年的資料，但數字較往年大幅增加，估計有關內容與反送中運動（2019年6月初起）以來出現的問題有關，可能政府基於政治原因而提出移除的要求多了。莫乃光續指，他每年都會要求政府提供執法機關向資訊及通訊科技公司提出披露資料要求的詳情，但政府一般不會披露， [保安局](https://web.archive.org/web/20201020003040/https://www.info.gov.hk/gia/general/202001/08/P2020010800639.htm) 轄下部門，尤其是警方，透露資料的程度愈來愈少。

莫乃光表示，未有收到市民就網上發布內容被移取或遭政府索取資料而向他求助，「一般市民如果遇到呢啲情況，可能會驚，或者覺得移除就移除啦，（申訴）可能要錢，或者可能有其他後果，實際上係一種白色恐怖，好多（市民）都未必會求助。」

莫乃光分析指，一般而言，內容涉及鼓吹自殺、教人製作炸彈等明顯涉及刑事罪行，或提出要求的一方具備法庭手令等，Google、Facebook等企業移除內容的機會較大，但如果內容不涉及干犯一些明顯的罪行、暴力等，則未必會遵照政府提出的要求。莫乃光以周庭個案為例解釋，當Facebook等大公司接到香港政府索取或移除資料的要求後，往往會向香港政府表示，要求索取涉及的內容並非存放於香港，指示政府向資料存放國家的法庭提出申請，如果申請獲當地法庭批准，香港辦公室才會根據法律互助原則，配合香港政府的要求，指國際大公司一般都會採取類似做法，作為「防火牆」。
