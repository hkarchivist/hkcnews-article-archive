# 習總聽不到的聲音　寄去瀋陽給曉波

抵港訪問的國家主席習近平，「聽不到」在機場守候的記者提問何時釋放諾貝爾和平獎得主劉曉波；相隔7年，300多人昨晚聚集在終審法院前，要向領導人清晰表達「釋放劉曉波！釋放劉霞！」的訴求，他們寫下明信片，準備寄到劉曉波正治療末期肝癌的瀋陽中國醫科大學附屬第一醫院。

「多謝你繼續溫柔地對抗政權！我們是愛國，但是對政權非常失望，但對民主、自由，香港人一直沒有放棄。」「人民沒有敵人，人民沒有忘記。」有人如是留言。有人記掛病情，寫著「請平安歸來」，旁邊一張明信片英文手書「Please release this great man immediately（請立即釋放這個偉人）」。

2010年劉曉波獲得諾貝爾獎，杜先生當時帶著矛盾的心情參與支聯會集會，來到當時的立法會大樓前「慶祝」中國公民得獎。七年過去，支聯會在同一個地點再次集會，立法會大樓變成了終審法院，大會重播劉曉波得到諾貝獎新聞片段中，關注劉曉波維權律師許志永被扣、陳光誠排除萬難到海外，中國外交部發言人也由姜瑜換了兩代發言人。

杜先生在明信片寫下「早日康復，早日得到自由」。他慨嘆，習近平雖然來到香港，可是聽不到港人聲音，港府不會轉達港人對釋放劉曉波訴求：「一個人被困了8年，還患上了重病，這反映了制度問題。到底內地是否依法治國呢？這不是法治，而是人治！」

孔先生則是比較現實，「官方還他清白是不可能的，只能祝福他」，所以他在明信片寫上《聖經》金句：「為義受壓的人有福了，因為天國是他們的」。「劉曉波沒有做錯事，只是要求基本權利，沒有要推翻共產黨，沒有組織起來推翻共產黨，不應被監禁。」他說：「（劉曉波）他的訴求也是很多市民的訴求，中國政制可以民主化、轉型，好像台灣一樣，共產黨仍然可以生存到，至少唔會傷害人。」

雖然多方要求劉曉波出國治療，但內地獨立記者高瑜引述內地司法部副部長向美國、德國、歐盟駐華官員稱劉因為病情不能轉移，聲稱劉曉波家人滿意治療方案。資深傳媒人程翔分析說內地顧忌劉曉波諾貝爾和平獎得主身分，擔心到了外面不斷發聲，所以抗拒容許劉曉波保外就醫。他直言，目前情況屬不樂觀，只能靠各界繼續發聲施壓。對於有人猜測內地當局有意釋放和風，程翔批評，「人都就死，呢個仲係和風？」
