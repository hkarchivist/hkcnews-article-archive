# 六四28周年　11萬燭光照亮維園　警方稱1.8萬

<figure>
<img src="https://web.archive.org/web/20210712005420im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/06/六四28周年-20170604225750_1820_large.jpg" alt="">
<figcaption>六四點點燭光，照亮維園，照亮香港。何君健攝</figcaption>
</figure>

點點燭光，在六四28周年，燃亮6個維園足球場及草地，也照亮香港。今年支聯會繼續在維園舉辦六四悼念晚會，主題是「平反六四，結束專政」。多個大專團體今年繼續杯葛晚會，大會宣布有11萬人參與，數字是自六四20周年以來的新低。警方表示最高峰時有1.8萬人，數字也是新低。但不論人數多少，六四悼念燭光依然照亮人心。


| **年份** | **支聯會公布人數** | **警方公布高峰期人數** |
| --- | --- | --- |
| 2009年(六四20周年) | 15萬 | 6.28萬 |
| 2010年(六四21周年) | 15萬 | 11.3萬 |
| 2011年(六四22周年) | 15萬 | 7.7萬 |
| 2012年(六四23周年) | 18萬 | 8.5萬 |
| 2013年(六四24周年) | 15萬 | 5.4萬 |
| 2014年(六四25周年) | 18萬 | 9.95萬 |
| 2015年(六四26周年) | 13.5萬 | 4.66萬 |
| 2016年(六四27周年) | 12.5萬 | 2.18萬 |
| 2017年(六四28周年) | 11萬 | 1.8萬 |

晚會開始時，支聯會常委向民主烈士紀念碑獻花，其後支聯會主席何俊仁與一眾青年共同燃點火炬，然後全體默哀一分鐘，向六四死難者致敬。在火化弔唁冊時，有一名操普通話人士向台上拋擲白色卡片，以示不滿，但他很快被大會保安帶離場。

1989年到過北京天安門見證八九學運、時任學聯代表團成員林耀強，在晚會憶述六四屠城的經歷，他當時眼見士兵向手無寸鐵的工人和學生開槍，有一位北京學生對他說：「你們香港人為我們做的已經夠多，想活命就離開這裡，將這一切告訴全世界。」林耀強續說，28年間，有無數的記者、父母和老師堅持向大眾說出真相，寄語大家不要放棄平反六四。全場為此歡呼鼓掌。

對於中大昨日在社交媒體發表聲明，指「六四情不再，悼念何時了」，更說悼念六四是時候劃上休止符。「工學同行」的大專生蕭麗萍認為，不應低估六四事件和現今社會的連繫，近年發生的銅鑼灣書店事件、今年七一遊行少了維園作起步點等，可見香港習以為常的自由漸漸被扼殺。她認為，學生有責任承傳六四追求公義的精神，藉六四的回憶改變專制政權。

另外，大會今年邀請了青年樂隊Boyz' Reborn表演，他們為六四創作的歌曲《自由之歌》，講述北京八九民運與香港學生同樣追求自由的精神。樂隊主音郭日朗認為有需要悼念六四，他解釋：「香港反國教事件和雨傘運動等，與八九民運的理念相同，想居住的地方變得更美好、更民主、更自由。」

曾在8歲時畫出王維林擋坦克畫作的蘇兆禧，現時已是17歲中學生，他認為年青一代有責任承傳六四真相，又指平反六四並非一朝一夕，年輕一代必須堅持至平反六四的一天。

今年燭光有點不一樣，支聯會提倡環保，在社交網站提供手機電子蠟燭圖像供下載使用，但仍保留派發傳統蠟燭。響應環保的市民不算多，大部份市民手持傳統白蠟燭，點點泛黃燭光，夾雜白色手機燈光，照亮整個維園。香港市民28年後，不忘高舉燭光，相信星火燎原，有天定能達成「平反六四，結束專政」之願。
