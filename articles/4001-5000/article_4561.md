# 冇錢，英文不大好，點去旅行？

**編者按：62歲筆者Bebe，退休18天後，隻身踏足西班牙，自由行了46天，寫下 [8篇旅遊見聞](https://web.archive.org/web/20210629143024/https://www.hkcnews.com/blogger/BeBe-107) 與《眾新聞》讀者分享。 [上篇](https://web.archive.org/web/20210629143024/https://www.hkcnews.com/article/3191/%E5%8D%80%E8%A5%BF%E9%81%8A%E8%A8%98-%E8%A5%BF%E7%8F%AD%E7%89%99-3191/%E5%A6%82%E4%BD%95%E5%9C%A8%E6%AD%90%E6%B4%B2%E4%BE%BF%E5%AE%9C%E5%8F%8A%E5%B9%B3%E5%AE%89%E5%9C%B0%E6%97%85%E8%A1%8C) 他整理了在歐洲便宜及平安地旅行的實戰心得，本篇就介紹如何在冇乜錢、又唔係好識英文和當地語言，都可以暢遊歐洲。**

[上篇文章](https://web.archive.org/web/20210629143024/https://www.hkcnews.com/article/3191/%E5%8D%80%E8%A5%BF%E9%81%8A%E8%A8%98-%E8%A5%BF%E7%8F%AD%E7%89%99-3191/%E5%A6%82%E4%BD%95%E5%9C%A8%E6%AD%90%E6%B4%B2%E4%BE%BF%E5%AE%9C%E5%8F%8A%E5%B9%B3%E5%AE%89%E5%9C%B0%E6%97%85%E8%A1%8C) 介紹如何用最平的方法去旅行，係介紹冇錢，英文不大好，點去旅行前，先介紹西班牙首都馬德里比較好區的 apartment (如港式的多層單位）價錢。

### 點樣搵錢去旅行？

處境1：兩公婆住一單位，樓已供斷，二人已退休

1.1 把自己單位租出去兩年，每月租金，好多人都夠用平方法去旅行

1.2 若覺此法太激進，你不能接受，那你的單位有沒有多餘房間？若有：   
 。把房間租出去   
 。或學做Airbnb，在旺季把房間以Airbnb形式租給遊客，淡季生意不多，旺季賺D錢，淡季兩公婆去旅行

處境2：若你同子女住，又如何？

教你子女做Airbnb，旺季租出去，你們兩老旺季去旅行，淡季你們回來住

**Airbnb是什麼？**

Download 個 Airbnb Apps，然後輸入 Hong Kong，任意輸入 Check-In 及 Check-Out 日子，你便可以知道有幾多人在香港做Airbnb，以及香港不同地點的房間 Airbnb 可以收多少錢，這樣你就可以估計你的房間一晚可以租多少錢。

**如何了解 Airbnb 運作？**

去台灣或去深圳、廣州住兩、三間 Airbnb，留意他們由取鎖匙，到 check-out，你便了解很多 Airbnb 運作。

### 去邊度旅行好？

冇錢可以去亞洲二、三線城市，泰國不去曼谷，不去布吉，去泰北；越南不去胡志明市，不去河內，去二、三缐城市及農村，country side住屋和食物平到你唔信，坐廉航如HongKong Express、Asia Airline 等飛機票幾百港元有交易。

台灣去台中、台南已經好平，台北可以去，留短D，留幾天算了。

日本的九州、四國、北海道也很平。

### 唔識英文及當地話如何？

語文唔識就要膽色。

在西班牙的Granada 市美麗的雪山Sierra Nevada山腰，我在那裡的 Hostel 住了七個晚上，首天他們的德國藉經理告訴我，月前有位韓國人入住，他只能說少少英文，原來此英文識少少的韓國人，正從摩洛哥用雙腳行路回韓國。

而他們的德國老闆，告訴我有位18歲的台灣女子入住，原來她而一個人 backpacking 了一年。

德國老闆問這嬌小的台灣女孩，一年過去，為什麼妳仍能生存？

台灣女子回答：一年遊歷經驗，我覺得無論什麼國籍，什麼人種，人類根本上是善良及愛助人的。

很多國家的平民均喜歡同外國人交往，做超出自己範圍的工作幫外國遊客。

筆者西班牙文、法文、德文唔識，都可以去過咁多次。

主要是叫餸、問路、到餐廳，看到其他顧客吃什麼，向waiter指他們的碟，不會錯到哪裡。

現在有Google Map，問路機會少咗，還有Google Translate 可以文字、語音翻譯幾十種語言，我是從大陸同事身上學用 Google Translate 相似的 Apps。

我們只有一生人，六十歲以上的退休人士，無論你幾長命，很多人說72處或最遲75處後，你便不能再去旅行，因為到時不是你身體有事，便是你的配偶身體有事。

到時才待在家中等死，行得時，不去看看這美麗的世界其他國家，感受其他文化，嚐嚐各國不同美食，碰碰世界上其他陌生人，看看他們如何生活，而所需貴不過住在香港。

不要浪費你所餘不多的光陰！

### 我點樣計劃西班牙自由行

在此我分享我如何計畫去西班牙的旅程。起初我打算用兩個月走遍約三十個城市，西班牙是一個地理上差不多是四方形的國家，從那裡開始，盡量不走回頭路，從那裡完結，也要費心思。

我在網上看到很多不同人的介紹。由於我以前公事去過巴塞隆拿及華倫西亞數次，但馬德里只去過大半天，我打算在整個國家四方形中點的首都馬德里起步。

馬德里這種大城市，以我個人興趣，如倫敦、巴黎、紐約、巴塞隆拿、東京，甚至香港一樣，慢慢細玩二個月，才能了解及享受此城市。事前我又打算學西班牙文，香港找西班牙人教西班牙文好貴，經一段網上研究，結果我登記了在馬德里一間西語學校上課二個星期，學校更安排我住在一個西班牙家庭 host family中，這是最好不過的了解當地人生活、文化的方法。每日費用約220港元，這包括一個房間、早餐及晚餐。

當訂好在馬德里西語學校及住宿後，我一個聖誕查經班年約八十的好朋友阿 Joe，給我提供如下地圖的行程，我一看識途老馬Joe 的行程，便跟著他去：

 。最後在飲食聞名的巴斯克地區 Basque Country完結   
 。此行程包括在那個城市逗留多少天

Joe 乃香港成功商人，全球任何地方差不多已經去過，有很多已經去過數次，剛巧2015年夏天，他住在馬德里嫁了西班牙人的張大千孫女的家兩週，也在附近玩了一段時間，Joe 很有心，自己親手印了張西、葡地圖，把每個城市應該停多少天寫上，再用箭咀把每個城市所走路缐聯起來，Joe 一共推介了約廿個西班牙大小城市給我。

我二話不說，便用了Joe 給我的行程，以首都馬德里為起點。

如很多 Backpackers 說，先計劃好行程，但只訂第一個城市的房間，到了第一個城市後，再計劃下次行程，因為你到第一個城市後，再問當地人介紹去那裡，再修改你的行程，再者，若你很喜歡首站的城市，多留一，兩天，若不喜歡，可以早點離開此地，轉去下一站。一個人自由行，彈性由你決定。

如我到了西班牙後，我把我的行程（Joe 的地圖）給幾個西班牙人及熟悉西班牙的老外看過，包括我的前西班牙同事，我的兩位西行牙語老師，一位在西班牙住了三個月的澳洲同學及幾位在Seville 的Hostel 遇到的老外，每人都加一，兩個城市要去。

但到最後，如很多backpacker 說，你只能去到大部分或部分你原來計劃的地方及行程，但你仍然玩得很開心。
