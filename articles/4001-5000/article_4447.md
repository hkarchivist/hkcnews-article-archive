# 高盛同志員工及伴侶　可享代母及領養津貼

高盛、瑞信等12間國際金融機構介入同志伴侶簽證上訴案件，事實上，不少跨國企業已將同志伴侶和異性婚姻一視同仁。其中高盛的員工指引列明，同志伴侶領養孩子、僱用代母產子等，都可以獲得有薪假期和津貼。

> 員工可指定「伴侶」享福利   
>  如代母假、領養補貼

《眾新聞》向高盛查詢同志員工的待遇，公關回覆的指引稱，高盛員工可指定一人作為伴侶，不論同性或異性，即使沒有註冊結婚，只是同居都可以。高盛將按照兩人關係的忠誠度，例如：同居時間等，來判斷關係是否屬實。

當高盛員工指定了伴侶後，他可向公司申請家庭福利，包括不同假期及津貼。若員工和伴侶打算透過領養或代母產子，並作為主要照顧人的話，員工可以申請4個月有薪假，並申請最少1.5萬元津貼。

#### 高盛向員工提供以下福利：


| 產假 | 嬰兒出生後12個月內，可申請4個月有薪假 |
| --- | --- |
| 父母親侍產假 | 非嬰兒主要照顧人，可享有4星期有薪假 |
| 家庭緊急情況休假 | 每年會有5日，員工可以申請假期來處理家庭緊急情況，包括生病或意外等 |
| 領養假 | 若員工與伴侶合法領養孩子，並成為主要照顧人，將可申請4個月有薪假 |
| 領養津貼 | 每個領養小孩可以申請2000至6000美元，約1.56萬至4.7萬港元津貼 |
| 代母假 | 若員工與伴侶透過代母產子，將可申請4個月有薪假期 |
| 代母津貼 | 每個孩子可獲約3.9萬港元津貼 |
| 喪假 | 若家人或登記伴侶離世，將可申請5日有薪假 |


除了假期和津貼外，高盛其他福利亦會涵蓋同志員工及他們的伴侶和孩子，例如醫療保障。高盛特別強調，公司會向員工家屬提供醫療照顧，同志伴侶也不例外，內容甚至包括生育治療，或變性手術。員工亦可指定其同志伴侶，為退休後福利及保險受益人，但他們強調這項福利有機會被香港法庭推翻。

近年積極關注商界同志共融的社商賢匯，早前列出不少向同志員工提供福利的企業，當中除了高盛外，摩根士丹利、貝萊德、瑞信等國際金融機構都榜上有名。

本地公司方面，港鐵、渣打銀行等都有向同志員工提供福利。港鐵指，員工需要提交有效結婚證明，就可以申請配偶福利，包括醫療及免費乘搭港鐵，雖然同志婚姻在港未合法，但員工即使在海外註冊都會被承認。渣打則指，員工和其同志伴侶，和已註冊的異性戀夫婦，可享有同等待遇。

至於新鴻基地產和東亞銀行則回應指，公司設有反歧視政策，保障同事不受歧視，但沒有具體回應會否向同志員工提供福利。其他本地企業例如：新世界、恆基、長實等，未有回覆記者查詢。
