# 網上情——女生好騙？

「恨人間，情是何物，直教生死相許。天南地北雙飛客，老翅幾回寒暑。歡樂趣，離別苦，是中更有癡兒女。君應有語，渺萬里層雲，千山暮景，隻影為誰去……」~ 《摸魚兒》— 金代詩人元好問（1190-1257）。

有政黨半年內收到42宗求助，有關美容院用「暖男計」，在網上認識單身女生，誘騙簽單做美容，涉款210萬元，受騙人年齡由26至47歲。

去年兩名年輕姊妹給一名35歲的已婚大胖子騙了300萬元；該名大胖子是妹妹透過手機程式認識的。

84名年齡介乎25至73歲的女性，在網上被騙約6,000萬元，當中近半為白領，其中14%從事服務行業及6%為專業人士，其餘為退休人士及家庭主婦等，而更耐人尋味是這84名女生，從來沒有和網上男友正式見過面。

新世代的愛情遊戲方式，從以前的看花燈找對象，相親，自由戀愛，漸漸變成虛擬的網上交友。根據警方資料顯示，從2014年至2016年首10個月中，總共有181宗類似騙案，涉及1.36億元，每名受害人平均損失約75萬元，2016年首10個月已錄得90宗個案，比2015年同期42宗上升逾1倍，金額由數千元至910萬元。

跟電話騙案不一樣，網上情緣騙案不能只用傻、笨、「身有屎」去解釋，因為很多受騙人，都不是入世未深的青春少艾。那麼，如果不是無知，是什麼？女生是不是真的比較容易受騙？

女生比較感情豐富，比較容易哭，比較心軟，比較癡心，比較有愛心等等，都是眾所周知的事實。有心人利用女生這方面的特質，進行騙情、騙財，甚至騙色。

對於那些網上情緣騙案，琢磨了很久都不得要領，直到最近在飛機上看了BBC著名廣播人和自然學家David Attenborough的紀錄片，Planet Earth II第二集後才略有所悟。

這一集紀錄片講述，一隻母雪豹帶著一隻還沒成年的女兒。當時正值母雪豹的發情期，需要交配。但帶著女兒去交配是非常危險的事，因為雄雪豹會殺死不是自己所出的未成年雪豹。

如果用以上的現象去解釋網戀騙案的起因，可能有些女生不同意。不過，養過狗女都知道，牠們每年有兩次發情期，需要交配，不做絕育手術挺煩的。生育是所有動物包括人類的自然能力，萬物也是靠這樣的動力，才可以繁衍下去。雄性和雌性互相吸引是最自然不過的事，造物主在天地初開時，已經把程序編好在我們的DNA裡，不能抗拒。女生大概每28天一個週期，這段日子裡，荷爾蒙estrogen and progesterone水平不斷改變，身體準備好受孕，很多女生在這個時期也會煩躁不安 (Pre-menstrual Syndrome) ，沒看出來是因為人有能力控制情緒而已。可能有人說，就算這個論點成立，也不一定導致網戀吧。那麼，讓我們看看香港的男女比例（截至2016年10月）：

| --- | --- |
| 出生 | 1.12：1 |
| 0 - 14歲 | 1.12：1 |
| 15-24歲 | 1.08：1 |
| 25-54歲 | 0.74：1 |
| 55-64歲 | 0.94：1 |
| 65歲以上 | 0.89：1 |
| 整體人口 | 0.89：1 |


數據顯示適婚年齡的男女比例不平衡，女生比較難找到對像，因為男生太少；別說精挑細選，就算隨便找個「爛燈盞」都不夠分配。如果女生的學歷高，找對象更困難。在大學裡除了工科外，其他科一般都是女生比男生多。在社會上除了最高級的職位外，中高層的管理人很多是女生。（有讀者寫來說這個現象是因為女生學習能力，在中一派位前比男生早熟，所以進好學校的女生比男生多。因此，進大學的女生數目也自然多了，並非女生比男生強。如果這個論點成立，香港女生找對象就更難了。）

別以為男生都是英勇無比，流血不流淚。假的，其實很多都是玻璃心。大陸有句話教人如何維持和諧的男女關係，非常一針見血——「男要崇拜，女要愛」。要一個男生去追求一個學歷比他高，工資比他多的女生，您叫他的臉擱到哪裡去？想想，像陳淑莊那麼能幹的女生，誰敢追求？如果自己能力不夠老婆強，自然會擔心老婆不崇拜自己，還是找個比自己水平低的好。所以以前封建社會提倡「女子無才便是德」，就是害怕女生比男生強。

撇開大自然的呼喚，女生是喜歡被「暖男」去愛和關懷的。100毛的Braologie廣告裡，何超儀說：「有邊個女人唔中意畀男人愛？」一語中的。李澤楷的女友郭嘉文也很坦白地說：「被想念的感覺真好。」既然現實生活中找不到愛的對象，在網上碰到懂得呵護自己，噓寒問暖的男生，很容易被碰到內心軟處。為了維持關係，別說金錢，就算人都願意搭進去。在一個心理和生理都需要異性的時候，邏輯分析能力幾乎蕩然無存，一切只憑感覺去判斷。可惜，這樣的判斷往往是錯的。

這個幾千年來的遊戲，一直沒有改變過，因為造物主把我們的基因編排了傳宗接代，這個編排，讓我們很容易不知不覺地墮入情網，癡癡迷迷，完全盲目。當一個人愛上心儀的人時，別說區區幾百萬，就算是自己的性命，也願意無條件獻給自己心愛的人。莎士比亞的羅密歐和朱麗葉充分表現出，愛情是多麼浪漫、多麼陶醉、多麼徹底。

既然那麼容易遇人不淑，沒對象的，是不是都要像林振強填詞的《夏日寒風》：「狂呼我空虛、空虛，恨極為她（他）心碎，明知結局，何必去做玩耍器具。狂呼我空虛、空虛，怒罵是她（他）不對。強忍眼淚，從此我願獨在痛苦中活下去……」呢？
