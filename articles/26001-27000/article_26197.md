# 炊君大人的精準高度

<figure>
<img src="https://web.archive.org/web/20210706042343im_/https://www.hkcnews.com/news_新聞/hk-discuss/2020/01/荒腔走板-王浩鈞-20200119104527_28cc_large.jpg" alt="">
</figure>

催眠治療的學理中，   
 暗示，   
 是改變一個人行為的心理板掣。   
 這個心理板掣有個專用名稱～「心錨」。   
 比喻為一隻船停泊時放下船錨的「固定」作用。   
 一個臨床催眠專家可以在任何情況對指定的對象，   
 下一道或幾道心錨，   
 情況就像電腦程式，   
 1.Clear   
 2.X=0   
 3. Input X   
 4. If X=A goto 7   
 6.If X≠A goto 1   
 7.output B   
 程式中的A，   
 就是傳說中的心錨。   
 若然，   
 你聽到某首歌，   
 你就站起來離開。   
 若然，   
 你看見一個紅衣女人坐在你對面，   
 你就變成一個交際能手。   
 你如果對這現象嘖嘖稱奇，   
 那時，   
 催眠的老行尊會笑笑告訴你：   
 「人生無處不催眠。」   
 講得白一點，   
 每個人在一生中，   
 都不免受種族文化家庭傳承既定俗成的概念所影響。   
 像13號的星期五就是黑色星期五諸事小心。   
 像粵語片中陳寶珠的單身母親自小告訴她男人都是壞東西。   
 廚房的炊君神位放置不能矮過人頭。   
 這些是不是心錨？   
 是不是暗示？   
 當事人都會當input=A時，   
 直接output B，   
 不論你有多少高等教育學位，   
 不論你才智過人眉精眼企，   
 你都一樣output如儀，   
 這種直接反應，   
 如夢幻泡影，   
 如露亦如電，   
 問你奈得它，   
 甚麼何？
