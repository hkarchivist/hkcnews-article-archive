# 警擬引入電槍作日常裝備　人權組織憂不當使用致市民傷亡

有傳媒引述消息指，警方正研究引入電槍（stun guns）、網槍（net guns）等電擊武器（electroshock weapons），作為日常警務裝備，以應對持續的示威活動。保安局局長李家超未有否認引入電擊武器，並承認警方有研究其他裝備。人權組織及泛民議員均批評有關計劃，憂慮警方不當使用武器會對市民造成嚴重人身傷害，並預期警方引入新武器將激發更多衝突，令局勢惡化。

[無綫新聞](https://web.archive.org/web/20210630054442/http://news.tvb.com/local/5e1e80c1335d193d505e143f/%E6%93%9A%E6%82%89%E8%AD%A6%E6%96%B9%E7%A0%94%E5%BC%95%E5%85%A5%E9%9B%BB%E6%A7%8D%E4%BD%9C%E7%82%BA%E8%A3%9D%E5%82%99-%E5%88%9D%E6%AD%A5%E5%82%BE%E5%90%91%E8%A1%9D%E9%8B%92%E9%9A%8A%E5%85%88%E7%94%A8 ) 指，警方研究增加警務人員裝備選項，已接近完成階段，初步認為可引入電槍，供日常警務裝備，而並非只作防暴用途。初步傾先供衝鋒隊使用，再逐步擴展至其他隊伍。據 [《南華早報》報道](https://web.archive.org/web/20210630054442/https://www.scmp.com/news/hong-kong/law-and-crime/article/3046049/hong-kong-police-consider-arming-officers-electroshock?fbclid=IwAR3xLSAChub27KzGjgm1mX68A9Cc0HXWc3lquE_tmWTu1Lrk2t4RcnoZpoQ ) ，警方早於2016年農曆年期間爆發的旺角衝突之後，已經測試過電擊武器，但警隊基於人體健康及政治考慮，當時沒有引入電擊武器，至近期局勢持續惡化，警隊才重提有關計劃，並已比較不同型號的電擊武器及市面上的網槍。

《南華早報》引述警隊消息指，引入電擊武器的目的，是增加武器使用的選擇，而不是提高武力程度或引入致命武器。報道提及消息人士稱，電擊武器對警務人員及疑犯都有利，「因為疑犯被捕時反抗或掙扎時間愈長，雙方受傷的風險愈高。」另一報道中的消息人士表示，當示威者配備更佳的防護裝備，胡椒噴霧的效用已降低。「所有工具，包括警棍，如果錯誤使用，都具有殺傷力，但電擊武器屬主流及非致命的。我們從數據得知，每100萬人之中，只有13人死於或嚴重受傷於電擊武器。」他稱，電擊武器的武力程度低於胡椒噴霧。

民主黨立法會議員鄺俊宇周三（15日）在立法會質詢時，引述消息詢問警方是否打算引入電槍。保安局局長李家超回應時未有否認，並指任何方法令警方更有效處理「暴徒」的暴力，而又可以將大家受傷的風險減低，保安局均支持，「當然警方會研究清楚每一樣工具，對佢哋執法能力所產生嘅風險係會點，但係我知道警方所研究嘅任何裝備，其實喺外國、先進國家都係用嘅裝備，呢啲裝備過往有良好紀錄，喺適當時候適當使用，係可以減少多方面受損傷嘅效果。」

電擊裝備是以電流作攻擊武器。當有人開動電槍，槍口的金屬針會如飛鏢彈出，射向攻擊對象，使其在極短時間內失去對肌肉的控制能力，因而無法作出反擊。網槍的原理相若，但發射出來的是一個電網。

### 美國逾百人遭電擊致死

《南華早報》引述國際特赦組織數據指，在2001至2012年間，美國有至少500人在被捕或在囚期間被電槍所傷後死亡，組織呼籲限制有關武器的使用。報道引述人權監察警告，當疑犯有心臟疾病等潛在健康問題，警方使用電擊武器有機會造成無法預計的後果，包括威脅疑犯的生命。

此外， [路透社2017年報道](https://web.archive.org/web/20210630054442/https://www.reuters.com/investigates/special-report/usa-taser-911/) 指，自2000年起，美國錄得至少1,005宗死亡個案涉及警方的電槍使用，路透社取得當中712宗的驗屍結果，得悉在153宗個案中，電槍攻擊被認為是導致死亡的成因或促成死亡的因素。

民權觀察重申，不當使用低致命武器可造成嚴重的人身傷害，甚至致命。民權觀察批評，警方至今仍未公開武力指引，對去年6月以來警察濫權問題充耳不聞，亦未設立有效機制向違規警員問責，引入新武器只會引起群眾不滿及激發更多衝突。

### 泛民強烈反對引入電擊武器

民主黨及公民黨對警方引入電擊武器作常規裝備表示強烈反對，並敦促林鄭政府回應市民訴求。

民主黨立法會議員林卓廷指，在現時警民關係緊張之際，加裝備對解決問題是雪上加霜。他引述外國例子指，電槍有可能令人心臟停頓，甚至死亡，非常危險。林認為，現時警察經常不按指引鎮壓示威者甚至無辜市民，形容警方「濫權濫暴嚴重」，如提升武力將會令局勢惡化。

法律界立法會議員、公民黨郭榮鏗指，民調顯示有一半市民給予警隊0分，顯示市民對警隊失去信任，倘若警方在此時新增武器，只會加深不信任和撕裂，對解決社會問題完全沒有幫助，反而火上加油，「更是給予警員多一件工具去施行警暴」。

另一公民黨立法會議員楊岳橋批評，政府所謂「止暴制亂」的措施，包括添置新型武器，是將納稅人的金錢用得不明不白，十分荒謬。他指出，財政司司長可以根據《公共財政條例》第8條，獲立法會財委會轉授權批發1000萬元以下撥款的能力，他批評政府以「拆件上」的方式， 將大額撥款分拆成不超過1000萬元的款項，以轉授權力批出，繞過財委會監察。根據李家超周三（15日）予楊岳橋查詢的 [書面答覆](https://web.archive.org/web/20210630054442/https://www.info.gov.hk/gia/general/202001/15/P2020011500382.htm) ，警方過去多次按《公共財政條例》第8條獲批撥款，包括在2019-2020年度批出3.2億元的逾時工作津貼，以及100萬元的水警船艇增加開支。

### 美電槍生產商擬落戶香港

資料顯示，以開發電槍品牌Taser聞名的美國公司Axon，近期擬拓展北亞地區業務，並計劃將北亞洲區總部設於香港。Axon北亞區域經理 Chris Brand去年8月來港，接受 [《PCM》](https://web.archive.org/web/20210630054442/https://www.pcmarket.com.hk/2019/08/17/axon%E9%9A%A8%E8%BA%AB%E9%8F%A1%E9%A0%AD-%E7%9B%A3%E5%AF%9F%E8%AD%A6%E5%AF%9F%E6%9A%B4%E5%8A%9B/?fbclid=IwAR3555_DMGQi4OAg2KRl1AoS30HNv0E-ptOQSglhjAs6JO8TwP3TmPnqWdE ) 、 [《經濟日報》](https://web.archive.org/web/20210630054442/https://sme.hket.com/article/2456021/%E5%BC%B7%E5%8C%96%E9%9A%A8%E8%BA%AB%E6%94%9D%E9%8C%84%20%20%E7%89%A9%E8%81%AF%E7%B6%B2%E6%8F%90%E5%8D%87%E4%BF%9D%E5%AE%89%E7%AE%A1%E7%90%86 ) 、 [《IT Square》](https://web.archive.org/web/20210630054442/https://www.it-square.hk/archives/9345 ) 等傳媒訪問，主要推廣其隨身鏡頭（body camera）產品及相關服務。 [《PCM》引述Brand](https://web.archive.org/web/20210630054442/https://www.pcmarket.com.hk/2019/08/17/axon%E9%9A%A8%E8%BA%AB%E9%8F%A1%E9%A0%AD-%E7%9B%A3%E5%AF%9F%E8%AD%A6%E5%AF%9F%E6%9A%B4%E5%8A%9B/?fbclid=IwAR3555_DMGQi4OAg2KRl1AoS30HNv0E-ptOQSglhjAs6JO8TwP3TmPnqWdE ) 指，Axon未有與香港警方接觸。

[Axon網頁](https://web.archive.org/web/20210630054442/https://buy.axon.com/ ) 顯示，Taser電槍有 [不同型號](https://web.archive.org/web/20210630054442/https://buy.axon.com/ccrz__ProductList?viewState=ListView&cartID=&portalUser=&store=&cclcl=en_US&categoryId=a4af3000000Dj0eAAC ) ，主要供應予組織或公司使用，另有 [出售予一般市民作自衛用途](https://web.archive.org/web/20210630054442/https://www.axon.com/ ) 。網頁未有提及電槍可產生的電流，但報稱可令攻擊對象的肌肉活動受限制5秒至30秒，使持電槍者有時間逃生。民用電槍的售價，每枝由449.99至1,399.99美元不等（相當於港幣約3,500至1.1萬元）。

另外，新界北總區刑事高級警司陳天柱及爆炸品處理課警司李展超周三（15日）表示，警方周二（14日）首次在香港檢獲「水喉通土製炸彈」。警方在旺角通菜街一間劏房，檢獲一個重1.5磅，由8寸長金屬水喉通製成的土製炸彈，拘捕3名年齡介乎21至22歲的大專生。警方之後在上水河霸村一間村屋，破獲相信是製作爆炸品的場所，再拘捕一名29歲男子。惟被捕者的家人指，警方從上水村屋檢獲的硫磺是用作「趕蛇」。陳天柱回應，爆炸品內的材料可能是合法物品，也可用作製成其他物料，警方會調查含量及裝置，是否足以製作爆炸品。

警方繼周二拘捕上述4人，周三再以涉嫌非法集結、串謀製作爆炸品和藏毒等罪名，拘捕6名男子，年齡介乎17至23歲，2人是學生，1人是幼稚園教師，1人是咖啡師，2人無業。陳天柱表示，相信爆炸品是一個勇武組織，計劃在大型公眾活動襲擊警員或警方設施，意圖造成大量傷亡。陳天柱稱，警方從情報得悉組織存在，相信部分成員涉及元旦日非法集結。他說，疑犯將爆炸品藏在民居，公眾安全受到極大威脅。
