# 鄭泳舜、鄧家彪、陳家珮的政治聯繫如何違反《基本法》？

【撰文：郭倫】   
 ***作者為 [中文及通識科網誌](https://web.archive.org/web/20210712013546/https://www.facebook.com/中文科閱讀理解應試技巧-164354330288453/) 作者，著有《圖解「今日香港」 》、《為甚麼我考不好中文》等***

香港眾志的周庭被選舉主任取消參加立法會補選的資格，理由是她的政治連繫是香港眾志，眾志主張自決，而根據2016年人大第五次釋法的內容，選舉主任再「暗引」了全國人大常委副秘書長李飛講及釋法時，強調自決即等同「港獨」作為取消資格的理據，不相信周庭真心擁護《基本法》。

原來政治聯繫也可以作為取消參選資格的理據，那麼好玩了。究竟鄭泳舜、鄧家彪、陳家珮的政治聯繫有沒有違反《基本法》呢？

鄭泳舜、鄧家彪的政治聯繫同樣是民建聯（正確一點說， 鄧家彪屬於「兩姓家奴」，同是民建聯和工聯會成員）。在香港主權易手之初，民建聯支持07 / 08雙普選，這還寫在其黨章之中。在2004年的立法會選舉前，民建聯更在一個選舉論壇上表示支持07 / 08雙普選。可是，2004年人大第二次釋法，否決了07 / 08年雙普選。當日梁天琦參加立法會選舉，選舉主任追溯了他以前支持港獨的言論，作為取消他參選資格的理據。若按此邏輯，選舉主任理應追溯民建聯當年支持07 / 08年雙普選的言論，不相信這政黨的成員真心擁護《基本法》。

#### 請選舉主任取消鄭泳舜、鄧家彪的參選資格。

至於陳家珮，其政治聯繫是新民黨，新民黨主席是葉劉淑儀。當年葉劉任保安局局長時，硬推《基本法》二十三條立法。最終，田北俊辭職，特區政府終止立法程序。但請不要忘記，《基本法》第二十三條寫明：「香港特別行政區應自行立法禁止任何叛國、分裂國家、煽動叛亂、顛覆中央人民政府及竊取國家機密的行為。」重點是《基本法》要求特區政府「自行立法」，政府卻主動終止立法程序。投票被否決是一回事，但是政府竟然主動收回議案，這又是另一回事。那麼，陳家珮背後的「政治聯繫」當然沒有擁護《基本法》。

#### 請選舉主任取消陳家珮的參選資格。

說回周庭的情況，或許有人認為她可以申請司法覆核，覆核選舉主任的決定。但是當日梁天琦被取消參選資格前，也曾申請緊急司法覆核，覆核選舉確認書。結果，覆核被高等法院駁回，法官區慶祥指出參選人的提出名不獲確定為有效，參選人可在選舉後提出選舉呈請，不會對參選人造成無可彌補的影響。

參選人在選舉之後才提出選舉呈請，即使呈請得直，而且最後當選，也會少做了數年議員，議員任期不會因而延長。

法官說不會造成無可彌補的影響？這只騙得了三歲小孩。你相信香港有法治？相信法官中立？我呸，法治你個屁！
