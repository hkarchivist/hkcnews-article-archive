# 從未消失的金漆招牌

【撰文：陳零／攝影：Mike Tam】

**吳文正 ( Simon Go ) 曾在一個訪問中說：「我驚自己老了，一個人坐在公園，一個人在後悔，後悔後生時沒去闖新生活。」於是，他人到中年，跑去創辦文化葫蘆，出書、辦活動探討關於這城市的「消失」。**   
 **最近，他還辦起攝影展，把 60 間老店定格、放大。**   
 **你看相展，相中人在看你。**   
 **This city is dying？怎會呢？**   
 **一百個人都可以有一百種方法去守護這城市。**   
 **只要，你願意。**

### 相機失打的魔法

走進《金漆招牌–街坊老店》攝影展覽的場地，有種置身於哈利波特霍格華茲魔法學院的氣氛。看著那些照片，相中人好像就站在你面前，讓你拍照、跟你聊天，說著他們的故事。   
 f22 foto space 創辦人，也是這攝影展策展人蘇彰德 ( Douglas ) 解說了展覽不少心思：「相片特意不用玻璃框的。」那樣才有種近在咫尺的效果，但當然看著參觀者太貼近照片，隨時像會弄花照片時，亦是一幕幕的驚心動魄。   
 場地對燈光、濕度的關顧，原因亦不用多說，反而相片布局的思量，就凸顯兩位男士細膩的一面。   
 不過，不想劇透，以免削減逛展的趣味，但不妨留意展覽打頭陣的兩幅大相，然後順時針繞場，其中四幅照片的特色；還有展覽二樓美都餐廳和 Vinyl Hero 阿 Paul 的相中亮點 … … 準會讓你會心微笑。   
 那就是攝影凝固那一刻的魔法。

當然，每張相片都盛載著照片人物和 Simon 的故事，有各自本身的，也有交互的。   
 Douglas 說，不把照片放成這樣大，就看不到店和人物每個趣味細節。   
 誠然，當 Douglas 讀著傘王何希記相中的兩句細字：「『何人任收天下傘　希望終於有傘神』，這兩句頭一個字正是何希的名字。」聽罷，有種難以言喻的震撼。

每張相，都如尋寶圖般，可以在每個角落找到店主的心意。   
 也許，不只是展覽中的照片，而是在香港不同的角落，那些持續經營三、四十年，甚至過百年的家店 ( Family Shop ) ，都各自藏著寶庫。

### 非常香港

用 Simon 的話，家店正是香港的文化特色，呈現年代遞進的社會面貌。「從前店面大，前舖後居，一家人住在一起，睡醒便開店；生意可能只夠交租、餬口，養活一家人。」然而，家人就緊靠在一起，同喜同悲。   
 流動小販檔亦別具深情，譬如太太煮好飯，便拿著飯壺給在守著檔的丈夫，午飯就是站著吃完，太太就拿著飯壺回家，又為丈夫、子女預備晚餐。   
 平凡的生活，簡單的幸福。   
 「都不是想著發達的。」Simon 這個項目延展了13 年，那是他的觀察，也是一番體會吧。

他說所接觸的家店店主，透過他鏡頭，都表現了「只得我有」的自豪和歸屬感。不管是後巷有煙頭、有滴水的二手租書店，又或是唱片塞滿店空間的黑膠唱片店，「每間店都自有特色，也是店主引以為傲之處。」他笑指鴨寮街 Vinyl Hero 店主阿 Paul 跟他說：「除了北極、南極，全世界都有人來找他。」各地的同好者在網絡世界得知香港有這位黑膠唱片藏家，不少都遠道而來拜訪他。   
 老店雖小，豐富的內涵卻載著無限；內涵需要年月累積。

<img src="https://web.archive.org/web/20210629150925im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/12/老店-好好故事-吳文正-20171226215210_1495_large.jpg" alt="">

相片也看到中國人的含蓄。   
 「上一代人拍照都比較拘謹，先生絕少會搭著太太膊頭。」Simon 說跟每間店拍照前，是先要交個朋友，而交朋友是用年月付出真心的。   
 所以，前後花上 13 年，才拍得 300 間家店的照片，絕對是一項記錄。   
 令他窩心是，有些家店是每隔兩三年，就再拍一張，每張都見證家店成員的變化。   
 其中，創於 1927 年的明鏡，店主已 95 歲，四代同堂。 Simon 將為該店拍第四次照片，而拍下的三張都放在店中。拍這張大合照，更是家族盛事，難得的聚首一堂。

### 屬於你和我的

Simon 為香港所做的功課，實始於 2009 年和太太自掏腰包辦的社區活動 「牛下開飯」，是告別牛頭角下邨的文化活動。當年，他邀請學生拍攝牛下的點滴，又辦講座和展覽，捧場者眾。他才意識到本土文化活動的火熱。   
 同年，他放棄了攝影記者的工作，創辦非牟利機構文化葫蘆。   
 此前，他已出版了 《街訪老店》一書。

他說自己在黃大仙長大，童年就在黃大仙、九龍城這些舊區游走，每天見到就是那些家店。只是，近年總像一覺睡醒，便有幾間家店結業。   
 城區不斷蛻變，只能與時間競賽。   
 他用相機，把時光凝固，那是唯一可與歲月抗衡的舉動。

家店的裝潢，一般雅緻也談不上，但憑甚麼讓你依依不捨？   
 代表著一個年代流逝的紙皮石、瓦仔牆磚、木凳？   
 都是，都不是。   
 大概是那份如家人般的情。   
 那就是家店的金漆招牌。   
 這份情包含了誠信、刻苦耐勞，在自家範疇做到最好。   
 這是對社區的承諾，因為社區就是左鄰右里。   
 這亦是一眾家店擁抱的價值觀，代表著一個年代的香港精神。

對這一代的我們來說，香港的金漆招牌又是甚麼？
