# 有一種成就叫勇敢

[請支持《眾新聞》，成為訂戶](https://web.archive.org/web/20210629150947/https://www.hkcnews.com/aboutus/#subscribe)

【撰文：陳零／攝影：Lewis Wong／影片：Trevor Tse、Ivan Yiu】


{{< video src="https://web.archive.org/web/20210629150947if_/https://www.youtube.com/embed/Av6xcibCX-A" >}}


**她是個愛笑的女孩。**   
 **她的笑容很有感染力，就如她的插畫般，密密麻麻，七彩繽紛，充滿喜悅的氣氛。**   
 **她坦言不愛讀書，只愛畫畫；為了畫畫，會到茶餐廳當侍應，找來生活費。**   
 **為了畫畫，她會一個人跑到波蘭，爬上18 樓，畫外牆。**   
 **她說自己是個很勇敢的女孩。**   
 **的確是。**   
 **追夢的人，都需要勇敢。**

### 哪一天我們會飛

你可以叫李美欣做 Messy Desk ( 她的品牌 ) ，或者Mei Mei 、Jane 。   
 她不介意告訴你，會考11 分，家人希望她繼續升中六，走姐姐的路，教書賺多個錢。   
 她亦無隱瞞自己不愛讀書，同學上堂抄筆記，自己亦不停手，不過是在畫畫畫。   
 會考過後，她亦沒有理會家人反對，報讀IVE ( 香港知專設計學院 ) 視覺傳意高級文憑課程。

她的插畫未必屬於殿堂級水平，但風格獨特 — 所有畫作都是密密麻麻，但色彩卻繽紛。   
 她說自己的畫作有如尋寶圖，可以在當中找到她創作的卡通角色。   
 她的作品其實充滿電影感，就如一幕一幕的場景，時而太空，時而童話故事，有時會找到旋轉木馬、蘑菇雲、南瓜車、蘋果屋，有時會見到太空船。當然，還有香港的建築物和人。   
 你會驚嘆，這樣密集的素材，要多久才能完成。   
 你看過她即場作畫，你或者會更驚嘆。

她亦提到自己畫風跟歐洲流行漫畫 [Bande Dessinée](https://web.archive.org/web/20210629150947/https://fr.wikipedia.org/wiki/Bande_dessinéeBande Dessinée) 相近，所以與歐洲結緣。   
 讀書時，她經常參加不同的校外比賽，已在網上平台My Space 發表自己畫作，因而結識了不少歐洲藝術家，互相交流。   
 第一個驚喜，英國廣告公司買下她其中一幅畫作3 年版權，賺了8,000 多港元。   
 後來又曾獲法國出版社Na Edition 邀請，參加全球第二大漫畫節—安古蘭漫畫節 [The Angoulême International Comics Festival](https://web.archive.org/web/20210629150947/https://en.wikipedia.org/wiki/Angoulême_International_Comics_Festival) ，並獲得漫畫新秀獎。

### 付出，可能是無止境的

很順利嗎？   
 畢業後，她沒打算去打工，由當freelancer ( 自由工作者 ) 開始。   
 頭三年，她日間到附近的茶餐廳工作，賺點生活費，放工便繼續作畫。   
 因為法國藝術雜誌《Get Freaky》 的賞識，她先後到過法國、美國、英國、德國等地舉辦展覽。   
 她自從2010 年，幾乎年年往外跑，頭幾年，甚至一年跑3 個展覽。   
 這種應該叫做付出。

2015 年8 月，她獲邀參與波蘭一項街頭藝術活動「 Fundacja Urban Forms 」，爬到波蘭38米高 ( 18層 ) 的羅茲大學語言中心，用自己的畫風繪畫該大廈外牆，足足畫了3 個星期。她是唯一也是第一位參與的香港插畫家。   
 她說時沒有吐氣揚眉的自傲，亦沒有不獲家人支持的憤憤不平。   
 她形容自己是個勇敢的女子，就是在別人不看好時，爆發力量，做出成績。

### 夢想，要一步步行過去

今天的Messy Desk ，還沒有如麥嘜般的家傳戶曉。   
 不過，每次見到李美欣，還是那樣保留著突然會大笑、傻呼呼的情懷。   
 然後，每次請她即場作畫，都是那末認真，一分鐘可以畫滿半張畫紙。   
 再然後，問她合作過的品牌，怎樣數都數不出來；但其實已經有Air France 、Starbucks 、RTHK 、地鐵公司……

她說希望Messy Desk 可以帶來正能量和快樂。   
 她說希望把自己的畫作，帶到全世界。   
 說出夢想，只是廉價的口號。   
 身體力行，達成夢想，其間需要的付出，未必人人看到。   
 Jane 就只管勇敢地向前行。   
 然後，一步步實現心中所想。   
 你呢？   
 說了，就去做吧！
