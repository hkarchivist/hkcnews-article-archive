# 萬宜水庫的夜與日


{{< video src="https://web.archive.org/web/20210706165858if_/https://roundme.com/embed/181373/469046" >}}


萬宜水庫西壩的星空──月黑風高，萬籟俱寂，只有天上星閃閃，黑夜過後迎來晨曦，金光灑在平靜水面，又新的一天開始。但願眼前美景不會被人相中，要把它填平建住宅。

**星空的拍攝數據：Nikon D800 Sigma 10mm鏡，ISO:1250; F:3.2 ; S:8秒**

**日出的拍攝數據：Nikon D800 Sigma 10mm鏡，ISO:200; F:9 ; S:1/100**
