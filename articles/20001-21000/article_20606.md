# 滅蟲滅蝨特攻隊

閱昨天某網媒，閱讀一尊貴立法會議員在其Facebook 講述其日前探訪大角咀劏房，看見年70高齡長者備受蟲蝨患困擾的情況 ; 在本人未看見有任何報導，指出如何助此些基層的困難無助者解決助滅蟲蝨及向有關當局反映以助處理下，更看到他也指出此是「國際都會背後的悲哀」之言，頓覺令我感到他旣是社工，為何不予援手手以發揮社工助解困難的天職，只令我感到此是「議會背後的涼薄」 。

本人認識此位尊貴議員已有數拾年，一向深覺他是一位為市民福發聲請命的議員，如為罕有病而怒斥政府不是(此估計是會令他可有背後一群罕有病者支持的政蹟，以贏得聲譽？！)

由於地球暖化、本港氣温日見溫暖炎熱及潮濕、人口居住面積縮小、加上老年人口增多，他們鮮有力執拾及清潔居所……等原因，本港蟲蝨患漸見猖獗，在當局只顧蚊患及鼠禍下，此日漸令受患者朝夕家居生活質素大受影響，坐不寧、睡不佳是必然事。需知，另有新聞報導也指出蟲蝨咬人後，再咬其他人士也有可能傳播病菌 ; 試問早年香港推行「洗太平地」，不是鼓勵市民「洗床板」嗎？推行此舉，就是一個防疫措施的初心 ; 是否因蚊患及鼠禍曾引來災害，是否有待蟲蝨於社區廣泛傳播病菌才作「禍後補求」，不作處理此「防範未燃」的社區隱伏危機呢？

正視面對此日漸惡化的社區安居困擾，是要及早處理，故本人斗膽集一已之力，得到曾受蟲蝨患人士參與，組織一義務「滅蟲滅蝨特攻隊」，設立24小時「環保滅蟲滅蝨熱線」外，更聯絡各區服務機構，於社區提供環保滅蟲滅蝨教育講座（因坊間用含化學物質的產品），以避免受患者家居殘留化學物質，繼續傷害人畜，如孕婦、小孩及體弱者 ; 主力義工更自資100萬元貨值生態環保滅蟲滅蝨產品，希望透過社福及慈善團體之助，訓練義工到户有需要者，作出滅蟲滅蝨的服務（有關資料，請參看下述附件)。

就以上資料，去年在沒有重大議會政治鬥爭，建制及泛民尚可和平共處時，本人幸獲此議員接見，令「特攻隊」可在其面前陳述社區蟲蝨患惡劣情況，並提出以上我們可為的服務工作 ; 感謝他即時委派一職員全權跟進 ; 唯不久本人獲通知他們向政府有關部門查詢求助個案數目，自言說數目只是個位以下，故此後便失聯了，雖本人再三表示，以乃是「離地」、不符的數字！試問，此議員壓根兒是一名曾受過訓練的社工兼曾為大學講師，又旣是為民福祉而「力拼」港府權威的一員，也不顧「此群無聲强受困」的一群，旣然親身到户看見，怎只有是感受，而沒有「身同感受」，他口出指受害人之「環境重衰過露宿者」，而轉移讀者的關注去感受當局的不是 ; 也不借此Facebook撰出其如何可於議會中，向有關政府部門反映及作要任何要求行動，而只談以上「不關他事」的快閃說話，便作已探訪及關心基層者生活的「拿取市民支持」的積分呢？

港人福祉，尤以基層巿民的需要，作為社工界的一員及市民代表，盼有關當事人不忘投入社工的初心，在能力範圍内至少也轉介至關注民生的議會伙伴，不要漠視問題而不顧，而令活在「非人生活」的小市民感到「議會背後的涼薄」。

### 滅蟲滅蝨特攻隊

成立原因 :   
 鑑於社區蟲蝨患近年日漸嚴峻，加上本港社福界尚未有此關注民生家居生活質素的服務，也在當局並未對影響市民安居生活質素的蟲蝨患困擾，作出任何的宣傳教育工作；此義工隊遂在自發性下基於以上社區需要而創辦。

目前情況 :   
 我們是香港首隊旣提供教育活動、宣傳及諮詢熱線、到户服務，以及透過資助為貧困無助老者及劏房户，作出滅蟲滅蝨根治服務的多元化社區服務隊，全由義工組成。

義工隊成員包括 :   
 創辦人是一位退休裝修水電工程承辦商，從事義務工作近30年、師從加拿大一教授學習環保及愛惜地球的防治蟲蝨法，更多番往製造廠了解及學習，多年來經常住美加考察滅蟲滅蝨實際工作發展。

一退休社工主導，憑藉其30多年按社會轉變而衍生的社會服務開創經驗，冀望藉社會熱心人士之助力，打造一個全方位關注基層人士的蟲蝨困擾問題，以現時「特攻隊」的服務作為示範，讓社福機構參考作實務的推行 ;

一位年50歲多的女士，曾身受蟲蝨困擾志切滅蟲滅蝨教育及服務，持有理工大學之「害蟲防制及施用除害劑之安全與健康訓練課程」証書行政人員作為後盾支援「特攻隊」的工作推行（如教育講座等) ;

一位80多歲的長者，因生於戰火受難而得可生存，感社會之助才有今天的他，視回饋社會是己責。年前自正式退出職場後，除不斷參加課程進修充實自己外，並參與多項義務工作 ; 亦深覺蟲蝨為禍基層甚大，故抽空加入「特攻隊」學習，希望可運用餘生多做服務人群的工作。

「特攻隊」的主要工作：   
 1. 往各區社福機構、房協長者服務等團體，以及於網媒提供社區教育講座及資訊，以增加社會各階層人士認識蟲蝨患的成因、徵狀、治理及預防方法；並教育巿民選用生態環保滅蟲滅蝨劑，以免錯誤使用含化學物料，而殘留家居傷害人畜。

2. 遇有乏力滅蟲滅蝨的獨居無助者，經社工轉介會到户服務 ; 或聯同聯盟組織㩦手做滅蟲滅蝨的工作。

3. 推動社福機構招募及培訓義工，例如，怎樣偵查蟲蝨患及作為後盾顧問，以跟進他們工作時的問題，困難及需要。

4. 為經濟有困難、由社工轉介而來的個案，提供「滅蟲滅蝨援助」，聯同社工的參與，以資助方式藉此教導受惠者除治理蟲蝨患，也推動家居整潔關注的配合行動。

5. 設立 8100-0038 環保滅蟲滅蝨熱線，為香港市民提供24小時之蟲蝨患成因及防治的方法。
