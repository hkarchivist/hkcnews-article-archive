# 勤練太極　朱天才贏得長壽

【撰文：林德民】

今年77歲的朱天才，在內地武術界德高望重，是當今中國陳家溝陳式太極四大金剛之一 ; 他精神很矍鑠，中氣十足，聲線洪量，雙眼炯炯有神，令人一看便知是習武的人。

</figure>

從少喜歡動手動腳的他，10 歲時便正式拜其舅父陳照丕習陳式太極，朱大師謙說 : 「練習太極拳是無難度的，只要有恒心、堅持，你要練拳練到家，就要每天下功夫，下時間，日子有功，拳自然到來」。抱著堅持的他，孩童時候就是每天不停打，不停耍。

不但可令四肢活動靈活，年青至今向是身體健康，他嚴厲地說 : 「此特別對老人家有益，因為此可以延遲身體老化，你看我身體還可以嗎？」

朱大師常教導學員，生命在於運動，運動在於適度，因為太極剛柔並重，經前人花心血精神設計出來，有序地隨意甩甩手，是可以收按摩內臟之效，這和隨便甩甩手壓壓腿，實是有天淵之別的功效。

「耍太極時，我們的心情必要放鬆，不要天亮到天黑多憂多慮，飲食要有規律，清淡一些較好 ; 睡覺要有規律，人自然長春長壽，身體也會少些病痛。」此確也是我們的養生延壽的一大方法啊。

太極是深藏道家學說的陰陽平衡，處處蘊藏陰陽的道理，道是學，拳是術，耍太極就是要把這個陰陽的道理耍出來 ; 唯太極並不是柔弱乏力，其實太極拳處處充滿攻防意識，平時耍起來慢吞吞，但一出手快如閃電，連消帶打，連防帶攻可將對手打倒在地上，所以朱大師每次傳授太極時，必先向年青學員強調心靜平和，不是單純練拳強身或防身，而是傳承一種中華文化，一種陰陽平和的文化。

朱大師指出，有關陳式太極和其他太極流派的分別之處，就是陳式太極特點在於「纏絲」兩個字，要耍太極養身有道必須理解纏絲，要把纏絲耍出來，外練筋骨皮，內練一口氣，這口氣就是纏絲，練拳要重視腰跨，其實是練腎，他耍拳的心得更是一個圈加兩個點，及在於纏絲。練太極先練纏絲一個圈最重要， 是圓圈的上下兩個點，就是頭和跨。

向以推動太極文化的朱大師，近年吸引不少海外人士遠道向他拜師學習，他自言不單是教授，更是悉心向外推廣太極文化，期望學員回到他們原來的國家後，將中華文化散播開去，讓外國人也能強身健體 ; 故他每年必會出外教授，支持各地多與内地合辦太極比賽，以提高各地的水平，期待一天可將太極拳列入奧運項目的宏願 ; 深信如朱大師習拳的堅毅手，太極成為奧運項目之一，是可以指日可待的。
