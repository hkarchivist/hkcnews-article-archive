# 對太極拳知行合一的一點了解

【撰文：張達揚師傅，陳氏太極拳第二十代傳人】

要求知而能行，行亦能知，否則終不免自欺欺人，妄語叢叢，言之多無邊際 ; 知行二字，雖然簡單，實則繁難，所謂知難行易者：凡對一門技術有深刻之功夫，亦有相當之效果，而因知識所限，不能道其所以然者皆為知難行易，若有識别之能而無功力，則可謂知易行難 ; 倘無功力又乏知識，則知行二字兩不可能。

只有知行合一，才能到達成功之路 ; 學術本無止境，其有若干知或有若干行，行到如何地步、知到怎樣程度，方算真知真行？則實不敢加論定，然應以能知者即能行，能行者亦能知，始可謂知行合一。

拳術一道，首要明理，更須切實用功 ; 若不首先明理，不知用功切要之所在，用功也是枉然，故要有明白的老師，將拳理拳經明明白白地告知，使學員不走彎路，不會渺茫，一步一步走入高深的境界。

**知行合一口訣：**

**練拳先從無極始，意守丹田自始於；**   
 **導引吐納先天術，氣行周身脉絡通；**   
 **虛靈頂勁頭要直，全身節節須放鬆；**   
 **呼吸自然丹田氣，兩腎生氣往下行；**   
 **屈膝鬆胯沉足底，仙氣再升脊骨中；**   
 **任督二脉通七窍，氣運全身轉不停；**   
 **周身循環無止境，練精保氣自然生；**   
 **無極混圓常不懈，神行玄妙在體中；**   
 **無極而後衍太極，太極功夫上一層；**   
 **靜中求動動求靜，動靜互變太極功。**
