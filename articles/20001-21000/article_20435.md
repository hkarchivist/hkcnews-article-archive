# 拜師是學習傳統技藝的頭等⼤事

【張達揚師傅，陳氏太極拳第二十代傳人】

中國很多傳統的項目都講究拜師學藝，如戲曲、書法、國畫、中醫、針灸等傳統文化。武術也不例外，沒有師傅就等於「無源之水，無本之木」，嚴格的說沒有師承，沒有師父不算入門，行裡的人認為不是門裡出身，俗稱地講「不正規」。

武術界向來講究拜師儀式，師徒的師承關係，名份極嚴而著稱於世，要想成為師門的正式弟子，須經過師父幾年的反復全面考察，確定品行端正，符合條件得到認可後才決定是否收他 ; 本人寫拜師申請，師父認可後必須舉行隆重的拜師儀式。在拜師儀式上要當眾宣讀「拜師帖」，要表明拜師者願意拜某人為師，入門後學習弘揚本門功夫的决心。同門師兄當眾宣讀「規門師訓」，並對其提出入門後要遵守法紀、尊師重道、武德武風、文明禮貌、刻苦練功等方面的要求。

**一般拜師儀式分成三個程序：**

**第一，拜祖師、祈求祖師爺保佑，使自己學有所成。**   
 **第二，行拜師禮，一般是師父、師母坐上座，徒弟行三叩首之禮，然後跪獻紅包和投師帖子。**   
 **第三，師父訓話，宣布門規。**

拜師儀式上徒弟向鼻祖、師爺、師父、師母行過三拜大禮，這是接受一次傳統文化的洗禮，是我們繼承傳統文化的重要部分 ; 在儀式上，師父及兩名以上見証人在「拜師帖」上簽字等一系列手續後，方能進入師門，成為師父的入門弟子 ; 入門弟子名嫡系弟子，最先進入師的入門弟子，稱為大師兄，又稱開山弟子，最後進入師門的入門弟子，稱為小師弟，亦稱關門弟子；同時入門的年長者為師兄，年輕者為師弟，未經過上述程序，跟師父學過拳且拳技較好者，得到師父承認的，也是師門弟子，但只能稱作記名弟子。不是入門弟子，也不是記名弟子，但只是同師父學過拳的，稱為學員、學生或學徒。

另外，有的同時進入兩個師門的問題，原則上師父在世就不可再拜第二個師父，一個徒弟進兩個師門學同一種拳是犯忌的 ; 但不是同一個拳種是可以的，拜其他老師學另一拳種要先得到目前師父的同意，有不少老師高風亮節，不但支持自己弟子再拜高人，有時還為自己弟子推薦武技高超的老師。

老師對於學生，可以教也可以不教，沒有這種拜師的緊密關係。拜師的弟子像家人一樣，老師有甚麼事要當成自己的事對待，你的事老師也當成自己兒子的事一樣關心；入門後師父要因才施教，盡心傳授，對於學生的保密，對弟子就不保密，弟子對老師也得負責，出去以後你的行為表現，功夫的展示，社會對你的口碑等各方面都代表着師門，關係着師門的榮辱，在做人和功夫上要做到能夠為本門爭光。

「傳人者」，能夠傳承之人也，在武術界能夠稱得上傳人的都非泛泛之輩。首先，作為（傳人）要繼承老師的傾囊之授，理法合一的將老師傳授真正學到手，其次要把作學消化吸收成為自己的東西。再者，就是要發揚光大，不但要有所創新而且要使之能傳承下去。

一般而言，跟老師學過拳可稱為學生；正式拜師的稱入門弟子；弟子之中敏而好學，窺得堂奧，有所造詣者，稱入室弟子；入室弟子中與老師關係密切，受到老師賞識的可稱得意弟子；得意弟子中，盡得所傳真髓且深受老師器重，老師一般會欽定為衣鉢傳人，也有部分入室弟子能夠把老師所授發揚光大，且在武林之中有一定影響，而被公認為某老師的傳人，不過被稱做傳人的，一般而言沒有自封的，大多是得到同門師兄弟首肯或是得到武林同道認可的。
