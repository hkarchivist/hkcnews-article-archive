# 【疫苗保障基金】半年接獲338宗申請　至今僅批出22宗　死亡個案未獲批

<figure>
<img src="https://web.archive.org/web/20220123192706im_/https://www.hkcnews.com/news_新聞/hk-discuss/2021/09/疫苗-疫苗接種-20210901170804_2fa4_large.jpg" alt="">
<figcaption>眾新聞製圖</figcaption>
</figure>

疫苗接種計劃展開逾半年，超過六成合資格市民經已接種第一針，總接種劑數逾750萬。根據食物及衞生局最新數字，截至8月23日，共接獲328宗受傷個案申請，已處理196個個案中，只有22宗成功獲批（約佔11.2%）、174宗申請被拒（約佔88.8%）；另外有10宗屬於死亡個案申請，8宗已處理，無一成功。

政府2月份設立10億元疫苗保障基金，為接種後出現嚴重異常事件的合資格市民提供財政援助。眾新聞向食物及衞生局查詢最新申請數字、個案情況，獲回覆指截至8月23日共接獲328宗申請。數字詳情如下：


|  | 死亡個案(宗) | 傷害個案(宗) |
| --- | --- | --- |
| 申請個案 | 10 | 328 |
| 處理中個案 | 2 | 132 |
| 被拒個案：   
 不完整申請 | 1 | 100 |
| 被拒個案：   
 委員會判斷與疫苗無關   
 或「不可分類」 | 7 | 73 |
| 被拒個案：   
 嚴重程度評估暫不涉長期   
 或永久身體損害 | 0 | 1 |
| 獲批個案 | 0 | 22 |
| 合共發放金額 | $ 0 | $ 2,630,000 |


食衞局指，獲批的22宗個案中，涉及速發嚴重過敏反應、住院治療及貝爾面癱個案，而被拒的不完整申請個案，包括醫護人員未有呈報為疫苗接種嚴重或非預期異常事件、呈報個案不屬於專家委員會通過的「接種新冠疫苗嚴重或非預期異常事件列表」或「接種新冠疫苗關注事件列表」載列的事件、或未能提供相關證明文件。

然而，關於個案受害人接種的是哪一款疫苗、基本資料包括年齡、性別、有否長期病患、接種後的反應等，局方均未有提供。

眾新聞 [早前報導](https://web.archive.org/web/20220123192706/https://www.hkcnews.com/article/44455/%E7%96%AB%E8%8B%97-%E5%BE%A9%E5%BF%85%E6%B3%B0-%E6%AD%BB%E4%BA%A1-44459/%E3%80%90%E5%AE%B6%E5%B1%AC%E7%94%B3%E8%A8%B4%E3%80%9135%E6%AD%B2%E7%94%B7%E5%AD%90%E7%96%AB%E8%8B%97%E6%B3%A8%E5%B0%84%E5%BE%8C%E5%85%A9%E5%80%8B%E6%9C%88%E4%BA%A1-%E9%86%AB%E8%AD%B7%E6%8C%87%E5%B7%B2%E9%81%8E14%E6%97%A5-%E4%B8%8D%E9%80%9A%E5%A0%B1%E3%80%81%E6%8B%92%E7%99%BC%E5%9A%B4%E9%87%8D%E4%BA%8B%E6%95%85%E8%AD%89%E6%98%8E) ，一名35歲男子接種兩劑復必泰兩個月後死亡，但其家屬未能申請疫苗保障基金，因為醫護人員拒絕發出嚴重事故證明書。
