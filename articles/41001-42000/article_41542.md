# 【NFT拍賣熱潮】蕭雲作品《我們的獅子山》2.8萬元賣出　競投者：相片代表港人重要記憶

近期NFT橫空出世，成為了數碼藝術的潮流，不少創作人都會在網上的 NFT 銷售平台出售個人作品。經常在抗爭現場採訪的獨立攝影師蕭雲，日前將一張名為《我們的獅子山》的相片放到 NFT 銷售平台 Opensea 拍賣，該照片最終由一位香港、台灣兩邊走的傳媒、創作人Thomas Lee 以 1.05 個以太幣（Ethereum, ETH）投得，折合約27,800港元，蕭雲表示將會捐出拍賣所有收益，「希望能夠幫助到香港與同道」。有份競投蕭雲作品的Liker Land營運總監余家齊認為這幅照片具有巨大的意義價值，該份意義包括香港人一片重要的記憶、一種身份的表現及一份信念的堅持。

### 何謂NFT？

非同質化代幣（Non-Fungible Token，NFT）可以是一幅jpg或Gif圖、任何遊戲道具或數碼藝術品等，這些數碼圖像透過區塊鏈技術產生獨有的標記，從而認證物主的擁有權。這代幣被記錄在區塊鏈的數碼分類帳中，以儲存有關NFT的詳細信息，包括所有權的轉變。NFT亦代表着它不能互相取代、且無法分割，像是每個 NFT 都有專屬於自己的編號（代表每個都是獨一無二的），所以這個世界上每一個人的NFT都是不同的，換言之，即使我們在 NFT 上購買了同樣的商品，但因為產品編號不同，因此我們手中的 NFT 依然是不同的東西。 簡單點來說，NFT就是數位世界的真品證書，令由作者認證的檔案成為數位世界中唯一的存在。

蕭雲這張賣出的《我們的獅子山》是第一幅以高價賣出，以 2019 反送中運動為主題的新聞照片 NFT。拍賣的平台為 Opensea，經歷了 17 口叫價，最終由一位香港、台灣兩邊走的傳媒、創作人Thomas Lee 以1.05以太幣（折合約27,800港元）投得。扣除平台所收取的2.5%手續費，蕭雲可獲得約27,000港元。

**蕭雲亦曾在Facebook分享過《我們的獅子山》背後的故事：**

***「夜晚八點獅子山腳早已水泄不通，一步一停，等候山上的市民下來。筆者等到凌晨兩點半才從山腰登頂，可是山頂依然塞滿了人，沿著山徑排隊，成為獅子山的歷史奇觀。***

***但筆者並非隊尾最後一人，身後還有一位叔叔，揹著龐然大物的他舉步維艱，因而不時墜後，幸有同行的市民協助。***

***原來他搬的是一支長逾一米，貌似火箭筒的巨型投射器，先到獅子山脊架設裝備，再將『我願榮光歸香港』和『SOS HK 光復香港』兩款標語投射到獅子山頭。***

***叔叔自號「熊本熊」，身材亦恰如其分。他解釋全套設備重二十公斤，可惜可攜的電源有限，每幀標語只能維持約數分鐘。當電量告盡，他便毫不無遲疑揹裝備下山。***

***筆者詢問他為何不惜千辛萬苦換來幾分鐘。他的回答只有一句反問，『你爽唔爽呀？』」***

協助蕭雲把此照片鑄成 NFT，並在 OpenSea 上拍賣的 Liker Land 營運總監余家齊 （Edmond）向記者透露，他也是競投者之一，因為他對於該張相片有一份「情意結」，「當晚其實我都喺獅子山、蕭雲又係我朋友、我失意嘅時候佢又幫過我、佢話會捐出所有收益俾嗰個機構又係我本身想捐嘅，所以呢張相本身對我嘅誘因係好大。」Edmond表示，若他可以競投獲得這幅NFT照片，它將永存在他的電子錢包中，全世界只此一幅，可以擁有這歷史性的一刻，他又解釋競投NFT的價值：「如果將蕭雲呢張相放落區塊鏈，其實同普通下載嘅 jpg，睇落係無分別，但係區塊鏈嘅 NFT，係能夠睇到張相過去及依家嘅擁有者，資料不可銷毀，不可竄改，呢樣嘢正正就係NFT嘅價值所在。」該照片最終由創作人 Thomas Lee 投得，Edmond坦言「真係有啲失落！」

Edmond 看到記者臉上的不解，笑笑口繼續解釋，「即係等於我哋雖然可以喺網上免費 download 蒙娜麗莎名畫嘅高解像度JPEG，並將佢Copy好多張，甚至印出嚟掛喺屋企埲牆上，但收藏喺羅浮宮嗰幅蒙娜麗莎真品嘅價值係唔會受到影響，因為嗰幅畫是唯一真正出自達文西手筆嘅作品。所以Thomas Lee買咗蕭雲嗰幅相，就等同收藏喺羅浮宮嗰幅蒙娜麗莎真品一樣，只此一幅，其他再copy出嚟，都只係一幅copy出嚟嘅《我們的獅子山》，並唔係真正嘅《我們的獅子山》。」

Edmond 表示基於區塊鏈技術提供的公信力，NFT 的出現為數位作品提供了獨一性。他指過往若要認證藝術品是否屬於真品，需經有公信力的公證行發出證書，而金額較小的商品如個人限量作品，則靠作者在作品上簽名等方式認證；然而數位作品卻一直沒有有效的防偽方法，但直至 NFT 技術出現，令作者能產出唯一的數位作品。因為 NFT 基於區塊鏈技術，本質是一張智能合約，由於區塊鏈不可竄改的特性，令數位作品的 NFT 一經生成便在網絡中永久存留。而因為其唯一性，NFT 就像藝術品一樣擁有升值空間，購入者可以通過轉售賺更多的錢。

以這次蕭雲所利用的拍賣平台OpenSea為例，來詳細說明如何購買 NFT。

OpenSea 是目前最大的 NFT 交易平台之一，你可以在上面交易各式各樣的 NFT，也能製作自己的 NFT 作品販售。OpenSea就像一間線上拍賣行一樣，在這裏客人可以競標物品或以固定價格「立即購買」。OpenSea 的通用貨幣是以太幣（Ethereum, ETH），OpenSea也支持許多數字錢包，其推薦使用 Chrome 瀏覽器的 MetaMask 數字錢包插件。

要進行拍賣競投，客人首先需要創建一個數字錢包，並連接到 OpenSea 的帳戶，然後購買一些可以在 OpenSea 上購買 NFT 的以太幣，客人可以通過各種渠道購買加密貨幣，譬如主流加密交易所 Coinbase、Voyager 等。再把以太幣轉換成 wETH（wrapped ETH，即「包裝過的以太幣」），符合 ERC-20 標準，之後便可以在 OpenSea上競標購買 NFT 了。

值得一提的是，在OpeaSea競投的過程中，當中不少程序都牽涉礦工費 (gas fee)，包括：把以太幣轉換成 wETH（包裝的以太幣），以符合 ERC-20 的要求、首次競投以及取消競投，而Edmond競投《我們的獅子山》的時候礦工費就大約$150港幣。Edmond 補充指，礦工費的每次估算都會因為當時不同的網絡情況令數值有少許不同，因此以上數字僅為約數。

他亦表示想趁此機會釐清一個坊間最大的謬誤：Bitcoin、ETH、Likecoin 等並不應該叫「虛擬貨幣」，應該叫做「加密貨幣」或者「密碼貨幣」，「因為如果叫虛擬貨幣，就俾人感覺好似覺得呢啲錢係假咁，因為『虛』啊嘛，有虛就有實，就好似假錢同真錢嘅分別咁。但事實上呢啲錢並唔係假錢嚟㗎嘛，係絕對可以係市面上流通嘅一種貨幣，可以買可以賣。」

Edmond 又提到，中原地產創辦人施永青今年初在其創辦的報章《AM730》以 [〈比特幣缺乏貨幣應有的特質〉](https://web.archive.org/web/20210712061940/https://www.am730.com.hk/column/新聞/比特幣缺乏貨幣應有的特質-251879) 為題撰文，質疑Bitcoin（比特幣）缺乏普遍的流通能力，缺乏貨幣的最基本特質，又指「賣一碗雲吞麵才幾十塊港元，比特幣一個單位都幾萬美元，用真鈔找贖風險很大」。Edmond 笑稱施永青就是典型對加密貨幣「無知的人」，「佢用買雲吞麵嘅例子都幾差，你正常拎張美金去香港嘅雲吞麵鋪，你都好難買得到一碗雲吞麵，純粹係因為人哋唔收美金，而唔係因為美金唔係錢，同樣嘅道理放落 Bitcoin 亦都一樣。」他強調，香港有很多外幣兌匯店讓人可以將美金兌換成香港流通的港幣，而同樣香港亦有加密貨幣交易所讓人將 Bitcoin 等加密貨幣轉換成港幣。

NFT的熱潮去年起開始爆發，至今年初數碼藝術家 Beeple 以6,900萬美元（約5.4億港元）的天價賣出一幅JPG作品《Everydays: the First 5000 Days》震驚全球，令人對這種新的創作者經濟模式引頸以待。而過往媒體行業的案例，有《紐約時報》以 350 枚以太幣 （當時值 56 萬美元）售出一則商業版專欄的截圖、美國商業雜誌《福布斯》以 333,333 美元售出一期雜誌封面、活躍於香港的新加坡華裔創作歌手陳奐仁（Hanjin）在 OpenSea 發佈新曲《Nobody gets me》，作品的 NFT 最終以 7 個以太幣（15 萬港元）成交。

Edmond 認為 NFT 所帶來的商業模式，讓新聞照片能繼續流傳於公眾，同時為新聞業者創造收入，但與一般藝術作品有點不同，新聞照片最重要的質素是反映跟人類息息相關的歷史事件，擁有一張新聞照片的 NFT，彷彿跟某個重要時刻結連，成為那記念品的唯一珍藏者。NFT 技術為新聞照片帶來的稀缺性令這意義變得獨一無異地珍貴，新聞照片 NFT 的前景令人期待，Edmond 認為蕭雲這張《我們的獅子山》算是「實驗成功」，希望能讓新一代的創作者及傳媒人建立出新的經營模式。

不過，Edmond 指 NFT 現時仍沒有有效方法避免有人假冒作者身份在網絡上發佈作品，這情況就如有人把蕭雲的相片複製，並宣稱贗品是真跡在市場拍賣一樣。針對此問題，LikeCoin 團隊正在研發基於區塊鏈的ISCN (International Standard Content Number) 技術，讓作者能簡單地為自己的作品註冊，修補原創者身份漏洞，再以 ISCN 編號發行作品 NFT，就如以 ISBN (International Standard Book Number) 編號出版書籍一樣，原創者的帳戶亦能得到認證，情況就如 Facebook 及 Instagram 的藍剔認證一樣。
