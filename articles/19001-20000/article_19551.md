# 【Fact Check譚惠珠】37國跟中國簽署並執行引渡條約　不包括英、美、加、德

政府建議修訂《逃犯條例》和《刑事事宜相互法律協助條例》，容許以一次性個案方式，將逃犯引渡至包括中國等目前未有跟香港簽訂長期引渡協議的國家，引起公眾憂慮。基本法委員會副主任譚惠珠周日（3月31日）出席商台節目時為港府修例 [說項](https://web.archive.org/web/20210712024926/https://www.speakout.hk/%E6%B8%AF%E4%BA%BA%E9%BB%9E%E6%92%AD/43383/-%E7%9F%AD%E7%89%87-%E6%83%B3%E9%91%BD%E6%B3%95%E5%BE%8B%E6%BC%8F%E6%B4%9E-STOP-%E8%AD%9A%E6%83%A0%E7%8F%A0-%E4%BD%9C%E7%82%BA%E4%B8%80%E5%80%8B%E5%9C%8B%E9%9A%9B%E9%87%91%E8%9E%8D%E5%9F%8E%E5%B8%82-%E5%94%94%E5%B9%AB%E5%8A%A9%E5%94%94%E5%AE%88%E6%B3%95%E5%98%85%E4%BA%BA%E5%96%BA%E9%A6%99%E6%B8%AF%E6%B4%BB%E5%8B%95-%E5%91%A2%E5%80%8B%E4%BF%82%E6%99%AE%E4%B8%96%E5%83%B9%E5%80%BC-%E4%B8%AD%E5%9C%8B%E6%B3%95%E6%B2%BB%E5%96%BA%E5%9C%8B%E9%9A%9B%E5%8D%94%E8%AD%B0%E4%B8%8A-%E5%B7%B2%E7%B6%93%E5%8F%97%E5%88%B0%E6%B0%91%E4%B8%BB%E5%98%85%E5%9C%8B%E5%AE%B6%E6%8E%A5%E5%8F%97) ，指中國近年推行「陽光司法」，大大提高司法透明度，更聲言中國法治受到先進民主國家所接受。

譚惠珠在節目上是這樣說的：

> **中國已經同36個國家簽咗國際司法協助協議，其中包括加拿大、美國、德國、法國、意大利、英國呢啲民主先進法治嘅國家，佢哋一樣有（押）解犯人返去俾中國。**

不過，眾新聞Fact Check譚惠珠的發言，發覺事實並非如此。根據 [內地檢察院資料](https://web.archive.org/web/20210712024926/http://www.gd.jcy.gov.cn/jcyw/sfxz/flfgytyxd/201812/t20181212_2440091.shtml) ，截至2018年10月，中國已與55個國家簽署引渡條約，當中37項已生效。而她列舉的6個西方國家，4個（加拿大、美國、德國、英國）都沒有跟中國簽署引渡條約。

前政務司司長陳方安生批評譚惠珠的言論是「侮辱香港人的智慧」，她引用黃秋生近日言論指：「當一個人沒有正義感，不能夠辨別是非黑白，就真是枉為人。」

根據 [廣東省人民檢察院陽光檢務網](https://web.archive.org/web/20210712024926/http://www.gd.jcy.gov.cn/jcyw/sfxz/flfgytyxd/201812/t20181212_2440091.shtml) ，截至2018年10月，中國與75個國家簽署司法協助協定，包括刑事司法協助條約、民商事司法協助條約、引渡條約、移管被判刑人條約等，涉及合共160項條約（128項已生效），其中，引渡條約有55項（37項已生效）。眾新聞整理已跟中國簽署引渡條約的55個國家及條約生效日期資料如下：

### [【與中國簽署引渡條約之國家一覽】](https://web.archive.org/web/20210712024926/https://docs.google.com/spreadsheets/d/1ISpDF0ErsRrhcnCIzKMfTzS_0_qmbOyGD3KOhBQqrSA/edit?usp=sharing)

截至2018年10月，譚惠珠提到的加拿大、美國、德國、英國，都未有跟中國簽署引渡條約，而意大利、法國確實與中國有引渡條約，有關條約分別於2015年7月17日及2015年12月13日生效。

### **有權拒絕引渡國民、異見者**

[《中華人民共和國和法蘭西共和國引渡條約》](https://web.archive.org/web/20210712024926/http://www.npc.gov.cn/wxzl/gongbao/2008-06/03/content_1463233.htm ) 及 [《中華人民共和國和意大利共和國引渡條約》](https://web.archive.org/web/20210712024926/http://www.npc.gov.cn/wxzl/gongbao/2012-03/05/content_1705132.htm ) 條文內容相若，均有訂明雙方的引渡義務，界定可引渡的罪行及拒絕引渡的理由。法國、意大利與跟中國簽署引渡條約均訂明，可引渡的犯罪條件包括：根據雙方法律均可判處1年以上徒刑的犯罪，或在提出引渡請求時，尚未服完的刑期至少為6個月。

相關引渡條約亦有提到，簽署條約的雙方均有權拒絕引渡本國國民，如被請求方認為引渡要求針對政治犯罪，或引渡人因種族、性別、宗教、國籍或政治見解而遭起訴或處罰，或該人在司法程序中的地位會因該些原因而受到損害，應當拒絕引渡。

中國與外國簽署並已生效的37項引渡條約，大部分屬亞洲或非洲國家（22國），包括哈薩克斯坦、吉爾吉斯、巴基斯坦、阿富汗、伊朗、韓國、突尼斯、南非、安哥拉、阿爾及利亞等；歐洲及美洲國家佔不足四成（14個國家），如墨西哥、秘魯、白俄羅斯、烏克蘭、立陶宛、阿塞拜疆、西班牙、葡萄牙等，但不包括加拿大、美國、英國、德國。此外，尚有18項已簽署的引渡條約尚未生效。

### **英、美、加僅簽刑事司法協助條約**

廣東省人民檢察院陽光檢務網顯示，譚惠珠提到的加拿大、美國、英國，均有跟中國簽署刑事司法協助條約，不過，根據相關條約內容，刑事司法協助範圍僅涵蓋送達刑事訴訟文書或法庭紀錄、獲取有關人員的證言或或陳述、提供證物等，但並不包括譚所說的押解犯人。以英國為例， [《中華人民共和國和大不列顛及北愛爾蘭聯合王國關於刑事司法協助的條約》](https://web.archive.org/web/20210712024926/http://www.npc.gov.cn/wxzl/gongbao/2015-11/10/content_1951893.htm) 適用範圍為：

**一、雙方應當根據本條約的規定，在刑事偵查、起訴和審判程序中相互提供最廣泛的司法協助，包括限制、凍結、扣押和沒收犯罪所得和犯罪工具。**

**二、協助應當包括：**   
 **（一）送達刑事訴訟文書；**   
 **（二）獲取有關人員的證言或者陳述；**   
 **（三）提供文件、記錄和證據物品；**   
 **（四）獲取和提供鑑定結論；**   
 **（五）查找和辨認人員；**   
 **（六）進行勘驗或者檢查；**   
 **（七）安排有關人員作證或者協助調查；**   
 **（八）進行查詢、搜查、凍結和扣押；**   
 **（九）獲取銀行資料；**   
 **（十）涉及犯罪所得和犯罪工具的協助；**   
 **（十一）通報刑事訴訟結果和提供犯罪記錄；**   
 **（十二）交流法律資料；**   
 **（十三）不違背被請求方法律的其他形式的協助。**

**三、本條約僅適用於雙方之間的相互司法協助。本條約的規定，不賦予任何個人以取得、排除任何證據的權利或者妨礙執行請求的權利。**

**至於德國，截至2018年10月，仍沒有跟中國簽署任何司法協助協定。**

### **陳方安生：譚惠珠言論侮辱港人智慧**

前政務司司長陳方安生周二（4月2日）與立法會法律界議員郭榮鏗、資訊科技界議員莫乃光見記者，總結訪美行程。陳方安生主動批評譚惠珠的言論：「這番說話簡直是侮辱香港人的智慧，正如黃秋生最近說：『當一個人沒有正義感，不能夠辨別是非黑白，就真是枉為人。』」

相關報道： [陳方安生：美國對港政策法已亮紅燈　郭榮鏗：美方恐逃犯條例修訂波及美國人安全](https://web.archive.org/web/20210712024926/https://www.hkcnews.com/article/19541/%E9%99%B3%E6%96%B9%E5%AE%89%E7%94%9F%E8%A8%AA%E7%BE%8E-%E9%83%AD%E6%A6%AE%E9%8F%97-%E7%BE%8E%E4%B8%AD%E7%B6%93%E6%BF%9F%E8%88%87%E5%AE%89%E5%85%A8%E5%AF%A9%E6%9F%A5%E5%A7%94%E5%93%A1%E6%9C%83-19543/%E9%99%B3%E6%96%B9%E5%AE%89%E7%94%9F%EF%BC%9A%E7%BE%8E%E5%9C%8B%E5%B0%8D%E6%B8%AF%E6%94%BF%E7%AD%96%E6%B3%95%E5%B7%B2%E4%BA%AE%E7%B4%85%E7%87%88-%E9%83%AD%E6%A6%AE%E9%8F%97%EF%BC%9A%E7%BE%8E%E6%96%B9%E6%81%90%E9%80%83%E7%8A%AF%E6%A2%9D%E4%BE%8B%E4%BF%AE%E8%A8%82%E6%B3%A2%E5%8F%8A%E7%BE%8E%E5%9C%8B%E4%BA%BA%E5%AE%89%E5%85%A8)

黃秋生不收片酬接拍新導演陳小娟的長篇電影《淪落人》，飾演一名下半身癱瘓的失婚中年漢，並憑該角色獲得香港電影評論學會、香港電影導演會及香港電影編劇家協會頌發的獎項。他接受 [《星島日報》訪問](https://web.archive.org/web/20210712024926/https://hd.stheadline.com/life/ent/realtime/1469718/ ) 時被問他曾因敢言而遭封殺，接連得獎可有吐氣揚眉之感，他回應指，一個人有怨憤才會覺得吐氣揚眉，他不斷提醒自己不要這樣。記者追問，如果能夠返轉頭，他會否做另一個選擇，他明言會做相同的事，「社會都磨不平我的稜角，如果一個人連正義感都沒有，是非黑白都分不到，就真的是枉為人。」
