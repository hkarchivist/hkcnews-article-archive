# 何時可得著太極的好處？

【撰文：陳雄飛】

***作者獲2018年首屆太極拳交流大會一等獎***

很多初學太極的人都會問：「太極可助圓滑人生？ 甚麽年齡學太極方可有此得著？」

其實甚麽時候學太極都不晚，只要開始學了，一切都剛剛好。 在我們的生活中，緊張和疲勞是在所難免，學習輕鬆自在是很有必要。

學習太極，將太極融入生活，輕鬆自在地活，太極自然幫到我們應對生活。太極，虛實變化，隨風擺柳，是有助化解生活壓力。要做到遇事淡定，待人圓整，穩穩的靜靜的鬆鬆的柔柔的，保持在一個好的狀態上 ; 只要穩定下來，才能鞏固下去，要穩定這個模式習慣，才能遇事淡定得住。

將太極融入生活，並非易事，是要不放棄，不貪多，不急成。 學習太極循序漸進，依各招式套路而練習，令我們的生活能夠尊重規律和原則，做到有理有據，遵循條理，並從中得以習慣和發揮，令人從生活壓力中獲得自由釋放，並遊刃有餘應付生活困難。

太極拳是端正大方，中正安舒的，也能令我們生活中有著不偏不倚，堂堂正正，就能有氣勢和精神； 太極是安靜、鬆弛的 ; 在生活中助我們捨去計較和爭鬥，從嘈雜和繁華中逃脫出來，讓身心靈得以呵護和保護、輕鬆自如，從而令我們獲得更多空間和領悟。

太極拳是自然、流暢的 ; 在日常生活中我們不妨多去體驗那份隨意和順勢，融入當中境界，不去強求和硬闖，就會感受到和諧與愉悅； 太極是圓潤、虛實的 ; 進而於生活中在於進、也在於退、欲上還下、欲左還右、轉換有餘。

此實在是會令人平和進而協調，而沒有永恒不變的彊化人生； 太極拳是捨己從人的，讓我們學會依附和借力，揚長避短，捨小我換大我的精神，令習拳者有更充分的自我和更大的價值所在 ; 太極拳著重還原，哪裏來哪裏去，無論氣息還是身位，在生活中使我們學會感恩和回歸，不斷如此循環往覆，周而複始，令人心境平靜和諧。把握初始也融入過程，最後也完善結果， 這樣的生活才會圓滑完整，讓人融入生活的和諧中。
