# 申訴專員：投訴房屋署個案最多　選舉事務處、懲教署首列十大

申訴專員公署今天發表2017至18年度截至3月的的工作報告，公署過去一年接獲4,826宗投訴，與去年的4,862宗相若。選舉事務處及懲教署首次成為十大最多人投訴的政府部門；房屋署代替食環署，成為最多人投訴的機構。公署解釋，2017是「選舉年」，行政長官選舉期間發生的電腦失竊事件，令選舉事務處的投訴數字增加。公署稱，懲教署的投訴數字增加是「好現象」，代表在囚人士能善用申訴專員公署的渠道。

另外，涉及未能取得公開資料的投訴個案達91宗，是歷年最多，其中33宗被投訴的政府部門有失當之處，最常見是部門無理拒絕市民索取資料的要求。公署認為，這顯示政府部門對《公開資料守則》缺乏全面了解。

**最多人投訴的十個政府部門：**


| **機構** | **申訴專員接獲投訴個案 (宗)** |
| --- | --- |
| 房屋署 | 432 |
| 食物環境衛生署 | 323 |
| 地政總署 | 213 |
| 屋宇署 | 210 |
| 康樂及文化事務署 | 134 |
| 運輸署 | 121 |
| 選舉事務處 | 109 |
| 社會福利署 | 78 |
| 懲教署 | 75 |
| 水務署 | 72 |


選舉事務處及懲教署，首次「上榜」成為十大最多人投訴的部門。香港眾志秘書長黃之鋒，早前向懲教署投訴調查組投訴他在囚時的待遇不公，最後被調查組以「被投訴人否認」、「當日進行搜查的職員否認有關申述」為由，裁定投訴不成立。申訴專員公署又能否有效處理涉及懲教處的投訴？

申訴專員劉燕卿表示，因為懲教署屬較密封的環境，加上在囚人士的身份與一般投訴的市民不同，因此公署有特別留意在囚人士的情況。即使沒有投訴個案，公署留意到傳媒報道在囚人士的待遇後，有主動接觸懲教署人員，確保在囚人士能夠自由地投訴，因此，懲教署的投訴數字增加是「好現象」，代表在囚人士能善用申訴專員的渠道。劉燕卿指，公署調查涉及懲教署的投訴案件時是用「窮追猛打」的方式，以增強阻嚇力。公署在調查任何案件時，都會盡量要求投訴人及機構提供證據，以了解哪一方的說法更近乎真相。

房屋署代替食環署，成為過去一年最多人投訴的政府部門，投訴個案較多是有關嘈音及輪候公屋時間。被問到可有人投訴新移民輪候公屋的時間，公署指，並沒有特別多類似投訴。至於涉及食環署的投訴個案，較多是冷氣機滴水問題，公署亦就此展開主動調查。劉燕卿指，機構投訴個案數目多，並不代表其表現最差，要視乎機構的工作性質，房屋署要管理多個屋邨，接獲投訴個案多可被理解。

申訴專員劉燕卿表示，更重要的是，機構被投訴後，有否就公署提出的建議作出改善。公署的數據顯示，接近九成的建議都有被政府部門接納，接近六成則已落實，公署有一直跟進政府部門的改善進度，若部門持續不改善，公署會考慮去信行政長官辦公室。

另外，涉及公開資料的投訴個案達91宗，是歷年最多，其中33宗 （約佔36%）發現所涉及的政府部門有失當之處，其中一個主要問題，是部門無理拒絕市民索取資料要求。公署指，數字反映出市民對開明問責的期望上升，但政府部門對《公開資料資料守則》的精神及原則缺乏全面了解。公署有鼓勵部門向員工安排相關培訓，亦會繼續留意政府有關公開資料立法工作的進展。

公署過去一年完成了12項主動調查，是歷年來最多。主動調查的題目是依據傳媒的報道、立法會議員及市民的意見而定。公署希望透過主動調查，可以探究個案的問題根源，以制定長遠的改善措施。

**已完成的12項主動調查報告：**

- 香港機場管理局簽發機場禁區通行證的機制

- 政府對工廠食堂的規管

- 房屋署及水務署就公共屋邨公共地方和空置單位的水費交收安排

- 食物環境衞生署發出「食物安全命令」後公布商號名單與否的決定準則

- 康樂及文化事務署採購和註銷圖書館資料的準則和程序

- 地政總署對某違規村屋的管制行動

- 運輸署對長期被圍封及閒置道路的處理

- 食物環境衞生署對進口蔬果安全的監控制度

- 社會福利署為有或懷疑有精神健康問題的人士其家人/照顧者及鄰居所提供的支援服務

- 政府對私人土地傾倒建築廢物及堆填活動的規管

- 食物及衞生局及衞生署處理違例吸煙的機制

- 水務署對政府水管的修護和危機處理

申訴專員劉燕卿於今日宣佈，就康文署公眾游泳池的泳線分配制度是否恰當展開主動調查。早前有報道指，體育總會分配泳線給其屬會的機制不公，扼殺小型泳會的發展空間，當中更有懷疑涉及利益輸送和違反租用條款等違規問題。她指，泳線屬公共資源，康文署應確保泳線能被善用。她歡迎市民9月17日或之前就此調查以書面向申訴專員公署提供意見。

在處理投訴方面，過去一年有237宗投訴個案是以調解方式進行，是歷年來最多，比去年有近八成的增幅。參與調解的政府部門及機構共28個，較去年增加6個。案件進入調解工序前，首先要由公署檢視個案的性質，不嚴重的個案，例如是部門及機構延誤、沒有採取行動等，才可以以調解方式處理；之後，投訴人及機構雙方均需願意調解；最後，由公署作為中間人，為雙方協調調解的過程。公署指，調解可以減少公署的調查工作，以分配更多公署人手協助主動調查，部門及機構亦可減卻撰寫報告的行政工作，令處理投訴的過程更快速，同時令投訴人及機構都得到彼此滿意的結果。

劉燕卿將於明年3月底約滿後離任申訴專員一職，原因是希望投放更多時間在家庭。行政長官林鄭月娥指，下任申訴專員將會由特設的 [遴選委員會](https://web.archive.org/web/20200919220501/https://www.info.gov.hk/gia/general/201806/19/P2018061900811.htm) 以公開形式招聘，預計將於8月刊登招聘廣告。
