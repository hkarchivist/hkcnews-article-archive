# 【共享隨筆】 官字兩把口，說話有兩手

這篇文是筆者近月最難寫的一篇文章，難寫的原因不是因為找不到角度去評論《旅館業條例》修例，而是因為開放短期民宿市場的好處明明已放在眼前，但不知怎地，這些原因都會被視而不見。

『我們需要創造一個有利創科發展的環境，一個對創新更友善的政府。我們的政府應該修改過時條例從而促進科技應用。』（原文）We need to create an environment which is conducive to the development of innovation and technology in Hong Kong. We need to have an environment that the government is more friendly in updating some obselete legislation to facilitate the application of technology. 兩周前，特首出席年度創科界旗艦會議 - RISE Conference 如是說道。

  
 說話言猶在耳，幾天後，民政事務局局長在立會首讀旅館業條例，收緊條例。常言道『官字兩把口，說話有兩手』我想這是我能想到對這個情況最合適的形容。

對此，共享經濟聯盟甚為失望。失望不是因為政府仍抱殘守闕，失望是因為，若然條例通過，香港又要在創新科技之路上走很多冤枉路。

作為緊貼本港共享經濟發展的一員，看到這幾年間不同共享經濟平台已由原本打著破壞性創新變為共融。在不同場都向政府表達願意配合政府倡議的智慧城市發展，每一次都希望能與政府正面溝通，共建合理短期住宿框架，但回頭卻被一招殺死，最耐人尋味是衝著旅遊平台執法的修例竟然可以說成不是針對網上平台的措施。

開放短期民宿市場對香港的影響，在以前的文章和今年5月發佈的白皮書內已多次提及，提高旅館住宿供應彈性，彌補業界人手不足，刺激旅遊業再發展，這等的都是開放短期民宿市場對香港的影響。而近日亦再有民調指出77%港人認為民宿有助增強本港旅遊業，可供外來訪港旅容多一個選擇。

說到底，短期民宿平台的興起不只是一間公司的作為，而是現今旅客對旅遊需求根本改變的體驗。比起傳統旅遊模式，現今旅客追求的更是與當地人交流的深度旅行。

而短期民宿正正是能提供到更多元化的旅遊體驗給全球旅客，這才是短期民宿平台能在整短時間內擴張全球的原因。

而事實是，短期民宿平台並不是唯一看到這趨勢的人，就連酒店業本身也開始留意旅遊模式的改變而加入短期民宿市場。萬豪國際酒店集團在英國就與短租住宿公司Hostmaker合作進攻短租民宿市場，法國雅高酒店集團擁有短租平台Onefinestay，凱悅酒店集團亦有投資初創短租平台Oasis。

可惜的是，共享企業知，酒店業知，政府仍是選擇擰轉頭，不正視趨勢轉變，繼續收緊共享經濟企業的生存空間。

如果條例獲三讀通過，將會令平台上房源大比數下跌，旅客不能再入住已經是少事，消息放出了，又給一個理由各不同機構評選香港落後鄰近地方創新發展。香港落後，是大家都不會想看到的情況。
