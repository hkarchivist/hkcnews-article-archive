# 假如今天左禍重臨，香港怎麼辦？──紀念67暴動50週年的現實意義（下）

紀念67暴動50週年的第三個現實意義，是藉此反思一下，假如當年港英失敗了，被左派鬥垮了，香港今天會是怎麼樣？更重要的問題是：假如今天大陸左禍重臨，香港怎麼辦？

歷史沒有如果。假如當年港英被鬥垮了，今天的香港會是一個怎麼樣的香港？相信沒有人能給出一個確切的答案。但是，有一位當年暴動的積極參與者（他也曾因此而入獄）對我說：「我很慶幸當年我沒有成功，否則我就永遠對不起香港人，我會一輩子感到遺憾」。從他這句話可以看出，他自己心中也知道，如果當年左派勝利了，則香港將不可能有今天的政治、經濟、社會各方面的成就。澳門在「12.3」事件之後的發展就是一面鏡子。

#### 擺在我們面前更重要的問題是：假如今天大陸左禍重臨，香港怎麼辦？

首先，這是一個很實際的問題，因為今天中國大陸文革的思潮有迴流之勢。自從薄熙來在重慶大搞「唱紅打黑」之後，文革的陰魂迅速傳遍神州大地，令很多學者驚呼「亞文革」的來臨。最早提出這個趨勢的是 [維權律師呂良彪（阿呆）](https://web.archive.org/web/20210712004830/http://www.open.com.hk/content.php?id=2285#.WQNVS1WGOpo) ，以至前總理溫家寶在其任內最後一次記者招待會上憂心忡忡地說：

「粉碎『四人幫』以後，我們黨作出了關於建國以來若干歷史問題的決議，實行了改革開放。但是『文革』錯誤的遺毒和封建的影響並沒有完全清除。隨著經濟的發展，又產生了分配不公、誠信缺失、貪污腐敗等問題。我深知解決這些問題，不僅要進行經濟體制改革，而且要進行政治體制改革，特別是黨和國家領導制度的改革。現在改革到了攻堅階段，沒有政治體制改革的成功，經濟體制改革不可能進行到底，已經取得的改革和建設成果還有可能得而復失，社會上新產生的問題也不能從根本上得到解決，『文化大革命』這樣的歷史悲劇還有可能重新發生。每個有責任的黨員和領導幹部都應該有緊迫感。」（見 [《溫家寶總理答中外記者問》(全文)](https://web.archive.org/web/20210712004830/http://www.gov.cn/2012lh/zhibo/zhibo_20120314zldw.htm) ，2012年3月14日）

這種文革回潮的可能性，隨著習近平出任總書記後日漸加劇，最能客觀反映這個現象的，是在官方《人民日報》上「共產主義」這個詞的使用頻率日漸上升。根據香港大學新聞與傳媒中心的錢鋼老師的數據化研究，這個詞在過去30年間出現明顯的變化：

是什麼原因導致 「共產主義」這個詞在2013年以後日漸多地出現在公共話語領域？相信這同習近平上台以來採取的種種政策有關，筆者嘗試把他上台以來的一些「向左轉」的重要言行表列如下：


| **習近平的言/行/政策** |  | **造成「亞文革」的原因** |
| --- | --- | --- |
| **2013年1月** |  |  |
| 提出「兩個三十年互不否定」的說法 |  | 實質上改變了中共1981年《關於建國以來黨的若干歷史問題的決議》的精神，使文革不再被否定 |
| **2013年4月** |  |  |
| 發出《中共中央9號文件》，列舉了7個方面的議題不准討論，包括：憲制民主、普世價值、公民社會、自由主義、新聞自由、權貴資本主義、黨的歷史錯誤。 |  | 這些「禁議」範圍，實際上就是使中國回復到「文革」時代全國只有一個腦袋的日子。 |
| **2013年8月19日** |  |  |
| 習近平在全國宣傳工作會議上發表講話，使全國進入「亞文革」狀態的一篇重要講話。他在講話中把意識形態工作提升到與經濟工作同等重要的位置。他說：「經濟建設是黨的中心工作，意識形態工作是黨的一項極端重要的工作」。 |  | 這就是說，意識形態已經恢復為一項與經濟建設同等重要的工作，或者是「雙中心」。這是對鄧小平「一個中心」主張的最大的修正，直接促使文革時期十分強調的「階級鬥爭」理論復活。 |
| **2013-2016** |  |  |
| 習近平重新構建中央高度集權的領導體制，改變了黨章關於集體領導的規定。習上台後，出任10多個最高領導小組的負責人，計有：總書記、國家主席、軍委主席、國安委主席、深化改革小組組長、財經小組組長、網絡安全小組組長、對台小組組長、外事小組組長、軍改小組組長。 |  | 為避免重蹈文革覆轍，中共黨章第二章第十條規定：「（五）黨的各級委員會實行集體領導和個人分工負責相結合的制度」。對照黨章，習近平的做法顯然是違背了黨章有關「集體領導」的規定。 |
| 2016年1月7日 |  |  |
| 習近平重新構建一個 「黨政不分、以黨代政」的領導架構。2016年1月7日在中共中央政治局首次會議上，習近平強調堅持黨中央集中統一領導，他說：「黨政軍民學，東西南北中，黨是領導一切的」。為了貫徹這點，習近平前所未有的召集黨政六套班子集體向習近平彙報工作的做法以體現這一原則。 |  | 這種六套班子向一個人彙報工作的情況只有毛澤東時代才出現過，毛澤東以後三十多年再未有過。這是回到「文革」時代「黨政不分、以黨代政」的領導架構。針對「文革」的災難，鄧小平在1980年發表了 [《黨和國家領導制度的改革》](https://web.archive.org/web/20210712004830/https://weblearn.ox.ac.uk/access/content/group/506bb161-3b6d-4880-8707-cc78aeecfb03/%E5%85%9A%E5%92%8C%E5%9B%BD%E5%AE%B6%E9%A2%86%E5%AF%BC%E5%88%B6%E5%BA%A6%E7%9A%84%E6%94%B9%E9%9D%A9.pdf) 一文，嚴厲批判了這種領導模式。現在又恢復過來。 |
| 2016年1月29日 |  |  |
| 習近平提出「核心」觀點，要構建「習核心」，他在中共中央政治局會議上說，每個黨員要有「政治意識、大局意識、核心意識、看齊意識」這「四大意識」。這是習近平親自要封自己做「核心」的最明顯證據。 |  | 中共黨章第二章第十條規定：「（六）黨禁止任何形式的個人崇拜」。對照黨章，習近平的做法顯然是違背了黨章有關「禁止個人崇拜」的規定。 |
| **2016年1月1日** |  |  |
| 習近平頒佈《中國共產黨紀律處分條例》，其中第46條第2款提出禁止黨員「妄議中央大政方針，破壞黨的集中統一」。 |  | 「文革」的一個特點就是「黨內鴉雀無聲、黨外輿論一律」。習近平上台後，也開始朝這個方向走。禁止「妄議中央」，將在黨內營造鴉雀無聲的局面。 |
| **2016年2月20日** |  |  |
| 習近平在「黨的新聞輿論工作座談會」上講話，強調「黨和政府主辦的媒體是黨和政府的宣傳陣地，必須姓黨」、「體現黨的意志、反映黨的主張，做到愛黨、護黨、為黨」。 |  | 文革的另一個特點是「黨外輿論一律」，媒體必須姓黨，實際上就是要營造一個「輿論一律」的環境。 |

凡此種種措施，使「文革」50年後，神州大地又再現「文革」時代的氛圍，這些「文革」元素的回潮，使人對中國前途不敢樂觀。

香港的「67暴動」，既然是中國大陸「文革」的延伸，如今大陸出現「亞文革」，其禍害會不會又延伸到香港？筆者認為，大陸「文革」回潮，再影響蔓延到香港的可能性是不能排除的。事實上，在過去幾年我們在香港已經開始感受到一些不尋常的情況，例如：

─出現「鼓動群眾鬥群眾」的事情，例如「愛」字頭組織對不同政見團體的肢體衝突、滋擾及破壞別人正在舉辦的活動。「鼓動群眾鬥群眾」是「文革」期間一個很普遍的行為，香港以前很少見到，今天卻不幸在香港大量地出現。

─出現「文革式大批判」的事件，例如《文匯》、《大公》兩報對陳文敏的攻擊（一年之內兩報發表超過400篇文章密集地批判他），導致香港大學校委會改變決定不再委任他為大學副校長；

─出現衝擊學術研討會的事件，例如對中文大學文革50週年的衝擊，使一些院校不願意再公開借出場地以供高政治敏感度的學術活動，有些院校則保密地把活動悄悄放在校外舉行；

─出現內地「群眾性運動」封殺不同政見的藝員、歌星事件，迫使主辦單位取消邀請或者中止合約；

─出現網民群體對台灣網絡集體出擊的「帝吧出征」事件，充分暴露了紅衛兵式的愚昧行為；

─出現內地支持的團體衝擊本港合法組織的事件，例如對法輪功的挑釁衝擊。

這些事情，在過去五年間頻頻發生，可以說是內地「亞文革」延伸香港的跡象。

50年前文革延伸香港時，有港英以一個國家政權的力量強勢頂住，直到中共下令香港土共收手為止，這就保護了香港免受文革蹂躪。今天，假如「亞文革」波及香港，特區政府能否像當年港英一樣非常強勢地抗衡這股歪風？對此筆者是悲觀的，因為作為一個權力來源來自北京，而在日常的運作中又備受中聯辦節制的特區政府，是無法抵擋來自大陸的任何負面影響的。所以，假如今天發生「亞文革」延伸香港，唯有靠全體市民團結一致去抗衡這些惡勢力。

怎樣去抵制、抗衡「亞文革」襲港？筆者認為最有效的辦法是強化、壯大香港的公民社會的力量，只有當每一個人都有健康的公民意識時，才能防禦「亞文革」的侵犯。而要培育強大、健康的公民意識，必須從自己做起。

筆者從建制派身上的種種醜態看到要有健康的公民社會，每個人都需要：

● **時刻警惕「統戰」，才能保持知識分子「獨立之精神、自由之思想」；不媚權，不媚俗的風骨；所以每一個人都要充分瞭解「統戰」對人性的腐蝕作用。**

● **時刻警惕不要遺忘歷史，只有牢記中共建國60多年來的歷史，才能看到一個強大的公民社會對防止國家走向災難的重要性。**

● **時刻警惕政治冷漠，因為今天我們如果對錯誤的政策不發聲，它就會持續下去，到了尾大不掉的地步時，已經造成很大的迫害。**

● **時刻警惕「平庸之惡」，所謂「平庸之惡」，就是不加思考地執行一些明顯錯誤的政策。「平庸之惡」的實質是起到助紂為虐的禍害。**

● **時刻警惕滑向邪惡。政治心理學家發現，一個人走上邪惡之路，往往是從「不假思索」地走上錯誤的第一步開始，有了第一步，就會不斷地向下滑行，終於自己成為邪惡的一部分。**

（以上這五點，都是筆者基於非常深刻的感受總結出來的，篇幅關係無法在這裡展開來談，他日有機會可以專門論述）。

只有建構一個強大的公民社會，我們才能夠抵擋住「亞文革」的來襲。這是紀念「67暴動」50週年的最大現實意義。
