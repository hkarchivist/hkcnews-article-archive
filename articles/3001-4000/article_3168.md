# 特朗普射飛彈

習總訪美，兩大強國頭目碰面，本來外界關注都放在這，想不到美國突然向敘利亞政府軍發射近60枚導彈，習特會的風頭立即被蓋過。言歸正傳，特朗普在上台前發出過很多減少干涉外國事務和親俄的信息，很多評論包括筆者都提及過，現在卻向俄羅斯的盟友敘利亞動武，還說該國總統巴沙爾冇得留低 (事實上直至幾日前美國外交官還在說無意推倒巴沙爾)，可能顯示隨著他走入建制，外交政策終將走回較傳統的共和黨總統的對外強硬路線。

是甚麼令他的外交政策有變呢？他在見記者時提及見到無辜婦孺受害的照片，感到很痛心。敘利亞內戰無辜婦孺受害其實不是新事，但我們也不可否定，他當上了總統後比較留心看國際新聞。又可能，眾所周知，他很聽他女兒伊萬卡的話，說不定伊萬卡看了新聞後覺得小孩子死亡的照片很痛心向爸爸告狀。

個人還認為早兩日宣布的國安委 (national security council) 人事變動也可能有關，即被視為特朗普智囊的白宮首席策略師 (White House Chief Strategist) 班農被降職一事。班農和原本獲邀出任國安顧問 (national security advisor) 的前中將弗林，被視為親俄派和獲得特朗普寵信。但弗林因事被炒，班農又降職，特朗普身邊的國防外交團隊，相信會變成國防部長馬蒂斯 (前上將) 和新國安顧問麥馬斯特 (現役中將) 有較大影響力，而兩人都是反俄派。

從大圍環境看，昨日北韓又射飛彈，今次特朗普主動出擊也可能是顯國威兼為自己造勢的一招，想對北韓、伊朗、甚至是來訪的習總作出震懾：我特朗普在外交上不會放軟手腳架。特別是，以往奧巴馬說過動用化武是「紅線」，但當敘利亞政府真的用化武時卻沒行動，講得出做不到，現在特朗普強硬得多。對內這也有助鞏固他在美國建制派內的形象，和減輕歐洲和日韓等盟友對於他親俄或孤立主義的憂慮。我們接下來需要留意美國國內不同政要、歐洲、俄羅斯、當然還有習總的反應，以及習特會如何收場。
