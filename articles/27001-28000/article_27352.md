# 口罩質素參差　囤貨抬價成風　消委會：若屢勸不改　會公開點名譴責商戶

眾新聞早前訪問到數名外判清潔工，一星期只獲發兩片「薄到透光」口罩。究竟這些薄口罩是否有用？能否有效阻隔到病毒？醫學會會長何仲平表示，普通市民分辨口罩是否有用，主要視乎口罩是否有三層，外層是否防水，中層是否不織布。至於薄口罩，只要符合以上條件其實也有用，不過可能會較容易濕，要更頻密地替換。

至於市面上的口罩質素參差、價格十分懸殊之外，有商店更懷疑囤貨，損害消費者權益。消費者委員會回覆查詢時指，截至2月20日，總共收到218宗有關口罩的投訴，消委會表示，一直與執法機關緊密聯繫，透過訊息通報和相關部門的執法，直接以法例遏止歪風。而在處理投訴個案時，消委會將警告有關不良營商的商戶，若屢勸不改，經過詳細的審視程序，消委會會行使既定的點名機制，公開點名譴責有關商戶。

眾新聞取得外判清潔工獲發的「薄到透光」口罩，另外有讀者報料指長沙灣一間雜貨店出售一款土耳其製口罩，十分薄，效用令人質疑。記者在2月14日到訪時發現該款土耳其口罩只賣剩三、四盒，每盒50個，售$220，店員說較早前賣每盒售$250，已賣了超過幾百盒，暫時不會再入貨，因物流方面有困難。

該款土耳其口罩包裝上大部分是土耳其文，只有2層設計，並非獨立包裝，標榜可阻隔99%細菌，記者之後搜尋包裝盒上的 [生產商官網](https://web.archive.org/web/20210712040401/http://drprufer.com/urunler) ，發現官網未有出售該款口罩，但一個 [土耳其購物網](https://web.archive.org/web/20210712040401/https://www.estoptan.com/dr-prufer-cift-katli-lastikli-maske-pmu691) 上，卻有出售同款式口罩。記者以電郵向生產商查詢，不獲回覆。

另外記者發現佐敦一間特賣場有出售印尼製粉紅色的口罩，同樣只得2層設計，每盒50個，售$150，而50個口罩分開兩份，並由兩個透明膠袋包裝，包裝盒上沒有標明可過濾細菌效率（BFE）等資料，只是寫上「To protect body & respiratory system from bacteria virus, dust, micro particle, and bad odour.」（保護身體和呼吸系統免受細菌、病毒、灰塵，微粒和惡臭的侵害）。

記者將外判清潔工的「薄」口罩，連同上述兩款口罩以及網民熱議的仿日中國製「救救」口罩，向醫生查詢，測試4款口罩是否合規格。醫學會會長何仲平接受訪問時表示，普通市民分辨口罩是否有用，主要視乎口罩是否有三層，外層是否防水，中層是否不縮布，而內層應該是吸水層，以吸收佩戴者呼出的濕氣。

在防水測試方面，4款均有基本的防水能力，何仲平認為合格。但三層設計方面，土耳其製口罩和印尼製口罩只得兩層，何仲平認為沒有過濾層，效果會較差，其餘兩款則合格。何仲平解釋，病毒主要靠飛沫傳播，防水層可以阻隔水珠，以防止病毒進入，若真的穿過防水外層，中間的不織布帶有靜電，可以吸附和凝結水珠。他補充指，過濾層成功阻隔病毒的百分率需要靠科學儀器，普通市民難以測試。

至於口罩透薄的問題，何仲平認為，只要符合以上條件其實也有用，不過較薄的口罩可能較容易弄濕，需要更頻密地替換。

眾新聞向消委會查詢，消委會回覆指，今年截至2月20日共接獲218宗到有關口罩之投訴，而過去3年分別只接獲2至7宗，投訴分類如下：


| 銷售手法 | 60 |
| --- | --- |
| 缺貨 / 送貨延誤 | 55 |
| 價格 / 收費爭拗 | 38 |
| 貨品質素 | 37 |
| 懷疑假貨 | 12 |
| 其他 | 16 |
| 總計 | 218 |


消委會稱，在這段期間多次派員到各區視察口罩等防疫用品的供應和價格，其中在上水28間藥房，發現售賣口罩的價格十分懸殊。已促請有關商戶不要趁火打劫。

消委會亦曾派員初步視察防疫產品的售價，發現部份小型的藥妝店、藥房、或以「快閃」形式出現的不知名店舖，防疫及衛生產品售價確實存在「海鮮價」，甚至大幅抬價，例如口罩更被「拆散」出售，難以辨認其貨源和品牌真偽，情況極不理想。

消委會指出，雖未有法定權力搜集數據及證據，但一直與執法機關緊密聯繫，透過訊息通報和相關部門的執法，直接以法例遏止歪風。消委會表示，將繼續密切留意有關商戶的銷售手法，同時再三促請商戶應竭盡其社會責任，致力採購及供應有質素的口罩之餘，也要保持以合理和可負擔的價格出售。在處理投訴個案時，消委會將警告有關不良營商的商戶，若屢勸不改，經過詳細的審視程序，消委會會行使既定的點名機制，公開點名譴責有關商戶。

眾新聞再向海關查詢，近期巡查中有否發現任何與 《商品說明條例》及《消費品安全條例》不符的口罩在市面上出售，並作出檢控，海關在2月17日發新聞稿稱，截至2月16日為止，已就市面上出售的外科口罩試購超過80個樣本。當中40個樣本的測試合符含菌量標準，未有發現違反《消費品安全條例》的情況出現。
