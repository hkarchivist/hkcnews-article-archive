# 【財政預算案】赤字$1391億卻派錢　陳茂波甚麼理財哲學？　學者：政治掛帥

財政司司長陳茂波昨發表新一份 [財政預算案](https://web.archive.org/web/20201021224553/https://www.budget.gov.hk/2020/chi/speech.html) ，他首次向全民派錢。凡於明年3月31日或之前年滿18歲的香港永久居民，可領取1萬元，預計今年7月初接受登記，目標在暑假期間陸續發放款項，或以長者優先。

香港民意研究所名譽總監鍾劍華認為，派錢是考慮到9月的立法會選舉，冀為建制派製造有利氣氛，「政府整體都係想建制派多一、兩個『成功爭取』嘅項目。」中大政治與行政學系副教授馬嶽指，即使政府有這盤算，但根據派4,000元的失敗經驗，今次是否能夠在選區前發放款額成疑，政府有機會弄巧成拙。馬嶽也認為，財政預算案予人感覺是「政治掛帥」。

香港過往連年庫房「水浸」，陳茂波一直沒有應民間要求派錢，強調「 [政府資源應該要針對性地使用在有需要的人身上](https://web.archive.org/web/20201021224553/https://www.info.gov.hk/gia/general/202001/18/P2020011800514.htm ) 」、「 [資源用得比較準確點會好些](https://web.archive.org/web/20201021224553/https://www.info.gov.hk/gia/general/201804/21/P2018042100446.htm) 」、「 [公帑是要善用，是要用得其所，是要用在一些我們（港府）認為聚焦、有針對性的一些政策目標](https://web.archive.org/web/20201021224553/https://www.info.gov.hk/gia/general/201803/05/P2018030500494.htm) 」，並形容 [「一刀切」全民派錢並「不合適」](https://web.archive.org/web/20201021224553/https://www.info.gov.hk/gia/general/201803/05/P2018030500494.htm) 。

本年度政府庫房15年來首次「見紅」，約有378億元赤字。陳茂波在新一份 [財政預算案](https://web.archive.org/web/20201021224553/https://www.budget.gov.hk/2020/chi/speech.html) 預測，未來5年赤字持續，估計下年度赤字更高達1,391億元，創歷史新高。在目前巨大赤字壓力下，陳茂波一反過往的「理財哲學」，決定向每名成年香港永久居民大手派1萬元，開支涉及約710億元，連同其他一次性紓緩措施，構成下年度赤字的主要部份、涉及超過1,200億元。他解釋，這是 [針對目前「特殊情況」而推行的「特殊措施」](https://web.archive.org/web/20201021224553/https://www.hkcnews.com/article/27276/%E8%B2%A1%E6%94%BF%E9%A0%90%E7%AE%97%E6%A1%882020-%E5%85%A8%E6%B0%91%E6%B4%BE%E9%8C%A21%E8%90%AC-27276/%E5%85%A8%E6%B0%91%E6%B4%BE%E9%8C%A21%E8%90%AC) ，不會對財政造成長遠負擔，認為在經濟下行而財政儲備仍然充裕的時候，可以在公共開支方面加大力度，以刺激經濟，與市民共渡時艱。他回應傳媒提問時指，全民派1萬元是他與特首林鄭月娥商量後得出，屬二人的共識。

中大政治與行政學系副教授馬嶽認為，今次財政預算案予人的感覺是「政治掛帥」，不能再以公共財政原則、哲學來討論，「因為好明顯同前一、兩年嘅原則好唔一樣，好多盈餘嘅時候唔派，反而赤字時派，咁個原則係乜嘢？係咪之後都係咁嘅原則？」他續指，一般理解財政預算案是落實施政報告或其他政府政策，「好得意，上星期先派完300億（防疫抗疫基金），今個星期又派。好多嘢理論上可以喺財政預算案度做，但唔係，而係各有各搞，令人感覺成個政府有啲亂嚟，變咗好難再講有乜理財哲學等，可能純粹係面對當前危機佢哋諗到嘅方法。」

### 鍾劍華：想建制派多一兩個「成功爭取」項目   
 馬嶽：派錢唔代表啲人會感激你

全民派錢計劃一公布，建制派第一大黨民建聯隨即在 [Facebook](https://web.archive.org/web/20201021224553/https://www.facebook.com/dab.hk.1992/photos/a.1567308223509702/2529233263983855/?type=3&theater) 自居「得咗啦」、「#成功爭取」。香港民意研究所名譽總監鍾劍華分析，香港及至中央政府在各項部署上，都考慮到9月的立法會選舉，冀為建制派製造較有利的氣氛，「肯定係top agenda」，「政府整體都係想建制派多一、兩個『成功爭取』嘅項目」，而派錢的安排亦是「有諗過」。鍾解釋，如果純粹從刺激消費的角度，派消費券效果更佳，因為政府發放現金，有市民會選擇儲起，刺激消費的效能打了折扣；不過，派消費券的行政手續繁複，估計需時較長，「用（全民派錢）呢個方法，我相信都有選舉考慮。」

馬嶽表示，陳茂波或有相關盤算，但他評估派錢對選舉的影響輕微。「大家本來有個期望，你唔派可能就鬧你；你派呢，唔代表啲人會感激你。」馬嶽質疑陳茂波派錢的時間表，「你派到就話啫，4,000蚊都派咗成年喇。佢可能預計8月（發放款項），但派唔到嘅話呢，咁可能調番轉頭，仲弄巧成拙。經過派4,000蚊經驗，佢係咪真係派得到、（時間）好準確，大家都好懷疑。」

政府去年透過關愛共享計劃，派發最多4,000元現金津貼，去年2月1日起接受申請，惟手續繁複，而負責審批的在職家庭津貼辦事處更一度表示， [有約1.2萬份已輸入初步資料的申請表，或因夾雜在其他已處理的申請表中，暫時未能尋回](https://web.archive.org/web/20201021224553/https://hk.appledaily.com/local/20191002/TBJAOBKQ4VMHBEFN2MH3JA25U4/) 。截至去年11月，仍有 [約3萬申請人未獲發放款項](https://web.archive.org/web/20201021224553/https://hd.stheadline.com/news/realtime/hk/1631812/%E5%8D%B3%E6%99%82-%E6%B8%AF%E8%81%9E-%E7%8D%A8%E5%AE%B6-%E4%B8%89%E8%90%AC%E4%BA%BA%E6%9C%AA%E6%94%B64000%E5%85%83-95-%E7%84%A1%E4%BA%A4%E8%BA%AB%E5%88%86%E8%AD%89%E5%89%AF%E6%9C%AC) 。

陳茂波過去一直被傳是特首「黑馬」，有機會接替林鄭。他今次大灑710億元，能否催谷民望呢？馬嶽回應指，不認為派錢對陳茂波的仕途有很大幫助，林鄭的接任人選主要視乎中央的盤算，官員本身的民望為次要，「佢（陳茂波）自己可能咁樣諗（派錢有利個人仕途）啦，冇辦法估。但宏觀少少睇，現班子整體民望都好低，只係程度問題。」馬嶽表示，無從得知中央揀人的標準，陳茂波雖為現屆班子其中一名高官，但其民望從來不高，「如果中央揀到冇得揀，但你喺原班子度揀，冇一個人嘅starting point係高嘅。」馬嶽續分析，如果林鄭的接任人在政治上沒有大改變，繼續跟隨中央路線，民望亦會很快走下坡。
