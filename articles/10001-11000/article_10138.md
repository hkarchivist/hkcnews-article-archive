# 入境處45萬元包港龍機遣返20名越南籍人蛇　人均開支逾兩萬元

入境處處長曾國衞周二率領9名副處長及助理處長，主持2017年工作回顧新聞發布會。曾國衞講述入境處今年的「重點工作」時提到，處方在最近3個月，兩次採用新設的包機形式遣返非法入境者，最近一次是本周一（2月12日）。

眾新聞查詢後得悉，入境處周一的行動，涉及45萬元公帑開支，安排包機將20名越南籍非法入境者遣返越南，行動中有40名入境處人員隨機押送。換言之，入境處用於每名被遣返人士的開支高達2.25萬元。

入境處去年12月首度採用包機，將自願被遣返的越南籍非法入境者送回國，當時一次過遣返68人，人均開支不到5,000元，不及今次行動的四分之一。入境處上次採用越南航空包機，並沒有入境處人員隨機押送。

據悉，入境處周一行動採用的包機屬國泰港龍航空。對於為何今次遣返20人要用包機，而且要特別安排入境處人員隨機押送，曾國衞解釋：「我哋每次安排都視乎實際情況，今次包機安排，有部分有啲……以前有啲刑事紀錄，甚至乎有啲有暴力嘅情況，所以我哋今次係多咗同事去管理。否則的話，我哋最主要考慮，一方面係顧及到佢本人個人身安全，亦都係顧及到我哋個行動需要，同埋我哋伙記嘅人身安全。」至於被遣返者涉及什麼刑事紀錄，曾國衞回應指：「呢個唔係好方便講。」眾新聞其後再向入境處查詢，被遣返者中有幾多人有刑事紀錄、去年68名被遣返的非法入境者是否有人有刑事紀錄，未獲處方回覆。

**入境處兩次安排包機遣返越南籍非法入境者詳情**


|  | **2017年12月28日行動** | **2018年2月12日行動** |
| --- | --- | --- |
| **被遣返者人數** | **68（30男、38女）** | **20（16男、4女）** |
| 被遣返者年齡分布 | 3名兩歲以下幼童，其餘主要介乎18至64歲 | 介乎21至56歲 |
| 自願／強制遣返 | 68人均自願被遣返 | 部分為強制遣返 （入境處未有提供具體自願及強制遣返人數） |
| 被遣返者是有否刑事紀錄 | 入境處沒有提供資料 | 部分人有刑事紀錄 |
| 隨機押送的入境處人員人數 | 0 | 40 |
| 提供包機的航空公司 | 越南航空 | 國泰港龍航空 |
| **行動開支** | **33萬元** | **45萬元** |
| **人均開支** | **4,853元** | **2.25萬元** |


過去一年被遣送離境的非華裔人士有4,139人，較前年的2,922人按年增逾40%。目前仍有約1,800人尚待遣返，他們主要來自印度、巴基斯坦、孟加拉等國家。入境處處長曾國衞介紹今年重點工作時表示：「我哋會繼續按實際情況，利用切實可行嘅方法，執行有效率嘅遣返工作，務求喺最短時間之內，將免遣返聲請不獲確立嘅人士，遣送離境。」他指，其中一個方法，是採用包機遣返。

入境處去年12月28日首次以包機，一次過將68名越南籍非法入境者遣離香港，行動開支約為33萬元，由越南航空提供包機。翻查報道，上次包機遣返行動中，入境處毋須派員隨機「護送」68名屬自願遣返的人士回到原居地，而是由航空公司加派較多男性艙務員在航班上執勤，確保航班和執行遣送的安全。

非法入境、逾期逗留、免遣返聲請不獲確立的人士，最終都是由入境處負責將他們遣返回原居地。據悉，入境處採用包機方式之前，以往多年一般會為自願被遣返者個別買機票，處方不會派員陪同上機，成本較低。根據 [保安局呈交立法會的資料](https://web.archive.org/web/20210712013842/https://www.info.gov.hk/gia/general/201802/07/P2018020700716.htm) ，須被遣返的人如持有有效旅遊證件，入境處會為他們安排航班回國。資料指，入境處可以為被遣返者個別購買機票，但基於市場情況或航空公司所施加的限制，入境處能就每個航班購得的機票往往有限。入境處亦可以商用包機航班一次過遣返較大批人，在某些情況下，包機形式能確保遣返行動得以盡早完成，但保安局未有明確指出有哪些情況。

至於須被強制遣返者，即拒絕接受遣返安排、甚至以激烈手法拒絕離境的非法入境者，入境處會與相關領事館人員及有關航空公司聯絡，派員乘搭同一航機以遣送有關人士離開香港。入境處早前交代首次包機安排時向傳媒透露，去年曾進行7次非包機的強制遣返行動，每次即使只是遣返1人，也要4至6名入境處人員押送回國，以確保安全。入境處助理處長（執法）馮伯豪解釋：「不自願遣返者，會在登機前或於機上激烈掙扎和叫囂，機長會因安全理由拒絕讓他們上機。」以往入境處採用非包機遣返的做法，未見有通報發生重大意外事故。

眾新聞搜尋越南航空票價，下月非公眾假期由香港飛往胡志明市的經濟客位單程機票，不超過1,500元；前往首都河內價錢相若。如果用非包機的方法將自願被遣返的68人送回越南，估計用於購買機票的開支約為10.2萬元，較去年包機花費33萬元，節省22.8萬元。

不過，個別買機票遣返的做法，因為機位安排、安全等理由，未必能一次過將整批被遣返者送回原居地，可能需要分開多次進行。若要分批遣返68人，部分遲回國者留在香港的生活開支，亦有機會要動用公帑。如果非法入境者或逾期逗留者有提出免遣返聲請，社署資助的香港國際社會服務社便會每月向他們提供人道援助金，每人每月平均3,740元。以他們平均在港額外逗留1.5個月計，涉及人道援助金額為38.1萬元，連同機票開支，則涉及公帑48.3萬元。

曾國衞周二公布，入境處在周一再次採用包機形式執行遣送行動，合共將20名越南籍非法入境者遣返越南。他在會上透露，今次開支為45萬元，「呢45萬包括送嗰啲越南人士（回國），同埋我哋伙記來回（機程）。」他補充指，今次有40名人員隨機押送。換言之，行動中每名被遣返人士的人均開支超過2萬元。

曾國衞稱，今次20名被遣返者，部分人屬被強制遣返，「有部分有啲……以前有啲刑事紀錄，甚至乎有啲有暴力嘅情況，所以我哋今次係多咗同事去管理。」但他未有提供被強制遣返或有刑事紀錄者的具體人數及資料。

據悉今次包機為國泰港龍航空旗下航機，估計包機收費亦較越南航空為高。國泰港龍回覆眾新聞查詢指，未能提供個別包機安排詳情。入境處未有明確回應今次行動由哪家航空公司負責，但處方在新聞發布會上展示的圖片所見，今次包機為港龍航機。至於入境處為何不選用收費較低的航空公司，而要在農曆新年前、工作回顧新聞發布會前選用國泰港龍包機安排遣返行動，處方未有回應。

若按以往多年採用非包機的做法，周一行動涉及20人要被遣返、動用40名入境處職員，假設該20人可以一批過被遣返，不涉及額外留港的人道援助金開支。入境處若選用越南航空的話，網上資料顯示，單程票約1500元，來回約2000元（入境處職員須購買來回票），估計非包機行動開支約為11萬元；若乘搭國泰港龍航空的話，單程票約5000元，來回約7500元，非包機行動則需要40萬元。

入境處周一選用國泰港龍航空包機的行動共花費45萬元，按上述估算，貴過採用非包機。不過，若在非包機行動中，需要出動更多入境處職員負責押送的話，非包機開支則有機會較高。

被問到為何要採用包機、包機會否成為恆常安排，曾國衞回應指：「包機係其中一個方法。實際上嚟講，過去兩次行動上面嚟睇，亦都係好成功，亦都好順利。咁嚟緊亦都會，一方面進一步加強同各個使領館聯繫，盡快等佢哋接收啲咁嘅人士。同時間，採取遣返行動上，我哋會考慮唔同嘅方式去進行，視乎實際情況決定用咩方式。我哋唔排除包機將來繼續係一個行動嘅方法。」但他未有具體說明，包機在安排遣返的過程中，可以快幾多及提高幾多效率。
