# 暴動罪審訊　控方將傳97名證人　多為警員

<figure>
<img src="https://web.archive.org/web/20210705041659im_/https://www.hkcnews.com/news_新聞/hk-discuss/2018/02/梁天琦-暴動罪-20180221193554_225a_large.jpg" alt="">
<figcaption>梁天琦被控四項罪名。何君健攝</figcaption>
</figure>

前年年初二的旺角衝突，前本土民主前線發言人梁天琦等6人被控暴動罪，案件繼續在高等法院開審。昨日選出的4男5女陪審團，今日正式宣誓，控方在庭上透露，將傳召97名證人作供，大部分為警員。

本案被告為梁天琦、李諾文、盧建民、林傲軒、黃家駒和林倫慶。其中，梁天琦被控兩項暴動罪、一項煽惑暴動罪和一項襲警罪，是控罪最多的被告，他早前已承認一項襲警罪。至於另一被告黃家駒，早前承認一項暴動罪，須押後判刑。

控方透露將傳召97名證人作供，大部分為警員，包括梁天琦早前承認襲擊的警員文錦璣；其他證人包括食環署小販管理隊人員、市民，以及2名法證專家。控方又透露，呈堂證物涉及大量錄影片段，包括警務人員和食環署人員拍攝的片段、位於砵蘭街的瑞興麻雀館後巷的閉路電視片段、兩段行車紀錄儀片段，以及警方從公眾平台取得的新聞片段。除此之外，證物亦涉及屬於黃台仰的駕駛執照、部分被告的警誡會面紀錄、15本照片冊及盾牌。

控方其後讀出開案陳詞：


| 時間 | 事件 |
| --- | --- |
| 2月8日   
 晚上8時 | 控方指，食環署小販管理隊當時在旺角近砵蘭街一帶執勤，當時沒有作任何驅趕、票控小販的行為。大約8時多，有小販在瑞興麻雀館的後巷擺賣，當時身穿藍色、寫有「本土民主前線」上衣的黃台仰，被拍到與小販有接觸。   
   
 控方又指，與此同時，有些人開始在砵蘭街聚集，但形容當時砵蘭街「仍然好似新年、多車多人、道路暢順。」 |
| 2月8日   
 晚上9時 | 控方指，大約9時多，有小販從後巷推車，沿砵蘭街、往奶路臣街方向推進，有藍衣人士跟隨。其他小販亦開始將販賣車推進，並在砵蘭街莎莎化妝品店、雅蘭商場Collect Point外聚集。   
   
 控方指，有證據顯示，當時小販管理隊人員被群眾包圍，包括身穿「本土民主前線」藍色上衣的人士，期間有推撞，聚集的人亦愈來愈多。控方續指，當時行人路塞滿人，汽車不能通過，形容與8時多的情況完全不同。   
   
 控方又指，當時有人報稱，有的士撞到行人，警方接報到場，但因大批人士聚集，警方無法上前處理。 |
| 2月8日   
 晚上約10時 | 控方表示，警方曾派機動部隊警員幫助已到場的警員，嘗試勸喻人群散開，讓交通警員處理事件。惟黃台仰用擴音器向群眾呼籲，導致現場開始失控，有人開始衝擊警員。   
   
 從呈堂片段顯示，當時站在電箱上的黃台仰一度用擴音器說：「你向後行，咁啲人咪散，去食嘢囉」；亦有不少人在現場高叫「收皮」、「X你老母」等字眼。   
   
 控方指，約10時半，群眾聽從黃台仰的指示，讓的士逆線離開，但黃台仰同時呼籲群眾阻止、不要讓警方向前走，亦有人開始在馬路上擺放雜物。控方形容，當時道路癱瘓、情況惡劣。 |
| 2月8日   
 晚上近12時 | 控方指，警方曾嘗試將高台推進砵蘭街，讓警員在高台上向民眾呼籲，但遭民眾阻礙、衝擊警方防線。有人更開始向警方防線掟雜物，當時黃台仰站在客貨車頂，不斷作出挑撥言論。從呈堂片段所見，有人一度舉起垃圾桶，警方亦舉起紅旗警告。   
   
 控方續指，警方期間不停受衝擊，開始退回山東街方向，惟仍受到民眾撞擊和投擲雜物，形容警方迫於無奈下施放胡椒噴霧。控方又指，黃台仰一直慫恿集結群眾衝擊警方和掟雜物。 |
| 2月9日   
 凌晨近1時 | 控方播放另一條呈堂片段，形容片段「拍攝到不尋常嘅事發生。」控方指，當時黃台仰仍坐在客貨車頂上，有人同時將「盾牌」、「盔甲」搬到車上，亦有人開始穿上「盔甲」。控方又指，警方嘗試派警員與黃台仰對話，但黃台仰沒有合作。   
   
 與此同時，梁天琦手持擴音器與群眾說話，稱「呢個地方係屬於香港人，唔係屬於公安同城管」、「如果係香港人就留低」、「我哋香港人同本民前一直都會留係到。」 |
| 2月9日   
 凌晨近2時 | 控方再播放另一條呈堂片段，梁天琦當時多次稱，他正進行選舉遊行，黃台仰則用擴音器說：「如果你哋（警方）要玩，我哋香港人，同本土民主前線一定同你玩大佢」、「如果你可以就拉晒所有人。」   
   
 控方向陪審團提出質疑：「如果話係30人嘅選舉遊行，點解黃台仰會叫警方有能力就拉全部人？」他建議陪審團「深思熟慮考量下，呢啲言論，同黃台仰及第一被告人（即梁天琦）嘅言論同行為係咪一致？」   
   
 控方又指，大約1時40分，警方向群眾發出10分鐘內離開現場的警告，但警告沒有太大效用，群眾反而向警方掟更多雜物，甚至有人手持長棍、盾牌站在警方前。控方形容，當時黃台仰和梁天琦不斷用擴音器發出言論，被警方認為言論屬挑撥群眾，向他們發出警告，但他們沒有理會。   
   
 控方表示，警方發出警告的時間即將完結時，黃台仰發出號令，與群眾衝向警方防線，有人用長棍插向警方，亦有人投擲物品和使用盾牌。 |


另外，陪審團今日宣誓前，一名男子向法庭表示，所屬公司規模較小，擔心擔任陪審員後，60日的審訊會對公司營運造成影響。法官彭寶琴表示，香港市民有責任擔任陪審員，一般工作、生活的不便不是構成豁免的原因。她又指，陪審員受《陪審團條例》保障，假如僱主以任何方式作歧視、威脅，有可能違反相關法例，甚至受相關法律制裁。一名女子表示，已經訂好機票，將於5月29日到澳洲一星期。彭官表示，預計審訊到五月中結束，假如審訊有延誤，會為她作遷就。彭官最終安排4男5女的陪審團宣誓。

彭官亦用了不少時間提醒，陪審團是唯一事實裁決者，進行事實裁斷時，不受控辯雙方甚至法官的影響。陪審團必須按法官提供的法律指引行事，並以法官的法律觀點為依歸。彭官指，陪審團在審訊過程須保持開放態度，並將之前在電視、報章、網絡等平台接觸過的相關資訊和觀點拋諸腦後。她稱，審訊期間會出現不同報道，但陪審團不應用作考慮，否則是違背誓言和制度的核心原則，陪審團亦不應自行就案件進行調查。

她又提醒，陪審員只能在所有陪審員在場之下討論案件，不應進行小組討論，亦不應與陪審員以外的人討論，或透過社交網絡、電話、文字訊息等平台與外界進行資料交換。

案件明早續審。梁天琦離開犯人欄時，有旁聽人士高叫「正義必勝」，又稱為梁天琦預留利巿，梁天琦則面露笑容，向旁聽人士做出恭喜手勢，便轉身步入拘留室。
