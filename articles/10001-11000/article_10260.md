# 新聞價值的反思－－人情趣味的探討

【撰文：皇甫河旺教授】   
 ***作者生前為香港珠海學院新聞及傳播學系系主任***

摘要

本文從一些感動人心、振奮心靈及令人開懷的新聞，反思這些具有潛移默化影響力的人情新聞的新聞價值。本文將過去30多年來中所蒐集香港、台灣報紙的人情與趣味新聞，歸納為逗笑溫馨、神奇巧合、感動人心、激勵心靈等四類，每類擇兩則新聞為例。這些新聞在每天滿是煽情新聞之外，也是大眾感興趣及需要的精神食糧。本文認為新聞媒體宜多重視、發掘人情及趣味新聞，並用圖片及寫作技巧來吸引讀者、深入人心。

關鍵詞：新聞價值、人情新聞、趣味新聞、感動人心、振奮心靈

### 壹：新聞史上的人情趣味新聞

美國的大眾化報紙始於1833年由班哲明．戴  ( [Benjamin Day](https://web.archive.org/web/20210712013941/https://en.wikipedia.org/wiki/Benjamin_Day_(publisher)) ) 創辦的紐約《太陽報》，每份售價一分錢，又稱為一分錢報  ( [penny press](https://web.archive.org/web/20210712013941/https://en.wikipedia.org/wiki/Penny_press) )。《太陽報》創刊四個月銷數達五千份，成為當時銷數最大的報紙。該報發行三週年時，銷數達三萬份。該報所以能開啟大眾化報業，除了售價低、印刷精緻、寫作簡明，新聞內容還偏重本地新聞及人情趣味故事。 **[1]** 該報從創刊就發現，人情趣味新聞可以吸引讀者，形成了新型態又賺錢的新報業。 **[2]**

該報採用一種特寫，稱為「人情趣味故事」。這是一種短文，其趣味不在故事之本身有何重大意義，而在引起一般人之興趣、新奇、哀憐與同感。這種故事必須以雋永的文筆才能表達出來，否則即平凡無奇。 **[3]**

人情趣味新聞無疑是滿足人性需要、引起讀者興趣的題材。但是隨著報業商業化，一些報紙走向以犯罪、災難、衝突、緋聞等激情新聞來吸引讀者，競逐銷數。到了十九世紀末，美國大城市的報紙更激起煽色腥 (sensational) 的「黃色新聞」氾濫，人情趣味新聞幾乎被埋沒。世界每天發生無數大大小小的新聞，包括人情與趣味新聞。但是被發掘、刊登出來的人情趣味新聞却少之又少，觸動人心的影響力似乎已沒有受到重視。

### 貳：人情趣味新聞的分類

中、英文新聞學著作談到新聞價值，通常把人情趣味 (human interest) 歸為一類。其實，人情與趣味是不同性質。趣味新聞包括逗笑、溫馨、神奇、巧合等新聞，讓人讀了心情愉悅、開心、輕鬆。人情新聞包括愛的故事、愛的奉獻、愛的尋覓等新聞，令人感動落淚、心靈激盪；還包括身殘肢障卻超越常人的不凡故事，令人感到激勵、發奮。本文將人情趣味新聞歸納為逗笑溫馨、神奇巧合、感動人心、激勵心靈等四類，每類舉兩則新聞實例。

**（一）逗笑溫馨**

（1）「查理咬我﹗」英國一對小兄弟，5歲半的哥哥哈里抱著2歲的弟弟查理坐在沙發面對鏡頭。哈里貪玩，將右手食指塞進弟弟嘴裡。弟弟一口咬著不放。哈里先是皺眉頭，被咬越來越痛，開始大叫想哭。好一會兒才把手指從弟弟口中拔出，責罵弟弟：「查理，那很痛呢。」查理望了哥哥，咧嘴大笑，笑了又笑，十分可愛。這段短片全長55秒，2007年5月拍攝，至2010年7月，被點擊了超過2.5億人次，成為歷來最多人看過的YouTube短片。目前每天還有成千上萬人瀏覽他們的網誌。有人留言說，在過了沮喪的一天後，看看查理和哈里，就會開心起來。 **[4]**

（2）八歲小學生一再質問，美總統無法證明身分。1991年3月11日，8歲的小學生韓德生要求到學校訪問的布希總統（港譯：布殊）證明他是總統時，布希幾乎無法招架。韓德生說：「我怎麼知道你是總統呢？」布希問道：「你認為我是一個偽裝總統的人嗎？」布希向韓德生展示了他的駕駛執照。無奈駕照上未證明職業。布希又從皮夾中拿出了信用卡、停放的總統黑色轎車等各種招數來證明自己是美國總統。但是韓德生仍半信半疑。布希總統到巴克羅夫小學訪問，是向自願來此小學協助教學的美軍陸戰隊員致敬時，遇上這件趣事。 **[5]**

**（二）神奇巧合**

（1）金戒被偷，15年後從魚肚中尋獲。2006年6月1日，一則來自美國的新聞報道，美國漁民在肯塔基州中北部捕魚時，從一條鯰魚肚中發現一玫鑲有寶石戒指，指環刻有「84年畢業塞爾坦」等字。漁民恐怕來自屍體，於是報警。警方從學校畢業生中終於找到失主，塞爾坦小姐的戒指是1991年被偷。雖然物歸原主，但是留下謎團 －－ 戒指在15年之間，如何從俄亥俄州去了肯塔基州的魚肚裡？ **[6]**

（2）機率百萬分之一的黑白雙胞胎。1995年1月24日，台北《民生報》刊登一幅膚色一黑一白雙胞胎男嬰的圖片，他們倆膚色不同，長得一模一樣。他們的父親是黑人，母親是白人，在倫敦的一所醫院誕生。2006年10月23日，香港各報刊出澳洲布里斯本一所醫院誕下一黑一白的雙胞胎。母親是牙買加人，父親是澳洲白人。醫學界指出，誕下黑白雙胞胎的機會是百萬分之一。四天後，《明報》又刊出英國米德爾斯伯勒的理查森太太誕下一黑一白雙胞胎。每篇報道都附有大幅圖片。

**（三）感動人心**

世間每當不幸與災難發生，往往造成死亡，因而成為大大小小的新聞。 生死瞬間，總是承載著愛的故事。例如：事禍中，祖母以身護孫而亡；地震中，慈母捨身擋塌樓而喪命；大火中，小童救老邁爺爺不及逃生；母親街頭遇搶劫，寧死抓住手袋不放，只因手袋中的手機錄了剛夭折小女叫媽媽的聲音和笑語。這樣洋溢著情和愛的故事述說不盡，本文僅摘錄兩則新聞。

（1）    癌母留12信，陪稚女長大。2005年8月13日，台灣《蘋果日報》大篇幅刊出兩則年輕母親患癌症末期的新聞。一位媽媽得知自己來日不多，無法看著8歲女兒長大。她撐著孱弱身體，在病牀上 [寫了12封信](https://web.archive.org/web/20210712013941/https://tw.news.appledaily.com/headline/daily/20050813/1974173/) ，並編號交給丈夫，做為女兒未來12年的生日禮物。一年一封，讓女兒生日都有母親的陪伴與祝福。另一位年輕母親蘇婉婷，二年半前在女兒三歲時不敵乳癌摧殘，臨終時不捨女兒沒有母親陪伴成長，在醫院請護士幫她買了 [60張各式各樣卡片](https://web.archive.org/web/20210712013941/https://tw.news.appledaily.com/headline/daily/20050813/1974178/) ，一字一淚寫下對女兒的祝福和期許。

（2）    2012年2月12日即情人節前夕，《蘋果日報》刊出全版有圖片的專題報道，標題是 [「邂逅九巴女神30分鐘，痴情叔苦等30年」](https://web.archive.org/web/20210712013941/https://hk.news.appledaily.com/local/daily/article/20120212/16063110) 。內文說，情人節前夕，九龍旺角行人專用區街頭出現一個身體告示牌。一位頭髮稀疏、兩鬢斑白的62歲阿叔，掛著寫滿字句的白布，尋找一位30年前在九龍巴士33M (已停駛) 上、邂逅30分鐘、只說過兩句話的少女。62歲的阿叔施純立說，他當時乘巴士回葵涌住所。在巴士上，他遇見一位年約17歲、穿白色衣裙校服的女學生，目光碰觸，改寫了他的一生。

**（四）激勵心靈**

世界上有千千萬萬先天或後天遭遇不幸的人。這些不幸的人有些度過悲慘的一生，有些卻活得精彩萬分。他們人生的分別也許是有沒有接觸激勵心靈的新聞。在澳洲出生的力克 (Nick Vujicic)，生出來就沒有手沒有腳，他曾說「我那好到不像話的人生」，就是一例。由於身體極度殘障，常遭同學欺凌，他十歲前曾三度自殺未逐。直到13歲，他閱報看到報道，一名殘障人士也可以幫助他人，才改變他消極的想法，決定以自己的生命影響其他生命。 **[7]**

（1）    缺三肢華裔女子，在美成兒科醫師。2007年5月29日，《星島日報》以全版報道華裔女子林凱莉的故事。她在美國密西根州出世、成長。八歲時候患了細菌性腦膜炎。為了保命，她接受截肢手術，雙腳、右手及左手三根指頭被截除。凱莉肢體極度殘缺，但並未自暴自棄，立志要成為兒科醫師。凱莉於2007年終於從加州大學洛杉磯分校 (UCLA) 醫學院畢業，在附屬醫院當兒科醫師。報道還刊登了凱莉除下義肢游泳、用左手拿著聽筒為兒童診病的圖片。凱莉除了生活、學習、工作，還學會了游泳、雙人跳傘、騎馬。

（2）    無臂鋼琴王子，「中國達人秀」奪冠。2010年10月12日，《明報》以全版刊出無臂鋼琴王子劉偉的報道，還登了他用雙腳彈琴的圖片。「中國達人秀」是上海東方衛視的真人秀節目。劉偉在決賽中壓軸出場，自彈自唱James Blunt 的「You're Beautiful」。1987年出生的劉偉來自北京，10歲時玩捉迷藏時觸及高壓線，結果要截肢，失去雙臂。劉偉受到北京一位無臂畫家的啟發，他學會用腳刷牙、吃飯、寫字。他12歲開始學游泳，19歲起學鋼琴。大腳指比琴鍵寛，用雙腳彈琴難度很高。劉偉每天自學苦練超過7小時。彈琴使劉偉的雙腳更靈活。他還兩度創下用腳打字的世界紀錄，每分鐘先後達231字和252字。

### 叄：人情趣味新聞的採訪及寫作技巧

世界之大，人事變化之速，每天發生的人情與趣味新聞絕不限上述四類，每一類之中絕不限兩例。這些新聞不見得每天都發生，也不至於完全沒有發生。新聞媒體要發掘、捕捉這些新聞，首先必須重視這一類新聞。

新聞媒體在當前市場競爭中，無疑最重視的是所謂「人咬狗」的反常、激情新聞。但是，每天充滿了這些令人沮喪、挫折、憂慮的負面新聞之外，能令人感動、振奮、開懷的人情、趣味新聞，也是大眾感興趣及需要的精神食糧。媒體重視人情、趣味新聞，記者自然會產生人情新聞的價值觀與敏銳度，從每天的新聞中發掘。災難與不幸中，總是有人性光輝的鏡頭；社會弱勢的角落，常見激勵心靈的故事。人情新聞貴在真實，不煽情，不誇大。

人情新聞並不是滿足大眾的知，而是感動人心，撫慰心靈、振奮精神。這類新聞採訪須深入，資料蒐集十分充實，擷取精華才有生動的內容。感動人的報道，有些是題材感人；有些則是記者的寫作技巧動人。寫作技巧不是用文字浮華裝飾，而是多用事實，文字簡潔有力。

趣味新聞就容易多了。在網絡時代，各地有趣的事、物都有人放到網上，如孩子的天真、寵物的逗笑、絕技的表演、神奇、巧合等等不可思議的圖片及視頻。本文引述「查理咬我」的趣聞就是從YouTube下載的例子。趣味適於用畫面、視頻來表視。報紙報道趣味新聞常以圖為主，文字為輔，才能突顯開懷的效果與吸引力。

在目前網絡時代，網上充滿了來自全球的趣味的視頻。新聞媒體取材毫不費力。人情新聞則有待媒體去發掘。人世間不平凡、感動人的故事時常發生。新聞媒體只要重視這類新聞，就不難發掘。撰寫感人、動人的新聞，需要具悲天憫人的心，以及平實有力文字技巧。
