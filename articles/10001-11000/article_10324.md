# 巴士車長新指引加入「特別更次」　更時上限維持14小時工會批「玩數字遊戲」

大埔公路九巴嚴重車禍造成19人死亡後，運輸署今日交代建議 [修訂《巴士車長工作、休息及用膳時間指引》](https://web.archive.org/web/20200131063801/https://www.info.gov.hk/gia/general/201802/23/P2018022300542.htm) 的內容，將舊指引中沒有的「特別更次」納入新指引之中。「特別更次」一般指工時特別長的更次，每更中段車長通常有3至4小時休息時間，一般是「因應上下午繁忙時間的服務需要」而安排。

運輸署新指引列明，「特別更次」的駕駛時間上限為14小時，比一般更次多2小時，更與舊指引下的每日最長工時一樣。巴士業職工會聯盟主席、新巴車長林錦標批評，巴士公司在新指引下可以不斷開設「特別更次」，車長仍然會疲累駕駛，所謂縮減更次時間及駕駛時間，只是「玩數字遊戲」、「誤導公眾」。他更擔心車長被安排駕駛「特別更次」，有機會被減人工。

**運輸署《巴士車長工作、休息及用膳時間指引》主要修訂內容：**


|  | **[舊指引](https://web.archive.org/web/20200131063801/http://gia.info.gov.hk/general/201710/25/P2017102400694_270495_1_1508913615137.pdf)** | **[新指引](https://web.archive.org/web/20200131063801/http://gia.info.gov.hk/general/201802/23/P2018022300542_278730_1_1519376581645.pdf)** |
| --- | --- | --- |
| 更次時間上限 | 14小時 | 特別更次：14小時   
 一般更次：12小時 |
| 每更駕駛時間上限 | 11小時 | 10小時 |
| 每駕駛6小時後休息時間 | 30分鐘 | 40分鐘 |


綜合九巴僱員工會主席郭志誠、巴士業職工會聯盟主席林錦標、城巴職工會外務理事長羅志輝提供的資料，目前只有九巴有「特別更次」的安排，現職特別更車長約有1,000人；新巴及城巴則沒有設「特別更次」。

「特別更次」與一般更次的主要分別，是特別更中間會有一段長約數小時的休息時間，行內俗稱 [「Tup」車](https://web.archive.org/web/20200131063801/https://www.facebook.com/kmbhongkong/videos/1324783787579721/) 。

巴士車長 **舉例** 說明，「特別更次」及一般更次的分別如下：


|  | **特別更次** | **一般更次** |
| --- | --- | --- |
| **時間分配** | **6小時駕駛**   
 **+**   
 **1小時午飯**   
 **+**   
 **3小時休息**   
 **+**   
 **4小時駕駛** | **5小時駕駛**   
 **+**   
 **1小時午飯**   
 **+**   
 **5小時駕駛** |
| 更次時間 | 14小時 | 11小時 |
| 駕駛時間 | 10小時 | 10小時 |


巴士車長指出，在一般更次之下，午飯時間是納入工時計算。由於九巴、新巴、城巴的底薪以每日基本工時8小時計算，若按以上的時間表計，一般更次有10小時駕駛時間及1小時午飯，總工時為11小時，減去8小時的基本工時，即車長會獲3小時超時補水，九巴每小時補水96元起、新巴城巴每小時70元起。舊指引規定，車長每駕駛6小時後，要有至少30分鐘的休息時間，這類休息時間與午飯時間性質相同，一樣會納入工時計算。

不過，新指引之下的「特別更次」，中間的休息時間，巴士公司未必會視作工時，例如九巴就休息時間只會提供俗稱「Tup車錢」的津貼，每小時僅約20元，遠低於超時補水的時薪價。若以上述時間表計算，「特別更次」長達14小時，比一般更次長3小時，車長每更僅多賺約60元。

九巴僱員工會主席郭志誠指，有車長為賺取較高工資，仍願意駕駛「特別更次」。

巴士業職工會聯盟主席林錦標及城巴職工會外務理事長羅志輝均指，新巴一直沒有「特別更次」的安排，城巴10多年前曾經有此做法，但在工會強烈反對下已取消，目前新巴及城巴都沒有「特別更次」。林錦標批評，「特別更次」的安排只是「玩數字遊戲」，更次時間上限仍然是14小時，而車長根本難以利用中間數小時的「休息時間」來休息，「嗰3個鐘可以去邊？又冇休息室，喺街何來休息？車長仍然係疲勞駕駛。」

林錦標及羅志輝又表示，新巴、城巴向來將午飯時間、休息時間納入工時計算，但若新巴城巴公司日後仿照九巴「特別更次」的做法，就長更中間的休息時間只計「Tup車」津貼，長更時的車長可能變相被減人工。林錦標形容，新指引容許「特別更次」是「倒退」，他對運輸署的做法感到非常失望。他們表示本月底會跟公司開會，屆時會了解公司因應新指引作出的編更安排。

運輸署指，明白新的編更安排可能會影響巴士車長的實際收入，署方已責成巴士公司必須妥善處理有關的事宜，所有巴士公司已承諾會積極探究不同的方案，以制定維持和進一步改善車長實際收入的通盤計劃，並會盡早與員工溝通及商討，「據運輸署了解，專營巴士公司均表示會視乎增聘車長的進度，計劃逐步將特別更次的工作時數由最長14小時縮短至13小時。」

署方表示，各專營巴士公司須敲定實施修訂《指引》下實際的編更細節安排，巴士公司承諾會盡早約見車長，就營運情況向車長解釋修訂《指引》的實施細節。待理順有關編更細節安排後，巴士公司的目標是今年第二季開始，可陸續實施修訂《指引》，明年第二季內全面實施。

九巴、新巴、城巴均表示會積極配合新指引。
