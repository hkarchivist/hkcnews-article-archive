# 謝安琪質疑流感疫苗　醫學界反駁七個謬誤

冬季流感肆虐，專家呼籲市民接種流感疫苗，加強個人保護。不過，家長群組昨日起流傳一條聲稱是歌手謝安琪的錄音，錄音內容就流感疫苗的成份、製作過程、成效、風險等方面提出連串質疑。謝安琪今早在 [個人facebook專頁](https://web.archive.org/web/20201022203650/https://www.facebook.com/kaytsemusic/posts/1708194162599696) 承認，錄音是她和朋友之間「一個私人群組的交流」，本來無意公開，「更無意影響大眾對流感疫苗的看法」。她表示非常著重家庭健康，時常參考醫學文章、自然療法等不同來源的資料。

有關錄音廣泛流傳，引起醫護衛生界關注，權威醫生紛紛撰文及發言澄清。眾新聞綜合 [衞生署](https://web.archive.org/web/20201022203650/https://www.info.gov.hk/gia/general/201802/08/P2018020800557.htm) 、港大李嘉誠醫學院微生物學系傳染病學講座教授 [袁國勇](https://web.archive.org/web/20201022203650/https://m.mingpao.com/ins/instantnews/web_tc/article/20180208/s00001/1518051774226) 、醫學界立法會議員 [陳沛然](https://web.archive.org/web/20201022203650/https://doctordaddysoccer.blogspot.hk/2018/02/flu-vaccine.html?m=1) 、醫生組織 [「杏林覺醒」](https://web.archive.org/web/20201022203650/https://www.facebook.com/enlightenedhealers/posts/1528287837289081) 的解說，以及食物衞生局的資料，逐點釐清七個謬誤。

#### 謝安琪：今年疫苗成效低，少於10%。


| **食衞局** | 過去本地研究顯示，流感疫苗對預防兒童因感染流感而入院的效能介乎約40%至80%。中心早前分析2011/12至2016/17年度季節性流感疫苗對在院舍長者的效能，發現疫苗對預防因感染流感而需入住深切治療部或死亡的效能約介乎37%至69%。另外，中心與醫管局合作分析2015/16冬季流感季節曾因呼吸道病徵入住公立醫院的病人數據，發現該季的流感疫苗對預防65歲或以上長者因流感入院的效力約40%。外地研究亦顯示為學童接種流感疫苗能減低缺席率，及對其他高危群組（例如老年人）有保護作用。 |
| --- | --- |
| **袁國勇** | 現時疫苗效力有50%至55%。 |
| **陳沛然** | 根據世衞的資料，當疫苗抗原與流行的流感病毒相當吻合時， 滅活流感疫苗對65歲以下人士提供的保護效用可達70%。 |


####   
 謝安琪：疫苗裡面有水銀和鋁等抗菌劑。


| **衞生署** | 現時供香港使用的流感疫苗，無論在「政府防疫注射計劃」及私人醫療市場，全部為單劑量預充式注射劑，並不含水銀化合物（例如：硫柳汞）作為防腐劑，亦沒有鋁的成分。 |
| --- | --- |
| **袁國勇** | 加入疫苗的物質並非我們所說的水銀，因為水銀進入人體後不能排出，但這種物質可以排出體外，且這種物質是對人體無害的防腐劑。疫苗內沒有鋁。 |
| **杏林覺醒** | 所有疫苗都無水銀及鋁。部分疫苗含硫柳汞 (Thimerosal)，主要用於防腐，多數是用在多劑量瓶中 (Multi-dose vial，即用同一樽疫苗為多人注射)，研究顯示對人體無害並不會引致自閉症。現在本港所用的獨立包裝一劑裝流感疫苗不含硫柳汞。至於磷酸鋁 (Aluminum phosphate) 含鋁的無機化合物，用以增強身體對疫苗的反應，使身體產生更強的免疫能力。鋁化合物作為疫苗成分已經安全使用超過半世紀，臨床及流行病學數據均支持其安全性，而人亦會在環境中自然接觸到鋁，包括空氣、食物和食水裡，甚至部分嬰兒配方奶中也含鋁。但現時本港所用的流感疫苗也不含磷酸鋁。 |


#### 謝安琪：科學家每年「買大細」估流感，來製作疫苗。


| **袁國勇** | 世衛很多科學家，包括香港衛生署病毒學家，每年在瑞士開會，分析過去一年實驗室的病毒基因排序，測試抗原，再決定哪種疫苗種子最適合加入做疫苗，過程非常繁複仔細。因病毒變化快，專家有時會估錯哪種疫苗 ，但4種疫苗起碼有3種選中。 |
| --- | --- |
| **陳沛然** | **世界衛生組織如何制定流感疫苗的建議？用許多不同的數據和信息來確定，包括：** 全球流感監測和應對系統網絡的監測數據，包括國家流感中心、世衛組織合作夥伴、世衛組織 病毒的抗原表徵 人類血清學研究 病毒的遺傳表徵 病毒健康預測 抗病毒抗藥性 疫苗有效性 |
| **杏林覺醒** | 流感疫苗之成份年年不同，WHO每年也會為南半球及北半球預測該年會流行哪一隻病毒，再將毒株發給疫苗製造商製造……現時所用之四價疫苗含H1、H3、維多利亞乙型及山崎乙型合共四隻毒株。但流感病毒不斷變種，故大半年前預測的毒株可以與該年冬季流行之病毒不同。不時也會出現疫苗毒株與該年主要流行之病毒不吻合之情況，但疫苗成效要到流感季節中後期或完結才知。如該年吻合，成效能高達 50-70% 之保護作用。即使不吻合，多數只是一隻不吻合，仍能對其他三隻流感病毒有保護作用……2017/18 的北半球四價流感疫苗中所含之乙型流感是布吉型 (B/Phuket/3073/2013)，屬山崎型。與現在流行之山崎乙型流感吻合。 |


#### 謝安琪：疫苗的製作過程，是先將病菌打入母雞體內，讓帶有病毒的母雞下蛋，再從雞蛋抽病菌出來。


| **衞生署** | 流感疫苗內雖含有卵清蛋白（即雞蛋白質），但疫苗製造過程經過反覆純化，卵清蛋白的含量極少，即使對雞蛋敏感的人士，在一般情況下亦能安全接種。 |
| --- | --- |
| **袁國勇** | 專家是用雞蛋製造疫苗，並非母雞。專家把疫苗種子打入雞蛋，令雞蛋製造疫苗，用雞的胚胎製造出來，而雞蛋製造疫苗後，會放入機器將疫苗的病毒殺死，再用化學方法把病毒表面抗原分離，並依據劑量放入注射器，消除大部分雜質。雖然可能留有小量雞蛋蛋白，但不會對人體有害。 |
| **陳沛然** | 不是用母雞，而是用蛋。流感疫苗是在雞蛋、細胞培養或人工重組蛋白質而制成的。流感 (influenza) 是病毒，不是細菌。細菌 (bacteria) 千份之一厘米，可以用顯微鏡看到；病毒 (virus) 比細菌更細小一百倍，要用電子顯微鏡才能。細菌和病毒都有可能致病，但是一般人都會將兩者混淆。 |
| **杏林覺醒** | 現時香港普遍使用的均為滅活流感疫苗(inactivated influenza vaccines)。流感疫苗之毒株用雞蛋來繁殖，然後將病毒殺死 (inactivated)，再將病毒分解 (split virions)。故疫苗內並無活病毒，即使打了疫苗後也不會傳播變種病毒。流感疫苗內含有卵清蛋白 (ovalbumin，一種雞的蛋白質)，但疫苗製造過程經過反覆純化 (purification)，故卵清蛋白之含量極少，連對雞蛋敏感者也能安全接種。 |


#### 謝安琪：製作疫苗最長只用3個月，疫苗內細菌的數量及活躍程度、效用，未經安全檢測，只是「搏一搏」。


| **衞生署** | 流感疫苗已沿用七十年，疫苗經過反覆試驗及品質檢測，證實非常安全可靠。 |
| --- | --- |
| **陳沛然** | 疫苗有做安全檢測的。每次新流感疫苗，先要得到美國食品和藥物管理局（FDA）的批准。在批准後這兩個機構不斷監測疫苗的安全性。 如果疫苗發現問題，美國疾病控制與預防中心（CDC）和FDA將通知衛生官員，醫護人員和公眾。 |
| **杏林覺醒** | 世衛一般會於每年二月為北半球國家預測，到九月底便會推出市場，製造時間約七個月。製作方法基本上年年相同，只是每年用不同的毒株。而疫苗也經過反覆試驗，證實非常安全。 |


####   
 謝安琪：細菌在雞蛋變種，發生「突變」，變成更強的細菌，所以很多人打流感疫苗，身體已經承受不了而生病，再將病菌在社區內散發開去。因此，今年的流感有一部分責任來自疫苗。


| **衞生署** | 香港現時使用的流感疫苗均為滅活流感疫苗，疫苗內並無活病毒，接受流感疫苗不會引致感染流感病毒，更不會傳播變種病毒。 |
| --- | --- |
| **袁國勇** | 不會有這情況，因只是將病毒表面抗原分離出來再放入注射器，是滅活流感疫苗，疫苗內並無活病毒。 |
| **陳沛然** | 疫苗裡的病毒都是死的和被分解的，何來突變 mutation呢？只有流感病毒才能變種，疫苗是不能自動突變的。 |


#### 謝安琪：打疫苗風險很高。


| **衞生署** | 接種疫苗是預防季節性流感及其併發症的最有效方法之一，亦可減低因感染流感而入院留醫和死亡的風險。今季疫苗的抗原與香港今年冬季流感季節流行的流感病毒吻合。 |
| --- | --- |
| **陳沛然** | 至今12宗兒童嚴重個案，90%無打針。成人嚴重個案194宗，111人死亡，大部分是長期病患者，四分三人無打預防針。 |
