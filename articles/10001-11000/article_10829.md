# 運輸署數字顯示　九巴交通意外宗數近兩年回升

眾新聞 [報道](https://web.archive.org/web/20210712014354/https://www.hkcnews.com/article/10797/%E4%B9%9D%E5%B7%B4-%E9%81%8B%E8%BC%B8%E7%BD%B2-%E4%B9%9D%E5%B7%B4%E8%BB%8A%E7%A6%8D-10797/%E6%9D%8E%E6%BE%A4%E6%98%8C%E4%B8%8A%E4%BB%BB-%E4%B9%9D%E5%B7%B4%E3%80%8C%E5%AE%89%E5%85%A8%E5%8F%8A%E6%9C%8D%E5%8B%99%E8%B3%AA%E9%87%8F%E9%83%A8%E3%80%8D%E6%B6%88%E5%A4%B1-%E9%81%8B%E8%BC%B8%E7%BD%B2%E8%A2%AB%E8%B3%AA%E7%96%91%E6%B2%92%E7%9B%A3%E7%AE%A1) ，現任九巴董事總經理李澤昌2015年上任後，2010年九巴內部成立的「安全及服務質量部」消失，運輸署被質疑沒有跟進及監管九巴的安全管理及運作。運輸署周五回覆眾新聞查詢，提供2010年至2017年涉及九巴的交通意外宗數（包括在車廂內發生的意外，例如乘客在車廂內樓梯 / 上落車時失平衡跌倒 ）。數字顯示，2016年、2017年的意外宗數回升：


| **年份** | **涉及九巴的交通意外（宗）** | **升／跌幅** |
| --- | --- | --- |
| 2010 | 923 | -- |
| 2011 | 959 | +3.9% |
| 2012 | 1040 | +8.4% |
| 2013 | 1083 | +4.1% |
| 2014 | 949 | -12.4% |
| 2015 | 853 | -10.1% |
| 2016 | 891 | +4.5% |
| 2017 | 976 | +9.5% |


九巴的「安全及服務質量部」2010年成立，據悉是因一架行走692線的九巴，2009年11月在將軍澳翻側，造成2死30多人受傷的車禍後，運輸署關注九巴為何沒有專責部門，負責安全風險管理及制定安全政策。運輸署回覆查詢指，「2009年11月在將軍澳發生巴士意外後，因應市民對巴士服務安全的關注，政府已即時與各主要巴士公司跟進，而巴士公司其後亦有就監察車長安全駕駛，加強改善措施。」

被問到九巴的「安全及服務質量部」近年消失，運輸署被質疑監管不力。署方回應指「有關巴士公司架構內個別部門的具體職能，屬公司的行政管理安排。」發言人其後補充稱：「運輸署強調重視巴士的服務安全，過去亦一直有因應需要要求巴士公司加強或引入相應的安全措施。」指據署方了解，九巴現時車長安全駕駛態度由九巴公司的「培訓及服務質量部」負責管理及監察，至於其他車務相關的安全事宜則由副車務總監管轄。 眾新聞早前報道提及，以往九巴的「安全及服務質量部」主管是向時任車務總監Owen Eckford負責，有時也會直接向時任董事總經理何達文匯報。

九巴在眾新聞報道刊出後，周五早上傳來電郵，提及以下措施：

「九巴一直有部門負責安全管理事宜，安全及服務質量部的職能一直存在並進行。在安全方面，2015年 ，相關職能升格由副車務總監及車務總監管轄，其中提升了用作記錄操作數據的『黑盒』，監察車長的行車情況，包括急煞、轉向、 加速等，如監測到車長駕駛不當會進行調查， 如核實會作出紀律行動。其他設施提升包括： 安裝駕駛提示器提醒車長保持安全行車速度， 以及下層車廂設有連續式扶手和扶手柱， 讓乘客可在巴士入口至出口車門之間暢通流動。2017年成立『培訓及服務質素管理部』， 部門職能加入包括為新入職及現職車長提供訓練及培訓課程， 及車長表現管理及監察，目的是進一步強化駕駛安全的訓練和監察。

服務質素提升方面，過去幾年， 九巴研發巴士到站時間預報服務系統，車廂提供充電插座及Wi- Fi上網服務，全線車隊配備低地台設施有助輪椅使用者及長者， 巴士站加設座椅便利長者及有需要人士。此外， 過去幾年車隊的機件可靠性持續改善，由2014年每行走44, 952公里發生一次故障，提升至2017年每行走61,033公 里才出現一次故障。九巴巴士車隊平均車齡，亦由2014年的11年降至2017年的7.5年。

上述例子說明了九巴重視安全， 不會因為資源或成本而影響安全運作。」
