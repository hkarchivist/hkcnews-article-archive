# 藍絲也懂得拒絕向太的捐款，民主黨呢？

【撰文：郭倫】   
 ***作者為 [中文及通識科網誌](https://web.archive.org/web/20210712014513/https://chinesereading.tumblr.com/) 作者，著有《圖解「今日香港」 》、《為甚麼我考不好中文》等。FACEBOOK專頁： [中文科閱讀理解應試技巧](https://web.archive.org/web/20210712014513/https://www.facebook.com/%E4%B8%AD%E6%96%87%E7%A7%91%E9%96%B1%E8%AE%80%E7%90%86%E8%A7%A3%E6%87%89%E8%A9%A6%E6%8A%80%E5%B7%A7-164354330288453/)***

日前民主黨黨慶，特首林鄭月娥出席其宴會，並捐了三萬元給民主黨，結果備受抨擊。民主黨主席胡志偉回應指林鄭月娥的捐款並不設條件，該黨會繼續監察和批判政府，不會因三萬元改變看法。民主黨接受林鄭的捐款，其實毫不令人意外，因為民主黨早前已接受過領展在黨慶場刊「落廣告」。

</figure>

七警捐款事件，我相信大家記憶猶新。當日七警入獄，藍絲組織敬言仁基金替七警籌款，向太和楊受成也有捐款，但是敬言仁基金董事會成員譚惠珠表示，已拒絕三份來自演藝界的捐款。為甚麼要拒絕？因為是捐款者是向太和楊受成，接受者是警察，不能因為一句「捐款不設條件」而接受。

當然，我不會天真地相信該批捐款會退回，只是可能令捐款者臉上無光，他們或者會用另一個方法捐款而已，但是起碼不是公開。

胡志偉聲稱民主黨會繼續監察和批判政府，但是會盡了多少力，只有天知、地知、你們自己知。只是你們一句「問心無愧」，憑甚麼要市民大眾相信你們？

有一句關於法律的名言：

> **Justice must not only be done but should manifestly and undoubtedly be seen to be done.**


公義不但要秉行，還要使其有目共睹，民主黨究竟懂不懂？

最可笑是民主黨副主席羅健熙認為事件對民主黨影響有限：「你請她來，她要舉手認捐好，捐款去獻捐箱好，我們阻止不了。」

對民主黨影響有限，這一點或許是事實，因為很多人對民主黨已經不予期望。信任，就是這樣一點一滴地失去。民主黨日漸萎縮，不是沒有原因的。

議員被取消資格，律政司堅持對雙學三子上訴，立法會被閹割，在這個劍拔弩張的形勢之下，民主黨仍然有這樣的判斷，叫市民怎樣相信你們民主黨？

林鄭只是用三萬元做一場戲而已，敬請民主黨退回那三萬「戲金」。
