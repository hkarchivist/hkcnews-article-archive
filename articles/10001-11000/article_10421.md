# 海洋公園連續兩年虧損逾2億　獲撥款3.1億發展教育旅遊

財政司司長陳茂波今日公布 [財政預算案](https://web.archive.org/web/20210712014047/https://www.budget.gov.hk/2018/chi/budget16.html) ，以多元經濟、投資未來、關愛共享為「三大目標」。其中，在支持旅遊業發展方面，政府宣布將於未來數年撥款合共3.1億元，以支持海洋公園發展教育旅遊項目，而海洋公園將於未來1年提供1萬張入場券予本地的中小學生。不過，全港有近71萬名中小學生，1萬張入場券可怎樣分配？

海洋公園近年業績倒退，入場人次持續第4年錄得下跌，在過去兩個財政年度更每年虧損逾2億元。海洋公園將會如何運用這筆3.1億元的公帑、怎樣發展教育旅遊？

根據 [教育局資料](https://web.archive.org/web/20210712014047/http://www.edb.gov.hk/tc/about-edb/publications-stat/figures/enrol-by-level.html) ，全港中、小學生各有約35萬人，總人數達70.9萬。換言之，海洋公園提供的1萬張入場券，僅夠供應1.41%中小學生。至於1萬張入場券怎樣分配，海洋公園回覆眾新聞指，園方現正在籌備相關計劃，稍後將公布有關細節。

其實海洋公園向來有提供 [學生團體票](https://web.archive.org/web/20210712014047/http://media.oceanpark.com.hk/files/s3fs-public/StudentGroupTickets_ApplicationForm.pdf) 優惠，中、小學生票價分別為80元及70元，每訂購10張中小學生團體票，可獲贈1張老師入場券。若以學生團體票價計算，1萬張中小學生入場券只涉及70至80萬元。如果全港中小學生都獲贈團體票，總額才5,320萬元。


|  | 學生人數 X 團體票價 | 金額 |
| --- | --- | --- |
| 小學 | 352,500 x $70 | $2,467.5萬 |
| 中學 | 356,600 x $80 | $2,852.8萬 |
| 總數 |  | $5,320.3萬 |


海洋公園獲政府撥款3.1億元，將如何運用呢？海洋公園未有明確回應相關撥款會用於哪些項目、各項目所涉及的金額，只表示園方有意強化教育旅遊，把海洋公園「打造成一個高科技的教育樞紐」，並計劃為公園的教育課程注入科技互動元素，促進STEAM，即科學 (Science)、科技 (Technology)、工程(Engineering)、文科 (Arts)和數學(Mathematics)，以及生物多樣性學習。

海洋公園為非牟利機構，早年由香港賽馬會資助及政府免費撥地建成，並由根據《海洋公園公司條例》成立的法人團體「海洋公園公司」管理，董事局成員由行政長官委任。現屆董事局主席為盤谷銀行資深副總裁、西九文化區管理局董事局成員孔令成；副主席是華人置業集團主席及行政總裁劉鳴煒。董事局成員中的政府代表有旅遊事務專員黃智祖，以及發展局常秘（工務）韓志強。

[海洋公園2016/17年度業績報告](https://web.archive.org/web/20210712014047/http://media.oceanpark.com.hk/files/s3fs-public/ophk_ar16-17.pdf) 顯示，近年公園有兩項大型重建計劃均由政府貸款，包括分期擴展海洋公園，政府提供一筆25年期（2031年到期）、固定年利率5厘的13.9億元附屬貸款，以及一筆20年期（2033年到期）、浮動利率相當於政府外匯基金財政儲備利率的22.9億元附屬貸款。

翻查資料，海洋公園近年營業表現倒退，2016/17年度的入場人次為580萬，按年跌20萬人次，是持續第4個年度錄得下跌。其中，入場人次以2015/16年度的跌幅最大，達18.9%，按年跌140萬人次，海洋公園在 [年報](https://web.archive.org/web/20210712014047/http://media.oceanpark.com.hk/files/s3fs-public/ophk_ar15-16.pdf) 中解釋，該年訪港旅客人數出現跌幅，令公園的入場人次受影響。

**海洋公園最近5年入場人次**


| 年度 | 入場人次 | 增幅／跌幅 |
| --- | --- | --- |
| 2016/17 | 580萬 | -3.33% |
| 2015/16 | 600萬 | -18.92% |
| 2013/14 | 740萬 | -2.63% |
| 2012/13 | 760萬 | -1.30% |
| 2011/12 | 770萬 | +8.45% |


海洋公園的收入近亦見萎縮，在過去兩個財政年度，每年更虧損逾2億元。

由於11歲或以下的小童，需由15歲或以上的人士陪同方可進入海洋公園。海洋公司向小學生派發免費入場券，或可帶動家長入場人流。假如1萬張門票中，有半數（5,000張）派發予小學生，再假設每個小學生均有雙親陪同入場，即合共1萬名家長，小學生連家長總數才1.5萬人，再加上另外5,000名中學生，亦只涉及2萬人次，是上年度的入場人次跌幅20萬的一成。眾新聞向海洋公園查詢預期計劃帶動的入場人次，園方未有明確回覆。
