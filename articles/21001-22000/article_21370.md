# 「反送中」被捕懶人包之「我無嘢講」

【撰文：陳零】   
   
 614 晚上，在Facebook出完個 po後，有朋友馬上私訊提醒，記得過幾日要刪文；加上有Telegram 群組管理員被警方以「串謀公眾妨擾」罪名拘捕，白色恐怖，人心惶惶。再講，今次參與「反送中」的示威者，不少是未滿18歳的學生，還有許多第一次上街的市民，任何情況下，最緊要是「prepare for the worst」。不如讀一讀義務律師(C) 的建議，其中最關鍵那句是：「我無嘢講。」

**問：在社交平台寫過或share過「反佔中」帖文、更換黑紫荊頭像等等，會否被秋後算帳？**   
 C：請如常生活吧。   
   
 **問：加入了WhatsApp「反送中」相關群組，是否應該退組、刪組，毀屍滅跡？**   
 **C：→ 請如常生活吧。**   
 → 不過，建議日後在群組留言，可以用「聽講」、「好似」等字眼，毋須句句都「我會去放火」那麼實在（而且只不過出於一時自 high）。   
   
 **問：「出得嚟（遊）行」，有甚麼可以保護自己？**   
 C ：  → 戴口罩  ( 理論跟「速龍小隊」無警員號碼一樣，無仇報。)   
 → 記得離開示威區才取下口罩。   
   
 **問：「出得嚟（遊）行」之前，除了帶水、雨傘、口罩，還有甚麼要預備？**   
 C ： → 把姓名和身份證號碼交給可信賴的朋友、律師或可信任的支援組織；一旦被捕，律師亦較容易在各區警署「撈」到你。   
 → 把重要的聯絡電話，包括律師及相關支援組織等，寫在紙上，好好收起，又或者寫到手上。   
   
 **問： 被警方拘捕，第一件事該做甚麼？**   
 C： → 要求通知律師，在律師到來前，任何情況下，都只說：「我無嘢講」，亦不要簽署任何文件。   
 → 未必一定可見家人，可請律師代通知家人。   
   
 **問： 警方在拘捕你後，是否有權搜身？**   
 C：→ 有，但搜身需由相同性別警員進行。   
 → 一般搜身是拍拍衫袋褲袋，除非懷疑藏有毒品或危險物品，才會要求除衫褲及內衣褲，而如有此情況，必須要求警員落簿記錄，保留日後投訴的權利。   
   
 **問：警方在拘捕你後，是否有權沒收手機？**   
 C： 有，但無權要求你解鎖，你可要求警方封袋處理。   
   
 **問：律師到來前，要做甚麼？**   
 C：→ 就所有要求，都要警員落簿記下，例如10:00至13:00，要求吃飯無飯吃，或者要求驗傷不獲處理，要求見律師話律師不會見你等等，方便律師寫到誓章，日後有機會呈上法庭。   
 → 如果警員無落簿，自己要想方法記下參考時間。   
   
 **問： 見到律師前，會遇到甚麼情況？**   
 C ： → 你會好驚， 然後向警員如數家珍般，和盤托出參與的來龍去脈。請謹記， **任何情況下， 只需說：「我無嘢講。」**   
 → 警員會有不同技巧盤問你，請謹記， **你只需說：「我無嘢講。」**   
 → 沒收手機後，你會好驚，但你是有對外聯絡的權利，請冷靜找出那張緊急聯絡電話的紙，然後聯絡律師。   
 → 不要以為你只可以打一次電話，你是有權隨時要求見律師。   
   
 **問：律師來到，會問甚麼、幫到甚麼？**   
 C： → 基本資料，包括姓名、年齡、身份證號碼等（18歲以下或通知有跟進個案的社工）   
 → 可代通知家人、朋友   
 → 有無受傷或不適 (例如給警察踢了幾多腳，可以日後投訴)，可要求到醫院驗傷   
 → 額外處理事項，包括有否被沒收眼鏡 ( 連律師個樣都看不清楚)、可有被搜身、搜屋等   
 → 有無投訴 (無飯吃、無水飲、無得去洗手間、無毛氈禦寒等等)   
 → 特殊需要 (例如定時需要服用藥物、聽障等)   
   
 **問：落口供時，應該如實道來嗎？**   
 C ：→ **只須說：「我無嘢講」 。**   
 → 如果夠硬淨，可以自己落，記得只說：「我無嘢講」；如果驚，就請律師陪同。   
 → 記得取口供副本。   
   
 **問：是否要求保釋？**   
 C ：→ 建議「踢保」（即不接受保釋，「坐爆」48小時）；電視、電影都睇得多，一般情況下，警方不能拘留你多於48小時；坐夠48小時，身份就回復 innocent person，亦等於無被拉過；而48小時是由警方說拘捕時間開始計，一般是上警車起跳，如果之後要到醫院，時間亦計在內。   
 → 不建議要求保釋，因為需要定時返警署報到，帶來不必要麻煩。   
   
 **問：不保釋期間，會遭受甚麼對待？**   
 C ： → 警方在拘捕你後，有權沒收手機；在沒有手機的48小時，正常的你應該會悶到發癲。   
 → 會安排到羈留室（俗稱臭格），內有一張 bench（水泥椅）和一個蹲廁，不是密封的。   
 → 可要求吃飯、飲水、毛氈（因為冷氣大）。   
   
 **問：家人來到警署，應該注意甚麼事項？**   
 C：→ 你只是被拘捕，來落案檢控、未定罪，所以不要問律師你會否有案底、要坐幾耐、會否影響到海外升學（可能出世以來都未諗過）、會否影響移民申請（同樣可能從未諗過）等。   
 → 家人請謹記配合被捕者的那句： **「我無嘢講。」**   
 此時此刻，沉默是金。   
   
 **問：假如朋友或家人懷疑你被拘捕，可以怎樣做？**   
 C： 通知律師，提供全名及身份證號碼（所以咪話在參與示威、遊行前要交低），一般律師團隊會到各區警署查詢，大約半小時可以找到。   
   
 註：在此感謝義務律師提供的建議，但有關建議並非正式法律意見。如有需要，請向閣下的律師諮詢。
