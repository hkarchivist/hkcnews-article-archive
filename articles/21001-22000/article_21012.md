# 【引渡修例】反修例聯署遍及社區、海外　六四前夕簽名突破23萬人次

協力：實習記者黃思銘

政府提出修訂《逃犯條例》，引起極大民意反彈。學界、業界、社區群體及至海外港人相繼發起聯署，表明反對修例。眾新聞自5月30日起搜集網上反修例聯署，每日統計聯署數據，製作成 [「反引渡修例聯署合集」](https://web.archive.org/web/20210924162102/https://bit.ly/2JYghFf ) 互動專題。截至6月3日晚上8時，合集收錄332項聯署的數據，簽名總數突破23萬人次。

<img src="https://web.archive.org/web/20210924162102im_/https://www.hkcnews.com/news_新聞/hk-discuss/2019/06/逃犯條例-網上聯署-20190604235943_1dba_large.jpg" alt="">
<figcaption>六四前夕，網上逾330個反引渡修例聯署，合共錄得超過23萬人次簽名。眾新聞製圖</figcaption>
</figure>

眾新聞從網上搜集不同界別團體發起的反對修例聯署，紀錄成聯署清單，再運用電腦程式，自5月30日起，每晚擷取清單上各項聯署的網頁內容，並紀錄各聯署公布的最新簽名數字，如聯署發起者沒有公布具體數字，記者則自行點算該網頁上簽名數字，作整合統計及分析。眾新聞會將新聯署加入清單，持續更新合集。由於一個人可以同時參與不同聯署，眾新聞無法核實重覆的人數，故以簽名人次為統計單位。

網上聯署浪潮最初以學界及專業界別為主。號召本地大學及中學師生校友、個別業界人士簽名反對《逃犯條例》修訂的多項聯署，5月中、下旬在facebook「洗版」。聯署戰線旋即擴大，至小學、特殊學校、社區層面、宗教組織、海外群體等，例如在5月28日， [全港九新界離島師奶](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/1uegu7LxhhBzdk2yiGMpdut-pXPiepZxhSZO2aPuYWl4/viewform?ts=5ced5787&edit_requested=true&fbclid=IwAR24rPt2tZcPDnLBt4i52hX6vUX3S74GlyIYqsX-v30CWtlhesK-tNfzDjI ) 、 [英國留學生](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSfdinABRL3XOkCR5h9HxJGK9sQnL2rNd8QPQZjOtQ8kbkQ2ww/viewform?fbclid=IwAR0D77qva8Zpr52iRfe6__7wMZaq4RjzBuNOTS3JwGTsje7TEe-UcD9lvLQ) 發起聯署；5月29日，盲人學校 [「心光學校」校友](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSdq2P5JvmLQnM8yUeOBmUy9IswRgeolweYGHxGSEsX5yJDDdg/viewform?fbclid=IwAR0-uTensy8vXsVPwKSKg_Cl9NwZM_xCWSZBWGo-vzVl8F0c0_acx_vhS_E ) 、 [大埔區居民](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSeTLp96thFyewGSTqgkB-nhtzGq2dvcAdURpFz57ifp4qGlwA/viewform ) 、 [澳洲港人組織](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSekf8Bj9xOHcqqoYGOuG_ChrbLRTDyqnWgLA3sbZQLp1vazSA/viewform?fbclid=IwAR2w8UE3CazkGi6ZY32t7k0CAKHeWsvZ9wjtsbSJKTk7MtJ2emmDFd8JGmY) 分別號召簽名；5月30日， [沙田區居民](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSdI6GvG0hmBX1heA_WfCuANJzerJJPyS-DDgo1Qj55TLsU3BQ/viewform ) 、 [賽馬界](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSeacLFZDMUDz2ghUqXcRM7I73Pf_rNTn1MCKfuot-T-BUnVSQ/viewform ) 參與聯署行列；6月1日， [「一群在港大陸新移民」](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLScsdw-szCWuPo91mG3QeKjzzqM68l_NIAdpmf1OT3LkZ3Yy_A/viewform?fbclid=IwAR311edUCNs5PsWMH-Hu37bkiXV-3Ft2MTN83kWHhz_ZFN_8KJXnc995d24 ) 加入發聲。

截至6月3日晚上8時， [「反引渡修例聯署合集」](https://web.archive.org/web/20210924162102/https://www.hkcnews.com/FOO-petitions/#/) 共紀錄332項聯署，當中以中學校友及師生聯署居多，共258項；其次是本地大專院校，有33項；其餘聯署包括小學、特殊學校、業界人士、宗教、社區及海外群體，合共41項。聯署聲明除了表達反對修例，大多同時呼籲聯署者出席6月9日、由民間人權陣線主辦的 [反「送中」大遊行](https://web.archive.org/web/20210924162102/https://www.facebook.com/CivilHumanRightsFront/photos/a.680202458693815/2281134281933950/?type=3&theater) ，部分表示會登報明志。

據眾新聞搜集的數據，目前以理工大學的聯署錄得最多簽名，超過1萬名學生、校友、教職員響應 [聯署](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/e/1FAIpQLSfpjEcJZoDgS0AMtyXGDI72E-JJt0HEQ9yFhgQpT6LAmdiniw/viewform?fbclid=IwAR0P61RUX7oLV_3-FQRwz6zOfFlJVD7j_HTmS1mm5jAACKW9kDXZTDdHl2M) 。聯署由理大學生校董李傲然；學生會會長廖建鈞、外務副會長李晃榮、外務秘書周美芝聯合發起，Google form共羅列5,498個名字，及其畢業年份、所屬學系的資料。

李傲然周一（3日）下午向眾新聞表示，他們四人於5月27、28日發起聯署。他形容，作為學生代表、學生會幹事，他們發起聯署是「應有之義」，「因為我哋都一致認為，呢條條例，如果修訂成功，對全香港人都係百害而無一利。」雖然網上顯示5,498個簽名，但李傲然指，截至他受訪之時，其實已有逾萬人實名聯署，不同年份畢業、不同學系的學生校友均有參與，「Google form（展示資料的空間）已經爆咗，唔俾我哋再加名落去。」理大師生、校友、教職員仍可繼續參與聯署，惟李傲然等發起人已暫停更新聲明顯示的聯署名單。眾新聞根據李傲然的說法，紀錄理大聯署在6月3日的數字為1萬人次。

李傲然表示，希望透過聯署，喚起同學、校友對《逃犯條例》修訂的關注，並借助簽名在公眾層面鼓勵更多人參與反修例，「我哋個睇法係，做得幾多得幾多，盡力做，希望可以喚醒多啲同學對呢件事嘅關注，甚至借呢個機會，喺校內宣揚我哋聯署想帶出嘅訊息。就我角度而言，同學好願意踴躍參與，反映佢哋對《逃犯條例》修例嘅關注，我哋嘅同學、校友對呢件事相當重視，亦都令到呢件事喺社會上醞釀咗咁耐都係得到公眾關注。」

李傲然提到，除了網上聯署，他們還有計劃籌辦一些後續行動。「嚟緊我哋都會同同學同行，喺6月9號都會一齊上街，希望借上街展示群眾嘅力量，亦都希望藉此話俾香港政府聽，我哋好多人都唔同意今次嘅修訂。」

眾新聞錄得簽名數字排行第二的是「 [全港九新界離島師奶反送中聯署](https://web.archive.org/web/20210924162102/https://docs.google.com/forms/d/1uegu7LxhhBzdk2yiGMpdut-pXPiepZxhSZO2aPuYWl4/viewform?ts=5ced5787&edit_requested=true&fbclid=IwAR24rPt2tZcPDnLBt4i52hX6vUX3S74GlyIYqsX-v30CWtlhesK-tNfzDjI) 」，截至6月3日有5,931個聯署。發起人黃彩鳳上周接受眾新聞 [訪問](https://web.archive.org/web/20210924162102/https://www.hkcnews.com/article/20896/黃彩鳳-師奶-師奶反送中聯署-20896/) ，表示她在5月28日看到母校反對修訂《逃犯條例》聯署後，認為該聯署聲明未有寫出她「心入面嗰啲嘢」，遂聚集一眾師奶朋友的意見，寫出訴說師奶心聲的聯署聲明，內容得到不少網民讚譽，當中以「得20日諮詢？ 一次月經嘅週期都不足呀！」、「師奶罷工，暴政必亡。」經常獲「貼堂」。

黃彩鳳留意到，其聯署觸發更多自發行動，「有個朋友畀咗意見之後，就係咁喺自己Facebook講，話啲人攻擊彩鳳乜乜乜，佢自己又出一個post。大家都好野貓式，就住呢件事elaborate，我哋真係無組織。甚至乎我見有啲人留言話，不如印（聯署聲明）出嚟我哋擺街站啦，我唔知會唔會咁樣做。就係大家自己再演繹，全部都係一個網絡咁散出去。」

律政司司長母校嘉諾撒聖瑪利書院，師生、校友聯署長居聯署榜前十名，截至6月3 日錄得2,100個簽名。聯署由Alison Hui及Rebecca Ho兩名校友發起聯署，聲明批評政府漠視港人在多月以來就修例反映的擔憂及疑問，以高傲的態度一意孤行。「特首以及校友鄭若驊司長並沒有提供合理理據解釋為何要急切推行修訂《逃犯條例》。對於香港管治團隊繼續無視香港價值及民心所向，我們深表遺憾及痛心。」
