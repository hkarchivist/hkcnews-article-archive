# 大數據系統顯示「反送中」遊行人數逾75萬人　慎防「佔中2.0」騎劫遊行

#   
 關於6月9日的「反送中」大遊行有多少人參與，事前沒有任何機構作出預估，民陣或一些召集人的目標設定30萬人。為此我嘗試以專門以大數據追蹤趨勢移動TMA（TRENDING MOVING AVERAGE）系統進行演算。

#   
 系統對比了2013年至2019年的6年數據，顯示這次「反送中」遊行人數有機會超逾75.5萬人次，幾乎肯定超過2014年71大遊行時的51萬人。

> #   
>  據民陣在2014年估計，當年71參與遊行的人數達到時51萬人，以港大民意研究計劃及警方的的估算則分別達到17.2萬人及高峰期9.86萬人。TMA系統估算當日遊行人數介乎40萬至53萬人。值得留意的是，系統顯示今次「反送中」遊行的人數遠高於2014年7月1日，介乎48.7萬至75.49萬人，成為近15年以來最多人上街的一次。

#   
 TMA系統對比了遊行前一天以「遊行」作為關鍵詞在GOOGLE搜尋的數據，再進行交叉對比及調整，得出介乎48.7萬至75.49萬人的數據。

#   
 對於遊行或創下15年以來新高，我不感恩惠，事關遊行人數多少與政府是否聽從民意沒有直接關係，最大機會是「意見接納態度依舊」，相反激進組織或挾這75萬人次的遊行而提出激烈訴求，包括重演2014年7月1日後的佔中行動，上演「佔中2.0」。

#   
 目前多個激進組織呼籲支持者在遊行後不要解散，以「不散」作為口號，並強調有重要事情宣布，以過激行動挾遊行以令政府，騎劫遊行，最終釀成流血衝突，這才是我最憂慮的事情，還請各方三思，而遊行人士請注意個人安全，以免遊行和平訴求的目的被盜。
