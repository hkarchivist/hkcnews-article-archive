# 怎樣練好太極拳？

【撰文：張達揚師傅】   
 ***陳氏太極拳第二十代傳人***

俗語說：無規矩不能成方圓。練太極拳應一招一式，由大圈到中圈，由中圈到小圈，由初級到中級，由中級到高級，應循序漸進，就是這個意思。拳論中講：「由著熟而漸悟懂勁，由懂勁而階及神明。」練拳可分為三個階段：著熟至懂勁至神明。

第一階段練習，就是著熟，此正也就是練架子之意。先將拳架一招一式地學會，並反覆練習，要求動作規範、定型，以柔為主、柔中帶剛，在練習每個動作時，該柔則柔、應剛則剛。在練發勁時，要體現出鬆、快、猛，也就是每個發勁動作必須在放鬆的基礎上，迅速將勁發出，要具有速度和威力，不論是身體那一部位，只要用發勁，均要體現出蓄而後發。當然初學者剛開始練習時不會放鬆，也鬆不下來，勁也發不出去，有剛無柔是必然現象，防治之法是要全身放鬆，同時又要做到鬆而「掤勁不丢」。

陳家先輩講「掤勁不丢，周身如同一個充滿空氣的氣球，外行飽滿，內氣鼓蕩，筋骨要鬆，皮毛要攻，周身貫穿，虛靈在中」。這就是以外形引內氣的練習方法，如果沒有這段柔中帶剛的練習，周身關節鬆不開，內勁就引不動。在這段練習中，勢頭要細心揣摩，承上啟下時要處處留心。不揣摩就不會明白其中的道理，不留心則來脉不真、轉關不靈，必須經過千錘百煉，剛歸之於柔，柔造至於剛，剛柔得中，方見陰陽。若此段成功則具體表現為：「金剛搗碓聲似雷，掩手肱拳帶風旋；二起上縱輕靈捷，跌岔舖地起穩健；獨立蹬腿如釘立，平縱穿飛遠輕快」。下盤穩如泰山。手的動作，由順變逆，由逆變順，必須做到沉穩不飄。

第二階段練習，就是懂勁。勁指氣，要辨明濁氣與清氣，濁氣下降降於湧泉，清氣上升升於百會。運用丹田之氣，內外結合，以外形引內氣，以內氣推動外形 ; 需知氣不到，形是寂然不動，氣到了，形是隨氣而動。要上虚下實，輕沉兼備，逢上必下，周身有上升部位必有下沉部位。腰以上螺旋上升，腰以下，螺旋下沉。上有虚領頂勁，下有氣沉丹田，五指抓地。對拉拔長，穩固下盤，襠走鍋底型。逐步作到上肢如風吹楊柳，下盤穩如泰山。手的動作，由順變逆，由逆變順，必須做到沉穩不飄。外柔內剛，連绵貫穿，圓轉自如。第二階段主要是練習肩、肘、胯、靠的功夫 ; 兩臂開展之中，有相係相合之意感 ; 而肩、胯、肘、膝、手、足之間要有六合規矩 ; 敞胸、敞襠，都是開展有餘，緊湊不够，脚該扣未扣，手型散也不可，四肢總須保持掤圓，掤勁不丢 ; 處處做到開中有合，合中有開，要做到舒展之中有團聚之意。兩膝兩足時常保持裡合。

第三階段已經是：「千回萬轉多多演，功到熟時巧自生」，方可完全剛柔相濟，渾然一圓之狀態。由中圈進入小圈，此時功夫已經滲透臟腑、肌膚毫毛之內，完全是靜如山岳，動如閃電之態了。練拳時柔和自然，交手時如迅雷不及掩耳，意到勁隨。至此階段功夫，太極已經練到化境了。
