# 元朗區議會抽起撤回修例動議　泛民質疑中聯辦、政府干預議會

民間反送中抗爭行動如火如荼，泛民主派區議員亦紛紛提出在區會上討論修例事宜，惟在多區均受阻撓。今日（25日）舉行的元朗區區議會大會，原訂討論由民主黨區議會黃偉賢提出、要求政府撤回有關逃犯條例修訂的動議，惟元朗區議會主席沈豪傑今早突然抽走該項動議，事前未有諮詢或知會區議員。泛民區議員在會上質疑是中聯辦及港府施壓，主席未有正面回應。

北區早前有區議員提出譴責政府強行修例的臨時動議，因建制派大比數反對而遭否決。大埔及西貢分別有區議員提出反對修訂、譴責警察開槍的動議，但遭兩區區議會主席拒絕，無法在7月的會議上討論。

民主黨元朗區議員黃偉賢指，他早於3個星期前，已提出促請政府撤回《逃犯條例》修訂的動議，得到民主陣線麥業成和議，有關動議已排定於今日（25日）舉行的區議會大會上討論。元朗區議會網頁上載的議程，截至下午4時仍可見該項動議。不過，會議今早9時半開始前，區議員在現場收到的議程，卻抽走了黃偉賢的動議。元朗區議會傍晚再 [更新議程](https://web.archive.org/web/20210712030559/https://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/dc_meetings_agenda/Agenda_25062019_chi_V3.pdf ) ，才將移除有關動議。

主席沈豪傑在會上解釋：「政治問題留番喺其他政治平台討論，我唔希望一個推動地區事務嘅平台，淪為一個政治角力嘅戰場。現在我哋最緊要係向前看，修補社會挪裂，停止內耗，讓社會盡快回復平靜。大家重新出發，重建互信，以一顆包容同體諒嘅心，一齊做好地區嘅民生工作。」麥業成在會上問他是否受到來自政府、中聯辦的壓力，沈未有正面回應，僅表示：「我作為主席，亦都要頂住呢啲壓力，就算係爭議性事情，我都必須要有個決定。」數名泛民區議員隨即離場抗議。

黃偉賢接受眾新聞查詢時表示，早前有聞全港區議會都不容討論修例事宜，故他在上星期特別致電沈豪傑查詢其動議是否可以如期討論，黃引述沈當時回應指：「未有決定。」

黃偉賢又指，他原本獲批於上一次、4月16日的大會上宣讀其反對修例的聲明，惟該次會議在他宣讀聲明前，建制派區議員「拉隊離場」，導致流會，主席當日承諾下一次（即今日）大會處理該次會議未完成的議案，包括黃偉賢的聲明。不過，在今日的大會上，黃偉賢並未獲准宣讀聲明。此外，麥業成聯同無申報政治聯繫的杜嘉倫、陳美蓮，上周五（21日）向元朗區議會提出在今次大會上宣讀聲明，今日同樣不獲准宣讀聲明。

黃偉賢指，他做區議員30年，過往只要預早一個工作天向元朗區議會入聲明，便可以在會議上宣讀聲明，強調麥、杜、陳的做法合乎會議常規。惟主席以黃、麥、杜、陳的聲明均關乎反對修例，一律否決，未有具體交代原因。

元朗區議會主席沈豪傑填報為律師，沒有申報政治聯繫。黃偉賢懷疑，沈今次打壓討論是受到政府或中聯辦的壓力。「佢（沈豪傑）係粗暴地、濫權地取消會議常規賦予我哋嘅權力，佢都話有壓力，不過唔肯透露邊度有壓力。我估計就係，能夠郁到18個區議會主席，一係政府，一係中聯辦，冇其他機構可以逼到18個區議會主席都就範。」18區區議會主席上月曾 [聯署](https://web.archive.org/web/20210712030559/https://www.inmediahk.net/node/1064243) 支持《逃犯條例》提上立法會大會討論。

6.12金鐘佔領翌日、13日，北區區議會大會上，民主黨陳旭明提出臨時動議，要求強烈譴責政府強行修訂《逃犯條例》，漠視6月9日103萬人和平請願的訴求，並於6月12日向要求撤回修例的群眾施放催淚彈及開槍，導致多人受傷，指政府漠視民意、罔顧人命，行動可恥，建議予以強烈譴責。該項臨時動議獲黨友林卓廷、新民主同盟陳惠達、申報獨立的劉其烽和議。林卓廷指，《逃犯條例》修訂在社會上的爭議非常嚴重，嚴重威脅香港的法治及人權狀況，情況危急，故希望在區議會會議上作詳細討論，表達北區區議會的意見。北區區議會主席、民建聯蘇西智在會上表示，根據會議常規，如得到過半數議員同意，方能納入議程。陳旭明的動議最終在4票贊成、16票反對、1票棄權下被否決，無法在區議會大會上討論。

西貢及大埔區議會，早前亦有區議員提出譴責警察開槍、反對修例的動議，本來可以在7月2日及4日的大會上討論。不過，提出譴責警察動議的范國威在 [fb](https://web.archive.org/web/20210712030559/https://www.facebook.com/124853124283126/photos/a.1904932052941882/1904932112941876/?type=3&__xts__%5B0%5D=68.ARDjunj_vSXrAB4L0xvmRgkZHW_viu5IAjwmYP2UdgaX-arMOabpkzb4aeh2RRnsiNq1njOi1ArcJlHffb7aSTlamTZS9GlqSNTPiLoCtZJmQhYSlPnuNEgG8Km7wn1mr1uuFE5wkcLm0_3SbZ5xWIe5Hctx-KowOPMr8o_-d9PzMGVplr8MGf4zbaafS_xLhnrh6q-K4LymkAdpAweHMzzw-_chGX0zKcecUSCgsZ3rziwMkX6XsYE4Jf5yoyLeWiX9XvfFqKsC3talxSY0kvB2OZKmXPpwss8xYG6hJpRlKgYN4-HhT3d_Wiqf9V7GBToXBWGx84-yNSjLudvQoc3C_w&__tn__=-R ) 指，西貢區議會主席吳仕福以議員可向警察投訴科投訴為由拒絕將其動議納入議程。據 [《東方日報》報道](https://web.archive.org/web/20210712030559/https://hk.on.cc/hk/bkn/cnt/news/20190618/bkn-20190618215855740-0618_00822_001.html) ，大埔區議會主席黃碧嬌指警方早前的記者會已經回應了區議員大部分問題，認為不宜在大會討論有關動議。
