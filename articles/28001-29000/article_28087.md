# 【武漢肺炎】本港單日增48宗　七成人曾外遊、多留學生　跨黨派促交代全面封關

武漢肺炎全球肆虐，本港昨單日新增 [48宗確診個案](https://web.archive.org/web/20210918144203/https://www.info.gov.hk/gia/general/202003/20/P2020032000843.htm) ，累計257宗。新增個案涉及26男22女，年齡介乎4至69歲，當中35人曾於潛伏期有外遊紀錄，包括多名留學生，也有規劃署員工、空中服務員等。   
   
 雖然政府周五凌晨起已經訂立措施，強制從中國以外的所有地區的抵港人士進行14日檢疫，但措施是否需要加強，甚至效法其他國家全面封關，禁止所以非香港人入境？

港大感染及傳染病中心總監何栢良形容，香港已進入戰時狀態，認為應安排返港人士到酒店作獨立隔離。公民黨立法會議員郭家麒促請政府全面封關，即所有非香港居民不能入境。立法會跨黨派議員要求港府交代會否全面封關，禁外國人入境。

食物及衞生局局長陳肇始僅表示，當局會繼續監察，不排除會加強現有的措施及禁止非香港居民入境。

衞生署衛生防護中心傳染病處主任張竹君公布，截至昨下午4時，新增確診個案已有48宗，年齡介乎4歲至69歲，大部份有外遊，多數是歐洲，還有美國、加拿大、新加坡、菲律賓、印尼、泰國等地。被問到會否參考一些國家，全面禁止非本地居民入境，張竹君回應指，48名確診者中，只有1人是轉機的外籍乘客，其他人全部持有香港身分證，指即使封關也未必能幫助減低確診宗數，只說希望巿民減少社交活動。

被問到應否恢復在家工作的措施，張竹君表示，如私人企業能在家工作當然好，而公務員在家工作與否，視乎政府安排。

港大感染及傳染病中心總監何栢良，態度較強硬，建議港府安排返港人士到酒店作獨立隔離，又呼籲市民未來2至3星期不要到健身室或酒吧等場所，建議業界在未來2至3星期停業。他認為，無戴口罩者，應被禁止乘坐公共交通工具或進入大廈。他甚至形容，香港現時是「病毒大軍壓境」，已進入戰時狀態。

本身是泌尿外科醫生的公民黨立法會議員郭家麒形容，新增確診數字驚人，但家居檢疫和電子手帶措施形同虛設，要求政府全面封關，即所有非香港居民都不能入境，而每位返港的香港居民都要接受病毒測試，否則的話，短時間內會出現第二波社區爆發。郭家麒又說，接獲很多求助指，酒店拒絕接待返港留學生，認為政府應徵用酒店供回港人士隔離。

另外，民主派全體22位議員聯署要求立法會主席梁君彥，因應新一輪疫情危機，暫停所有與疫情無關會議。下周一、二沒有立法會會議。

建制派在昨日的立法會衛生事務委員會會議上，要求政府交代會否考慮封關，禁止非香港居民入境，包括民建聯葛珮帆、陳凱欣。《文匯》昨 [刊登社評](https://web.archive.org/web/20210918144203/http://paper.wenweipo.com/2020/03/20/WW2003200001.htm) 指，政府應採取更嚴格的口岸檢疫，包括果斷拒絕除內地、澳門、台灣以外的其他國籍人士入境；要求酒吧、健身中心等非售賣必需品的商戶停業；要求市民減少社交聚集；指政府「不要再「研究」，要當機立斷馬上做、今天就做。」

陳肇始回答議員提問時僅表示，過去兩星期大部份的確診個案，屬外國返港的香港人，當局會繼續監察整體情況和非香港居民的入境數字。她又稱，非港人的入境人數一直有下跌，但不排除會加強現有的措施及禁止非香港居民入境。

**部分已經實施「全面封關」的鄰近國家／地區如下：**


| 台灣 | 所有外籍人士不得入境（包括中港澳） |
| --- | --- |
| 澳門 | 禁止所有非澳門居民入境（中國、香港、台灣及外地僱員除外） |
| 菲律賓呂宋島 | 境內的機場全面關閉 |
| 馬來西亞 | 禁止外國遊客入境 |
| 新西蘭 | 禁止所有非新西蘭公民和永久居民入境 |
| 澳洲 | 禁止所有非澳洲公民和永久居民入境 |
| 俄羅斯 | 全面禁止外國人入境 |
| 歐盟27個國家 | 除了歐盟公民以外，一律禁止入境歐盟 |
