# 【數說國際疫情】歐洲人口最老化國家　意大利武漢肺炎死亡率達9%

武漢肺炎席捲歐美。繼中國之後，意大利、美國、西班牙、德國、伊朗、法國的累計確診個案數字相繼破萬。其中，意大利的死亡個案數字，上周已超越中國，本周二（24日）更突破6,000宗，死亡率高達9%，比中國的4%大幅高出5個百分點。

數據反映，老年人口是疫情的致命傷。意大利的死亡個案中，有一半是80歲或以上的長者。意國本身有近四分一人是65歲或以上，人口年齡中位數為44.3歲，是全歐洲人口最老化的國家。首先爆發疫情的中國，確診個案年齡及人口結構均較意大利年輕。

中國的確診個案數字2月起漸趨平穩，死亡數字亦在3月明顯緩和。然而，比中國遲兩個多月爆發疫情的意大利，雖然確診個案未及中國，但死亡個案已於3月中超越中國，且未見緩和跡象。緊隨意大利，美國及西班牙的確診個案數字亦見 [指數成長（exponential growth）](https://web.archive.org/web/20210712041338/https://doctordaddysoccer.blogspot.com/2020/01/maths.html ) 趨勢。

從累計死亡個案數字圖表可見，意大利的疫情惡化速度明顯高於中國。以各國突破第100宗確診個案起計，不足1個月，意大利的確診個案雖然少於中國，但死亡個案已達中國的3倍。對比中國及意大利，美國及西班牙目前仍處於爆發早期，死亡個案相對確診個案的比率（死亡率）往往小於疫情中後期的數字，現時數字難以作準。

### 為何意大利的死亡率遠高於中國？

若從意大利的死亡個案著手了解，可見長者是意大利的致命傷。意大利疾病防控部門統籌的公共衛生組織 [EpiCentro整合的數據](https://web.archive.org/web/20210712041338/https://www.epicentro.iss.it/coronavirus/bollettino/Infografica_22marzo%20ENG.pdf ) 顯示，該國的死亡個案中，有一半是80歲或以上的長者。根據3月22日的數字，意大利4465宗死亡個案中，有1806宗介乎80至89歲，佔整體死亡個案最大比例，為40.4%，其次是70至79歲，佔35.5%；愈年輕的群組，佔比愈低，50歲以下的患者群組，均佔不足1%。

愈年長的群組，死亡率愈高。在意大利，90歲或以上的確診個案中，有24.1%死亡；80至89歲的確診個案中，有23.3%死亡。個案死亡率隨年齡減少而顯著遞減，50至59歲的確診個案，死亡率為1.3%；30至39歲的患者，則有0.3%死亡。意大利未有錄得30歲以下的死亡個案。

對比意大利及中國的確診個案年齡分布，可見意大利的患者年齡，本身普遍高於中國的。 [EpiCentro的數據](https://web.archive.org/web/20210712041338/https://www.epicentro.iss.it/coronavirus/bollettino/Infografica_22marzo%20ENG.pdf ) 顯示，當地確診患者的年齡中位數是63歲，50歲以上的患者佔整體74%，最老的70歲或以上群組佔36.1%。

根據 [《刺針》發布的一項研究](https://web.archive.org/web/20210712041338/https://www.thelancet.com/journals/landig/article/PIIS2589-7500(20)30026-1/fulltext) 及 [中國疾病預防控制中心的資料](https://web.archive.org/web/20210712041338/http://weekly.chinacdc.cn/en/article/id/e53946e2-c6c4-41e9-9a9b-fea8db1a8f51 ) ，中國確診患者的年齡中位數是46歲，50歲或以上的患者佔整體53%，70歲或以上群組僅佔12%。

### 意大利北部死亡個案大增　院舍經理指多死者有病徵但無檢測

意大利的武漢肺炎確診及死亡數字雖高，但仍被質疑未反映疫情實況。 [路透社3月19日報道](https://web.archive.org/web/20210712041338/https://www.reuters.com/article/us-health-coronavirus-italy-homes-insigh/uncounted-among-coronavirus-victims-deaths-sweep-through-italys-nursing-homes-idUSKBN2152V0 ) ，引述意大利北部倫巴第大區城鎮貝加莫（Bergamo）的市長Giorgio Gori指，死於武漢肺炎的實際人數，遠比官方公布為多。據Gori指，今年3月首兩個星期，貝加莫有164人死亡，較去年同期56人死亡，多出108人；164人之中，只有31人被視為與武漢肺炎相關個案。

報道提到，意大利疫情爆發之際，當地護老院的死亡數字，正無聲無息驟增。這些死者並沒有接受病毒檢測，無從確定是否死於武漢肺炎，不會被納入官方疫情數字。路透社訪問倫巴第大區另一城鎮克雷莫納（Cremona）、一個提供460個床位的院舍經理Emilio Tanzi。 Tanzi形容，自3月初以來，當地有明顯而異常（anomalous）的死亡個案增長，單計訪問前一星期，其院舍有18人死亡，而他們都有呼吸困難、發燒等情況，由於死者沒有經過檢測，院舍亦不知道他們是否死於武漢肺炎病毒。Tanzi批評，疫情主要影響老年人，而護老院處於疫情的前線，卻得不到足夠的支援。路透社引述當地衛生部門指，根據政府的檢測及治療指示，只有嚴重呼吸道症狀（significant respiratory symptoms）的病人才接受住院治療。
