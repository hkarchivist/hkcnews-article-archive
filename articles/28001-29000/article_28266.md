# 【數說亞洲疫情】南韓及早密集驗病毒　每百萬人逾6千檢測　扭轉疫情新增確診數回落

武漢肺炎在意大利、美國、西班牙等國大規模爆發之際，鄰近中國、疫情較早發的香港、澳門、台灣、日本、南韓，未見明顯指數成長跡象。其中，南韓的新增確診個案數字，在2月底、3月初從高峰回落，是繼中國之後的第二國。

及早進行大型病毒檢測，以阻截隱形病人在社區播毒，並讓他們盡快接受治療，被視為南韓扭轉疫情的關鍵。截至3月20日，南韓已進行逾31.6萬次病毒檢測，每100萬人口就有6,148人接受檢測，檢測率屬亞洲第一，亦是世界前列。

武漢肺炎疫情在中國爆發之後，鄰近的香港、澳門、台灣、日本、南韓首當其衝，相繼發現確診個案，南韓更出現大規模爆發。

意大利、美國、西班牙等國呈現 [指數成長（exponential growth）](https://web.archive.org/web/20210712041406/https://doctordaddysoccer.blogspot.com/2020/01/maths.html ) 趨勢之際，香港、澳門、台灣、日本、南韓仍未有明顯指數式擴散跡象，疫情相對受控。

雖然南韓的確診個案數字，在2月下旬至3月上旬一度「火箭式」急升，但隨後已明顯放緩（圖二）。到3月12日，意大利的確診個案（7375宗）超越南韓（7314宗），兩國疫情到達分水嶺。及後，西班牙、美國的確診個案數字亦相繼超越南韓。

在同一時間軸上，香港、澳門、台灣的確診個案數字並不明顯。單看港、澳、台（圖三），則可見三地的疫情走勢發展相似，1月底至3月初呈現較小幅度的線性增長（linear growth），到3月中、下旬，增長幅度顯著擴大，未見緩和跡象。

### 為何南韓能夠控制疫情？

參照疫情最嚴重的中國及意大利，兩國確診個案中，都是以長者為高危群組。根據聯合國今年的數字，南韓的人口結構，較意大利年輕、較中國年長。不過，就確診個案分布而言，南韓患者普遍比中國及意大利都年輕。意大利逾七成患者有50歲以上，而中國有四成多的患者屬50歲以下，南韓更有近六成患者不足50歲。

參照疫情最嚴重的 [中國](https://web.archive.org/web/20210712041406/https://ourworldindata.org/uploads/2020/03/Covid-19-CFR-by-age-vs.-US-Seasonal-Flu-3.png) 及 [意大利](https://web.archive.org/web/20210712041406/https://www.hkcnews.com/article/28226/%E6%84%8F%E5%A4%A7%E5%88%A9-%E4%B8%AD%E5%9C%8B-%E6%AD%A6%E6%BC%A2%E8%82%BA%E7%82%8E-28243/ ) ，兩國確診個案中，都是以長者為高危群組。根據聯合國今年的 [數字](https://web.archive.org/web/20210712041406/https://population.un.org/wpp/Download/Standard/Interpolated/ ) ，南韓的人口結構，較意大利年輕、較中國年長。不過，就 [確診個案分布](https://web.archive.org/web/20210712041406/https://www.statista.com/statistics/1102730/south-korea-coronavirus-cases-by-age/ ) 而言，南韓患者普遍比中國及意大利都年輕。意大利逾七成患者有50歲以上，而中國有四成多的患者屬50歲以下，南韓更有近六成患者不足50歲。

與此同時，南韓的病毒檢測率，屬世界前列，高於歐美多個先進國家的檢測率。 [Our World in Data統計數據](https://web.archive.org/web/20210712041406/https://ourworldindata.org/grapher/covid-19-tests-country?country=AUS+AUT+BEL+BRA+COL+FIN+FRA+DEU+NOR+PAK+RUS+KOR+GBR+USA+VNM ) 顯示，截至3月20日，南韓已進行逾31.6萬次病毒檢測，相當於美國的三倍。 [每百萬人口計](https://web.archive.org/web/20210712041406/https://ourworldindata.org/grapher/covid19-tests-per-million-people?country=FRA+DEU+IRN+JPN+KOR+GBR+USA+HKG+ESP+ITA+China%20-%20Guangdong+TWN ) ，南韓有6148人接受檢測，而香港只有698人接受檢測。

據 [《紐約時報》](https://web.archive.org/web/20210712041406/https://www.nytimes.com/interactive/2020/03/19/world/coronavirus-flatten-the-curve-countries.html?_ga=2.101978118.609338834.1584975500-130615368.1541657002  https://www.bbc.com/news/world-asia-52001837 ) 及 [BBC報道](https://web.archive.org/web/20210712041406/https://www.bbc.com/news/world-asia-52001837 ) ，南韓每日生產10萬盒檢測試劑，每日接受檢測的人數近兩萬。該國早已建立一個公營及私營實驗室的網絡，足以支持大量市民進行檢測。

南韓當局視儘早及頻密的檢測為抗疫重點措施。南韓外交大臣康京和（Kang Kyung-wha）日前接受 [BBC訪問](https://web.archive.org/web/20210712041406/https://www.bbc.com/news/av/world-asia-51897979/coronavirus-south-korea-seeing-a-stabilising-trend?te=1&nl=the-interpreter&emc=edit_int_20200321 ) 時指，檢測是南韓低死亡率的關鍵：「檢測是至關重要的，因為可以及早發現，最大程度地減少擴散，並且可以快速治療被病毒感染的人。」為了避免醫院及診所爆滿，政府額外開設600個檢測中心，以便盡量快、大量地進行檢測。

[《紐約時報》分析](https://web.archive.org/web/20210712041406/https://cn.nytimes.com/asia-pacific/20200324/coronavirus-south-korea-flatten-curve/zh-hant/ ) 認為，除了大規模檢測，南韓能夠及早發現及治療感染，將輕症患者隔離到特殊中心，將醫院留給重症患者；而且政府大力宣傳，提醒、鼓勵公眾參與控制疫情的工作，都是南韓為外國提供的經驗。
