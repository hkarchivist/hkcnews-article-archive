# 中共極權統治下的威權香港

  
 【撰文︰李卓人】   
 ***作者為工黨常委***

今年十一國慶，剛好是在中共十九大之前。十九大必然大事吹捧習近平為核心，將是自毛澤東以來，又再一次大搞造神運動。對香港人來說，只有國殤，沒有國慶，年輕一代甚至不認中國為國家，談不上國殤還是國慶。但無論大家對中國有什麼看法，香港人肯定不希望香港成為另一個中國城市，不想香港沉淪為「威權香港」。為喚醒香港人認清威權新形勢，民間團體連同政黨團結起來，一起發動市民參與十一反威權大遊行，向極權中共在香港推行的威權統治說「不」。現在只有港人自救，香港才有希望，我們必須在社會各領域組織反威權力量，為持久戰作好準備。

### 威權就是要人民活在恐懼中

可能有人會覺得威權一詞較空泛，難以用作動員。大遊行發起團體考慮到這點，也會同時要求袁國強下台，抽出破壞香港的元兇，作短期目標。但我們更希望港人認識到威權已殺到埋身，必須以反威權作為未來民主運動的方向。

威權，就是高舉「依法管治」與《基本法》的至高地位，以法律壓逼港人追求民主、權利與尊嚴的意志；威權，就是以謊言圍攻政權的眼中釘，大搞批鬥；威權，就是用政治檢控拉人封艇，威嚇市民，製造寒蟬效應，威逼個人放棄追求公理，屈從專制自我噤聲；威權，就是要人民活在恐懼之中。

### 倒退還是進步？

香港人追求的是自由民主，但中共是極權統治，我們是倒退到新加坡威權模式，還是走向自由民主？以下表列，希望可以道出香港的沉淪危機。


|  | 極權中共 | 威權香港 | 自由民主 |
| --- | --- | --- | --- |
| 憲政 | 一黨獨大 | 三權合作 | 三權分立 |
| 政制 | 共產黨專政 | 半普選 | 真普選 |
| 法制 | 法律為鎮壓工具 | 法律淪為政治檢控工具 | 法治保障人權 |
| 統治手段 | 敵我矛盾鬥爭、批鬥異己 | 統戰、表態成風 | 表達自由 |
| 內部安全 | 維穩費高於軍費 | 維穩費參透政黨、社團，攪分化 | 槍口對外 |
| 學術自由 | 控制思想、大學奉行「七不講」# | 滲透大學、打壓學術自由 | 大學自主、學術自由 |
| 社會文化 | 謊言治國、人民不敢講真話 | 謊言扭曲真相、人民講真話抗衡 | 真相鬥爭謊言 |
| 結社自由 | 禁獨立工會、境外非政組法例監控非政府組織 | 自由結社但正以政治立場收緊 | 自由結社 |
| 網絡自由 | 以「九不發」*控制網絡、一人違法群組當災 | 設網上罪案組、查閱網上言論作刑事檢控用 | 百花齊放 |

### 文革式批鬥載耀廷

其中最恐怖的是中共那套敵我矛盾的鬥爭策略，最近就開動全副鬥爭機器對付戴耀廷，除刑法侍候外，更要封殺他的教席。而港大法律學院就更是眼中釘。幸好有何君堯的殺無赦的瘋狂言論，暴露出中共那套文革式鬥爭手法，提醒港人必須提高警覺，頂住威權，撐住香港。
