# 友善的愛爾蘭人

我的中學同學老袁告訴我，愛爾蘭人好友善 friendly。

我的太太退休不足四個星期，我們在2017年9月15日上午11時經倫敦抵達愛爾蘭的Shannon機場，一出機場，認識了十多年的愛爾蘭朋友Kieran已經在機場等我們。

Kieran 一早從他都柏林 Dublin 的家，駕車三小時到 Shannon機場接我們，我們拿了租車後，我跟隨他的車（我太太坐在他車上），再駕40分鐘到他哥哥的渡假屋 vacation home，跟著他告訴我渡假屋的鎖匙，保安系統密碼等，和我吃完午飯後，他再駕車三小時回Dublin 看他兒子的足球比賽。

<img src="https://web.archive.org/web/20220816082436im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/10/區西遊記-旅遊-愛爾蘭-20171002193845_29bf_large.jpg" alt="">

Kieran 可以不用花一整天來機場接我們，如我們住在美、加、英國朋友的家，他只要給我他渡屋地址，我用Google Map 可以駕車到他們的家。可見愛爾蘭人友善一班。

渡假屋在愛爾蘭西岸的 Clare郡，叫 [Liscannor](https://web.archive.org/web/20220816082436/https://en.wikipedia.org/wiki/Liscannor) 村，愛爾蘭西岸臨大西洋，飽食幾百萬年的大西洋西北風，加上大西洋在海邊拋浪頭，造成西岸很多漂亮驚人的臨海懸崖。很多崖高千多呎，沿海連續長達幾哩。

渡假屋就離愛爾蘭地標的 [Cliff of Moher](https://web.archive.org/web/20220816082436/https://en.wikipedia.org/wiki/Cliffs_of_Moher) 「海崖之母」只有5公里。

我們在渡假屋住5個晚上，再到首都都柏林 Kieran 的家住三個晚上。愛爾蘭人👍👍🙏🙏。

港人及華人喜歡買磚頭，這些渡假屋點賣？

我朋友渡假屋，自建，約2500呎實用，半畝地，可養牛、鷄。約卅萬歐元，或270萬港元。

而一房單位，8萬歐元有交易，或70萬港元而已。香港買房子手續費，也不只此數。

他們收入和香港如何比較？用一般GDP per Capita 國民人均收入為標準，以下乃Google 數字：


| 愛爾蘭 | US$61,600 |
| --- | --- |
| 美國 | US$57,466 |
| 新加坡 | US$52,960 |
| 香港 | US$43,680 |
| 英國 | US$39,899 |


愛爾蘭人均收入，比香港及英國高約一半，比新加坡和美國還要高，農村房價低，農村收入，可能只有平均的一半，或US$30,000一年，這房價，相對港、星、倫敦、紐約等超級大城市，仍然是相當affordable。

能習慣大風，小陽光，冷的天氣，愛爾蘭會是安居之地😝😜🤑。

有沒有平D的渡假屋選擇？

照片和我太太合照的女士，她手中也拿了一大包黑草莓，主動請我們吃，又可見愛爾蘭人的友善。👍👍👌👌

她住在我們渡假屋旁的 mobile home, 她告訴我們每年三月至九月常來此住，mobile home 每年十月至三月，mobile 渡假屋關閉，由於太冷，時常大風及大雨。她的 mobile 渡假屋，有二房，約500呎實用，每年（不是每月）二千歐元，港幣一萬八千一年租金。

這裡 mobile 渡假屋有近一百家，有一房，二房，三房單位，有D有海景，還加個露台，也有給小孩玩的地方。

在 mobile 渡假屋也碰到另一女士，她家在一小時車程外，她和家人夏天每週末均來住。

我以為地中海區國家的人如西、葡、意、希臘等人懂得享受人生，生在氣候不大好的愛爾蘭人，也懂享受人生。

怪不得以前奶媽時代的港督，在新界有渡假屋，香港英資大公司，以前也在大嶼山有渡假屋給他們的高級經理。

中國的海南島，現在已經發展了豪華的渡假退休房地產業了。

渡假屋外有一條行一小時，下山到海邊，再上小小山轉回渡假屋，首天下午行這小路時，遇到照片中的二人愛爾蘭女士，她們主動請我吃她們剛採摘的野生黑草莓blackberry。
