# What's in a name，有英文名又如何？

前特首梁振英昨日出席一論壇時， [大談](https://web.archive.org/web/20210712014929/https://www.facebook.com/notes/cy-leung/%E7%AC%AC%E5%8D%81%E4%BA%8C%E5%B1%86%E5%8F%B0%E6%B8%AF%E8%AB%96%E5%A3%87-%E9%9D%92%E5%B9%B4%E9%A0%98%E8%A2%96%E8%AB%87%E6%9C%AA%E4%BE%86/601156853580129/) 對下一代的看法，提到3名子女傳昕、齊昕和頌昕，都有一個「昕」字，寓意「希望」，「我預期他們一代的未來充滿希望」。他又強調， 三人都沒有取英文名字，也沒有拿外國護照，認為他們「未來不必有英文名字，更不要外國護照」。梁振英以全國政協副主席身份，談年輕人未來，有更好、更大的舞台，即是中國，但又扯到英文名字和外國護照，令人摸不著頭腦，思「歪」的話，會誤以為下一代要北上國家發展，最好不要改洋名， 要用英文名也只用名字的簡稱，例如C. Y.、C. H.，不要Carrie、Donald名字，不要外國護照， 拿了也最好放棄。

英文有一句名句，What's in a name（名字有什麼關係），來自莎士比亞話劇《羅密歐與朱麗葉》內一句說話，「名字有什麼關係？我們稱之為玫瑰的東西，叫別的什麼名字還不是照樣芬芳？」（What's in a name? That which we call a rose, by any other word, would smell as sweet.）有些父母十分緊張子女的名字，更準確的說，是他們的未來，希望他們開開心心的，或會把將子女名字叫「開心」；也有覺得叫什麼名字沒有什麼關係，也不想給太大的壓力， 要振什麼建什麼。香港超過九成人口是華人， 但有百多年殖民地歷史，加上地理因素，經濟、社會發展關係，華洋雜處，不少新一代港人有英文名字，例如Peter, John, Mary，大家習以為常，目的是方便溝通。

回歸後4位特首，董建華英文名用簡稱C.H.，但董太叫Betty。梁振英是C.Y.，其餘兩位特首曾蔭權及林鄭月娥，很多人都直接稱呼他們的英文名Donald和Carrie。 香港兩位中方器重的女士，前人大常委范徐麗泰，英文名是Rita， 另一位基本法委員會副主任梁愛詩，很多人叫她Elsie， 另一位親中紅人譚惠珠，英文名是Maria。歷屆和現屆政府班子不少人也有英文名字，親中政黨民建聯創黨主席曾鈺成學貫中西， 英文名Jasper有「珍貴石頭」的意思，在18世紀頗受英國上流社會歡迎，他的徒弟李慧琼，選擇有「星」味的英文名Starry。港人有英文名頗普遍，有歷史文化原因，也有實際功能考慮，喜歡或不喜歡用，屬個人決定，父母為子女決定，也是自己家事， 但梁振英講年輕人未來，提到「未來不必有英文名字，更不要外國護照」，容易令人聯想到，說到底，是「去殖民地化」的「老問題」。

1997年6月30日凌晨，英殖民管治已結束，至今快21年，仍有一些人對帶殖民地色彩的東西看不過眼，不能接受已過去的歷史事實；事實是估計超過1百萬港人擁有外國護照，有英文名字的更多，是歷史，也是香港特色，正如街道名字，皇后大道東、英皇道及地方名，例如麥理浩徑，都有殖民地色彩。每年人大政協兩會在北京召開，往往有愛國者提出建議，特區政府去除及減少殖民政府時留下的符號，由本港地名和道路名稱開始，大部份人的回應是4個字：毋須理會。原因很簡單，叫什麼名字有什麼關係？What's in a name？
