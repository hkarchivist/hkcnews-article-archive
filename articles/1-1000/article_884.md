# 【再拆港大民調】梁振英的0分和100分

《眾新聞》向港大民意研究計劃，索取過去4年半政府主要官員民調的原始數據，本周分析行政長官梁振英的民望。梁振英本周三將發表任內最後一份《施政報告》，他在過往多次民調的得分（0分最低、100分最高），均處於50分以下的低位。《眾新聞》分析數據進一步發現，受訪者給予梁振英最低0分的比率，在2014年佔領運動後升至兩成；給予他100分比率則長期維持在5%以下。相比之下，勢將參選特首的林鄭月娥及曾俊華，0分比率長時間低於一成，100分比率同樣維持約5%。不論是梁振英、林鄭月娥、曾俊華，他們三人各自所得的0分及100分，都各有民眾階層特點。

港大民研每個月進行的民調中，要求受訪者給予主要官員評分，0分代表「絕對唔支持」、100分代表「絕對支持」，50分則代表「一半半」。（註：支持或不支持是指續任其職位）

梁振英上任四年半以來，民望一直未見起色。港大民研在梁振英2012年7月上任時，首次錄得他的評分為53.8分，這是他上任以來所得的最高評分。

《眾新聞》分析了港大民調原始數據，顯示梁振英2012年上任之初，只有約一成人給他0分，但在2014年佔領運動後，持續有近兩成人給他0分，過去一年這比例更升至兩成三。受訪者中，以年輕人、高教育程度、高收入群組給予梁振英0分的比率明顯上升。


| 給予梁振英0分比率較高的受訪群組（約兩成半至三成） |
| --- |
| 18-29歲、30-39歲 |
| 大專以上教育程度 |
| 行政人員及專業人士、學生 |
| 月入超過4萬元、月入介乎2萬元至4萬元 |
| 居於私人樓宇 |


在18-29歲及30-39歲兩個年齡群組，給予梁振英0分的受訪人數比率最高。佔領運動之後，0分比率由以往在兩個群組都錄得的一成多，分別升至三成及兩成。過去一年，90後及80後給予梁振英0分的人數比率更升至三成。與此同時，在40至69歲之間合共三個年齡群組當中，錄得的0分比率亦增至兩成至兩成半不等。

若按職業劃分，在「行政人員及專業人士」和「學生」兩大群組，梁振英錄得的0分比率，較其他職業明顯高。專業人士給他0分的比率，在佔領後升至兩成半水平，過去一年更多次突破三成。

受訪者收入方面，佔領後在月入2-4萬元及4萬元以上的兩個群組中，錄得梁振英0分比率，由一成衝上兩成五，去年底更進一步貼近三成半。梁振英宣布不連任後，今年1月的民調結果顯示，他所得的0分比率輕微下跌，但仍達兩成二。

至於給予梁振英100分的比率，則由他上任至今一直維持在5%以下；給予他50分的比率，由2013年初高峰期三成多，下跌至去年底不足兩成。

給梁振英100分的受訪者，多是70歲以上長者、小學教育程度、勞動工人、收入低於1萬元及無收入者，以上群組給予梁振英100分的比率，高於其他組別1至3個百分點不等。梁振英宣布不連任後，100分的走勢沒有大變動。（註：每次民調給予100分者，只平均約佔總受訪人數約5%。）
