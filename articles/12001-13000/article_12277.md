# 【社運抗爭資料庫】一周整理　旺角衝突案梁天琦、盧建民暴動罪成

<figure>
<img src="https://web.archive.org/web/20210702222852im_/https://www.hkcnews.com/news_新聞/hk-discuss/2017/10/umbrella%20revolution-公民抗命-社運抗爭資料庫-20171001035020_d067_large.png" alt="">
</figure>

前年農曆年初二的 [旺角衝突](https://web.archive.org/web/20210702222852/https://www1.hkcnews.com/legalcasedb/events/59c4e23e1cfa7a0ad8cc88a6) ，本土民主前線前發言人梁天琦，以及李諾文、盧建民、林傲軒、林倫慶、黃家駒，6人分別被控以煽惑暴動、暴動、襲警、非法集結等罪名。案件在高等法院經歷逾50天的審訊，陪審團上周退庭商議20多個小時後，終於上周五（18日）作出裁決，其中梁天琦、盧建民各被裁定一項暴動罪罪名成立。眾新聞已更新 [【社運抗爭資料庫】](https://web.archive.org/web/20210702222852/https://www1.hkcnews.com/legalcasedb/) 的相關內容。

**裁決一覽**


| **被告** | **控罪** | **裁決** |
| --- | --- | --- |
| 梁天琦 | 煽惑暴動罪 | 罪名不成立 |
| 暴動罪（砵蘭街） | 無法達成裁決 |  |
| 暴動罪（亞皆老街） | 罪名成立 |  |
| 李諾文 | 暴動罪（砵蘭街） | 無法達成裁決 |
| 盧建民 | 暴動罪（砵蘭街） | 罪名成立 |
| 林傲軒 | 非法集結 | 罪名不成立 |
| 暴動罪（砵蘭街） | 無法達成裁決 |  |
| 林倫慶 | 暴動罪（亞皆老街） | 罪名不成立 |
| 暴動罪（山東街） | 罪名不成立 |  |
| 暴動罪（花園街） | 罪名不成立 |  |


此外，梁天琦早前已承認一項襲警罪。案中其中一名被告黃家駒亦被控一項暴動罪，他早前已認罪。黃尚有一項襲警罪，留檔處理。

法庭將於本周一（21日）處理梁天琦、盧建民的求情。至於無法達成裁決的控罪，法庭將於下月11日再提訊。

同案原本尚有黃台仰、李東昇、容偉業及袁智駒4人。黃台仰、李東昇缺席預審，疑棄保潛逃，法庭已向他們發出拘捕令。而容偉業、袁智駒則排期於今年7月18日預審、11月8日至12月19日審訊。
