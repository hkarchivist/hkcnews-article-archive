# 聽 Head-Fi 容易墮入的「情網」

  
 近年 Head-Fi 非常受歡迎，街頭上可觀察到，愈來愈多人擁有一副「吸睛」的耳機，手上的 music player，組合不同硬件，能有意想不到的效果。與 Hi-Fi 相比，現可把心愛的音樂帶到室外欣賞。不過，想盡情享受音樂的音質，真是值得花數千甚至過萬港元購買？

坊間售賣五花八門的軟件檔案格式，例如：wav﹑flac﹑dsd 等，相信已流行一段時間，衝破一般 CD 使用 16-bit、44.1 kHz 的格式。（一眾 head-fi 友可能會話：「晨早無玩 CD 好耐喇！」）理論上，bit depth 數值愈大，代表音樂的 dynamic range （大細聲）變化更大，所謂動態相對較高。而 sample rate 數字愈高，表示收集了更多資料，聲音應愈準確，能播放高於一般 CD 的頻率。

由此定論，軟件檔案的「數字」到天文數字般高，那麼不就是最好嗎？

<figure>
<img src="https://web.archive.org/web/20210628213702im_/https://www.hkcnews.com/news_新聞/hk-discuss/2018/05/head%20fi-音樂-20180526051146_1714_large.png" alt="">
</figure>

數字只能作參考，但絕對不可定其「生死」，對於選取器材如是。每專輯的錄音器材、錄音環境、錄音師使用的方式亦不同，因此萬萬不能一概而論。尤其是錄音所用的格式，如果該音樂在錄音室採用的是 24-bit、48 kHz，我們購買它更高的格式（如：32-bit、96 kHz)，音質上會自動提升嗎？當然不會！音質仍然只能停留在原來的質素。加上現時市場產品參差不齊，明顯地大量 remastered、reissued 的新版本與舊版分別太大，讓不少 fans 卻步。

因此，訓練我們的耳朵變得特別「重中之重」。不管任何現場演出地點：街頭表演、紅館、博覽館、外國場地等，任何音樂類型如：古典、爵士、電子、搖滾等，都一拼實際考察過，再與 Hi-Fi 或 Head-Fi 器材比較，才能領會。筆者亦同時學習中，一切要從經驗得回來！
