# 「上帝」改變了林鄭

好幾年前當林鄭還是發展局局長的時候，跟她有幾面之緣。她不苟言笑，給人的感覺是很有威嚴，不容易妥協。不過，她不是深藏不露的人，所以當立法會議員例如黃毓民指責當時還是政務司司長的她時，她會黑臉；不會像梁振英的奸笑和皮笑肉不笑。

林鄭是天主教徒，天主教徒從來不叫神做上帝。她宣佈參選特首時，記者問她不是公開說了退休嗎？為什麼突然宣佈參選呢？她說是「上帝」叫她辭職參選。她是聰明人，用「上帝」這個詞就是很坦白告訴大家，是中央叫她參選。為了忌諱，大陸人稱中央為天朝，天朝的大帝當然就是「上帝」了。

這個「上帝」把她改變得很厲害。林鄭在沒打算參選特首之前，說話是不會不停眨眼的，例如她以政務司司長身份，在2013年在融·愛藝術 @ autism開幕禮致辭，和2013年嘉諾撒聖方濟各書院校舍重建籌款晚會致辭。說話大大方方，笑容可恭，眼神不會閃爍不定。

可是，21日她會見傳媒，為司法覆核一事作出辯護時，大家可以看見她不停地眨眼，而且她學會乾笑和嘴藐藐。總的來說，她是一個演技很差的人。除非她上任後患了眼疾，因而不能自我控制地眨眼，否則只有兩種情況下才會不停眨眼；一是壓力很大，二是在撒謊。聽了她二十多分鐘的說話，根本就是在撒謊、就是強詞奪理。

讓我們看看林鄭對有關司法覆核的論點：


| 林鄭論點 | 社會觀點 |
| --- | --- |
| 1. 上訴庭的對兩單司法覆核的裁決是抗議、遊行行為違反法律，因此需要受到制裁； | 還用說嗎？原訟法庭已經裁定有關人士違法，而且判了社會服務； |
| 2. 說當事人受到政治逼害，或是政治犯，是完全不正確； | 如果不是政治逼害，上訴庭大可以發還原訟庭重審； |
| 3. 批評律政司司長作出刑期覆核是政治檢控全無理據； | 如果批評是無理據，那麼批評無理據的批評也是無理據；為什麼早不提出覆核，而在快要補選立法會議席時才提出。除了白癡，沒有人會相信覆核是沒有政治考慮。 |
| 4. 律政司整個檢控和刑期覆核完全是法律考慮，絕對冇任何政治動機； | 廢話，理由同上； |
| 5. 上訴庭接受律政司的申請司法覆核是因為有法理依據； | 上訴庭如果覺得原訟庭判刑太輕，可以發還重審。不發還是害怕維持原判，於是親自動手； |
| 6. 在七月一號的演詞提到，香港的司法獨立在普通法的經濟體裡是全球第三；社會批評司法受到政治干預是不公平； | 還好意思說就職演詞說了司法獨立是香港的基石，這個基石就是給她破壞了； |
| 7.     為律政司司長袁國強辯護不是一言堂，不接受同事意見是因為法理的考慮； | 一意孤行還說不是一言堂？常言道：法律不外乎人情； |
| 8.     周永康、羅冠聰、黃之鋒四月份已放棄上訴，證明他們都同意自己是犯了法； | 他們沒有說自己從來沒有違法行為，而且接受了社會服務的刑罰； |
| 9.     10月21日同學聯五位同學對話包括周永康和羅冠聰，說無論理想多崇高，所有行為必須合情、合理、合法； | 重奪公民廣場是九月發生，十月才說等於沒說，別提自己也是母親好嗎？ |
| 10.  量刑覆核是有法理依據，否則法庭不會接受申請 | 問題不是有沒有法理依據，而是為什麼不發還重審，而自己審？ |
