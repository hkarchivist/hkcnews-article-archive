# 環保觸覺：梁任內百頃地劃住宅未紓房屋問題　促減內地來港人口

政府上星期公布「土地供應專責小組」名單，為土地供應提供意見，明天舉行第一次會議。環保觸覺根據梁振英政府任内改劃土地用途及人口增長的數據，發現逾百公頃土地被改劃成住宅用途，相等於約6個維園，而人口增長主要來自内地來港人士。環保觸覺建議，專責小組應先對人口控制方面進行研究，從源頭減輕土地壓力。

<figcaption>環保觸覺整理政府過去5年的土地改劃資料，發現過百公頃土地被改劃為住宅用途。（戴晴曦攝）</figcaption>

環保觸覺整理城規會及規劃署，由2012年7月至2017年6月，梁振英政府任期内全港「政府、機構或社區」、「綠化地帶」，及「休憩用地」被改劃成「住宅」的資料，發現共87幅總面積111.59公頃、相等於約6個維園的土地，被改劃成「住宅」用地：


| 改劃成「住宅」用地前的土地用途 | 幅數 | 面積（公頃） |
| --- | --- | --- |
| 政府、機構或社區（GIC） | 53（最多） | 60.42（最多） |
| 綠化地帶（GB） | 22 | 36.29 |
| 休憩用地（O） | 12 | 14.88 |
| 總數 | 87 | 111.59 |


環保觸覺表示，被改劃的用地大部分集中在新界，估計因爲新界有較多未發展的預留用地，故較易改動土地用途。該組織又指，上述三種用地較接近社區，對居民影響較大，亦有助緩和熱島及屏風效應。

至於人口增長方面，環保觸覺整理統計處及入境處資料，統計2012年至2017年的人口資料，發現本港人口由714萬上升至735萬，多了21萬人。另外發現本港出生率及死亡率相若，人口主要增長來自内地來港人口：


| 内地人口入境種類 | 過去5年的入境數字 |
| --- | --- |
| 單程證 | 235,898 |
| 内地專才計劃 | 45,068 |
| 内地生留港 | 42,481 |
| 總數 | 323,447 |


環保觸覺幹事譚凱邦指，根據差餉物業估價署，7月份的私人住宅售價指數連續第9個月刷出新高，批評政府即使增加住屋土地供應，房屋問題依舊未能紓緩，認爲問題來自人口政策出錯。

他表示，「土地供應專責小組」應考慮積極控制外來人口，建議將內地外來人口減半，降低住屋需求。他認爲，單程證不設財政審批，輸入專才則加劇本港就業市場競爭，故此不同意減少輸入外來人口會減慢經濟發展，以及降低本港的勞動力。
