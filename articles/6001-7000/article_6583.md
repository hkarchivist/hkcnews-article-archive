# 平庸邪惡的二次剝削

【撰文：鄧永謙】   
 ***作者為新福事工協會關懷貧窮學校事工主任***

路過嘅工友，樓下嘅保安，社區嘅街坊，成日都問我點解成日著住件「分配公義」嘅衫，其實一來屋企有好幾件係買嚟支持關綜聯（「關注綜援低收入聯盟」）嘅工作，二來我認同呢個理念，不過恐怕現時喺呢個世界未能實現，所以著多啲提醒人又提醒自己，唔好因為自己係既得利益者，忘記咗一班基層嘅街坊係處於水深火熱之間。

今朝一早出門去搭車，沿途碰見一個清潔工友，   
 同佢講 : 早晨呀阿姐！   
 工友 : 哎呀！早晨啦後生仔（哈，喺長輩面前我永遠都可以後生啲，呵呵！）   
 我 : 呀！阿姐，你喺邊個站開工架？   
 工友 : 哦，我喺必發道個站架！   
 我 : 咦！必發個站離呢度都幾遠吓，其實晏架街個站仲近啲啦，點解唔係佢哋負責嘅呢？   
 工友 : 唉！我都唔知呀，不過點分就亂咁嚟，我哋都理唔倒，有啲地方都係咁架啦！   
 我 : 哦，係呀，都聽過吓。   
 工友 : 後生仔，你做咩架！   
 我 : 冇呀，我哋做社福機構嘅，成日走去同你哋傾偈接觸街坊之嘛！   
 工友 : 哦，都睇倒嘅，見你著住件衫（分配公義）都應該係做呢啲嘢。   
 我 : 係呀。   
 工友 : 唉，我哋啲人工好低架，做嘅嘢又辛苦，俾多少少都唔肯，命苦命苦。你哋後生呀要幫下我哋啲老人家呀。

我哋繼續吹水下去………

街坊同樣認同衫上面四個字——「分配公義」嘅理念，佢唔需要識寫呢四個字，因為喺佢嘅處境，已經係好真實咁經歷緊呢樣嘢，同樣地，佢亦都覺得喺呢個時勢，係冇辦法去扭轉呢個被剝削嘅局面。咁樣睇，可能大家就會覺得連工友都認命啦，仲搞乜倡議呀，嘥氣啦，雞蛋又點可以敵得過高牆呀！若果我哋順住呢種嘅思維去睇，當然，呢種諗法最正路，因為我哋每個人身上都有種平庸邪惡嘅特質，制度係咁咪跟囉，訂立出嚟就梗係有佢嘅功能架啦。當強權sale 佢嘅理念同做法，順服就令到我哋要去跟隨佢，而呢種邪惡特質會令我地唔會有獨立思考判斷嘅動力，莫講話改變，連關係都會變成冷漠，正正就係社會現時嘅景況。

前兩日同事帶人落區接觸街坊，有位參與者平時都會跟唔同嘅團體去探街坊，而多數都會係以傳福音或分享信仰為導向，對於工友嘅處境就比較少提及，而今次俾呢位參與者睇見一樣嘢，佢反思到因為佢平時探訪嗰陣，冇討論到外判問題同工作環境呢啲嘢，完全唔了解到工友本身做緊嘅工作原來係處於被剝削嘅情況，咁佢就諗，如果我去探佢，連佢本身被剝削處境都唔了解，然後又話自己係去關心愛鄰舍咁，最重要嘅目標就係傳福音俾佢，要佢信，其實呢份關心又有幾關心，呢份所謂嘅同理心又有幾同理呢？

最後佢講到若果探訪只係為達到某啲目標而唔去了解街坊嘅狀況，其實我哋只係喺佢哋已有嘅被剝削處境底下進行第二次嘅剝削，而呢次嘅剝削（關懷）係對佢嘅處境全無幫助，而我哋就繼續以呢種平庸邪惡嘅關懷去進行無數次嘅探訪服務，滿足自己嘅慈心。因為我哋認為自己已經係做咗嘢，政策制度唔關我哋事嘅。咁難怪連工友都要認命囉！
