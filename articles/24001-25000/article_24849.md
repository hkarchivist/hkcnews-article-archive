# 港人權法案美參院啟動「熱線」　魯比奧：有望最快下周一通過

<figure>
<img src="https://web.archive.org/web/20200630001249im_/https://www.hkcnews.com/news_新聞/hk-discuss/2019/11/香港人權與民主法案-參議員魯比奧-美國參議院-20191115135549_1cf7_large.png" alt="">
<figcaption>美國參議員魯比奧在Twitter表示，若無人反對，預計法案有望最快下周一通過。</figcaption>
</figure>

美國國會參議院外交委員會主席里施及委員會成員魯比奧，已就《香港人權與民主法案》啟動「熱線」程序（Hotline），加速推動法案，並稱預計法案有望最快下周一在參議院通過。

魯比奧周四在Twitter說，「今天我們在參議院啟動了加速程序，推動通過香港民主法案，如果沒有參議員提出反對的話，法案最早可能將在下周一通過」（Today we set in motion an expedited process in the Senate to pass the #HongKong democracy act. If no Senator objects it could pass as early as next Monday）。

「熱線」程序是指在一些情況下，參議院領導層（包括多數黨領袖、少數黨領袖以及各委員會主席）可能會以「熱線」形式推出某項法案，最常見的情況，是他們希望針對某項法案迅速採取行動。具體做法是，國會領袖要求兩黨議員向參議員傳出信息，表示某項法案將在未經表決（無異議）情況下通過。只要有一位參議員對該法案有異議，即可提出反對並進行登記，該程序就會馬上停止。換句話說，在這種「熱線」程序程序下，只要有一位議員對《香港人權與民主法案》提出異議，全院將停止法案推進。

上月到訪香港的共和黨參議員霍利周四說，「我相信啟動了熱線，這代表有很大的希望可能會很快獲得通過。參議員可以提出反對，然後撤銷熱線」，「但我們也就知道這些議員是誰，而且他們還必須提出反對理由」。

由魯比奧提出的《香港人權與民主法案》，在參議院獲得兩黨40位議員聯署。魯比奧周四說，在當前兩黨之爭激烈的政治環境下，法案得到了兩黨議員的普遍支持。「好消息是這一法案得到強有力的支持。我認為法案通過這件事，本身顯然就是非常重要的，僅僅是看到參議院無異議或接近無異議通過，就會有很大的不同效果」。參議院的法案版本若獲得全院通過，之後必須與眾議院9月無異議通過的版本，就不同之處進行協調。之後整合一致的文本，才能送交白宮，由總統簽字成為正式法律。

不過，魯比奧並未明確表示，是否能在11月28日感恩節前完成立法程序。他說，這是很緊迫的時間框架，取決於眾議院方面是否願意以多快的速度審議和通過。「他們已經通過自己的版本。我們還必須努力協調解決兩個版本的差異，如果不順利的話，我們還有方案B和方案C」。

他說，替代方案包括將法案納入國會每年必須審議通過的《國防開支授權法》或其他政府開支法案。魯比奧稱，他曾經和行政部門談過這部法案，國務院等行政部會，都此抱持開放態度。

霍利則強調，他已多次與特朗普在不同場合談論香港目前局勢，如果法案獲得通過，他有希望認為總統將簽署這部法案，「我和總統談過這件事，我在多個場合跟他談過，我從香港一回來就告訴他，好像就從香港一回來的那天。我覺得可能還有其他人跟他提過這件事。但我對他將會簽署法案抱持希望」。
