# 科大生尚德停車場墮樓　兩度腦手術仍危殆　警否認阻撓救援

22歲科大二年級學生周梓樂，周一（4日）凌晨於將軍澳尚德邨停車場由3樓跌落2樓，重傷昏迷。當時防暴警察在附近施放催淚彈，現場人士懷疑周是因躲避催淚彈而墮樓，有第一時間趕到救援的急救員指消防處救護人員被警方的封鎖阻礙到場。截至昨日（5日）晚上，周梓樂在伊利沙伯醫院兩度接受腦部手術後，情況仍然危殆，於深切治療部留醫。

警方昨晚9時見記者交代事件，指警方已初步翻查停車場閉路電視，相信傷者在4日凌晨零時45分至1時之間墜樓，期間警方曾於唐俊街施放催淚彈，與墜樓位置相距至少120米。警方又強調沒有阻撓消防、救護人員及車輛進出，完全沒有阻礙救援工作。

[《蘋果日報》](https://web.archive.org/web/20210712033624/https://hk.news.appledaily.com/breaking/realtime/article/20191105/60232901 ) 引述消息指，周梓樂已接受兩次腦部手術，包括切除了部分腦組織，及引流腦積血，惟未能降低腦壓；醫生嘗試安裝心塞輔助器，亦告失敗；周的盆骨也嚴重骨折，致腹腔出血達5公升。《蘋果》指，周梓樂昏迷指數是最低的3分，眼晴對外界刺激並無反應，他對外界說話、疼痛刺激亦沒有反應，呼吸機能微弱，需用強心藥及接駁呼吸機維生，報道形容他情況持續危殆，病況嚴重且不斷惡化。 [《明報》報道](https://web.archive.org/web/20210712033624/https://news.mingpao.com/ins/%E6%B8%AF%E8%81%9E/article/20191105/s00001/1572940878754/%E6%B6%88%E6%81%AF-%E5%A2%AE%E6%A8%93%E7%A7%91%E5%A4%A7%E7%94%9F%E7%96%91%E7%8F%BE%E8%85%A6%E5%B9%B9%E6%AD%BB%E4%BA%A1%E8%BF%B9%E8%B1%A1?fbclid=IwAR2-x550iItY0CTtPfaMsSMaTwFA8Za4JXHh37i5C2Z-H5XMiighimJlryU ) 提到，手術仍未能控制腦水腫及出血情況，周梓樂的瞳孔一直沒有反應，更有瞳孔放大的情況，疑是腦幹死亡跡象。有醫護消息指，周尚未到做腦幹死亡測試的階段，但情況絕不樂觀。

大批傳媒及周梓樂的朋友、同學，周二長時間在深切治療部門外守候。周梓樂多名家屬下午先後進入病房探訪。 [《蘋果日報》](https://web.archive.org/web/20210712033624/https://hk.news.appledaily.com/breaking/realtime/article/20191105/60232901 ) 引述周梓樂姑媽探望周後表示，他仍未甦醒，需由醫生繼續觀察，「好嚴重，不過暫時冇事。」被問到有傳周將會拔喉，她澄清說，情況未有那麼嚴重，「嗰啲係假消息。」

科大學生會臨時會長黎煒進，傍晚在醫院與周梓樂家屬簡短溝通後，指網上流傳關於傷者的消息，大部分並非屬實。黎表示，因應家屬意願，學生會不便再對外評論傷者情況。學生會仍在留意周的消息，目前未有行動計劃，以免打擾其家人。對於領展指停車場閉路電視未有拍攝到周墜樓的經過，黎煒進指現場有不少車輛，他呼籲車主提供車輛錄影片段。黎引述校長史維的承諾，將盡力向消防、救護、警方詢問，救援工作是否曾受刻意阻撓。

### 史維：已去信領展索閉路電視　向警方查詢如何在示威現場自保

科大校長史維、周梓樂就讀的計算機科學及工程學系系主任楊瓞仁及學生事務處職員下午分別到醫院探望。史維逗留約10分鐘後離開，未有接受傳媒訪問。

據 [科大學生會編委會報道](https://web.archive.org/web/20210712033624/https://www.facebook.com/hkusteb/photos/a.195769800540041/2451873111596354/?type=3&theater) ， 史維在探望周梓樂後，向全體師生發公開信，表示校方已致函領展行政總裁及警務處處長，向領展索取相關閉路電視片段，並向警方查詢應如何在示威現場保護自己，尤其在警方使用催淚彈的情況下。史維周三下午5時將再次與學生對話。

特首林鄭月娥昨日在上海見記者時表示，她對有市民，包括一名科大學生，在衝突中嚴重受傷感到傷心，她向傷者致以深切慰問，希望他們可以早日康復。被問到有指事件是因警方胡亂發射催淚彈而起，林鄭回應稱，事件仍在調查，不能妄下評論，她強調警方非常重視事件，已交重案組調查。

### 警方：沒有趕走救援人員　沒有阻撓消防救護員進出現場

東九龍總區高級警司（行動）傅逸婷，以及東九龍總區刑事總部警司（行動）胡家欣，昨日晚上見記者交代事件。傅逸婷指，警方對於有人受傷感到非常難過、傷心，向傷者及家人致以深切慰問，警方非常重視案件。

傅逸婷表示，在事發前、11月4日凌晨零時20分左右，有過百人聚集在將軍澳唐明街及唐俊街一帶堵路，有人設路障、「雨傘陣」等，示威者向警員投擲硬物。另有示威者在附近的建築物高處，包括尚德邨停車場，向地面的警員投擲硬物。警方發出多次警告不果，遂施放44枚催淚彈、11枚橡膠子彈、3枚布袋彈及1枚海棉彈，以驅散示威者。警方知悉消防處在這段時間內，到停車場處理一宗一級火警事故，消防人員於凌晨零時55分到場，在幾分鐘後，有人向消防人員指出有人墜樓，消防人員於1時05分在2樓發現傷者，進行施救，並於1時11分要求救護車到現場增援。

綜合傅逸婷及胡家欣指，在消防人員發現傷者的同一時間、1時05分，警員從廣明苑方向進入停車場，嘗試從地面開始向上層驅散示威者。警方當時尚未知道有人受傷，直至警員到達2樓，見到消防人員及義務救護員為傷者急救。傅強調，警員並沒有影響、干預救援工作，沒有趕走任何救援人員，當時消防人員表示不需要警方協助，警員於是繼續原來的驅散行動，過程中沒有用到任何武力、彈藥。警員1時25分完成停車場內的行動，1時45分離開現場。

傅逸婷提到，警方事後知道，救護人員當晚在1時30分到場，而救護車在1時41分離開。她重申，警方完全沒有阻撓任何一名消防員、救護員，或消防、救護車接送傷者離開現場，警方在事件中完全沒有阻礙過救援工作。警方得悉事件後，高度重視事件，並交由東九龍總區重案組跟進，希望找出受傷原因。

被問到警方有否阻礙救護車到場，傅逸婷回應指，救護車進入現場時，唐明街及唐俊街一帶多處路面已被堵塞，故救護人員需要落車，再行入停車場。傅表示，救護人員進入時，沿途應該沒有接觸到警務人員。不過，記者問及有人拍攝到救護車在唐俊街時，前面有一批防暴警員，傅表示未能確定影像拍攝的時間，她不作評論。

胡家欣表示，重案組已初步翻查停車場閉路電視，綜合資料，警方相信傷者在凌晨零時45分至1時之間的15分鐘內墜樓。警方當時於唐俊街施放催淚彈，與傷者墜樓的位置相距至少120米，同一時間，有一批警員在寶康路戒備，但他們沒有放催淚彈或作任何驅散行動。

胡家欣指，停車場2樓及3樓都有矮牆，2樓的矮牆分隔泊車位及平台，但3樓的矮牆對出沒有平台，警方不排除傷者誤以為3樓矮牆後有平台，故跨越矮牆後不幸墜下。他續指，因為角度問題，在閉路電視看不到傷者墜樓、著地的情況，但警方會繼續翻查影片，他又呼籲在停車場泊車的車主及目擊者聯絡警方，提供資料，供警方調查事件。

被問到凌晨零時45分至1時之間、周梓樂墜樓的時間內，警方是否曾放催淚彈，傅逸婷回應指，警方在零時45分之前已施放催淚彈，直至1時許，曾經有少量催淚彈向停車場方向發射。至於傷者是否因為躲避催淚彈而墮樓，傅表示不排除任何可能性，但她引述消防人員指，他們在場救援期間，催淚煙的程度不高。
