# 風動旛動

要知天下最新的局勢，要先掌握、歸納所有風向指標。

#   
 圖一顯示過去14日大小不一的新聞互動率，所謂的互動率是指大眾在新聞網站或facebook對不同新聞的反應，這些反應包括Share、Comment以及Like等情緒表達。新聞互動率越高，顯示大眾對有關新聞的反應愈大。

#   
 過去14日大眾「最有反應」的新聞是《蘋果日報》題為 〈民陣召集人岑子杰遇襲　大量流血倒地　送院時清醒〉，互動率977次（每分鐘Share、Comment及Like合共977次），及後大眾對「反修例」運動的反應開始下降，到了這幾天多則新聞的互動率已跌至200次以下。   
 #   
 互動率減少的原因複雜，概有：   
 A）整體新聞減少；   
 B）缺乏涉及社會運動的重磅焦點新聞；   
 C）大眾情緒開始冷卻等等。

#   
 不過圖二告訴大家原因A）並不適用，因為過去24小時的新聞量不少，所以原因可以歸納到B）及C），一來欠缺重磅焦點社運新聞，二來大家的情緒的確有所冷卻，這個亦為前線或社會運動人士所憂慮。   
 #   
 說完客觀大數據意見後，講講個人夢境。   
 #   
 涉及社運的重磅焦點新聞減少，因為這星期沒有大型遊行示威活動，取而代之的是以「和理非」式抗爭維持有限度的社運能量，為什麼大型遊行示威活動暫停？   
 #   
 原因包括：   
 1）區選在即暴力事件下容易被利用繼續祭出緊急法；   
 2）美國參議院審議《香港人權與民主法案》前副總統彭斯開腔叫停暴力示威；   
 3）中共「四中全會」在10月28日至10月31日召開；

#   
 若大家看過前文拙作 [〈就係要你打砸搶燒！〉](https://web.archive.org/web/20210712033513/https://www.facebook.com/photo.php?fbid=2579168428802200&set=a.171499816235752&type=3&theater) ，會知道香港之亂始於A，BCDE及F（分開）。   
 #   
 「四中全會」的召開、中美貿易談判似有進展兼中美下月在智利開會簽約，似乎局勢有機會明朗化。關於這點大家可以從彭斯演講中的懷柔言論得出結論，彭斯2018年10月時演講非常強硬，所以我們不應該只著眼他強硬的陳詞，而要看放軟的言論（不過我更相信彭斯是欲擒故縱，美國國策大方向不會改變）。   
 #   
 作為棋子的香港不應過份期待美國可以左右大局，始終最重要是A，BCDE。   
 #   
 究竟「四中全會」會有什麼結論？我不知道。但我建議大家留意數則新聞：   
 1) 張曉明神隱多時終於浦頭，在人民大會堂宴請工聯、街總和婦聯聯合訪京團露面，大談「一國兩制」；   
 2) 華春瑩罕有撤回否定《金融時報》換特首的言論；   
 3)《南華早報》傳市場不滿監警會報告後會成立獨立調查委員會；   
 4) 運房局前局長張炳良指現階段無大台難重啟政改；   
 5) 林鄭月娥張建宗開始轉口風放棄警察（惹來屈穎妍口誅筆伐）；   
 6) 還有其他不能盡錄；   
 #   
 歸納123456，風在轉，至於會怎樣轉會如何轉甚至會否轉吹倒頭風我不知道，但幾乎肯定是達成了某些重要共識，包括特朗普與中國、短線最大機會是整個港澳系統及警察組織進行大洗牌重組，何X堯被用完即棄是我最期待的畫面之一。   
 #   
 未來風向怎轉，端視：   
 1) 選管會會否允許黃之鋒入閘（不要玩，黃之鋒能否入閘不會是18區選舉主任可以話事），可以看到各方妥協的程度有多大，誠意有多大。   
 2) 11月2日是「四中全會」後回歸港島的大型遊行示威集會，「輸家」會否就範視乎「四中全會」後會否開大片。   
 3) 如何處理陳同佳一事。   
 #   
 坦白說，我已講得太多，最後要集思廣益，特別是針對張炳良言論的建議。張炳良說現階段無大台難重啟政改，這確是政治現實亦為涉事者口實不推政改（林鄭換人一定會衍生政改議題）。   
 #   
 若香港人不希望永遠成為ABCDEF的棋子，唯一出路是如何明知不可為而為之，「偷跳」衝擊真普選，示威者或民間記者會要研究如何成立有足夠民意基礎的對話代表團（我不會用大台這個TERM），這個對代表團可以是流水式由8大專院校學生會或大專國際事務組織擔任代表（議員？不了。）這個代表只負責傳遞由民間記者會收集到的民意。   
 #   
 所謂的流水式是代表只輪流做1個月或2星期，令民意可以充份地反映之餘，不會被對方以大台之名進行統戰收編。   
 #

 【關於「風動旛動」】   
 六祖慧能在法性寺時，風吹旛動。一僧說是風動，一僧說是旛動，兩人爭論不休，慧能說：「不是風動，不是旛動，仁者心動。」意謂外境的變化，是自心投射的結果。見《六祖壇經・行由品第一》。後比喻限於外在現象不同意見的爭辯。如：「這種各執己見的辯論，正像風動旛動的爭辯一樣，誰也沒把握住問題真正的重點。」
