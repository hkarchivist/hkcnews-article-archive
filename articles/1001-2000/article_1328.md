# 點評鬍鬚林鄭葉劉 周永新：AO不是政治人才

**大年初一，老香港憶苦思甜，想起兒時身上補了又補的衫褲鞋襪，還有那封令窮小子難得樂透的一毛錢紅包。**

**今天香港社會雖然表面富足，但瑟縮邊緣的低下階層長期被主流社會遺忘，他們都不在平日忙碌的中環人群視線範圍。可是，近日因為特首選戰，窮人的身影突然出現在政治人物的facebook，變成一場又一場「親民騷」不可缺少的配角。但我們其實又曾幾何時見過鍾意坐紅色房車、唔知便利店無廁紙賣的高官，會到深水埗茶記或者天水圍公屋，親身了解窮人生活？**

**選舉暴露人性。70歲的香港大學社會工作及社會行政學系榮休教授周永新，最知道當中的諷刺，他幾十年來和無數高官打過交道，見盡窮人處境所折射出來的官場偽善。他在以往的公職生涯中，和3個AO政務官出身的特首候選人林鄭月娥、曾俊華、葉劉淑儀都有公事上的接觸，近年他最為人記得的一幕，是和林鄭月娥就全民退休保障開火。**

**AO出身，必屬佳品？周永新對此冷笑一聲，然後，嘆了長長一口氣。**

已表態參選特首的4個人當中，3個是AO出身。曾經何時，AO是大學生夢寐以求的筍工，不單人工高福利好，更加是精英尖子明日之星的身份象徵。AO平日返工有人接送又有秘書服侍，在政府總部大部分人都要聽佢話，開會有佢講無人講。AO為官之道最緊要係乜？就係循規道矩，唔好搞咁多無謂嘢、少做少錯、總之千祈唔好孭鑊，咁就可以無風無浪，定時定候升官發財搵十幾廿萬一個月直至退休之後再食長糧至百年歸老。至於要去立法會或者上電台就更加easy job啦，只要識得做人肉錄音機，夠厚面皮就可以話知你。

## 「我哋以前叫班AO做太監，佢哋最叻，就係奉旨行事。」周永新笑道。

曾蔭權2007年出來選特首的口號是：「做好呢份工」，反映他將AO打份工的根深柢固思維，等同他作為一個領導應有的視野。

曾蔭權是曾俊華為官的恩師，「煲呔曾」做特首之初，委任周永新做策略發展委員會成員。周永新因此與當時是特首辦主任的曾俊華有過一些行政交往，但不算深入接觸，曾俊華在會上也無講好多嘢。周記得，有一次開策發會時，他講到香港貧富懸殊非常嚴重，曾蔭權就話：「全世界都係咁啦，有乜特別。」

## 社福界好多人都話：「曾蔭權好憎社工」，周永新也說：「曾蔭權對福利無興趣，覺得福利多餘，我覺得，曾俊華同佢都一樣。」

「曾俊華是典型的財金官員，同陳德霖、許仕仁那些都差不多，自信心高，覺得自己係精英、high class。我做過強積金諮詢委員會主席，同一班財金官員有接觸，知道佢哋嘅mind set係重視商界利益，所謂『扶貧』對佢哋來講，就係只要搞好經濟、有錢就得，但佢哋唔會諗，有乜嘢長遠政策措施可以幫助弱勢社群。」周說，他倡議社福政策數十年來，鮮有財金官員主動接觸他。他其實並非對經濟一竅不通，他擁有英國曼徹斯特大學經濟碩士學位。

曾俊華做了財爺9年多，周永新說：「他做預算案咁多年，無就社福議題諮詢過我。」2011年，曾俊華公布預算案後出席電台節目，被聽眾問到一盒粟米斑塊飯幾多錢時答不出，後來又話：「我唔知粟米斑塊飯幾錢，因為我唔鍾意食粟米，亦唔鍾意食斑塊。」他當年因而被批評漠視民情。事隔6年，他身邊的一群政治化妝團隊，在他正式宣布參選特首前，帶他到黃大仙食豆腐火腩飯，並主動提到要50元「真不便宜」。帖文在facebook一出，不單沒有什麼人重提當年的粟米斑塊飯，他更因為眼前的一碟豆腐火腩飯，成功得到超過4700個likes。

「2011年那一份財政預算案，我最記得曾俊華本來話，向低收入打工仔強積金戶口注入6000元，我原本好支持。但後來因為有人批評，結果佢改為派6000蚊，我當時講過，覺得佢身為高官出爾反爾，應該辭職。」

周永新說不出曾俊華任內有何政績，「佢出來講參選嗰日，份稿完全無政綱無內容，但仍然有好多掌聲。我有上網，我知道佢係網絡好紅。你問我點解，或者係因為梁振英啦，而家只要有一個人出來同梁振英講相反的說話，就會有掌聲。」周永新分析，曾俊華當上特首之後，香港或只會重回曾蔭權的舊路。

**特首要親和**

林鄭月娥說過，為官最有滿足感，是在2000至2003年做社會福利署署長。周永新當年如何跟她交手？

首先是2003年，政府因通縮欲削減11.1%綜援金，「林鄭來港大找我，做晒功課帶晒圖表上來。但講吓講吓就知道，佢根本唔係來聽我意見，而係想說服我支持佢。當我講到傷殘和兒童的綜援金不可以削減時，佢起初都不接受，然後就講一大堆嘢例如話『你知啦，政府都有機制，我哋要跟機制辦事。』佢嘅態度係好堅持，唔讓步。」

另一次，林鄭月娥又走上港大搵周永新，今次是要說服他支持向社福機構整筆過撥款的改革，這一次周永新沒有太大意見，兩人的談話氣氛平和。但兩次的「摸底」游說工作，令周永新感覺到林鄭月娥是個充滿自信、強勢、按本子辦事的AO公務員。

「我同林鄭一直都係公事接觸。2012年梁振英做特首之後，林鄭打電話問我有無興趣做扶貧委員會和關愛基金，但我年紀大啦，無應承。」林鄭月娥之後邀請周永新做關於退休保障的研究：「交報告之前1年，我有同佢食過一、兩次飯，講吓進展，當時我哋仍未做完個研究，未有全民退保呢個建議，所以在飯局中無提到。」

「我承認我哋之前食兩次飯時，林鄭可能覺得我哋唔會go for 全民退保。結果我哋出報告時，我估林鄭有點caught by surprise。我哋建議全民退保，係我睇過晒數字之後拍板，當時我都無同林鄭講，直至我哋交個初稿俾政府時，林鄭領導個督導委員會班財金官員就好有意見，仲阻頭阻勢挑剔得好緊要，改埋我啲字眼，例如講人口老化，我哋話「it is a serious problem that we have to face」，班官就話要改做「a problem that we can no longer ignore」，仲堅持一定要出佢哋嘅版本。

結果2015年12月，政府推出全民退保諮詢時，林鄭月娥和周永新隔空開火，林鄭月娥批評周永新提出的全民退保方案稱政府不用加稅是「不切實際」，更說他不掌握公共財政、建議是「非認真學術研究應有態度」。周永新則反駁諮詢文件中的數字「不盡不實不公道」，例如將30萬沒領社會保障的老人計算在福利開支之內，將開支金額報大1倍至2萬億元。

駁火事件之後，周永新沒有再就退保議題接觸林鄭月娥，直至去年11月，林鄭為準備施政報告，邀請了周永新和其他界別的人到政府總部給予意見。周永新在會上問她，50年前，香港經歷六七暴動之後，港督麥理浩做了大量教育、醫療、房屋等大規模的民生改革。過去幾年香港社會發生各種風風雨雨，今天的政府，可會像當年麥理浩一樣，作一個翻天覆地的改變，做些有益民生的大事扭轉大局，攞返民心？

「最終林鄭點答我？佢話：『現屆政府做咗好多嘢，例如......』」又係回帶。

**誠信最重要**

## 「坦白講，我覺得林鄭月娥做AO好用力，由政改到西九故宮，大家都睇到佢有幾瞓身去做。但講到做特首，一個人的親和力就好重要。林鄭成日講凝聚共識，但你睇佢做官咁多年，有邊一件事可以將人拉埋一齊凝聚共識？我諗唔到一個例子，佢好多嘢係夾硬推出來，而唔係先聽人意見。」

林鄭月娥有幾真正關心窮人？「佢表現出來係關心，實際上我唔知，我亦recall唔到一些事，感覺到佢係有warm、有passion，我感受唔到，佢係好公事公辦。」

## 那麼葉劉淑儀又如何？「我80年代做入境處審裁員，開始同佢有接觸，一直到佢做入境處處長。佢個人做嘢算均真，但都係典型AO，覺得自己高高在上，要人聽佢講。但我覺得佢出嚟從政之後，因為接受過選舉洗禮，近年個人謙卑咗。」

## 不論是曾俊華、林鄭月娥、葉劉淑儀，周永新覺得，他們都離不開AO的mentality，基本盤是「曾蔭權2.0」，或者當中有人另加入了「梁振英2.0」的元素，「講到底，AO的訓練，是培養不到政治人才。」

剩番一個，「當下的香港，需要一個有誠信的人做特首。4個人當中，胡國興來自法律界又做過法官，比較之下我估他是相對有誠信的人。」周永新做人事登記審裁員時認識胡國興，但兩人並不相熟也沒有私交，他選胡官只是根據自己的直覺，雖然他和好多人一樣，不知道胡官出來參選的背後，是否另有盤算。

如果有普選的話，周永新的一票，在AO以外，相信有更多選擇。
