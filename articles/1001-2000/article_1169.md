# DQ特首

林鄭月娥「黑箱放暗箭」，遭正氣凜然的前選管會主席胡國興當頭棒喝，直斥「極不君子」有違選舉規則，慌忙漏夜澄清兼「耍冧」，原來當高官遇上法官，照樣腳軟。

縱使林鄭再三辯解，指自己只是「機械式事實陳述」，又望公眾包容她 3 6 年來首次由官變民，唔識講人話；但她推卸不了，在她說這番話之前，胡官、葉劉和財爺均已公開宣布參選特首，她仍猜測和質疑有人可能會不獲中央任命，不是針對三人是指誰？

特首選戰以來，西環中聯辦躲在暗處大力發功，假傳「聖旨」蠱言惑眾，甚至公然違反選舉規則，明目張膽教唆和誤導選委，可惜選管會「隻眼開隻眼閉」，市民心中有氣無處訴。今回真是：假選舉碰上真法官——邪不勝正。

林鄭閉門吹風恐嚇誤導傳媒當然抵鬧，這場公關災難可能仍會繼續發酵，但她卻預示了一個問題，值得警惕和深思：假如林鄭真的落敗，奪魁者並非中央欽定人選，北京是否真敢冒天下之大不韙拒絕任命， DQ當選的特首？若然如此，如何解決這憲制危機？重選？解散選委會？還是派個「欽差大臣」空降治港？翻查基本法，未見有關規定。

這絕對不是危言聳聽，杞人憂天。專權者不願接受甚至強行推翻民主選舉結果，北京不是史無前例，台灣就是明證。

2 0 0 0年陳水扁領導台灣民進黨首度執政，中共氣急敗壞，不惜出動軍艦和導彈隔海威嚇；去年蔡英文帶領民進黨重返凱達格蘭大道，因為不肯承認國共兩黨之間的「九二共識」，中共惱羞成怒，除出動航空母艦外，更蓄意限制大陸自由行來打擊台灣經濟，如果沒有美國第七艦隊保護，解放軍恐怕早已殺入台北，強壓蔡英文低頭降伏。

在民主政制下，選舉是人民的自由抉擇，不是當權者說了算；尤其是一個享有獨立管治權的地方，由誰執政當然由當地人民決定，政黨輪替也是正常規律，其他任何地方任何人都無權過問。但是，中共從來都沒有尊重過台灣人民的意願。

今日台灣會成為明日香港嗎？

當然，香港不是台灣，香港的主權在中國。絕大多數香港人也認同在基本法框架下，特區行政長官必須經中央政府任命，我們只是希望能夠投票選出自己心目中的理想特首，可是回歸二十年來，這個卑微要求仍然未能達到。

有人相信，在北京精心設計下，選委會不可能會選出一個中央不信任的特首人選，憲制危機微乎其微；但連建制派穩控大局的立法會都會出現政改方案「甩轆事件」，政治又有甚麼不可能？如果，有朝一日，1 2 0 0名選委真的不理會西環指示，按照全港市民的意願，選出不是北京欽定的特首，那北京會怎麽辦？西環又會做些甚麼？（忽然聯想起電視劇《選戰》的若干情節和胡官的坐駕日前被撞，不禁倒吸一口涼氣⋯⋯）

從另一角度，萬一，北京真的一意孤行DQ按民意當選的新特首，我們又怎麼辦？有多少人會為此再度走上街頭？在法律之下，我們可以做些甚麼？筆者不是法律專才，還望有識之士賜教。

未雨綢繆，這個問題的確需要深思。
